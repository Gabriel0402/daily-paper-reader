## 摘要
自动化奖励设计（ARD）旨在通过语言驱动的奖励函数合成，取代强化学习中的人工奖励工程。然而，现有基于大语言模型（LLM）的方法本质上仍由相关性驱动，依赖迭代的环境反馈来优化每个特定任务的奖励假设。这种范式不仅导致推理效率低下，还使LLM容易受到语义上看似合理但因果上虚假的奖励组件的影响，从而导致优化无效。为解决这些局限，我们提出因果奖励世界模型（CRWM），该模型通过对多任务交互数据进行离线预训练，明确建模候选奖励组件与任务目标物理变量之间的因果拓扑关系。基于从粗到细的预训练策略，我们引入了一个联合优化模块，该模块将显式机制解耦与置信感知软融合相结合，利用微观轨迹细化粗粒度结构先验，从而构建鲁棒且可解释的因果骨架。在推理过程中，LLM利用CRWM作为与任务无关的因果先验来约束奖励生成，从而实现零样本奖励函数设计。我们的工作为ARD问题开辟了新的白盒范式。在复杂连续控制基准上的大量实验表明，CRWM无需反馈驱动的奖励精化即可生成可执行的奖励函数，显著降低了获取新机器人技能的设计延迟，同时匹配或超越了最先进的性能，并进一步在未见过的任务和多样化的机器人平台上展现出强大的泛化能力。

## Abstract
Automated Reward Design (ARD) aims to replace manual reward engineering in reinforcement learning with language-driven reward function synthesis. However, existing approaches based on large language models (LLMs) remain inherently correlation-driven, relying on iterative environmental feedback to refine reward hypotheses for each specific task. This paradigm not only results in inefficient reasoning but also makes LLMs susceptible to semantically plausible yet causally spurious reward components, leading to ineffective optimization. To address these limitations, we propose the Causal Reward World Model (CRWM), which explicitly models the causal topological relationships between candidate reward components and task-targeted physical variables through offline pre-training on multi-task interaction data. Based on a coarse-to-fine pre-training strategy, we introduce a joint optimization module that integrates Explicit Mechanism Decoupling with Confidence-Aware Soft Fusion to refine coarse structural priors using micro-level trajectories, thereby constructing a robust and interpretable causal skeleton. During inference, LLMs leverage CRWM as a task-irrelevant causal prior to constrain the reward generation, enabling zero-shot reward function design. Our work opens up a new white-box paradigm for the ARD problem. Extensive experiments on complex continuous control benchmarks demonstrate that CRWM generates executable reward functions without feedback-driven reward refinement, significantly reducing the design latency for acquiring new robotic skills while matching or surpassing state-of-the-art performance, and further exhibits strong generalization capabilities across unseen tasks and diverse robotic embodiments.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：自动奖励设计（ARD）旨在用语言驱动替代人工奖励工程。现有基于大语言模型（LLM）的方法**依赖迭代式环境反馈**来精化奖励函数，效率低下；同时，LLM倾向于基于语义关联而非因果推理引入奖励分量，导致**优化崩溃**（奖励信号无效）和**规范博弈**（智能体利用虚假奖励）等系统性失败。
- **背景**：同一动力学系统内，不同任务的奖励组件往往共享因果结构。若能提取跨任务通用的因果先验，则可避免逐任务的迭代搜索。
- **目标**：构建一个因果奖励世界模型（CRWM），作为ARG的**白盒因果核**，使LLM能在零样本（zero-shot）下直接生成可执行奖励函数，无需反馈迭代。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- 离线多任务预训练，显式建模候选奖励组件与任务目标物理变量之间的**因果拓扑关系**，形成与任务无关的因果先验。
- 推理时，LLM被约束在此因果先验上，通过**因果剪枝**剔除虚假相关分量，实现零样本奖励生成。

### 关键技术细节
#### 阶段一：结构先验提取（粗粒度）
- **层次化数据采集**：对每个预训练任务，使用ARD框架（Eureka/URDP）提取k个物理奖励分量，然后系统性地扰动其权重（do操作），收集宏观级数据`D_macro`（权重配置→最终适应度分数）和微观级轨迹`D_micro`（结构变量逐步骤序列）。
- **因果基础模型**：使用预训练模型**LimiX**从`D_macro`中估计原始因果效应矩阵，得到粗结构先验`M_prior`和置信度矩阵`M_conf`。

#### 阶段二：联合优化模块（微观精化）
- **显式机制解耦（EMD）**：将状态相关瞬态物理交互（如接触）从持久因果骨架中解耦。构建瞬时拓扑`M_inst(s_{t-1}) = M_inv ⊙ (1 + M_trans(s_{t-1};θ))`，其中`M_inv`是可学习的可保留因果骨架，`M_trans`是MLP参数化的瞬态组件。
- **置信感知软融合（Confidence-Aware Soft Fusion）**：损失函数`L_soft = γ||M_conf ⊙ (M_inv - M_prior)||²_F`，仅对高置信度边施加强约束，允许低置信度边自由偏离，以避免继承先验中的结构幻觉。
- **联合目标**：
  ```
  min_{M_inv,θ} L = E[(s_t - M_inst(s_{t-1}) s_{t-1})²] + λ h(M_inv) + L_soft
  ```
  其中`h(M_inv)`为DAGMA无环正则化，采用增广拉格朗日法求解。收敛后`M*_inv`作为最终CRWM。

#### 阶段三：Causal-ARD（零样本推理）
- **因果图→提示翻译**：对CRWM边缘效应绝对值进行2-means聚类，提取活跃因果边，序列化为三元组(源, 目标, 效应值)，嵌入系统提示中。
- **因果剪枝**：LLM根据提示，仅保留存在有效因果路径到“score”节点的奖励分量，将虚假相关分量系数置零，生成一次性Python奖励函数。

- **白盒可解释性**：CRWM显式输出因果图，推理过程透明。
- **零样本效率**：部署时无需环境交互，显著降低新技能获取延迟。
- **任务无关通用性**：一次预训练，可复用于同一动力学系统内的任何新任务。
- **鲁棒性设计**：EMD解耦瞬态与持久因果，软融合灵活利用先验，联合优化避免结构幻觉。
- **实验全面**：覆盖仿真（多任务）、跨域、真实世界、消融、超参数和模型替代对比。
- **适用性广**：在Dexterity、ManiSkill2和真实机器人上均有效，且对LLM和因果模型选择不敏感（DeepSeek/Qwen/Llama3, LimiX/AVICI结果一致）。

## 8. 不足与局限
- **离线预训练成本高**：需要4200次RL策略训练，对算力要求大，且需构建多任务交互数据集。
- **泛化受组件池限制**：原子奖励组件池仅覆盖常见运动学/动力学变量，无法处理流体、变形体、非刚性接触等新型物理变量。
- **无双手真实机器人验证**：虽然仿真实验包括双手任务，但真实世界仅测试单臂操作（CASBOT W1），缺少双手hardware验证。
- **真实世界实验范围有限**：仅评估奖励对齐（AUC），未进行完整RL策略训练→部署的端到端实验，存在sim-to-real动态差距未充分暴露。
- **DAG约束的取舍**：施加无环正则化可能排除合理的物理反馈回路（虽然通过瞬态组件部分补偿）。
- **因果发现假设**：依赖因果基础模型（LimiX）的准确性和干预数据的质量，在干预覆盖不足时可能引入偏差。
- **未见任务数量**：零样本评估仅6个任务，泛化结论的统计稳健性可进一步加强。
