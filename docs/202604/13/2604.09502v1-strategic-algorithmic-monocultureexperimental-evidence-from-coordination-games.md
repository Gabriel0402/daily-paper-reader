## 摘要
AI 智能体越来越多地在结果取决于协调的多智能体环境中运行。我们将原生算法单一化（即基准行为相似性）与策略性算法单一化（即智能体根据激励调整相似性）区分开来。我们设计并实施了一个简单的实验，能够清晰地分离这两种力量，并将其应用于人类和大语言模型（LLM）受试者。研究发现，LLM 表现出高水平的基准相似性（原生单一化），且与人类相似，它们会根据协调激励来调节这种相似性（策略性单一化）。虽然 LLM 在相似行为上的协调能力极强，但在奖励差异化的情况下，它们在维持异质性方面落后于人类。

## Abstract
AI agents increasingly operate in multi-agent environments where outcomes depend on coordination. We distinguish primary algorithmic monoculture -- baseline action similarity -- from strategic algorithmic monoculture, whereby agents adjust similarity in response to incentives. We implement a simple experimental design that cleanly separates these forces, and deploy it on human and large language model (LLM) subjects. LLMs exhibit high levels of baseline similarity (primary monoculture) and, like humans, they regulate it in response to coordination incentives (strategic monoculture). While LLMs coordinate extremely well on similar actions, they lag behind humans in sustaining heterogeneity when divergence is rewarded.

*   **研究动机**：随着 AI 智能体在招聘、定价和交易等高风险经济活动中的普及，它们如何相互协调成为关键问题。
*   **核心概念**：论文区分了两种“算法单一化”：
    *   **原生算法单一化（Primary Monoculture）**：智能体在无激励情况下的基准行为相似性。
    *   **策略性算法单一化（Strategic Monoculture）**：智能体根据环境激励（如需要一致或需要差异）主动调整其行为相似性的能力。
*   **研究意义**：探讨 LLM 是否像人类一样拥有“谢林点”（Schelling points）意识，以及它们在需要“协调发散”（即各做各的，避免冲突）的场景下是否会因为过于相似而失败。

### 2. 论文提出的方法论
*   **核心思想**：通过经典的**协调博弈（Coordination Games）**框架，对比人类与 LLM 在不同激励下的表现。
*   **博弈类型**：
    *   **选择（Picking）**：无特殊激励，仅需提供一个有效答案。
    *   **协调（Coordination）**：如果两个智能体选择相同，则获得奖励。
    *   **发散（Divergence）**：如果两个智能体选择不同，则获得奖励。
*   **关键指标——一致率（Agreement Rate）**：定义为两个独立智能体提供相同答案的概率。公式上体现为响应分布的内积。
*   **文本推理分析**：利用 LLM 的思维链（CoT）功能，通过语义相似度分析和“LLM 评委（LLM-as-a-Judge）”方法，量化智能体在决策时的战略意图（如是否提到了“显著性”、“冷门”或“随机化”）。

    *   不同信息反馈（告知对方是“副本”还是“人类”）。

### 6. 主要结论与发现
*   **高原生相似性**：在无激励时，LLM 的行为一致性远高于人类（14% vs 58%），证实了严重的“原生算法单一化”。
*   **具备策略调节能力**：LLM 能够识别激励。在协调任务中一致率上升（至 72%），在发散任务中一致率下降（至 27%）。
*   **非对称表现**：
    *   **协调强项**：LLM 在“寻找彼此”方面远超人类，能精准锁定极具代表性的答案（如字母 A、数字 7）。
    *   **发散弱点**：在需要“避开彼此”时，LLM 表现糟糕。即使被要求不同，它们的一致率（27%）仍远高于人类（3.5%）。
*   **推理与行为脱节**：文本分析显示 LLM 知道应该选“冷门”答案，但在实际选择时仍难以摆脱高频词的吸引力。
*   **温度调节局限**：调高温度虽能降低一致率，但无法完全消除 LLM 之间的发散赤字。

### 7. 优点
*   **理论创新**：首次将算法单一化细分为“原生”和“策略性”，为多智能体风险研究提供了新视角。
*   **实验严谨**：对比了 16 种不同家族的模型，结论具有普适性。
*   **多维度验证**：结合了博弈论选择数据和自然语言推理数据，揭示了 LLM “言行不一”的有趣现象。

### 8. 不足与局限
*   **任务简单化**：实验基于静态的单轮开放式问题，尚未涉及复杂的动态博弈或长期经济互动。
*   **文化偏差**：人类样本仅限美国，LLM 的训练数据也以英语为主，可能无法代表全球多元化的协调逻辑。
*   **RLHF 的副作用**：论文指出 LLM 的发散困难可能源于强化学习（RLHF）将其行为推向了“人类最喜欢的标准答案”，这种训练偏差难以通过简单的 Prompt 消除。
