## 摘要
视觉-语言-动作（VLA）模型通过大规模预训练推动了机器人操控的发展，但由于部分可观测性和延迟反馈，其在现实世界中的部署仍面临挑战。强化学习通过价值函数来解决这些问题，价值函数能够评估任务进度并指导策略改进。然而，现有的基于视觉语言模型（VLM）构建的价值模型难以捕捉时间动态，从而削弱了在长程任务中进行可靠价值估计的能力。在本文中，我们提出了 ViVa，这是一种视频生成价值模型，它将预训练的视频生成器重新用于价值估计。ViVa 以当前的观测结果和机器人本体感受作为输入，联合预测未来的本体感受和当前状态的标量价值。通过利用预训练视频生成器的时空先验，我们的方法将价值估计建立在预期的具身动力学基础之上，超越了静态快照，将价值与前瞻性内在耦合。通过集成到 RECAP 中，ViVa 在现实世界的纸箱组装任务上取得了显著改进。对所有三项任务的定性分析证实，ViVa 产生了更可靠的价值信号，能够准确反映任务进度。通过利用视频语料库中的时空先验，ViVa 还能泛化到新物体，凸显了视频生成模型在价值估计方面的巨大潜力。

## Abstract
Vision-language-action (VLA) models have advanced robot manipulation through large-scale pretraining, but real-world deployment remains challenging due to partial observability and delayed feedback. Reinforcement learning addresses this via value functions, which assess task progress and guide policy improvement. However, existing value models built on vision-language models (VLMs) struggle to capture temporal dynamics, undermining reliable value estimation in long-horizon tasks. In this paper, we propose ViVa, a video-generative value model that repurposes a pretrained video generator for value estimation. Taking the current observation and robot proprioception as input, ViVa jointly predicts future proprioception and a scalar value for the current state. By leveraging the spatiotemporal priors of a pretrained video generator, our approach grounds value estimation in anticipated embodiment dynamics, moving beyond static snapshots to intrinsically couple value with foresight. Integrated into RECAP, ViVa delivers substantial improvements on real-world box assembly. Qualitative analysis across all three tasks confirms that ViVa produces more reliable value signals, accurately reflecting task progress. By leveraging spatiotemporal priors from video corpora, ViVa also generalizes to novel objects, highlighting the promise of video-generative models for value estimation.

*   **研究背景**：视觉-语言-动作（VLA）模型在机器人操控中取得了进展，但在面对部分可观测性和延迟反馈的现实环境时，仍难以准确判断任务进度。
*   **核心问题**：现有的价值函数（Value Function）多基于视觉语言模型（VLM）构建，而 VLM 主要在静态图像上预训练，缺乏对物理世界时间动态（Temporal Dynamics）的理解。这导致模型在长程任务中无法可靠地估计价值，难以区分有益行为与错误行为。
*   **核心思想**：价值估计本质上是一个“预见未来”的问题。论文提出利用预训练视频生成模型（Video Generative Models）的时空先验知识，将价值学习重新表述为未来预测任务。

#### 2. 方法论
*   **核心架构**：基于预训练的视频扩散 Transformer（Wan2.2）。通过“潜空间注入”（Latent Injection）技术，将非图像模态引入模型，而不改变其核心架构。
*   **关键技术细节**：
    *   **输入注入**：将当前多视图图像、机器人本体感受（Proprioception）和标量价值映射为潜帧（Latent Frames）。本体感受通过重复填充（Repeat-padding）嵌入，标量价值通过广播（Broadcast）嵌入。
    *   **联合预测**：模型输入当前观测作为“干净”的条件帧，同时预测两个“噪声”目标帧：**未来的本体感受状态**（$K$ 步后）和**当前状态的标量价值**。
    *   **价值定义**：采用归一化回报（Normalized Return）。成功轨迹的价值在 $[0, 1)$ 之间，失败轨迹通过终端惩罚偏移至 $[1, 2)$，从而在价值空间中强制拉开成功与失败的距离。
    *   **训练目标**：采用流匹配（Flow Matching）目标函数，联合优化未来状态预测损失和价值预测损失。

#### 6. 主要结论与发现
*   **性能提升**：在纸箱组装任务中，ViVa 将成功率从模仿学习的 53% 提升至 **73%**，吞吐量也显著增加。
*   **可靠性**：ViVa 产生的价值信号更平滑且更具预见性，能够准确捕捉任务里程碑，并在发生细微物理错误时产生明显的价值下降。

#### 7. 优点与亮点
*   **范式创新**：将价值函数从“静态分类/排序”转变为“动态未来预测”，充分利用了视频生成模型对物理规律的内在建模。
*   **具身耦合**：通过联合预测未来本体感受，强制模型内化机器人的动力学特征，使价值估计不仅基于“看”，更基于“动”。
*   **效率优势**：在提供更高精度的同时，训练和推理速度均优于传统的 VLM 方案。

*   **数据依赖**：模型性能高度依赖于预训练视频生成器的质量，且仍需要一定规模的带标签机器人演示数据进行微调。
