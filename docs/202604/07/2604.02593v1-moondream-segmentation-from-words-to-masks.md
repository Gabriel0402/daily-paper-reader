## 摘要
我们提出了 Moondream Segmentation，它是视觉语言模型 Moondream 3 的指代图像分割（referring image segmentation）扩展。给定一张图像和一个指代表达式，该模型自回归地解码一条向量路径，并迭代地将光栅化掩码细化为最终的详细掩码。我们引入了一个强化学习阶段，通过直接优化掩码质量来解决监督信号中的歧义。该阶段的展开（Rollouts）为细化器生成了从粗糙到真实标签（ground-truth）的目标。为了减轻多边形标注带来的评估噪声，我们发布了 RefCOCO-M，这是一个经过清洗的 RefCOCO 验证集划分，具有边界精确的掩码。Moondream Segmentation 在 RefCOCO (val) 上实现了 80.2% 的 cIoU，在 LVIS (val) 上实现了 62.6% 的 mIoU。

## Abstract
We present Moondream Segmentation, a referring image segmentation extension of Moondream 3, a vision-language model. Given an image and a referring expression, the model autoregressively decodes a vector path and iteratively refines the rasterized mask into a final detailed mask. We introduce a reinforcement learning stage that resolves ambiguity in the supervised signal by directly optimizing mask quality. Rollouts from this stage produce coarse-to-ground-truth targets for the refiner. To mitigate evaluation noise from polygon annotations, we release RefCOCO-M, a cleaned RefCOCO validation split with boundary-accurate masks. Moondream Segmentation achieves a cIoU of 80.2% on RefCOCO (val) and 62.6% mIoU on LVIS (val).

*   **整体含义**：本文通过将 Moondream 3 扩展为分割模型，提出了一种“从单词到向量路径，再到精细掩码”的两阶段方案，并引入强化学习（RL）来优化这一过程。

### 2. 方法论：核心思想与技术细节
模型由 **向量路径解码器** 和 **迭代掩码细化器** 两部分组成：
*   **向量路径表示**：模型首先自回归地输出边界框（Bounding Box）和一组 SVG 风格的指令（如 M, L, C, Z）及坐标，形成紧凑的中间表示。
*   **强化学习（RL）优化**：
    *   **解决歧义**：由于同一形状有多种向量表达方式，直接用 Token 交叉熵损失（SFT）会导致梯度方差大。
    *   **GRPO 算法**：采用群组相对策略优化（Group Relative Policy Optimization），直接针对光栅化后的掩码质量进行奖励。
    *   **分段奖励函数**：根据模型水平动态切换奖励重点：框 IoU（目标发现） $\rightarrow$ Tversky 指数（覆盖率） $\rightarrow$ 边界 IoU（边缘锐化）。
*   **迭代细化器（Refiner）**：
    *   接收粗糙掩码作为“密集提示”（Dense Prompt）。
    *   利用 Moondream 的冻结视觉特征（结合早期和晚期层特征以获取高频细节）。
    *   通过“掩码 Token + 质量头”架构，进行 $T=5$ 次迭代优化，不断修正边界。
*   **数据流水线**：构建了超过 1000 万样本的训练集，通过 VLM 集成标注、Moondream 验证和分割模型过滤，确保数据质量。
