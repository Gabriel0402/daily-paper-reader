## 摘要
视觉-语言-动作（VLA）模型通过大规模预训练推动了机器人操作的发展，但由于部分可观测性和延迟反馈，现实世界中的部署仍然具有挑战性。强化学习通过价值函数来解决这一问题，用以评估任务进展并指导策略改进。然而，基于视觉-语言模型（VLMs）构建的现有价值模型难以捕获时间动态，从而削弱了在长时任务中进行可靠价值估计的能力。本文提出了 ViVa，一种视频生成式价值模型，它重新利用了预训练的视频生成器用于价值估计。ViVa 以当前观测和机器人本体感知为输入，同时预测未来的本体感知以及当前状态的标量价值。通过利用预训练视频生成器的时空先验，我们的方法将价值估计建立在对预期体现动态的建模之上，从静态快照迈向内在地将价值与前瞻性相结合的框架。当 ViVa 集成进 RECAP 系统后，在真实世界的箱体组装任务中取得了显著改进。对三项任务的定性分析表明，ViVa 产生了更可靠的价值信号，能准确反映任务进展。通过从视频语料库中汲取时空先验，ViVa 还能够泛化至新物体，凸显了视频生成模型在价值估计中的潜力。

## Abstract
Vision-language-action (VLA) models have advanced robot manipulation through large-scale pretraining, but real-world deployment remains challenging due to partial observability and delayed feedback. Reinforcement learning addresses this via value functions, which assess task progress and guide policy improvement. However, existing value models built on vision-language models (VLMs) struggle to capture temporal dynamics, undermining reliable value estimation in long-horizon tasks. In this paper, we propose ViVa, a video-generative value model that repurposes a pretrained video generator for value estimation. Taking the current observation and robot proprioception as input, ViVa jointly predicts future proprioception and a scalar value for the current state. By leveraging the spatiotemporal priors of a pretrained video generator, our approach grounds value estimation in anticipated embodiment dynamics, moving beyond static snapshots to intrinsically couple value with foresight. Integrated into RECAP, ViVa delivers substantial improvements on real-world box assembly. Qualitative analysis across all three tasks confirms that ViVa produces more reliable value signals, accurately reflecting task progress. By leveraging spatiotemporal priors from video corpora, ViVa also generalizes to novel objects, highlighting the promise of video-generative models for value estimation.

### 研究动机
- **机器人强化学习的关键挑战**：真实世界机器人操作受限于**部分可观测性**和**延迟反馈**，使得任务进展与奖励信号之间存在时间延迟。
- **价值函数的重要性**：在强化学习（RL）中，价值函数衡量当前状态下未来预期回报，是策略改进的核心信号。
- **现有方法的瓶颈**：多数基于视觉语言模型（VLMs）的价值预测模型主要训练于静态图像-文本数据，对时序变化缺乏建模能力，难以反映**任务动态演化**（场景变化、操作顺序、动作后果）。
- **核心问题**：如何让价值函数具备**时间前瞻性（foresight）**和**体现动态（embodiment dynamics）**的能力，从而在长时任务中产生可靠、平滑的价值估计信号。

### 研究目标
- 将**价值估计（value estimation）问题转化为未来预测问题（future anticipation）**。
- 通过引入视频生成模型的时空先验，让价值预测具备对未来动态变化的感知。

## 二、方法论：ViVa 模型设计

### 核心思想
- **ViVa（Video-generative Value Model）**提出将一个预训练的视频生成模型（Wan2.2 diffusion Transformer）重新用于价值估计。
- 输入：当前多视角图像观测 + 机器人本体感知（关节角度、姿态等）。
- 输出：联合预测（1）未来的本体感知状态；（2）当前状态的标量价值。
- 通过**视频生成网络的时空建模能力**来获得价值预测的“前瞻感”。

### 模型结构与关键技术
1. **基于Diffusion Transformer的生成骨干**  
   - 使用 Wan2.2 模型作为视频生成核心，输入采用**潜变量注入（latent injection）**形式整合视觉与本体信息。
   - 当前观测与预测目标都被编码为低维潜空间（latent frames）。
2. **多模态潜变量设计**  
   - 图像通过时空VAE压缩；
   - 本体感（proprioception）通过“重复填充”（repeat padding）变换为与视频帧相同的尺寸；
   - 标量价值通过“广播扩展”（broadcast）注入为一个均匀潜变量帧。
3. **训练过程**  
   - 构造由条件帧（当前观测）和目标帧（未来本体 + 价值）组成的序列；
   - 使用**流匹配（Flow Matching）损失**优化，使模型学会从噪声反演出未来潜变量；
   - 目标函数包括价值预测与未来本体预测两部分：  
     \[
     \mathcal{L} = \lambda_{\text{prop}} \mathbb{E}[||v_\theta(z_q) - (z_1 - z_q^0)||^2] + \lambda_{\text{val}} \mathbb{E}[||v_\theta(z_v) - (z_1 - z_v^0)||^2]
     \]
4. **奖励与监督信号设计**
   - 定义特殊的分段奖励函数，使成功与失败轨迹在返回值（return）上相互区分，确保训练中有稳定的进展与失败边界。
   - 目标值为归一化回报 \( G_t \in [0,2) \)，成功轨迹为 [0,1)，失败轨迹为 [1,2)。

## 七、方法优势与创新点

- **创新视角**：首次系统地将视频生成扩散模型（Video Diffusion Transformer）用于强化学习中的价值函数建模。
- **时空一致性强**：通过未来状态预测建立“具身前瞻性”，使价值曲线平滑且敏感于任务进展。
- **架构兼容性好**：无需修改核心视频生成框架，采用潜空间注入（latent injection）即可扩展价值估计与本体预测。
- **泛化能力优**：具备对未知对象或不同任务阶段的动态理解力。
- **可解释性**：价值信号变化与实际动作事件（抓取、对齐、放置）高度对应，易于诊断行为优劣。
