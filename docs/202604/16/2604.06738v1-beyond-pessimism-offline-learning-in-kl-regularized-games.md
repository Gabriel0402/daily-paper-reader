## 摘要
我们研究了KL正则化的双人零和博弈中的离线学习问题，其中策略在相对于固定参考策略的KL约束下进行优化。以往的研究依赖于悲观的价值估计来处理分布偏移，仅能获得 $\widetilde{\mathcal{O}}(1/\sqrt n)$ 的统计率。我们针对KL正则化博弈提出了一种无悲观的新算法和分析框架，基于KL正则化最优响应的平滑性以及由偏斜对称性诱导的纳什均衡的稳定性性质。这使我们首次在KL正则化的零和博弈离线学习中达成了 $\widetilde{\mathcal{O}}(1/n)$ 的样本复杂度上界，完全不依赖于悲观假设。我们进一步提出了一种高效的自对弈策略优化算法，并证明其在与样本规模呈线性关系的迭代次数下，可以获得与极小极大估计器相同的快速 $\widetilde{\mathcal{O}}(1/n)$ 统计率。

## Abstract
We study offline learning in KL-regularized two-player zero-sum games, where policies are optimized under a KL constraint to a fixed reference policy. Prior work relies on pessimistic value estimation to handle distribution shift, yielding only $\widetilde{\mathcal{O}}(1/\sqrt n)$ statistical rates. We develop a new pessimism-free algorithm and analytical framework for KL-regularized games, built on the smoothness of KL-regularized best responses and a stability property of the Nash equilibrium induced by skew symmetry. This yields the first $\widetilde{\mathcal{O}}(1/n)$ sample complexity bound for offline learning in KL-regularized zero-sum games, achieved entirely without pessimism. We further propose an efficient self-play policy optimization algorithm and prove that, with a number of iterations linear in the sample size, it achieves the same fast $\widetilde{\mathcal{O}}(1/n)$ statistical rate as the minimax estimator.

## 二、方法论与关键技术  

### 1. 总体思路  
- 作者提出一种**无悲观（pessimism-free）离线学习框架**，直接在**KL正则化的双人零和博弈**中进行估计与优化。  
- 通过利用**KL正则化带来的几何结构特性**——包括**最佳响应函数的平滑性（smoothness）**与**由偏斜对称性（skew-symmetry）带来的纳什均衡稳定性（stability）**——取代传统的悲观性假设，从而避免保守惩罚和不确定性下界构造。  

### 2. 关键算法  
论文提出两种主要算法：  

#### （1）算法 1：最小二乘极小极大估计（Minimax Estimation without Pessimism）  
- **输入**：离线数据集 $\mathcal{D}$、参考策略 $\pi_{\text{ref}}$、正则系数 $\eta$、函数类 $\mathcal{G}$。  
- **步骤**：  
  1. 执行**最小二乘回归**以估计真实 payoff 函数 $g^\star$：  
     \[
     \hat g = \arg\min_{g \in \mathcal G} \sum_i (g(x_i,a_{i,1},a_{i,2}) - p_i)^2
     \]
  2. 在估计函数 $\hat g$ 上求解对应的正则化零和博弈：  
     \[
     \hat\pi = \arg\max_{\pi_1} \min_{\pi_2} J(\hat g, \pi_1, \pi_2)
     \]
  3. 输出均衡策略对 $(\hat\pi_1,\hat\pi_2)$。  
- **结果**：无需任何惩罚项即可在理论上达到 $\widetilde{O}(1/n)$ 样本复杂度。  

#### （2）算法 2：离线自博弈策略优化（Offline Self-Play Policy Optimization）  
- **核心思想**：基于 KL 约束的自博弈镜像下降（mirror descent-ascent）更新机制。  
- **更新形式**：每轮迭代中按照以下形式更新策略  
  \[
  \pi^{(t+1)}_1 \propto \pi^{(t)}_1{}^{1-\alpha_t/\eta} 
  \exp(\alpha_t \hat f^{(t)}_1) \pi_{\mathrm{ref},1}^{\alpha_t/\eta}
  \]
  对抗玩家采用对称的负效用更新。  
- **收敛性质**：当迭代次数 $T$ 与样本量 $n$ 成线性关系时，达到与算法1相同的 $\widetilde{O}(1/n)$ 差距。  

### 3. 理论创新点  
- 引入**Lemma 2–4**构建平滑性与稳定性理论支撑：
  - 最优响应偏差可表示为 KL 距离；

  - 纳什点的稳定性保证了小的单边估计误差即可限制总体收敛误差。  
- 利用**单边可控性（unilateral concentrability）**假设，确保即使只有一方偏离，也能由行为策略的数据集覆盖。
