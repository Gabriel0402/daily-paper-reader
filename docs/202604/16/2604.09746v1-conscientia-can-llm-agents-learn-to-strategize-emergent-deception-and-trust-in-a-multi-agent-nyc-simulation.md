## 摘要
随着大型语言模型（LLM）越来越多地被部署为自主代理，理解在多代理环境中战略行为的涌现已成为一个重要的对齐挑战。我们采取中立的实证立场，构建了一个可以直接观察和衡量战略行为的可控环境。我们引入了一个简化版纽约市模型中的大规模多代理模拟，其中由 LLM 驱动的代理在相互对立的激励下进行互动。蓝色代理的目标是高效地到达目的地，而红色代理则试图通过使用有说服力的语言将其引导至广告牌密集的路线，以最大化广告收入。隐蔽身份使得导航具有社会中介性，迫使代理决定何时信任或欺骗。我们通过迭代模拟管线研究策略学习，该管线在重复交互轮次中使用 Kahneman-Tversky Optimization（KTO）更新代理策略。蓝色代理的优化目标是减少广告牌曝光同时保持导航效率，而红色代理则不断调整以利用剩余的弱点。经过多轮迭代，最佳蓝色策略的任务成功率从46.0%提升至57.3%，但易受攻击性仍高达70.7%。后期策略表现出更强的选择性合作，同时保持轨迹效率。然而，安全性与有用性之间的权衡仍然存在：能够更好抵抗对抗性操纵的策略无法同时最大化任务完成率。总体而言，我们的结果表明，LLM 代理能够表现出有限的战略行为，包括选择性信任与欺骗，但仍高度易受对抗性劝说的影响。

## Abstract
As large language models (LLMs) are increasingly deployed as autonomous agents, understanding how strategic behavior emerges in multi-agent environments has become an important alignment challenge. We take a neutral empirical stance and construct a controlled environment in which strategic behavior can be directly observed and measured. We introduce a large-scale multi-agent simulation in a simplified model of New York City, where LLM-driven agents interact under opposing incentives. Blue agents aim to reach their destinations efficiently, while Red agents attempt to divert them toward billboard-heavy routes using persuasive language to maximize advertising revenue. Hidden identities make navigation socially mediated, forcing agents to decide when to trust or deceive. We study policy learning through an iterative simulation pipeline that updates agent policies across repeated interaction rounds using Kahneman-Tversky Optimization (KTO). Blue agents are optimized to reduce billboard exposure while preserving navigation efficiency, whereas Red agents adapt to exploit remaining weaknesses. Across iterations, the best Blue policy improves task success from 46.0% to 57.3%, although susceptibility remains high at 70.7%. Later policies exhibit stronger selective cooperation while preserving trajectory efficiency. However, a persistent safety-helpfulness trade-off remains: policies that better resist adversarial steering do not simultaneously maximize task completion. Overall, our results show that LLM agents can exhibit limited strategic behavior, including selective trust and deception, while remaining highly vulnerable to adversarial persuasion.

## 二、方法论与核心技术

### 1. 总体思路
构建一个**简化纽约市地图**的对抗性多智能体仿真环境：
- **蓝色代理（Blue）**：目标是高效到达目的地，避免经过广告牌区域。
- **红色代理（Red）**：使用说服性语言将蓝色代理误导至广告牌路线，以增加“广告收益”。

### 2. 核心算法：Kahneman–Tversky Optimization（KTO）
一种受前景理论启发的优化方法，用于在**无配对监督**的场景中细化代理策略。

#### 工作流程：
1. **数据生成**：使用基础模型 Qwen3-4B 生成初始交互轨迹。  
2. **标注与扩充**：将轨迹分类为“可取/不可取”（desirable/undesirable），由更大模型 Qwen3-14B 进行数据增强。每轮生成约 3,600 条可取样本、1,500 条不可取样本。  
3. **KTO 对齐训练**：基于以下优化目标：

   - 对可取样本提升概率，对不可取样本压低概率：
     \[
     L_{KTO} = E_{D+}[w^+ \log(1 + e^{-(r_\theta - z_0)})] + E_{D-}[w^- \log(1 + e^{-(z_0 - r_\theta)})]
     \]
     其中 \(r_\theta(x,y) = \beta \log \frac{\pi_\theta(y|x)}{\pi_{ref}(y|x)}\)。

4. **策略迭代**：一轮仿真 → 数据更新 → 再训练 → 再部署，重复 **10 代循环**。

### 3. 优化区别
相较于常见的 SFT、DPO、PPO：
- KTO 适合无配对监督的轨迹级判断，不需明确奖励函数或偏好对；
- 直接以行为结果为优化目标，适配复杂社会交互的延迟反馈。

## 三、实验设计

### 1. 环境与场景
- **模拟环境**：纽约市简化路网。  
- **代理数量**：150 个蓝方、100 个红方。  
- **交互目标**：蓝方导航至目的地、红方诱导蓝方至广告牌区域。  
- **迭代次数**：10 代对齐实验，每代都重新仿真整个交互。

### 2. 仿真指标体系
- **任务成功率 (TSR)**、**易受攻击率 (SR)**；
- **行为指标**：蓝–红抵抗率、蓝–蓝信任效率、过度拒绝率；
- **路径指标**：平均轨迹长度、冗余度、路径安全率；
- **综合效用**：蓝方效用 U_blue 与红方效用 U_red。

### 3. 对比与评估方法
- 采用量化指标（任务完成率、抗干扰性）与质性评估（LLM-as-a-Judge）。  
- 三种评审模型：**GPT-5.4-nano、Mistral-large-latest、Step-3.5-flash**。  
- 使用 Cohen’s kappa 和 Krippendorff’s alpha 衡量一致性。

## 六、主要结论与发现

1. **性能变化**：  
   - 蓝方任务成功率由 46.0% → 57.3%。  
   - 最低易受攻击率降至 70.7%（第 8 代）。  
   - 改进非单调：安全最优与任务最优代不一致。  

2. **行为演化与策略涌现**：  
   - 出现“选择性信任”与“延迟抵抗”的战略模式。  
   - 蓝方学会拒绝恶意建议但仍可合作于善意建议。  
   - 红方策略从“紧迫劝说”演化为“文化 / 景观框架欺骗”。  

3. **安全–有用性权衡**：  
   - 抗干扰性提升往往伴随任务效率下降。  
   - 表明真正的“稳健策略”仍未达成。

4. **长期脆弱性**：  
   - 延迟妥协现象显著：84.6% 初始拒绝的蓝方最终被操控。  
   - 多重红方接触与社会信任语言显著提高误导率。
