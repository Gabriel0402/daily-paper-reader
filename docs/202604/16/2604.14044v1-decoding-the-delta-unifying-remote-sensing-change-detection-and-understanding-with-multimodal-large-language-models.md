## 摘要
尽管多模态大型语言模型（Multimodal Large Language Models，MLLMs）在通用视觉-语言任务中表现出色，但它们在遥感变化理解中的应用受到一种根本性的“时间盲”限制。现有的架构缺乏内在的多时相对比推理机制，并且在精确空间定位方面表现不佳。为解决这一问题，我们首先提出了 Delta-QA，一个包含 18 万个视觉问答样本的综合基准数据集。Delta-QA 在双时相和三时相场景中统一了像素级分割与视觉问答任务，将变化解释结构化为四个渐进的认知维度。在方法上，我们提出了 Delta-LLaVA，这是一种专为多时相遥感解释设计的新型 MLLM 框架。该框架通过三项核心创新克服了简单特征拼接的局限性：一是“Change-Enhanced Attention”（变化增强注意力）模块，用于系统性地分离并强化视觉差异；二是“Change-SEG”模块，利用“Change Prior Embedding”提取可微分的差异特征作为大语言模型的输入；三是“Local Causal Attention”（局部因果注意力）机制，以防止跨时相的上下文泄漏。大量实验表明，Delta-LLaVA 在复杂变化推理与高精度边界定位任务上显著优于领先的通用 MLLM 和专用分割模型，建立了一个用于地球观测智能的统一框架。

## Abstract
While Multimodal Large Language Models (MLLMs) excel in general vision-language tasks, their application to remote sensing change understanding is hindered by a fundamental "temporal blindness". Existing architectures lack intrinsic mechanisms for multi-temporal contrastive reasoning and struggle with precise spatial grounding. To address this, we first introduce Delta-QA, a comprehensive benchmark comprising 180k visual question-answering samples. Delta-QA unifies pixel-level segmentation and visual question answering across bi- and tri-temporal scenarios, structuring change interpretation into four progressive cognitive dimensions. Methodologically, we propose Delta-LLaVA, a novel MLLM framework explicitly tailored for multi-temporal remote sensing interpretation. It overcomes the limitations of naive feature concatenation through three core innovations: a Change-Enhanced Attention module that systematically isolates and amplifies visual differences, a Change-SEG module utilizing Change Prior Embedding to extract differentiable difference features as input for the LLM, and Local Causal Attention to prevent cross-temporal contextual leakage. Extensive experiments demonstrate that Delta-LLaVA decisively outperforms leading generalist MLLMs and specialized segmentation models in complex change deduction and high-precision boundary localization, establishing a unified framework for earth observation intelligence.

## 一、核心问题与研究背景

- **问题定义**：多模态大型语言模型（MLLMs）在通用视觉语言任务中表现优异，但在遥感影像变化检测与理解中表现不佳。根本原因是这些模型存在“**时间盲区（temporal blindness）**” —— 缺乏多时相对比推理能力，不能真正理解地表变化。  
- **背景意义**：遥感影像持续记录地球多时态数据，对城市化、生态动态、灾害响应等至关重要。现有 MLLMs 能识别单时相图像内容（颜色、位置、数量），但无法比较不同时间段的图像，也缺乏精确的像素级定位能力。  
- **研究动机**：希望构建一个统一框架，将**变化检测**（Change Detection）与**语义理解**（Change Understanding）结合起来，实现同时输出文本说明与像素级分割结果，为地球观测智能奠定基础。

## 二、方法论：Delta-QA 数据集与 Delta-LLaVA 模型

### 1. 总体思路
作者提出**数据与模型双向改进**策略：
- 构建新的**Delta-QA 数据集**，系统组织多时相（双时相、三时相）视觉问答任务；
- 设计**Delta-LLaVA 模型架构**，通过差异增强模块实现真正的跨时相视觉语言推理。

### 2. Delta-QA 数据集
- 包含 **180,876 条多模态问答样本**，整合自 SECOND、Landsat-SCD 和 WUSU 三个主流遥感变化检测数据集。
- 支持 **点/框视觉提示（Visual Prompt）**、**像素级掩膜（mask）对齐**以及**多种 QA 类型**。
- 四个认知维度：
  1. **变化识别与分类（CIC）**：识别具体地表类别变化。
  2. **变化定量与统计（CQS）**：计算变化数量或面积。
  3. **变化趋势推理（CTI）**：分析演化趋势，如城镇化进程。
  4. **变化空间分析（CSA）**：空间分布模式分析。
- 同时支持双时相和三时相场景（如城市扩张、多阶段地貌变化）。

### 3. Delta-LLaVA 模型结构

#### 核心模块
- **Change-Enhanced Attention（CEA）**  
  通过计算两个时间阶段图像特征的 *绝对差值* 和多层交叉注意力，强化变化区域、抑制背景噪声。
  - 数学流程包括：
    - 差异特征提取：`F_diff = |F_T1 - F_T2|`
    - 差异增强：通过 MLP 和跨注意力更新特征，使变化区域权重升高。
- **Change-SEG 模块**  
  引入 **Change Prior Embedding（CPE）** 概念，生成差异导向查询向量 `Q_d`，将这些查询作为掩膜预测的先验。  
  - 可同时生成语义分割掩膜（Mask）和类别分数（Score）。
  - 与语言模型交互时，变化区域以 `[SEG]` token 表示，保证文本与掩膜一致。
- **Local Causal Attention（LCA）**  
  提出一种局部因果注意力机制，限制跨时相特征不互相干扰，防止信息“泄漏”。  
  - 在 token 序列中形成棋盘状注意力图，实现时间相位隔离。

#### 训练策略
- **两阶段训练**：
  1. **阶段一：视觉对齐预训练**  
     冻结 LLM 参数，优化视觉模块（CEA、SEG、投影器），通过 `Lpretrain = Ltext + Lreg` 进行特征对齐。
  2. **阶段二：多时相指令微调**  
     使用 LoRA 调整 LLM，综合文本生成、掩膜分割和类别分类损失：  
     `Linstruction = Ltext + α LFocal + β LDice + Lcls`。
