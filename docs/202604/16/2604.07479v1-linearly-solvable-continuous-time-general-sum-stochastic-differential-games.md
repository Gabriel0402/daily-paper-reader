## 摘要
本文提出了一类连续时间、有限参与者的随机一般和型微分博弈，其解可由精确的线性偏微分方程（PDE）系统得到。我们构建了一个分布规划博弈，利用交叉对数似然比（cross-log-likelihood ratio）来自然建模多智能体空间冲突，如拥塞规避问题。通过应用广义多变量的Cole-Hopf变换，我们将相关的非线性哈密顿-雅可比-贝尔曼（HJB）方程解耦为一个线性偏微分方程系统。该简化使得通过Feynman-Kac路径积分方法可以高效地、无需网格地计算反馈Nash均衡策略，有效克服维度灾难。

## Abstract
This paper introduces a class of continuous-time, finite-player stochastic general-sum differential games that admit solutions through an exact linear PDE system. We formulate a distribution planning game utilizing the cross-log-likelihood ratio to naturally model multi-agent spatial conflicts, such as congestion avoidance. By applying a generalized multivariate Cole-Hopf transformation, we decouple the associated non-linear Hamilton-Jacobi-Bellman (HJB) equations into a system of linear partial differential equations. This reduction enables the efficient, grid-free computation of feedback Nash equilibrium strategies via the Feynman-Kac path integral method, effectively overcoming the curse of dimensionality.

## 二、方法论与理论框架

### 1. 总体思想

- 将多智能体的博弈视为**分布规划问题**（distribution planning game），每个玩家控制其轨迹的概率分布。
- 通过引入**交叉对数似然比（cross-log-likelihood）**建模玩家间的相互作用：  
  - 当该项为正时，玩家倾向于避开其他玩家高概率的区域（拥塞规避）；  
  - 当为负时，则倾向于聚合（协同或追随）。

### 2. 核心结构与公式

- 每个玩家在时间区间 \([0,T]\) 内的状态遵循 Itô 随机微分方程：
  \[
  dx_t = f(x_t)\,dt + g(x_t)\,dv_t
  \]
  控制项通过改变输入的分布实现，即 \(dv_t = u_i(t)\,dt + dw_t\)。

- **代价函数定义**（衡量玩家 i 的目标）：
  \[
  J_i(P) = E_{P_i}\left[\int_0^T C_i(x_t)\,dt + \Psi_i(x_T)\right]
  + \alpha_{ii}E_{P_i}\!\left[\log\dfrac{dP_i}{dR_i}\right]
  + \sum_{j\neq i}\alpha_{ij}E_{P_i}\!\left[\log\dfrac{dP_j}{dR_j}\right]
  \]
  其中第二项是自信息熵惩罚，第三项为跨玩家交叉耦合项。

### 3. 关键理论步骤

1. **等价性建立**：  
   通过 Girsanov 定理将信息论形式转换为控制论形式，使相对熵对应于控制能量。
2. **推导耦合 HJB 方程**：  
   得到每位玩家的动态规划方程并构造其梯度耦合系统。
3. **多变量 Cole–Hopf 变换**：  
   定义转化变量 \(J_i = -\sum_j\alpha_{ij}\log Z_j\)，通过该变换非线性项完全消除，将耦合非线性 PDE 转化为独立的线性 PDE：
   \[
   -\partial_t Z_i = (f+\bar{u}_i g)^\top\nabla_x Z_i + \tfrac12\text{Tr}(gg^\top\nabla^2_{xx}Z_i)
   -\Big(\sum_j \beta_{ij}C_j(x_t)\Big)Z_i
   \]
4. **Feynman–Kac 路径积分求解**：  
   每个玩家的解可表示为期望形式，可直接通过 Monte Carlo 采样估计，无需网格。

5. **最优控制恢复公式**：
   \[
   u_i^*(t,x) = \bar{u}_i(t) + g(x)^\top\nabla_x\log Z_i(t,x)
   \]
   可通过路径积分的加权噪声均值近似得到。

## 三、实验设计

  - 交叉耦合矩阵 \(\alpha(\gamma)=\begin{bmatrix}1 & \gamma\\ \gamma &1\end{bmatrix}\)，其中 \(\gamma\in[-1,1]\) 控制合作或竞争强度。

- **Benchmark 对比**：  
  - 对比三种不同的交互情形：

    3. **排斥耦合**（γ>0），体现拥塞规避。

- **评估指标**：
  - 分布间的期望距离 \(D(t)=|\mathbb{E}[x_1]-\mathbb{E}[x_2]|\)；
  - 终端状态分布差异和目标偏离程度；
  - 控制策略下的轨迹与重加权分布对比。
