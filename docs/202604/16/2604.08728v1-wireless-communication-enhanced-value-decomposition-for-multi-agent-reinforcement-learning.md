## 摘要
在多智能体强化学习（Multi-Agent Reinforcement Learning，MARL）中，智能体间的通信有助于实现合作，但大多数方法假设理想化信道，现有的价值分解方法也忽略了哪些智能体之间成功共享了信息。我们提出了 CLOVER —— 一种合作型 MARL 框架，其集中式价值混合器（centralized value mixer）依赖于在真实无线信道下实现的通信图。该通信图为价值分解引入了关系归纳偏置（relational inductive bias），限制了个体效用的混合方式，使之取决于实际通信结构。混合器基于图神经网络（GNN），由一个具有置换等变性的超网络（Permutation-Equivariant Hypernetwork）生成节点特定权重；通信边上的多跳传播会重塑信用分配，使不同的拓扑结构引发不同的混合方式。我们证明该混合器具有置换不变性、单调性（保持 IGM 条件），且其表达能力严格强于 QMIX 类型的混合器。为处理真实信道，我们构建了一个增强型 MDP，将随机信道效应从智能体计算图中分离，并采用了随机感受野编码器（stochastic receptive field encoder）来处理可变大小的信息集，从而实现端到端可微训练。在 Predator-Prey 和 Lumberjacks 基准测试中，CLOVER 在 p-CSMA 无线信道下的收敛速度与最终性能均优于 VDN、QMIX、TarMAC+VDN 和 TarMAC+QMIX。行为分析表明，智能体学会了自适应的信号传递与监听策略，消融实验验证了通信图归纳偏置是性能提升的关键来源。

## Abstract
Cooperation in multi-agent reinforcement learning (MARL) benefits from inter-agent communication, yet most approaches assume idealized channels and existing value decomposition methods ignore who successfully shared information with whom. We propose CLOVER, a cooperative MARL framework whose centralized value mixer is conditioned on the communication graph realized under a realistic wireless channel. This graph introduces a relational inductive bias into value decomposition, constraining how individual utilities are mixed based on the realized communication structure. The mixer is a GNN with node-specific weights generated by a Permutation-Equivariant Hypernetwork: multi-hop propagation along communication edges reshapes credit assignment so that different topologies induce different mixing. We prove this mixer is permutation invariant, monotonic (preserving the IGM condition), and strictly more expressive than QMIX-style mixers. To handle realistic channels, we formulate an augmented MDP isolating stochastic channel effects from the agent computation graph, and employ a stochastic receptive field encoder for variable-size message sets, enabling end-to-end differentiable training. On Predator-Prey and Lumberjacks benchmarks under p-CSMA wireless channels, CLOVER consistently improves convergence speed and final performance over VDN, QMIX, TarMAC+VDN, and TarMAC+QMIX. Behavioral analysis confirms agents learn adaptive signaling and listening strategies, and ablations isolate the communication-graph inductive bias as the key source of improvement.

- **核心问题**：多智能体强化学习（MARL）依靠智能体间通信以实现协作，但已有方法大多假设通信信道是理想的（即消息总能可靠传递），忽略了在现实无线信道中「谁成功地与谁通信」的动态影响。

  - 现实环境中通信具有**随机性与动态性**（信号衰落、干扰、竞争）。

  - 通信拓扑本身可提供天然的**图结构先验（graph-structured inductive bias）**，这一点在现有研究中尚未充分挖掘。

## 二、方法论：CLOVER 框架

### 2.1 核心思想

提出 **CLOVER**（Communication-enhanced vaLue decOmposition oVer realistic wiReless channels）框架：
- 将真实无线通信信道下的**动态通信图（communication graph）**引入集中式价值混合器；
- 用该图结构约束各智能体效用值的混合方式；
- 在现实信道带来的随机丢包与接收不确定性下仍保持端到端可微分的训练流程。

### 2.2 关键技术组成

1. **增强型 MDP 建模（Augmented Dec-POMDP）**  
   - 将原环境状态空间扩展为 \( S = S_T \times S_C \)，其中 \(S_C\) 描述无线信道状态。  
   - 扩展动作空间 \(A = A_T \times A_C\)，其中 \(A_C\) 包含「是否传输」等通信动作。
   - 通过“因果状态对齐（causal state alignment）”策略，把通信结果延迟到下一时刻处理，使系统在梯度传递上可微。

2. **智能体网络结构**
   - **观测融合模块（Observation Fuser）**：独立编码游戏状态与无线观测（RSS、ACK 反馈）。
   - **随机感受野编码器（Stochastic Receptive Field Encoder）**：  
     - 处理变长、无序的接收消息集合。  
     - 采用「Sum-of-MLP」结构，保证**置换不变性与近似可逆性**。
   - **记忆式Q生成器（GRU-based Q-head）**：以历史观测与接收信息生成每个智能体的局部 \( Q_i(a^T_i, a^C_i) \)，并输出消息内容 \( m_i \)。

3. **通信图驱动的集中式混合器（GNN Mixer）**
   - 基于图神经网络（GNN）执行消息传递，将通信拓扑嵌入到全局价值 \( Q_\text{tot} \) 的计算中。
   - 每个节点的权重由**置换等变超网络（Permutation-Equivariant Hypernetwork, PEHypernet）**生成，可依赖全局状态 \( s \) 并保持置换不变。
   - 数学性质：
     - **置换不变性**（Theorem 2）：不受智能体编号影响；
     - **单调性**（Theorem 4）：保持 IGM 条件以支持分布式决策；
     - **表达能力严格强于 QMIX 型单调混合器**（Theorem 5）。

4. **训练算法**
   - 采用**集中式训练、分布式执行**（CTDE）范式；
   - 以时间差分（TD）损失为优化目标；
   - 每个智能体独立执行 ε-greedy 探索；
   - 所有非理想信道操作被视作环境随机因素，保证梯度在智能体模块中可传播。
