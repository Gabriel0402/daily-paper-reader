## 摘要
本文研究了动态控制系统的努力（effort）与韧性（resilience）概念，这两者由系统在满足给定有限时序逻辑（finite temporal logic）规范的条件下所能承受的最大扰动定义。对于给定的动态系统和规范，我们的目标是合成一个控制器，使系统在考虑输入约束的情况下既满足规范，又使其韧性最大化。此外，我们引入了一种新的指标，称为努力指标（effort metric），用于刻画在受扰系统中满足给定规范所需的最小输入边界。两个指标相关的问题均被形式化为鲁棒优化问题，其目标是在具有输入约束的系统中计算最大韧性或最小努力，同时合成相应控制器参数。我们进一步研究了韧性与努力之间的权衡关系，旨在最大化韧性并最小化控制努力。对于线性系统和线性控制器，我们针对闭环与开环系统的时变多面体（time-varying polytopic）规范类给出了精确解。对于非线性系统、非线性控制器以及更一般的规范情况，我们利用场景优化（scenario optimization）方法，提供了解的概率保证及计算可行性。最后，通过不同的案例研究展示了理论结果。

## Abstract
In this paper, we consider the notions of effort and resilience of a dynamical control system defined by the maximum disturbance the system can withstand while satisfying given finite temporal logic specifications. Given a dynamical system and a specification, the objective is to synthesize the controller such that the system satisfies the specification while maximizing its resilience, taking into account input constraints. In addition, we introduce a new metric, called the effort metric, which characterizes the minimal input bound necessary to satisfy a given specification for a perturbed system. The problem for both metrics is formulated as a robust optimization program where the objective is to compute the maximum resilience for the system with input constraints or the minimal effort while simultaneously synthesizing the corresponding controller parameters. Moreover, we study the trade-off between resilience and effort, where we seek to maximize resilience and minimize the control effort. For linear systems and linear controllers, exact solutions are provided for the class of time-varying polytopic specifications for the closed-loop and open-loop systems. For the case of nonlinear systems, nonlinear controllers, and more general specifications, we leverage tools from the scenario optimization approach, offering a probabilistic guarantee of the solution as well as computational feasibility. Different case studies are presented to illustrate the theoretical results.

## 一、研究背景与核心问题

- **研究动机**：  
  实际控制系统（如自动驾驶、工业机器人、电力系统）需在不确定和扰动环境下保持预期功能。传统鲁棒控制能保证系统在一定扰动范围内稳定，但无法**量化系统在保证规格满足的前提下可承受的最大扰动**。  
  研究者因此提出“**韧性（Resilience）**”的指标，用以度量系统在满足时序逻辑规范的条件下，能抵抗的最大扰动幅度。

- **问题定义**：  
  希望合成一个控制器，使系统在**满足有限时序逻辑（FTL / LTLf）规范**的同时：
  - 最大化系统韧性；
  - 或最小化实现规范所需的**输入努力（Effort）**；
  - 并研究韧性与努力之间的**加权权衡（trade-off）**。

- **研究意义**：  
  提供了一个**量化韧性与控制代价关系**的新理论框架，为未来自适应控制与安全关键系统的结构设计提供优化工具。

## 二、方法论与技术路线

### 2.1 核心思想

- 将韧性与努力问题统一表述为**鲁棒优化（robust optimization）**框架；
- 在输入和扰动受限的条件下，求解使**系统轨迹满足时序逻辑规范**的最优控制器；
- 定义三个指标：
  1. **韧性指标 \( g_\psi(x, \varepsilon_0) \)**：系统在输入约束下能承受的最大扰动上界；
  2. **努力指标 \( h_\psi(x, \mu_0) \)**：系统在给定扰动下，为满足规范所需的最小输入幅度；
  3. **韧性–努力权衡指标 \( p_\psi(x, w_1, w_2) \)**：通过加权 \( w_1\mu - w_2\varepsilon \) 形式实现多目标最优。

### 2.2 数学建模

- 系统模型：离散时间动态系统  
  \[
  x(k+1) = f(k, x(k), u(k)) + d(k)
  \]
  其中 \( d(k) \in D \)、\( u(k) \in U \)。

- 时序规范：有限时间逻辑（LTLf）公式描述的状态轨迹集合，如安全性（□）、可达性（◇）等。

- 优化形式：
  - 对于线性时变系统（LTV）与**线性控制器**：
    - 利用 **Farkas 引理** 将鲁棒约束转化为标准线性 / 多项式约束；
    - 闭环控制器形式： \( u(k) = \alpha_1 x(k) + \alpha_2 \)；
    - 开环控制器形式：输入序列 \( u = (u_0, \ldots, u_{N-1}) \)；
    - 问题最终转化为线性或多项式规划。

  - 对于**非线性系统与控制器**：
    - 基于 **场景优化（Scenario Optimization）**：
      - 通过在扰动空间随机采样有限个场景；
      - 构造近似优化问题；
      - 使用已知样本的解对整体分布进行概率性风险界定；
      - 提供解的**置信度下限**（如违约概率 ≤ b(s*, M)）。

### 2.3 算法实现与求解流程

1. 定义目标指标（韧性、努力或其加权组合）；  
2. 建立系统动力方程与规范约束；  
3. 将无限扰动约束转换为有限鲁棒形式（Farkas引理）；  
4. 构建优化问题：
   \[
   \max_{μ, ε, α, P} \; w_1μ - w_2ε \quad \text{s.t.} \; P A_b = μE(α),\; P B_b ≤ F(x, α, ε),\; P ≥ 0
   \]
5. 对线性系统用标准优化器（如线性规划或Knitro）求解；
6. 对非线性系统用场景优化+概率分析保证结果的置信水平。

## 五、主要结论与发现

1. **韧性与努力可统一为鲁棒优化问题，并可精确或概率求解**；  
2. **线性系统**下可得解析/精确解，**非线性系统**可得带概率保证的近似解；  
3. 提出韧性–努力的 **帕累托边界（Pareto Front）**，刻画其定量权衡关系；  
4. 实验表明：高韧性必然要求更高控制努力，符合能量–鲁棒度权衡原则；  
5. 多项式控制器在非线性场景中较线性控制器具有更高韧性；  
6. 方法能在合理计算时间内（<1分钟~30秒级）稳定求解。
