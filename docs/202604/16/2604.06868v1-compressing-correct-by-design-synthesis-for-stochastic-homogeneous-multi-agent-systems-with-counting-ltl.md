## 摘要
正确设计综合（Correct-by-design synthesis）为随机多智能体系统（MAS）建立形式化安全保证提供了一种系统化的框架。然而，基于有限抽象的传统方法在智能体数量和时序逻辑规范复杂度增加时，往往导致计算成本高昂。在本研究中，我们针对同质随机多智能体系统的计数线性时序逻辑（cLTL）规范进行研究，证明相应的满足概率可通过确定性有限自动机（DFA）获得结构化的张量分解。基于该结构，我们提出一种基于双树的值迭代框架，在动态规划过程中减少冗余计算。数值结果表明，所提出方法在处理复杂规范和大规模多智能体系统时具有良好的有效性与可扩展性。

## Abstract
Correct-by-design synthesis provides a principled framework for establishing formal safety guarantees for stochastic multi-agent systems (MAS). However, conventional approaches based on finite abstractions often incur prohibitive computational costs as the number of agents and the complexity of temporal logic specifications increase. In this work, we study homogeneous stochastic MAS under counting linear temporal logic (cLTL) specifications, and show that the corresponding satisfaction probability admits a structured tensor decomposition via leveraging deterministic finite automata (DFA). Building on this structure, we develop a dual-tree-based value iteration framework that reduces redundant computation in the process of dynamic programming. Numerical results demonstrate the proposed approach's effectiveness and scalability for complex specifications and large-scale MAS.

## 二、提出的方法论

### 1. 核心思想
- 将系统的 **规范满足概率（satisfaction probability）** 通过确定性有限自动机（DFA）建模；
- 发现满足概率具有可分离的**结构化张量分解形式**（structured tensor decomposition）；
- 基于此分解，设计**双树（Dual-Tree）值迭代算法**，降低动态规划的冗余计算与内存需求；
- 实现对 **同质随机多智能体系统的正确式策略合成**。

### 2. 方法框架
#### （1）张量结构分解
- 对于满足 cLTL 规格的系统，有限时间内的满意概率可写为多个**秩为 1 的张量（rank-1 tensor）**之和；
- 每个智能体对应一组分量向量，联合后形成外积，即  
  \[
  P_{M\times C_\pi} = \sum_{l[0,t]\in W} \bigotimes_{i=1}^{N} P^{C_i}_\pi(l_i[0,t])
  \]
  其内存成本由指数级 \(O(|X_c|^N)\) 降为线性级 \(O(N|X_c|)\)。

#### （2）基于 DFA 的约束策略
- 借助 DFA 将计数时序逻辑公式化；
- 控制策略 \(C_\pi\) 仅依赖于当前状态和 DFA 状态，避免依赖无限长前缀；
- 每个智能体策略定义为 \(C_i^{\pi}: X_c \times Q  \to A_c\)。

#### （3）双树值迭代（Dual-Tree Value Iteration）
- **多智能体树（Gₘ）**：描述系统在 DFA 状态空间的见证路径；
- **单智能体树（Gₛ）**：仅存储唯一的单智能体值函数；
- **双树映射关系（R）**：将每个“节点–智能体对”(z, i) 映射到单智能体树的节点 κ。  
- 通过此结构，实现张量分解的值迭代，每个单智能体值只需计算一次。

#### （4）辅助优化机制
- **BDD（Binary Decision Diagram）压缩**：将可满足计数命题的字母集合符号化表示，减少树宽度；
- **节点剪枝（Pruning）**：剔除低概率贡献的树叶节点，在保证概率下界的同时控制规模；
- **策略优化算法**：在双树结构上迭代更新 \(\pi_i^*\)，最大化期望满意概率。
