## 摘要
视觉-语言-动作（Vision-Language-Action，VLA）模型已成为具身智能的主流方向。近年来，VLA 模型的输入模态从单一的 2D 扩展到 2D+3D 范式，形成了多视觉模态的 VLA（MVLA）模型。尽管多模态融合提升了空间感知能力，但模态扩展带来的输入 token 数量激增，也使 MVLA 模型面临更高的加速需求。Token 剪枝是一种针对 MVLA 模型的有效优化方法。然而，现有的 token 剪枝方案主要针对仅使用 2D 模态的 VLA 模型设计，忽略了 2D/3D 模态显著性差异。本文结合 MVLA 模型中多模态数据的应用流程，提出了一种三阶段分析方法，用于捕捉 2D/3D 模态显著性的差异与动态变化。在此基础上，我们进一步提出了一种面向 MVLA 模型的三阶段 token 剪枝框架，实现最优的 2D/3D token 选择与高效剪枝。实验结果表明，该框架在仅增加 5.8% 开销的前提下，可在推理阶段实现最高 2.55 倍的加速，同时保持极小的精度损失。我们的代码将于近期发布。

## Abstract
Vision-Language-Action (VLA) models have emerged as the mainstream of embodied intelligence. Recent VLA models have expanded their input modalities from 2D-only to 2D+3D paradigms, forming multi-visual-modal VLA (MVLA) models. Despite achieving improved spatial perception, MVLA faces a greater acceleration demand due to the increased number of input tokens caused by modal expansion. Token pruning is an effective optimization methods tailored to MVLA models. However, existing token pruning schemes are designed for 2D-only VLA models, ignoring 2D/3D modality salience differences. In this paper, we follow the application process of multi-modal data in MVLA models and develop a tri-stage analysis to capture the discrepancy and dynamics of 2D/3D modality salience. Based on these, we propose a corresponding tri-stage token pruning framework for MVLA models to achieve optimal 2D/3D token selection and efficient pruning. Experiments show that our framework achieves up to a 2.55x inference speedup with minimal accuracy loss, while only costing 5.8% overhead. Our Code is coming soon.

## 一、研究背景与核心问题

- **研究背景**：  
  视觉-语言-动作（Vision-Language-Action, VLA）模型是具身智能（Embodied Intelligence）的关键技术，通过视觉观测和语言指令直接输出机器人操作动作。然而传统的 **2D-only VLA 模型** 在复杂空间任务中表现有限，难以充分理解三维结构，于是研究者发展出 **多视觉模态 VLA （MVLA）模型**，即结合 2D 图像与 3D 点云，以提升空间感知与操控精度。

- **研究动机**：  
  作者旨在分析 MVLA 模型中 2D/3D 模态的显著性规律，建立相应的 **模态-aware 剪枝机制**，在较低开销下实现高效推理与性能保持。

## 二、方法论与核心思想

### 1. 总体思路：三阶段模态显著性分析与剪枝
论文提出一个 **三阶段（Tri-Stage）框架**，对应 MVLA 模型的三大推理阶段：
1. **Stage 1：数据预处理阶段**  
   - 定量分析 2D/3D 模态在底层特征上的显著性差异；
   - 通过特征范数（L1-norm）计算各模态的 salience；
   - 引入双阈值机制（𝜏₂ᴰ, 𝜏₃ᴰ）确定剪枝候选集（2D-only / 3D-only / dual modality）。

2. **Stage 2：语义综合阶段**  
   - 基于注意力矩阵分解与聚类（K-Means），将 patch 划分为三类语义集：背景（bg）、机器人（rob）、目标物体（obj）；
   - 计算跨语义区域的模态显著性指标（MS₂₂ᴰ, MS₂₃ᴰ）；
   - 对 3D 注意力向量作并行与正交分解：正交部分代表独有信息，协同部分代表冗余信息；
   - 根据语义区域设定剪枝策略：
     - 背景区域随机剪枝 90%，
     - 机器人区域根据空间功能保留双模态，
     - 目标区域保护核心模态，冗余时单模态保留。

3. **Stage 3：动作迭代阶段**  
   - 考虑动作连续性引起的时序动态；
   - 使用 **指数滑动平均 (Exponential Moving Average, EMA)** 平滑前后帧的显著性预测；
   - 结合滑动窗口（长为 k）更新时序掩码，避免帧间抖动导致的剪枝不稳定。

### 2. 剪枝融合逻辑
- 各阶段均生成候选集（CS₁, CS₂），通过 **交集融合(Cfinal = CS₁ ∩ CS₂)** 生成最终掩码；
- 若冲突（交集为空），触发回退机制（fallback）保留 Stage 2 的结果；
- 输出最终的 token mask：  
  - Mask₂ᴰ = 1 表示保留 2D token；  
  - Mask₃ᴰ = 1 表示保留 3D token。

### 3. 算法机制简述
总体流程（对应 Algorithm 1）：
1. 在首次迭代（t=1）下，不进行剪枝（冷启动）。
2. 动作迭代阶段：计算时序平滑显著性量。
3. 数据预处理阶段：双阈值筛选模态候选。
4. 语义综合阶段：聚类语义集并应用语义约束规则。
5. 最终阶段：融合各候选集生成剪枝掩码并应用到视觉编码器。
