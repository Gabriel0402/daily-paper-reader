## 摘要
大型语言模型（LLM）智能体在需要战略性协调的多智能体系统中被广泛应用。尽管近期研究已经分析了LLM在双人博弈中的行为，但涉及$n$个智能体动态组建合作群体的联盟形成仍缺乏理论刻画。本文提出首个将LLM智能体网络中的联盟形成建立在享乐博弈理论（hedonic game theory）基础上的框架，并给出了形式化的稳定性保证。我们引入LLM联盟形成博弈（LLM Coalition Formation Game, LCFG），建立了达到Nash稳定划分的充分条件，并证明了相关复杂性结果。我们的分析表明，LLM智能体表现出受限理性，其偏好可由$ε$-理性偏好描述；我们提供了确定性的存在性保证以及基于一致性驱动的稳定性界限，这些预测与经验结果一致。基于GPT-4、Claude-3和Llama-3的2400个实验回合验证了我们的框架：在我们的“思维联盟”（Coalition-of-Thought, CoalT）协议下，LLM联盟在73.2%的情况下实现了Nash稳定，相比之下，“思维链”（chain-of-thought）下为58.4%，标准提示（standard prompting）下为41.8%（$p < 0.001$）。我们的框架为设计稳定的多智能体LLM系统提供了理论基础。

## Abstract
Large Language Model (LLM) agents are increasingly deployed in multi-agent systems requiring strategic coordination. While recent work has analyzed LLM behavior in two-player games, coalition formation, where $n$ agents dynamically form cooperative groups, remains theoretically uncharacterized. We present the first framework grounding coalition formation in LLM agent networks in hedonic game theory with formal stability guarantees. We introduce the LLM Coalition Formation Game (LCFG), establish sufficient conditions for Nash-stable partitions, and prove complexity results. Our analysis reveals that LLM agents exhibit bounded rationality characterized by $ε$-rational preferences; we provide both deterministic existence guarantees and consistency-driven stability bounds whose predictions are consistent with empirical outcomes. Experiments with GPT-4, Claude-3, and Llama-3 across 2,400 episodes validate our framework: LLM coalitions achieve Nash stability in 73.2% of cases under our Coalition-of-Thought (CoalT) protocol, compared to 58.4% under chain-of-thought and 41.8% under standard prompting ($p < 0.001$). Our framework provides theoretical foundations for designing stable multi-agent LLM systems.

## 二、方法论与核心框架

- **总体框架**：  
  提出“LLM联盟形成博弈”（LLM Coalition Formation Game, LCFG），在享乐博弈(hedonic games)理论下刻画LLM智能体偏好与联盟稳定性。

- **关键思想**：  
  - 每个LLM智能体视为具有能力向量的理性决策者；联盟的效用由成员能力组合和协调成本共同决定。  
  - 通过“受限理性偏好”模型（ε-rational preferences）刻画LLM智能体决策中的不确定性。  
  - 提出两类稳定性保证：  
    1. **确定性存在性定理**（当误差 ε < δ/2 时存在Nash稳定划分）。  
    2. **一致性驱动稳定性定理**（当 ε 较大时通过偏好一致性概率p预测稳定性下界）。

- **重要定义与公式**：  
  - 联盟价值函数：  
    \( v(S) = ϕ(\bigoplus_{a_i∈S} c_i) - ψ(|S|) \)，其中  
    - ϕ：聚合能力函数（如 ∥c∥₁/d），体现任务覆盖度；  
    - ψ：协调代价函数，采用 ψ(k) = α·k^β 建模任务沟通成本（默认 α=0.15, β=1.3）。
  - 智能体的效用：每个成员的**人均价值** \(v(S)/|S|\)。  
  - 稳定性概念：Nash稳定、个体稳定（IS）、核心稳定（Core）。重点分析可验证的Nash稳定。

- **算法流程：Coalition-of-Thought (CoalT)协议**  
  五步提示推理过程：  
  1. 能力分析（列出成员技能）；  
  2. 互补性评估（识别技能重叠与缺口）；  
  3. 价值估计（预测任务绩效）；  
  4. 协调成本分析（估计沟通负载）；  
  5. 偏好声明（给出最终选择）。  
  借助结构化推理提升偏好一致性而非简单求最优。

## 六、主要结论与发现

1. **理论发现**：  
   - 提供了Nash稳定划分存在的充分条件。  
   - 稳定性主要由“偏好一致性概率”p决定，而非完美理性。  
   - 提出稳定性下限公式 \( Pr[Nash] ≥ p^{K_{eff}} · γ(G) \)。

2. **实验结果**：  
   - CoalT协议下稳定率达到 **73.2%**，超越CoT（58.4%）与Standard（41.8%）。  
   - 混合架构（GPT-4 + Claude + Llama）联盟社会福利最高（0.81），体现能力互补效应。  
   - 稳定性随一致性提升近线性增长，验证理论预测。

3. **核心启示**：  
   - 在多智能体环境中，**一致性比最优性更关键**。  
   - CoalT显著提升LLM间稳定协作能力。
