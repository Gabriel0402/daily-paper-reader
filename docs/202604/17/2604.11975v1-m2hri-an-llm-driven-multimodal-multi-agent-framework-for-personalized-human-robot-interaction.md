## 摘要
多机器人系统在家庭和医院等社会环境中具有巨大潜力，但现有的多机器人研究通常将机器人视为功能上相同的个体，忽视了机器人个体身份如何影响用户感知，以及在这种个体差异存在时，协调机制如何塑造多机器人行为。为了解决这一问题，我们提出了 M2HRI——一种基于大型语言模型的多模态多智能体框架，为每个机器人赋予独特的个性与长期记忆，并加入了基于这些差异的协调机制。在一个受控用户研究（n = 105）的多智能体人机交互（HRI）场景中，我们发现由大型语言模型驱动的个性特征具有显著可区分性并提升了交互质量，长期记忆增强了个性化和偏好感知，而集中式协调显著减少了重叠并改善了整体交互质量。综合而言，这些结果表明，智能体的个体差异与结构化协调对于实现连贯且符合社会规范的多智能体人机交互至关重要。项目网站与代码：https://project-m2hri.github.io/。

## Abstract
Multi-robot systems hold significant promise for social environments such as homes and hospitals, yet existing multi-robot works treat robots as functionally identical, overlooking how robots individual identity shape user perception and how coordination shapes multi-robot behavior when such individuality is present. To address this, we introduce M2HRI, a multimodal multi-agent framework built on large language models that equips each robot with distinct personality and long-term memory, alongside a coordination mechanism conditioned on these differences. In a controlled user study (n = 105) in a multi-agent human-robot interaction (HRI) scenario, we find that LLM-driven personality traits are significantly distinguishable and enhance interaction quality, long-term memory improves personalization and preference awareness, and centralized coordination significantly reduces overlap while improving overall interaction quality. Together, these results demonstrate that both agent individuality and structured coordination are essential for coherent and socially appropriate multi-agent HRI. Project website and code are available at https://project-m2hri.github.io/.

## 二、方法论：M2HRI 框架设计  

- **核心思想**：  
  采用大语言模型（LLM）和视觉语言模型（VLM）为基础，在统一架构下融合**人格建模**（Personality）、**长期记忆**（Memory）和**中央协调机制**（Coordination）。  
  框架目标是同时保证“个体差异”（个性化表达）与“群体一致性”（协调化响应）。  

- **系统结构分层**：  
  每个机器人智能体 \( a_i = \langle \phi_i, \Lambda_i \rangle \)，其中：  
  - **身份层（Identity Profile）** \( \phi_i = \langle M_i, B_i, \Gamma_i \rangle \)：定义个体记忆 \(M_i\)、人格 \(B_i\)、动作能力集合 \( \Gamma_i \)。  
  - **认知层（Cognitive Loop）** \( \Lambda_i = \langle P_i, R_i, E_i \rangle \)：包括感知（Perception）、推理（Reasoning）、执行（Execution）。  

- **关键模块说明**：  
  1. **感知（Perception）**：通过VLM整合语音与视觉输入，将多模态信号转换为语义描述。  
  2. **记忆（Memory）**：包括三类记忆：  
     - 工作记忆（Working Memory）存储当前会话上下文；  
     - 语义记忆（Semantic Memory）用于保存事实性知识；  
     - 情节记忆（Episodic Memory）记录过往交互经验。  
     每个智能体拥有独立的长期记忆仓库，通过LLM代理控制调用与更新。  
  3. **人格（Personality）**：采用“五大人格模型（Big Five）”，即 **OCEAN（开放性、责任心、外向性、宜人性、神经质）**。各维度取值为1–5并映射为自然语言描述，以影响语言风格和行为决策。  
  4. **规划（Planning）**：由LLM驱动，根据感知输入、上下文、记忆以及性格参数生成行为策略 \( \pi_t = \{(a_1, \theta_1), …\} \)。  
  5. **执行（Execution）**：通过行动库执行语音、手势、移动等身体行为。  
  6. **协调（Coordination）**：  
     - 由中央协调器收集每个机器人观测结果与个性特征；  
     - 利用LLM模型计算每个智能体的响应概率 \( r_{i,t} = f_{coord}(Z_t)_i \)；  
     - 超过阈值的智能体按顺序响应，以避免重叠与冲突。  

- **实现细节**：  
  - 使用 **两台 NAO 机器人**。  
  - 感知采用外部VLM进行视觉语音整合。  
  - 记忆存储使用 **ChromaDB + LangGraph + text-embedding-3-large**。  
  - LLM 推理部分由 **GPT-4o-mini** 驱动，采用 **AutoGen AssistantAgents** 管理交互。  
  - 机器人执行层基于 **NaoQi Proxy** 实现。

## 三、实验设计  

- **研究问题（RQs）**：  
  - **RQ1**：人格对多机器人可区分性与互动参与度有何影响？  
  - **RQ2**：长期记忆如何影响个性化与上下文感知？  
  - **RQ3**：协调机制对整体交互质量的作用？  

- **实验场景与数据**：  
  - 采用 ***预录视频观察法***，展示人类与两台 NAO 机器人（性格/记忆/协调配置不同）的交互录像。  
  - 保持相同场景与台词，确保仅变量变化。  

- **样本与设计**：  
  - 105名参与者（年龄18–77岁），分布均匀。  
  - 共 **7个实验条件**：人格 5（O、C、E、A、N），记忆 1（有/无），协调 1（有/无）。  
  - 每组约15人，采用 **10分钟时长视频+问卷** 的方式收集主观感知。  

- **测评指标**：  
  - 人格：可区分性、一致性、参与感、特质识别准确率。  
  - 记忆：回忆准确率、用户偏好识别、交互自然度。  
  - 协调：对话流畅性、响应适配性、重叠回避率。

## 六、主要结论与发现  

1. **人格（RQ1）**  
   - 用户可清楚区分机器人性格（尤其开放性和神经质），但责任心差异较不明显。  
   - 五大人格皆能保持一致表现，性格稳定性高。  
   - 外显性格（如开放、神经质）显著提升互动参与度。  
   - 用户能准确识别目标人格（准确率66.7–93.3%）。  

2. **记忆（RQ2）**  
   - 长期记忆显著提升信息回忆与用户偏好感知（p < .001）。  
   - 与无记忆条件相比，个性化体验提升明显，交互自然感有中等提升。  

3. **协调（RQ3）**  
   - 集中协调显著改善对话流畅性、响应恰当性、避免重叠（p < .001）。  
   - 无协调时常出现同时发言、打断等混乱现象，破坏社交连贯。
