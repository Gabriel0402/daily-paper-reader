## 摘要
视觉-语言-动作（Vision-Language-Action, VLA）模型通过大规模预训练推动了机器人操作的发展，但由于部分可观测性和延迟反馈，现实世界中的部署仍然存在挑战。强化学习通过价值函数来解决这一问题，价值函数用于评估任务进展并指导策略改进。然而，基于视觉-语言模型（Vision-Language Model, VLM）的现有价值模型难以捕捉时间动态，从而削弱了在长时间任务中进行可靠价值估计的能力。本文提出了 ViVa，这是一种视频生成式价值模型，利用预训练的视频生成器进行价值估计。ViVa 以当前观测和机器人本体感知（proprioception）作为输入，联合预测未来的本体感知以及当前状态的标量价值。通过利用预训练视频生成器的时空先验，我们的方法将价值估计与对预期具体现象的动态进行关联，从静态快照转向内在地将价值与预测能力耦合。在 RECAP 框架中集成 ViVa，可在真实世界的箱体组装任务中取得显著改进。对三项任务的定性分析表明，ViVa 产生的价值信号更可靠，能够准确反映任务进展。通过利用视频语料中蕴含的时空先验，ViVa 同样能够泛化至新颖的物体，体现了视频生成模型在价值估计中的潜力。

## Abstract
Vision-language-action (VLA) models have advanced robot manipulation through large-scale pretraining, but real-world deployment remains challenging due to partial observability and delayed feedback. Reinforcement learning addresses this via value functions, which assess task progress and guide policy improvement. However, existing value models built on vision-language models (VLMs) struggle to capture temporal dynamics, undermining reliable value estimation in long-horizon tasks. In this paper, we propose ViVa, a video-generative value model that repurposes a pretrained video generator for value estimation. Taking the current observation and robot proprioception as input, ViVa jointly predicts future proprioception and a scalar value for the current state. By leveraging the spatiotemporal priors of a pretrained video generator, our approach grounds value estimation in anticipated embodiment dynamics, moving beyond static snapshots to intrinsically couple value with foresight. Integrated into RECAP, ViVa delivers substantial improvements on real-world box assembly. Qualitative analysis across all three tasks confirms that ViVa produces more reliable value signals, accurately reflecting task progress. By leveraging spatiotemporal priors from video corpora, ViVa also generalizes to novel objects, highlighting the promise of video-generative models for value estimation.

## 二、方法论：ViVa 模型设计

### 2.1 核心思想

- 提出 **ViVa（Video-generative Value model）**：  
  通过**重用预训练的视频生成扩散模型**（Video Diffusion Transformer）作为价值预测器，使模型能够：
  - 输入：当前多视角视觉观测 + 机器人本体感知（关节角度等）；
  - 输出：未来的本体感知预测（proprioception） + 当前状态的标量价值（normalized return）。

- 关键理念：  
  将价值估计问题转化为**条件视频生成**问题，使模型在预测未来动态的同时，内在地学习任务进度与执行质量的映射。

### 2.2 技术细节

- **模型基础架构**：  
  基于 **Wan2.2 Video Diffusion Transformer**（Wan et al., 2025）修改：
  - 增添输入/输出模态注入（latent injection）机制；
  - 将多模态输入（图像、本体感知、标量值）编码为统一的潜变量帧（latent frames）。

- **数据流流程（训练阶段）**：
  1. 将当前观测编码为潜变量帧；
  2. 拼接形成固定长度序列（conditioning + noisy target）：
     \[
     [z_{blank},z_{q_t},z_{o_t}^{(1)},z_{o_t}^{(2)},z_{o_t}^{(3)},z_{q_{t+K}},z_{v_t}]
     \]
  3. 条件部分保持“干净”，未来状态与价值加噪；
  4. 扩散 Transformer 学习从噪声中恢复目标潜变量，从而预测未来本体状态与当前价值。

- **推理阶段**：
  - 给定当前观测，仅提供条件帧；
  - 通过反向扩散生成未来本体感知与价值估计。

- **训练目标函数**（Flow Matching 形式）：  
  模型同时拟合未来本体状态与价值潜变量的流动向量：
  \[
  L = \lambda_{prop} \| v_\theta(z_q^\tau) - (z_1 - z_q^0) \|_2^2 + \lambda_{val} \| v_\theta(z_v^\tau) - (z_1 - z_v^0) \|_2^2
  \]

- **奖励函数设计**：  
  通过自定义逐步奖励 \(r_t\) 与回报 \(G_t\)，将任务进度标准化至区间 \([0,2)\)，保证成功与失败轨迹具有恒定可区分间隔，便于稳定的监督学习。

## 六、主要结论与发现

1. **视频生成模型可作为价值函数的自然基础**，因其具备时空动态预测能力；
2. **ViVa 能提供平滑、进度相关、灵敏的价值信号**，能实时反映任务进展与执行误差；
3. 在 RECAP 框架中：
   - 任务成功率由 58%（VLM 价值）提升至 73%；
   - 吞吐量由 11 提升到 14 次/小时；
4. ViVa 在未见过的新任务中仍能稳定输出合理的进度轨迹，表现出 **显著的泛化能力**；
5. 加入未来本体感知预测对检测微小错误尤为关键，能显著提高价值信号鲁棒性。

## 七、方法亮点

- **创新性**：首个将视频生成扩散模型用于强化学习价值估计；
- **架构优势**：以预测未来动态为基础，将价值与“行动后果”天然耦合；
- **时空先验利用**：从视频语料中迁移时序规律，使模型具备先见性；
- **高效性与可复用性**：直接基于已有视频扩散模型扩展，无需重新训练庞大VLM；
- **可解释性提升**：价值变化曲线可直观反映机器人执行进度与错误事件。
