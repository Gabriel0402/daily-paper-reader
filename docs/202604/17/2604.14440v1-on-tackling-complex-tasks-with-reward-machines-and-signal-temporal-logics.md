## 摘要
我们提出了一种基于强化学习（Reinforcement Learning, RL）的控制设计框架，用于处理复杂任务。该方法将奖励机器（Reward Machines, RM）的概念与可用于事件生成的信号时序逻辑（Signal Temporal Logic, STL）公式结合起来。STL的使用不仅能够更高效地表示复杂任务的奖励，还能够引导训练过程收敛到满足特定要求的行为。我们还提出了利用STL在线监测算法的框架实现方案。通过三个案例研究（minigrid、cart-pole和high-way环境）展示了该框架在非平凡任务中的应用。

## Abstract
We propose a Reinforcement Learning (RL) based control design framework for handling complex tasks. The approach extends the concept of Reward Machines (RM) with Signal Temporal Logic (STL) formulas that can be used for event generation. The use of STL allows not only a more efficient representation of rewards for complex tasks but also guiding the training process to converge towards behaviors satisfying specified requirements. We also propose an implementation of the framework that leverages the STL online monitoring algorithms. We illustrate the framework with three case studies (minigrid, cart-pole and high-way environments) with non-trivial tasks.

## 二、方法论与技术细节

### 1. 核心思想：RM 与 STL 的融合  
- **Reward Machine (RM)：** 一种基于有限状态自动机的结构，它在RL过程中作为高层任务描述器，允许奖励依赖于历史状态序列；  
- **Signal Temporal Logic (STL)：** 定义在连续信号上的时序逻辑，用于描述时域约束和行为规范；  
- **RM-STL 框架：**
  - 将**STL公式嵌入至RM的命题集**中，用以生成事件（Event Generation）；  
  - STL 判定出的逻辑事件触发 RM 状态的转移；
  - 奖励由 各个RM模块的加权和 计算；
  - 可并行组合多个RM，实现模块化与分层化学习。

### 2. 技术实现要点  
- **STL Robustness**：使用鲁棒度度量 \( ρ(ϕ, y, t) \)，描述信号满足时序逻辑的量化程度（正值表示满足，负值表示违规）；  
- **在线鲁棒监测**：利用引用文献 [25] 的算法，针对部分信号计算鲁棒区间 \([ρ, \bar{ρ}]\)，实现训练时动态监测；
- **状态增强与奖励机制：**
  - 观测空间扩大为 (环境状态, RM状态, STL鲁棒度)；
  - 通过滑动时间窗计算短期STL指标，既保留马尔可夫性，又保证适应在线训练；
- **RM-STL 工作流程（文字描述）：**
  1. 环境产生状态 \(s_t\)，动作 \(a_t\)；
  2. 更新观察序列 \(ξ_t = [s_0, a_0, \dots, s_t]\)；
  3. STL Wrapper 评估命题真值集 \(σ_t = L(ξ_t)\)；
  4. 各 RM 根据 \(σ_t\) 决定下一个状态 \(u_{t+1}\)；
  5. 计算多RM加权奖励 \(R_t = Σ_i w_i δ^i_r(u_i, u'_i)\)。

### 3. 算法特点  
- 支持任意标准RL算法（如 PPO、DQN）；
- 通过 RLRom 框架实现层级封装：
  - **STL Wrapper**：计算逻辑真值和鲁棒度；
  - **RM Wrapper**：生成奖励；
- 可解释性强：RM 状态对应任务阶段，STL公式明示约束逻辑。
