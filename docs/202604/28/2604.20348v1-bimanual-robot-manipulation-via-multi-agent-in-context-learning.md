## 摘要
大语言模型（LLMs）已成为具身控制的强大推理引擎。特别是，上下文学习（ICL）使现成的纯文本 LLMs 能够在无需任何任务特定训练的情况下预测机器人动作，同时保留其泛化能力。将 ICL 应用于双臂操作仍然具有挑战性，因为高维联合动作空间和紧密的手臂间协作约束会迅速使标准上下文窗口过载。为了解决这一问题，我们引入了 BiCICLe（双臂协调上下文学习），这是第一个使标准 LLMs 能够在无需微调的情况下执行少样本双臂操作的框架。BiCICLe 将双臂控制建模为一个多智能体主从问题，将动作空间解耦为顺序的、有条件的单臂预测。这自然地扩展到了“手臂辩论”（Arms' Debate）——一个迭代细化过程，并引入了第三个 LLM 作为裁判（LLM-as-Judge）来评估和选择最合理的协调轨迹。在 TWIN 基准测试的 13 个任务中进行的评估显示，BiCICLe 达到了高达 71.1% 的平均成功率，比表现最好的免训练基准方法高出 6.7 个百分点，并超过了大多数监督学习方法。我们进一步展示了在全新任务上强大的少样本泛化能力。

## Abstract
Language Models (LLMs) have emerged as powerful reasoning engines for embodied control. In particular, In-Context Learning (ICL) enables off-the-shelf, text-only LLMs to predict robot actions without any task-specific training while preserving their generalization capabilities. Applying ICL to bimanual manipulation remains challenging, as the high-dimensional joint action space and tight inter-arm coordination constraints rapidly overwhelm standard context windows. To address this, we introduce BiCICLe (Bimanual Coordinated In-Context Learning), the first framework that enables standard LLMs to perform few-shot bimanual manipulation without fine-tuning. BiCICLe frames bimanual control as a multi-agent leader-follower problem, decoupling the action space into sequential, conditioned single-arm predictions. This naturally extends to Arms' Debate, an iterative refinement process, and to the introduction of a third LLM-as-Judge to evaluate and select the most plausible coordinated trajectories. Evaluated on 13 tasks from the TWIN benchmark, BiCICLe achieves up to 71.1% average success rate, outperforming the best training-free baseline by 6.7 percentage points and surpassing most supervised methods. We further demonstrate strong few-shot generalization on novel tasks.

### 1. 核心问题与整体含义（研究动机和背景）
双臂机器人操作（如开瓶盖、抬托盘）要求两臂在位置、力量和时间上高度同步。传统的模仿学习（IL）或强化学习（RL）通常需要海量的任务特定数据。虽然大语言模型（LLM）的上下文学习（ICL）在单臂控制上展现了强大的少样本泛化能力，但直接应用于双臂操作面临两大挑战：
*   **维度爆炸**：双臂联合动作空间（$Z^{14}$）远高于单臂（$Z^7$），导致预测复杂度激增。
*   **协调约束**：简单的动作拼接会导致两臂动作不连贯或冲突（如左手还没接，右手就放了）。
该研究旨在开发一种**无需微调（Training-free）**的框架，利用 LLM 的推理能力解决双臂协调操作问题。

### 2. 方法论：BiCICLe 框架
核心思想是将双臂控制从“单智能体联合预测”转变为“多智能体主从协作预测”。

*   **主从分解（Leader-Follower）**：
    *   **阶段1（主臂预测）**：指定一个手臂为 Leader，LLM 根据场景观察和单臂演示预测其完整轨迹。
    *   **阶段2（从臂预测）**：Follower 智能体在预测时，不仅观察场景，还将 Leader 预测的轨迹作为输入条件，从而实现时空同步。
*   **手臂辩论（Arms’ Debate）**：一种迭代优化机制。Leader 和 Follower 轮流重新规划，每次都将对方最新的轨迹作为参考，通过多轮“对话”消除不一致性。
*   **最佳 N 选 1（Best-of-N）**：利用 LLM 的随机性生成多个候选方案，引入第三个 LLM 作为“裁判”（LLM-as-Judge），根据演示样本的模式对候选轨迹的协调性、碰撞风险等进行评分，选出最优解。
*   **表示方式**：将 3D 空间体素化（100³ 栅格），将动作和物体位置序列化为文本 Token，使纯文本 LLM 能够处理空间信息。

### 3. 实验设计
*   **数据集/场景**：主要在 **TWIN 基准测试**（包含 13 个双臂操作任务，如推箱子、交接物体、抬托盘、开抽屉等）上进行仿真评估。

### 6. 主要结论与发现
*   **性能卓越**：BiCICLe + Best-of-N 达到了 **71.1%** 的平均成功率，比最强的免训练基准高出 6.7%，甚至超过了许多需要大量训练的监督学习方法（如 ACT 和 PerAct 2）。

### 7. 优点与亮点
*   **架构创新**：首次将多智能体辩论和主从博弈思想引入双臂 ICL 领域，巧妙解决了上下文窗口限制和高维动作空间问题。
*   **无需训练**：极大地降低了机器人适配新任务的门槛，仅需少量演示即可部署。
*   **鲁棒的表示**：通过体素化和文本序列化，成功将复杂的 3D 物理协作转化为 LLM 擅长的模式匹配任务。

### 8. 不足与局限
*   **精度限制**：受限于体素化分辨率（100³）和离散化的旋转分箱（5°），对于极度精细的操作（如穿针引线、处理细绳）仍有提升空间。

*   **感知依赖**：目前依赖于准确的物体分割和质心提取，在视觉噪声剧烈的真实环境中可能受限。
*   **长程任务**：对于步骤极多的长程任务，LLM 的上下文窗口仍可能成为瓶颈。
