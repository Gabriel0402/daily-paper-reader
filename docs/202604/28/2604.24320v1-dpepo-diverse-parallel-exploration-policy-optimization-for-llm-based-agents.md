## 摘要
遵循序列化“先思考后行动”（reason-then-act）范式的大语言模型（LLM）智能体在许多复杂任务中取得了卓越的性能。然而，由于这些方法每步仅与单个环境交互，因此面临探索受限和环境理解不完整的问题。在本文中，我们首先引入了一种新颖的范式，使智能体能够同时与多个环境交互并共享跨轨迹经验。基于这一范式，我们进一步提出了 DPEPO，这是一种鼓励智能体进行多样化并行探索的强化学习（RL）算法。DPEPO 包含两个阶段：初始的有监督微调（SFT）赋予智能体基础的并行推理和动作生成能力，随后是采用分层奖励方案的强化学习阶段。我们设计了一个并行轨迹级成功奖励和两个步骤级奖励：多样化动作奖励和多样化状态转移奖励，旨在主动惩罚行为冗余并促进广泛探索。在 ALFWorld 和 ScienceWorld 上的广泛实验表明，DPEPO 达到了最先进的（SOTA）成功率，同时保持了与强序列基线相当的效率。（代码可在 https://github.com/LePanda026/Code-for-DPEPO 获取）

## Abstract
Large language model (LLM) agents that follow the sequential "reason-then-act" paradigm have achieved superior performance in many complex tasks.However, these methods suffer from limited exploration and incomplete environmental understanding, as they interact with only a single environment per step. In this paper, we first introduce a novel paradigm that enables an agent to interact with multiple environments simultaneously and share cross-trajectory experiences. Building upon this paradigm, we further propose DPEPO, a reinforcement learning (RL) algorithm that encourages the agent to perform diverse parallel exploration. There are two stages in DPEPO: initial supervised fine-tuning (SFT) imparts basic parallel reasoning and action generation, followed by reinforcement learning stage with a hierarchical reward scheme. We design a parallel trajectory-level success reward and two step-level rewards: Diverse Action Reward and Diverse State Transition Reward, which actively penalize behavioral redundancy and promote broad exploration. Extensive experiments on ALFWorld and ScienceWorld show that DPEPO achieves state-of-the-art (SOTA) success rates, while maintaining comparable efficiency to strong sequential baselines. ### 1. 核心问题与整体含义（研究动机和背景）
论文针对当前基于大语言模型（LLM）的自主智能体在复杂任务中存在的**探索受限**和**环境认知不全**问题进行了研究。
*   **背景**：主流智能体采用 ReAct（推理-行动）范式，这是一种线性的、单路径的交互方式。
*   **核心痛点**：在每一步中，智能体只能执行一个动作并观察一个反馈，这导致其对环境的理解是片面且容易产生偏差的。虽然可以通过多次独立采样来增加探索，但这种方式往往缺乏多样性（动作趋同）、效率低下（Token 消耗大）且不同轨迹间的经验无法共享。
*   **研究目标**：提出一种能够同时与多个环境交互、共享跨轨迹经验，并能主动进行多样化探索的并行智能体训练框架。

### 2. 方法论：核心思想与关键技术
论文提出了 **DPEPO（多样化并行探索策略优化）**，其核心思想是将“并行探索”引入智能体的决策流程，并通过分层奖励机制引导多样性。

*   **并行探索范式**：智能体在每一步 $t$ 动态选择一组环境子集，同时生成并执行多个并行动作，获取多个观察结果，从而构建全局环境认知。
*   **两阶段训练流程**：
    1.  **SFT 冷启动**：通过少量人工标注和模型合成的并行探索轨迹进行微调，使模型掌握基础的并行推理格式和动作生成能力。
    2.  **强化学习阶段**：基于 GRPO（组相对策略优化）算法，无需 Critic 模型，通过组内轨迹对比来估计优势函数。
*   **分层奖励方案（核心创新）**：
    *   **并行轨迹级成功奖励**：只要并行环境中的任何一条路径达成目标，即判定为成功。
    *   **并行步骤级多样性奖励**：
        *   **多样化动作奖励 (DAR)**：惩罚在单个环境内（深度）或跨并行环境间（广度）重复执行相同动作的行为。
        *   **多样化状态转移奖励 (DSTR)**：惩罚重复的“状态-动作”转移对，强制智能体尝试未探索过的路径。
*   **策略优化**：结合轨迹级和步骤级的优势信号，通过层次化监督引导模型在保证任务完成的同时，最大化探索的覆盖面。

### 7. 优点
*   **范式创新**：打破了传统的单线 ReAct 模式，提出了更符合人类“多路尝试”逻辑的并行探索范式。
*   **奖励设计精巧**：通过惩罚深度和广度的冗余，有效地解决了 LLM 容易陷入重复动作的顽疾。
*   **训练高效**：相比传统的 RL 方案，DPEPO 表现出极高的样本效率和更短的训练收敛时间。
*   **实用性强**：通过精心设计的 Prompt 压缩技术，解决了并行交互带来的上下文窗口压力。

*   **环境依赖性**：该方法依赖于能够被“并行实例化”的环境。在某些无法轻易复制状态的现实物理场景中，构建并行环境可能存在挑战。
*   **偏差风险**：论文在伦理考虑中提到，如果环境本身存在偏见，更广泛的探索可能会使智能体学到并放大这些偏见。
*   **应用限制**：目前主要在文本交互环境中验证，对于多模态或实时性要求极高的具身智能场景，其并行同步机制可能需要进一步优化。
