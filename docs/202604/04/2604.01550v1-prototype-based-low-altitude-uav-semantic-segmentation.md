## 摘要
低空无人机（UAV）图像的语义分割由于极端的尺度变化、复杂的物体边界以及边缘设备上有限的计算资源而面临独特挑战。现有的基于 Transformer 的分割方法虽然取得了显著的性能，但计算开销巨大，而轻量级方法则难以捕获高分辨率航拍场景中的细粒度细节。为了解决这些局限性，我们提出了 PBSeg，这是一个专为无人机应用量身定制的高效原型分割框架。PBSeg 引入了一种新颖的基于原型的交叉注意力（PBCA），利用特征冗余来降低计算复杂度，同时保持分割质量。该框架包含一个高效的多尺度特征提取模块，结合了可变形卷积（DConv）与上下文感知调制（CAM），以同时捕获局部细节和全局语义。在两个具有挑战性的无人机数据集上的实验证明了所提方法的有效性。PBSeg 在 UAVid 数据集上达到了 71.86% 的 mIoU，在 UDD6 数据集上达到了 80.92% 的 mIoU，在保持计算效率的同时展现了极具竞争力的性能。代码可在 https://github.com/zhangda1018/PBSeg 获取。

## Abstract
Semantic segmentation of low-altitude UAV imagery presents unique challenges due to extreme scale variations, complex object boundaries, and limited computational resources on edge devices. Existing transformer-based segmentation methods achieve remarkable performance but incur high computational overhead, while lightweight approaches struggle to capture fine-grained details in high-resolution aerial scenes. To address these limitations, we propose PBSeg, an efficient prototype-based segmentation framework tailored for UAV applications. PBSeg introduces a novel prototype-based cross-attention (PBCA) that exploits feature redundancy to reduce computational complexity while maintaining segmentation quality. The framework incorporates an efficient multi-scale feature extraction module that combines deformable convolutions (DConv) with context-aware modulation (CAM) to capture both local details and global semantics. Experiments on two challenging UAV datasets demonstrate the effectiveness of the proposed approach. PBSeg achieves 71.86\% mIoU on UAVid and 80.92\% mIoU on UDD6, establishing competitive performance while maintaining computational efficiency. Code is available at https://github.com/zhangda1018/PBSeg.

### 2. 方法论
作者提出了 **PBSeg** 框架，其核心由以下三个关键技术组成：
*   **原型交叉注意力机制 (PBCA)**：
    *   **核心思想**：利用特征冗余性，不再让查询（Query）与所有空间像素交互，而是与一组代表性的“原型（Prototypes）”交互。
    *   **流程**：首先计算空间特征与对象查询的亲和力矩阵；通过 `argmax` 操作为每个查询选择一个亲和力最大的空间位置作为原型；利用前一层的预测掩码（Mask）过滤背景干扰；最后通过元素级乘法和线性投影进行特征调制。这种设计将计算复杂度从像素级的二次方降低到了线性级别。
*   **高效多尺度像素解码器**：
    *   **上下文感知调制 (CAM)**：通过全局平均池化和 MLP 生成通道描述符，对骨干网络特征进行重加权，以极低的成本引入全局场景统计信息。
    *   **可变形卷积 (DConv)**：在特征金字塔聚合过程中使用 DConv，根据输入内容动态调整感受野，从而更好地适应无人机视角下不规则的物体形状。
*   **整体流程**：采用 ResNet-50 作为骨干网络提取特征，通过高效解码器构建特征金字塔，最后由 Transformer 解码器利用 PBCA 细化查询并生成最终分割掩码。

*   **效率平衡**：虽然 PBSeg 的参数量（35.5M）并非最低，但其在计算量（85.4 GFLOPs）和精度之间取得了极佳的平衡，尤其在处理高分辨率输入时比传统 Transformer 模型更高效。

### 7. 优点
*   **创新的注意力机制**：PBCA 巧妙地利用了语义区域内的特征冗余，解决了高分辨率图像下 Transformer 计算量爆炸的问题。
*   **针对性设计**：结合了 DConv 和 CAM，专门针对无人机视角的尺度变化和全局上下文需求进行了优化。
*   **小物体分割能力强**：在“人”和“车辆”等极小类别上的 IoU 提升显著，这在无人机应用中至关重要。

### 8. 不足与局限
*   **骨干网络依赖**：目前仍依赖于 ResNet-50，虽然性能好，但对于极度资源受限的微型无人机，可能需要更轻量（如 MobileNet 系列）的骨干网络支持。

*   **实时性挑战**：虽然比重型模型快，但在 3090 上 46.6 FPS 的速度在更弱的边缘芯片（如 Jetson 系列）上可能难以达到实时处理要求。
