## 摘要
视觉-语言-动作 (VLA) 模型通过大规模预训练推动了机器人操控的发展，但由于部分可观测性和延迟反馈，现实世界的部署仍具挑战。强化学习通过价值函数解决这一问题，价值函数评估任务进度并指导策略改进。然而，现有的基于视觉语言模型 (VLM) 的价值模型难以捕捉时间动态，损害了长时程任务中价值估计的可靠性。本文提出了 ViVa，一种将预训练视频生成器重新用于价值估计的视频生成价值模型。ViVa 以当前观测和机器人本体感受为输入，联合预测未来本体感受和当前状态的标量价值。通过利用预训练视频生成器的时空先验，我们的方法将价值估计建立在预期的具身动态之上，超越了静态快照，使价值与预见性内在耦合。集成到 RECAP 后，ViVa 在现实世界的纸箱组装任务中实现了显著改进。对三项任务的定性分析证实，ViVa 产生了更可靠的价值信号，准确反映了任务进度。通过利用视频语料库的时空先验，ViVa 还能泛化到新物体，凸显了视频生成模型在价值估计方面的应用前景。

## Abstract
Vision-language-action (VLA) models have advanced robot manipulation through large-scale pretraining, but real-world deployment remains challenging due to partial observability and delayed feedback. Reinforcement learning addresses this via value functions, which assess task progress and guide policy improvement. However, existing value models built on vision-language models (VLMs) struggle to capture temporal dynamics, undermining reliable value estimation in long-horizon tasks. In this paper, we propose ViVa, a video-generative value model that repurposes a pretrained video generator for value estimation. Taking the current observation and robot proprioception as input, ViVa jointly predicts future proprioception and a scalar value for the current state. By leveraging the spatiotemporal priors of a pretrained video generator, our approach grounds value estimation in anticipated embodiment dynamics, moving beyond static snapshots to intrinsically couple value with foresight. Integrated into RECAP, ViVa delivers substantial improvements on real-world box assembly. Qualitative analysis across all three tasks confirms that ViVa produces more reliable value signals, accurately reflecting task progress. By leveraging spatiotemporal priors from video corpora, ViVa also generalizes to novel objects, highlighting the promise of video-generative models for value estimation.

#### 1. 核心问题与研究动机
*   **核心问题**：现有的视觉-语言-动作（VLA）模型在处理现实世界机器人任务时，常面临部分可观测性和延迟反馈的挑战。虽然强化学习（RL）通过价值函数来评估任务进度，但现有的价值模型多基于视觉语言模型（VLM），其训练数据多为静态图像，缺乏对物理世界时间动态的显式建模，导致在长程任务中价值估计不准确、不连续。
*   **研究动机**：作者认为价值估计本质上是一个“预见未来”的问题。视频生成模型在大规模视频语料上预训练，具备强大的时空先验知识。因此，将视频生成器重新利用为价值函数，可以更好地捕捉物体运动、物理交互和场景演变，从而提供更可靠的价值信号。

#### 2. 方法论
*   **核心思想**：提出 **ViVa** 模型，将预训练的视频扩散 Transformer（基于 Wan2.2）改造为价值函数，通过联合预测“未来机器人本体感受”和“当前状态价值”，将价值估计锚定在预期的具身动态中。
*   **关键技术细节**：
    *   **潜空间注入（Latent Injection）**：为了不改变视频生成器的核心架构，将非图像模态映射为潜帧。本体感受（14维向量）通过重复填充（Repeat-padding）映射，标量价值通过广播（Broadcast）映射。
    *   **联合预测序列**：输入序列包含：[空白占位符, 当前本体感受, 多视图图像（3个视图）]。目标预测序列为：[未来第 $K$ 步的本体感受, 当前状态价值]。
    *   **训练目标**：采用流匹配（Flow Matching）公式。模型学习预测从噪声到目标潜帧的向量场。损失函数由本体感受预测损失和价值预测损失加权组成。
    *   **奖励与价值定义**：为了区分成功与失败，定义回报 $G_t$：成功轨迹的回报在 $[0, 1)$ 之间（反映进度），失败轨迹的回报在 $[1, 2)$ 之间。这种设计在成功和失败之间建立了 1.0 的恒定边际，解决了价值估计中的歧义。

#### 3. 实验设计
*   **实验场景**：三项具有挑战性的现实世界双臂协作任务：

    2.  **纸箱组装与包装**：长程、多阶段操作。
    3.  **卷纸整理**：涉及撕纸、丢弃、重绕和贴标签。
*   **Benchmark 与对比方法**：
    *   **纯模仿学习基准**：$\pi_{0.5}$ 和 Gigabrain-0。
    *   **RL 框架对比**：在 RECAP 框架下，对比了“基于 VLM 的价值函数”与“基于 ViVa 的价值函数”。
*   **泛化性测试**：在未见过的物体（折叠裤子）上测试模型的零样本泛化能力。

*   **推理效率**：ViVa 的推理延迟为 **0.18s/帧**，能够满足实时或准实时部署需求。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了三项复杂任务的定性价值轨迹分析，以及在最复杂的“纸箱组装”任务上的定量成功率测试。
*   **消融实验**：

    *   研究了预测步长 $K$（取值 25, 50, 75）对稳定性的影响。
*   **充分性评价**：实验设计较为全面，不仅有成功率对比，还通过大量的价值曲线图展示了模型在检测“抓取失败”、“对齐偏差”等细微错误时的敏感度。但受限于 RECAP 框架的长回放周期，大规模的真机统计数据（如数千次尝试）相对较少。

#### 6. 主要结论与发现
*   **可靠性更高**：ViVa 产生的价值信号更平滑且具有单调性，能准确反映任务进度。
*   **错误检测敏感**：当机器人发生误操作（如纸箱未对齐）时，ViVa 的价值估计会立即下降，而基于 VLM 的模型往往反应迟钝或产生错误波动。
*   **泛化能力强**：得益于视频生成器的时空先验，ViVa 在处理从未见过的物体（裤子）时，依然能准确识别关键里程碑。
*   **性能提升**：在纸箱组装任务中，ViVa 将成功率从 VLM 基准的 58% 提升至 **73%**，吞吐量也显著增加。

#### 7. 优点
*   **范式创新**：将价值估计从“静态分类/排序问题”转变为“未来动力学预测问题”，更符合物理世界的逻辑。
*   **时空先验利用**：成功证明了视频生成模型（而非仅 VLM）在具身智能价值评估中的巨大潜力。
*   **架构优雅**：通过潜空间注入技术，无需修改预训练视频模型结构即可实现多模态融合。

#### 8. 不足与局限
*   **任务覆盖范围**：虽然在三项任务上表现优异，但尚未在更广泛的 Open X-Embodiment 等大规模数据集上进行全面验证。
*   **计算开销**：尽管比某些 VLM 快，但扩散模型的多次去噪迭代（即使是 1 步推理）仍比简单的回归网络重。
*   **依赖预训练质量**：模型的表现高度依赖于底层视频生成器（如 Wan2.2）的质量，若生成器对某些物理现象理解有误，会直接影响价值估计。
