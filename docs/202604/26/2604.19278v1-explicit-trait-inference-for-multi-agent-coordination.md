## 摘要
基于大语言模型（LLM）的多智能体系统（MAS）在处理复杂任务方面展现出巨大潜力，但仍容易出现协作失败，如目标漂移、错误级联和行为失调。我们提出了显式特质推理（Explicit Trait Inference, ETI），这是一种基于心理学原理的改进协作的方法。ETI 使智能体能够根据交互历史，沿着两个公认的心理学维度——热情（如信任）和能力（如技能）——推断并追踪伙伴的特征，从而指导决策。我们在受控环境（经济博弈）中评估了 ETI，结果显示其将收益损失降低了 45-77%；在更真实、复杂的多智能体场景（MultiAgentBench）中，相对于思维链（CoT）基准，其性能提升了 3-29%（取决于具体场景和模型）。进一步的分析表明，性能提升与特质推理密切相关：ETI 档案可以预测智能体的行为，且信息丰富的档案驱动了性能的改进。这些结果凸显了 ETI 作为一种轻量且鲁棒的机制，能够提升多样化多智能体环境中的协作能力，并提供了首个系统性证据，证明 LLM 智能体可以：(i) 从交互历史中可靠地推断他人的特质；(ii) 利用对他人的结构化特质认知来进行协作。

## Abstract
LLM-based multi-agent systems (MAS) show promise on complex tasks but remain prone to coordination failures such as goal drift, error cascades, and misaligned behaviors. We propose Explicit Trait Inference (ETI), a psychologically grounded method for improving coordination. ETI enables agents to infer and track partner characteristics along two established psychological dimensions--warmth (e.g., trust) and competence (e.g., skill)--from interaction histories to guide decisions. We evaluate ETI in controlled settings (economic games), where it reduces payoff loss by 45-77%, and in more realistic, complex multi-agent settings (MultiAgentBench), where it improves performance by 3-29% depending on the scenario and model, relative to a CoT baseline. Additional analysis shows that gains are closely linked to trait inference: ETI profiles predict agents' actions, and informative profiles drive improvements. These results highlight ETI as a lightweight and robust mechanism for improving coordination in diverse multi-agent settings, and provide the first systematic evidence that LLM agents can (i) reliably infer others' traits from interaction histories and (ii) leverage structured awareness of others' traits for coordination.

#### 1. 核心问题与整体含义（研究动机和背景）
基于大语言模型（LLM）的多智能体系统（MAS）在处理复杂任务时，常因**协作失效**（如目标漂移、错误级联、信息共享不足和行为失调）而导致性能受限。
*   **研究动机**：借鉴人类心理学中的“社会评价”机制。人类在协作时会形成对他人的印象，从而决定信任谁、如何分配任务。
*   **核心目标**：赋予智能体“特质推理”能力，使其能从交互历史中识别伙伴的稳定行为倾向，并以此调整自身的协作策略。

#### 2. 方法论：显式特质推理 (ETI)
ETI 是一种轻量级的提示词（Prompting）与上下文管理流程，无需微调模型。
*   **核心思想**：将伙伴特征简化为心理学中经典的两个维度：**热情 (Warmth)** 和 **能力 (Competence)**。
*   **关键技术细节**：
    *   **特质维度**：细化为 8 个具体特质。热情维度包括：目标一致性、协作性、可靠性、恶意性；能力维度包括：执行能力、稳定性、适应性、效率。
    *   **算法流程**：
        1.  **观察**：智能体记录交互历史（动作、通信、结果）。
        2.  **推理**：在每轮交互后，智能体被提示生成伙伴的结构化 JSON 档案，包含 1-7 分的 Likert 评分及文字证据。
        3.  **注入**：将最新的特质档案附加到智能体的决策提示词中。
        4.  **决策**：智能体根据伙伴的特质（如“伙伴 A 能力低但热情高”）调整计划（如“增加对 A 的审核”或“重新分配任务”）。

#### 3. 实验设计
实验分为两个阶段，旨在验证推理的准确性及其对决策的实际贡献：
*   **受控实验（经济博弈）**：
    *   **场景**：迭代囚徒困境 (PD) 和 猎鹿博弈 (SH)。
    *   **设计**：与具有固定特质（如“高能力/低热情”）的脚本对手博弈。
    *   **对比**：对比 CoT（思维链）基准。
*   **复杂场景（MultiAgentBench）**：
    *   **场景**：代码协作 (Coding)、研究提案 (Research)、商务谈判 (Bargaining)、狼人杀 (Werewolf)。
    *   **模型**：Qwen 3-8B 和 GPT-4o-mini。
    *   **对比**：对比任务特定的 CoT 基准，并进行了跨模型验证（如用 Qwen 生成特质供 GPT 使用）。

#### 4. 资源与算力
*   **算力设备**：使用了 AWS EC2 g5.12xlarge 实例，配备 **4 张 NVIDIA A10G GPU**（共 96GB 显存）。
*   **时长与消耗**：总计约 **300-400 GPU 小时**。
*   **API 使用**：GPT-4o-mini 实验通过 OpenAI API 完成。
*   **成本**：ETI 会增加 token 消耗（在通信密集型场景下最高增加约 2 倍），但单次运行成本增量极小（约 $0.01-$0.13）。

#### 5. 实验数量与充分性
*   **实验规模**：在 MultiAgentBench 中，每个场景进行了 **100 次随机种子运行**；在经济博弈中，每个配置运行 25 次。
*   **充分性**：实验覆盖了从简单博弈到复杂协作/对抗任务的多种维度。
*   **客观性**：引入了跨模型测试（证明提升源于特质信息而非模型本身）、消融实验（预校准特质 vs 实时更新）以及针对狼人杀等场景的因果分析（逻辑回归验证特质评分对投票行为的预测力），实验设计严谨且公平。

#### 6. 主要结论与发现
*   **性能提升**：在经济博弈中减少了 45-77% 的收益损失；在 MultiAgentBench 中性能提升 3-29%，协作质量提升 6-42%。
*   **推理能力**：LLM 能够可靠地从历史中推断特质，且 ETI 显著增强了模型在噪声环境下的鲁棒性。
*   **行为引导**：特质档案能有效预测智能体行为（如在狼人杀中，高“恶意性”评分与被投票出局显著正相关）。
*   **模型差异**：Qwen 3-8B 生成的特质档案比 GPT-4o-mini 更具区分度和诊断性，后者存在明显的“积极偏见”（倾向于给高分）。

#### 7. 优点
*   **理论扎实**：基于成熟的社会心理学框架，具有良好的可解释性。
*   **轻量高效**：无需训练，通过简单的 Prompt 即可集成到现有 MAS 中。
*   **通用性强**：在协作、谈判、对抗等多种性质的任务中均表现出有效性。
*   **诊断价值**：特质档案可作为 MAS 运行过程中的透明监控工具。

#### 8. 不足与局限
*   **模型偏见**：部分模型（如 GPT 系列）存在“老好人”偏见，可能导致特质推理不够尖锐。
*   **适应滞后**：由于 ETI 强调“稳定特质”，当伙伴行为突然发生剧变（如突然背叛）时，智能体的反应可能存在滞后。
*   **Token 开销**：在长对话场景下，频繁生成和注入特质档案会显著增加 Token 成本。
*   **规模限制**：主要在较小规模的模型（8B 级别）上验证，在超大规模模型或异构系统中的表现仍需进一步探索。
