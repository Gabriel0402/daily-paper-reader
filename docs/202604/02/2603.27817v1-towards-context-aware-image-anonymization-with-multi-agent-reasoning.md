## 摘要
街景图像包含个人身份信息 (PII)，其中部分信息具有上下文依赖性。现有的匿名化方法要么对图像处理过度，要么遗漏了细微的标识符，而基于 API 的解决方案则会损害数据主权。我们提出了一个名为 CAIAMAR（基于多智能体推理的上下文感知图像匿名化）的智能体框架，用于上下文感知的 PII 分割和基于扩散模型的匿名化，该框架结合了针对高置信度案例的预定义处理以及针对间接标识符的多智能体推理。三个专业智能体通过轮询发言者选择机制在“计划-执行-检查-处理”(PDCA) 循环中进行协作，使大型视觉语言模型能够根据空间上下文（私人财产与公共财产）而非僵化的类别规则对 PII 进行分类。这些智能体实现了空间过滤的由粗到精检测，其中“侦察与缩放”策略用于识别候选目标，开放词汇分割处理局部裁剪区域，而基于 IoU 的去重（30% 阈值）则防止了冗余处理。具有外观去相关性的模态特定扩散引导显著降低了重识别 (Re-ID) 风险。在 CUHK03-NP 数据集上，我们的方法将行人重识别风险降低了 73%（R1：16.9% 对比 62.4% 基准）。在 CityScapes 数据集的图像质量保持方面，我们实现了 KID: 0.001 和 FID: 9.1，显著优于现有的匿名化方法。该智能体工作流能够检测跨对象类别的非直接 PII 实例，并保留了下游语义分割性能。该框架完全使用开源模型在本地运行，生成支持欧盟 GDPR 透明度要求的人类可解释审计轨迹，同时标记失败案例以供人工审查。

## Abstract
Street-level imagery contains personally identifiable information (PII), some of which is context-dependent. Existing anonymization methods either over-process images or miss subtle identifiers, while API-based solutions compromise data sovereignty. We present an agentic framework CAIAMAR (\underline{C}ontext-\underline{A}ware \underline{I}mage \underline{A}nonymization with \underline{M}ulti-\underline{A}gent \underline{R}easoning) for context-aware PII segmentation with diffusion-based anonymization, combining pre-defined processing for high-confidence cases with multi-agent reasoning for indirect identifiers. Three specialized agents coordinate via round-robin speaker selection in a Plan-Do-Check-Act (PDCA) cycle, enabling large vision-language models to classify PII based on spatial context (private vs. public property) rather than rigid category rules. The agents implement spatially-filtered coarse-to-fine detection where a scout-and-zoom strategy identifies candidates, open-vocabulary segmentation processes localized crops, and $IoU$-based deduplication ($30\%$ threshold) prevents redundant processing. Modal-specific diffusion guidance with appearance decorrelation substantially reduces re-identification (Re-ID) risks. On CUHK03-NP, our method reduces person Re-ID risk by $73\%$ ($R1$: $16.9\%$ vs. $62.4\%$ baseline). For image quality preservation on CityScapes, we achieve KID: $0.001$, and FID: $9.1$, significantly outperforming existing anonymization. The agentic workflow detects non-direct PII instances across object categories, and downstream semantic segmentation is preserved. Operating entirely on-premise with open-source models, the framework generates human-interpretable audit trails supporting EU's GDPR transparency requirements while flagging failed cases for human review.

### 论文总结：Towards Context-Aware Image Anonymization with Multi-Agent Reasoning

#### 1. 核心问题与整体含义（研究动机和背景）
街景图像（如地图服务、自动驾驶数据集）中包含大量个人身份信息（PII）。现有的匿名化技术存在三大痛点：
*   **处理失衡**：传统模糊处理（Blurring）会破坏图像的下游任务效用（如语义分割），且容易被逆向攻击；而现有的生成式方法往往只针对人脸或全身，忽略了间接标识符（如衣着、私人财产上的文字）。
*   **缺乏上下文感知**：无法区分公共标识（如交通标志）与私人标识（如私人住宅门牌），导致过度处理或遗漏。
*   **合规与主权风险**：基于云端 API 的方案存在数据泄露风险，且缺乏符合 GDPR 要求的可审计决策链。
**研究动机**：开发一个能够在本地运行、具备语义推理能力、且能平衡隐私保护与数据效用的自动化匿名化框架。

#### 2. 方法论：核心思想与算法流程
论文提出了 **CA-IAMAR** 框架，其核心思想是将确定性的目标检测与基于多智能体的启发式推理相结合：
*   **双阶段架构**：
    *   **第一阶段（确定性预处理）**：使用 YOLOv8 检测高置信度的直接标识符（行人、车牌）。行人采用 SDXL + OpenPose 进行匿名化，通过随机化衣着颜色实现“外观去相关”。
    *   **第二阶段（智能体推理）**：针对间接标识符，利用 AutoGen 协调三个专业智能体：
        1.  **Auditor（审计员）**：利用 Qwen2.5-VL-32B 对 PII 进行分类和质量验证。
        2.  **Orchestrator（协调员）**：管理工作流状态，强制执行 PDCA 循环。
        3.  **Generative（生成员）**：执行“侦察与缩放”（Scout-and-Zoom）分割，并利用 SDXL + ControlNet (Canny) 进行局部修复。
*   **关键技术细节**：
    *   **PDCA 循环**：采用“计划-执行-检查-处理”循环进行迭代优化，最多迭代 3 次以确保覆盖率并防止死循环。
    *   **空间过滤与去重**：基于 30% 的 IoU 阈值进行去重，防止重复处理；利用空间上下文判断是否属于私人财产。
    *   **审计轨迹**：生成人类可读的推理日志，满足 GDPR 的透明度和解释权要求。

#### 7. 优点
*   **上下文推理**：通过多智能体协作，实现了对 PII 的动态定义，而非依赖死板的类别标签。
*   **本地化与合规**：完全基于开源模型本地部署，支持生成审计日志，符合 GDPR 监管需求。
*   **外观去相关**：利用扩散模型在保留人体姿态的同时彻底改变外观特征，有效对抗自动化 Re-ID 系统。
*   **PDCA 机制**：引入工业质量管理中的 PDCA 循环，提高了检测的鲁棒性和完整性。

#### 8. 不足与局限
*   **实时性差**：每张图超过 2 分钟的处理时间，无法满足实时视频流或大规模实时处理需求。

*   **文化偏差**：实验主要基于西方城市街景，在不同文化背景下的 PII 定义可能需要重新调整提示词。
