## 摘要
强化学习（RL）范式在代码生成等推理密集型任务中展现出强大的性能。然而，有限的轨迹多样性往往导致收益递减，从而限制了可达到的性能上限。搜索增强的强化学习通过引入结构化探索缓解了这一问题，但其仍受限于单智能体策略先验。同时，利用多个相互作用的策略可以获得更多样化的探索信号，但现有方法通常与结构化搜索脱节。我们提出了 MARS$^2$（多智能体强化树搜索扩展），这是一个统一的强化学习框架，其中多个独立优化的智能体在共享的树状结构搜索环境中进行协作。MARS$^2$ 将搜索树建模为一个可学习的多智能体交互环境，使异构智能体能够在共享的搜索拓扑中协作生成并优化候选解决方案。为了支持有效的学习，我们引入了一种基于树一致性奖励塑造的路径级群体优势公式，这有助于在复杂的搜索轨迹中进行有效的信用分配。在代码生成基准测试上的实验表明，MARS$^2$ 在不同的模型组合和训练设置下一致地提升了性能，证明了将多智能体协作与树搜索相结合以增强强化学习的有效性。我们的代码已在 https://github.com/TsinghuaC3I/MARTI 公开。

## Abstract
Reinforcement learning (RL) paradigms have demonstrated strong performance on reasoning-intensive tasks such as code generation. However, limited trajectory diversity often leads to diminishing returns, which constrains the achievable performance ceiling. Search-enhanced RL alleviates this issue by introducing structured exploration, which remains constrained by the single-agent policy priors. Meanwhile, leveraging multiple interacting policies can acquire more diverse exploratory signals, but existing approaches are typically decoupled from structured search. We propose \textbf{MARS$^2$} (Multi-Agent Reinforced Tree-Search Scaling), a unified RL framework in which multiple independently-optimized agents collaborate within a shared tree-structured search environment. MARS$^2$ models the search tree as a learnable multi-agent interaction environment, enabling heterogeneous agents to collaboratively generate and refine candidate solutions within a shared search topology. To support effective learning, we introduce a path-level group advantage formulation based on tree-consistent reward shaping, which facilitates effective credit assignment across complex search trajectories. Experiments on code generation benchmarks show that MARS$^2$ consistently improves performance across diverse model combinations and training settings, demonstrating the effectiveness of coupling multi-agent collaboration with tree search for enhancing reinforcement learning. Our code is publicly available at https://github.com/TsinghuaC3I/MARTI.

*   **核心问题**：现有的搜索增强 RL（如 TreeRL）仍由单一策略驱动，搜索路径容易集中在少数高概率分支；而现有的多智能体 RL 缺乏结构化搜索机制，难以处理深层、多分支的推理任务。
*   **研究目标**：提出 **MARS$^2$** 框架，通过在共享的树状搜索环境中部署多个独立优化的智能体，结合多智能体协作与结构化搜索来突破单智能体探索的性能天花板。

### 2. 方法论
*   **核心思想**：将搜索树建模为一个可学习的多智能体交互环境，让异构智能体在共享的拓扑结构中协作生成和优化代码。
*   **关键技术细节**：
    *   **多智能体协作树搜索**：利用汤普森采样（Thompson Sampling）在每一轮扩展中选择最合适的“智能体-节点”对。支持“生成节点”（水平扩展新方案）和“优化节点”（垂直改进已有方案）。
    *   **树一致性奖励塑造（Tree-Consistent Reward Shaping）**：为了解决信用分配问题，定义了一个混合基准，结合了父节点的奖励和兄弟节点的平均奖励。通过计算当前节点相对于该基准的增益来调整原始奖励，鼓励智能体超越父辈并胜过同辈。
    *   **路径级群体优势（Path-level Group Advantage）**：扩展了 GRPO 的群体相对优势公式，将其应用于树结构。每个智能体根据其生成的节点在整棵树中的相对表现进行独立优化。
*   **算法流程**：
    1. 多个智能体共同构建一棵搜索树。
    2. 通过执行反馈（如编译器结果）获取节点奖励。
    3. 应用奖励塑造和树级优势计算。
    4. 各智能体基于异步更新机制，利用收集到的树节点轨迹优化各自的策略。

*   **性能提升**：MARS$^2$ 在所有模型规模上均显著优于 Vanilla GRPO 和 RS$^2$。例如，Qwen3-8B 在 MARS$^2$ 训练后 Pass@1 提升了 8.0%。

### 7. 优点
*   **架构创新**：成功将多智能体协作（MARL）与结构化树搜索（MCTS）在强化学习框架下统一，解决了单智能体探索受限的问题。
*   **信用分配精准**：提出的树一致性奖励塑造有效地利用了树结构的层级信息，使训练更加稳定。
*   **多样性增强**：通过异构策略的交互，显著提升了生成代码在算法逻辑和实现路径上的多样性。

### 8. 不足与局限
*   **训练效率开销**：由于树搜索存在顺序依赖（需先有父节点再有子节点），其训练时的并行度低于纯并行的 GRPO，可能导致训练耗时增加。
*   **弱智能体风险**：消融实验显示，如果引入的协作智能体能力过弱，可能会对强智能体的个体性能产生一定的负面干扰（尽管系统整体性能仍有提升）。
*   **应用限制**：目前主要在代码和数学等具有明确验证器（Executor/Verifier）的任务上验证，对于缺乏自动评估手段的开放式生成任务，其效果尚待验证。
