## 摘要
视觉-语言-动作（Vision-Language-Action, VLA）模型通过大规模预训练推动了机器人操作的进步，但由于部分可观测性和延迟反馈，实际应用仍然具有挑战性。强化学习通过价值函数来解决这一问题，价值函数用于评估任务进展并指导策略改进。然而，基于视觉-语言模型（Vision-Language Models, VLMs）构建的现有价值模型难以捕捉时间动态，从而削弱了在长时任务中进行可靠价值估计的能力。本文提出 ViVa，这是一种视频生成式价值模型，旨在利用预训练的视频生成器进行价值估计。ViVa 以当前观察和机器人本体感知作为输入，联合预测未来本体感知以及当前状态的标量价值。通过利用预训练视频生成器的时空先验，我们的方法将价值估计建立在预期的具身动态上，超越静态快照，将价值与前瞻性内在耦合。在集成到 RECAP 框架后，ViVa 在现实场景的装箱任务中取得了显著提升。对所有三个任务的定性分析表明，ViVa 能产生更可靠的价值信号，准确反映任务进展。通过利用视频语料中的时空先验，ViVa 还能泛化到新物体，凸显视频生成模型在价值估计中的潜力。

## Abstract
Vision-language-action (VLA) models have advanced robot manipulation through large-scale pretraining, but real-world deployment remains challenging due to partial observability and delayed feedback. Reinforcement learning addresses this via value functions, which assess task progress and guide policy improvement. However, existing value models built on vision-language models (VLMs) struggle to capture temporal dynamics, undermining reliable value estimation in long-horizon tasks. In this paper, we propose ViVa, a video-generative value model that repurposes a pretrained video generator for value estimation. Taking the current observation and robot proprioception as input, ViVa jointly predicts future proprioception and a scalar value for the current state. By leveraging the spatiotemporal priors of a pretrained video generator, our approach grounds value estimation in anticipated embodiment dynamics, moving beyond static snapshots to intrinsically couple value with foresight. Integrated into RECAP, ViVa delivers substantial improvements on real-world box assembly. Qualitative analysis across all three tasks confirms that ViVa produces more reliable value signals, accurately reflecting task progress. By leveraging spatiotemporal priors from video corpora, ViVa also generalizes to novel objects, highlighting the promise of video-generative models for value estimation.

## 一、研究问题与整体背景

### 研究动机
- 视觉-语言-动作模型（Vision-Language-Action, VLA）近年来在机器人操作任务中表现出色，但其在**现实环境中的强化学习部署**仍面临重大难题。  
- 挑战主要源自：
  - **部分可观测性**：机器人无法完整感知环境状态；
  - **延迟反馈**：动作的后果往往在多个时间步之后才显现；
  - **长时任务信用分配难**：静态图像理解无法捕捉动态演进。  
- 虽然已有研究探索用静态视觉语言模型（VLM）进行价值估计，但这些模型缺乏明确的时间建模能力，无法可靠预测长期任务的状态价值。

### 核心问题
> 研究核心问题：如何使机器人的价值估计具备“预见性”，即基于当下观测预测未来交互动态，从而提升强化学习中长期任务的评估精度。

## 二、方法与技术路线

### 核心思想
- 论文提出 **ViVa (Video-generative Value model)**，将**预训练视频生成模型**（Video Diffusion Transformer）重用于价值函数估计。  
- 核心理念：  
  > 将“价值估计”视为“对未来场景演化的生成”。  
  即，预测未来的机器人状态和任务进展是衡量当前状态价值的自然途径。

### 关键机制
1. **输入与输出设计：**
   - 输入：当前时刻的多视角图像（视觉观测） + 机器人自身传感（14维本体感知量）。
   - 输出：  
     - 未来时间步的本体状态预测（未来动作后姿态）；  
     - 当前状态对应的标量价值（归一化回报）。  

2. **架构基础：**
   - 采用 Wan2.2 视频扩散 Transformer 作为生成骨干。
   - 新增“潜变量注入（latent injection）”模块，将低维传感与标量值嵌入到视频潜空间中：
     - 本体状态 → **repeat-padding 编码**；
     - 价值 → **广播填充编码（broadcasting）**；
     - 图像 → **VAE 编码**。
   - 时序输入结构：
     ```
     [空白帧, 当前 proprio, 多视角图像, 未来 proprio, 未来 value]
     ```
     其中前半部分为条件，后半部分为噪声目标帧，用扩散模型去噪恢复。

3. **训练流程与目标函数：**
   - 采用**流匹配（flow matching）**形式的扩散重构损失，预测噪声插值路径方向：
     \[
     \mathcal{L} = \lambda_{prop} \|v_\theta(z_q^{\tau}) - (z_1 - z_q^0)\|^2 + \lambda_{val} \|v_\theta(z_v^{\tau}) - (z_1 - z_v^0)\|^2
     \]
   - 同时预测未来本体与价值信号，使模型内化机器人运动学联系。

4. **价值目标定义：**
   - 每个任务轨迹标注成功/失败；
   - 设计分段奖励：
     - 成功任务：逐步增加至 [0,1)
     - 失败任务：整体区间 [1,2)，留出固定惩罚间隔；
   - 保证回报空间能区分“任务进展行为”与“失败执行”。

## 三、实验设计与对比设定

### 实验任务与数据来源
- 共三项真实世界任务（均为双臂机械臂执行）：
  1. **衬衫折叠** — 柔性织物操作；
  2. **盒子装配/包装** — 多阶段硬体组装；
  3. **厕纸整理** — 软性物体+粘贴协调任务；
- 每个任务录制真实操作视频和配套感知数据，用作训练与验证。
- 另在“未出现物体类别”（如裤子折叠）上评估泛化。

### 对比方法
- **VLM-based baseline：**  
  - 参照 π*0.6 (Physical Intelligence, 2025) 的多任务价值分类器；  
  - 基于静态视觉语义。
- **ViVa（本模型）：**  
  - 替换价值函数部分，集成到同一 **RECAP (RL with Experience and Corrections via Advantage-conditioned Policies)** 管线；

- **额外 baseline：**  
  - 仅模仿学习的预训练策略 π0.5 (VLA model) 与 Gigabrain-0；

### 评价指标

- 吞吐率（Throughput：每小时完成任务数）；

## 六、主要结论与发现

1. **ViVa 显著提升真实任务性能：**
   - Box assembly 成功率从 58% (VLM-based) 提升至 **73%**；
   - 吞吐率从 11 → **14 tasks/hour**。
2. **价值曲线动态更稳定、与任务进展强相关。**
   - 能实时检测执行错误（如盒盖错位、力量不平衡、提前释放等）。
3. **优良的跨物体泛化性能。**
   - 对裤子折叠等 OOD 任务仍能准确锁定进度节点；
   - VLM 模型则失败或产生无意义波动。
4. **合理的中等预测时长可稳定价值信号。**
   - K=50 明显优于 K=25/75。
5. **附加未来本体预测对价值估计关键。**
   - 没有该分支，模型无法感知身体动作差异，丧失对错误行为的敏感性。
