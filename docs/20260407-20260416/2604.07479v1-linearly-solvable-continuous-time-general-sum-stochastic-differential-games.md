## 摘要
本文介绍了一类连续时间、有限参与者的随机一般和微分博弈，其解可以通过一个精确的线性偏微分方程（PDE）系统得到。我们构建了一个基于交叉对数似然比的分布规划博弈，以自然地建模多智能体空间冲突（如拥塞规避）。通过应用广义多变量 Cole–Hopf 变换，我们将相关的非线性 Hamilton–Jacobi–Bellman（HJB）方程解耦为一个线性偏微分方程系统。此简化使得能够利用 Feynman–Kac 路径积分方法高效且无网格地计算反馈纳什均衡策略，从而有效克服维度灾难。

## Abstract
This paper introduces a class of continuous-time, finite-player stochastic general-sum differential games that admit solutions through an exact linear PDE system. We formulate a distribution planning game utilizing the cross-log-likelihood ratio to naturally model multi-agent spatial conflicts, such as congestion avoidance. By applying a generalized multivariate Cole-Hopf transformation, we decouple the associated non-linear Hamilton-Jacobi-Bellman (HJB) equations into a system of linear partial differential equations. This reduction enables the efficient, grid-free computation of feedback Nash equilibrium strategies via the Feynman-Kac path integral method, effectively overcoming the curse of dimensionality.

## 二、方法论：核心思想与技术框架

### 1. 信息论—测度论建模
- 玩家通过选择作用在轨迹空间上的**概率测度 (path measure)** 来控制各自的随机状态演化。  
- 每个玩家的成本函数包括三部分：
  1. **轨迹成本**：个体运行与终端状态的代价；
  2. **自信息偏离项**：即自身策略与基准分布的 KL 散度；
  3. **交叉对数似然比项**：衡量不同玩家分布重叠程度，用来建模拥挤或对抗。

- 交叉项的符号（正/负）决定交互性质：
  - 正值：表现为**分离趋势**（拥塞避免）；
  - 负值：表现为**聚合行为**（协同或跟随）。

### 2. 等价控制形式化（Theorem 1）
- 通过 Girsanov 定理，将测度论问题等价为随机控制问题。每个玩家的控制代价与交叉项被显式重构为控制输入的二次型：
  \[
  J_i = E\left[\int_0^T C_i(x_t) + \frac{\alpha_{ii}}{2}\|u_i - \bar u_i\|^2 + \sum_{j\neq i} \text{interaction terms}(u_i,u_j)\, dt + \Psi_i(x_T)\right]
  \]
- 得到耦合的非线性 HJB 方程系统。

### 3. 广义多变量 **Cole–Hopf 变换**（Theorem 2）
- 定义新的变量 \(Z_i = \exp(-\sum_j \beta_{ij} J_j)\)，其中 \(\beta = \alpha^{-1}\)。  
- 该变换能**完全抵消 HJB 方程中的二次项和交叉项**，将系统化简为如下线性 PDE：  
  \[
  -\partial_t Z_i = (f+\bar{u}_i g)^\top \nabla_x Z_i + \tfrac{1}{2}\text{Tr}[gg^\top\nabla^2_x Z_i] - (\sum_j \beta_{ij} C_j)Z_i
  \]
  并且边界条件线性可求。

### 4. **Feynman–Kac 路径积分求解**（Corollary 1）
- 应用 Feynman–Kac 引理，将线性 PDE 的解表示为**期望形式**，可利用 Monte Carlo 采样直接计算：  
  \[
  Z_i(x) = E_{R_i}\left[\exp\left(-\sum_j \beta_{ij}\int_t^T C_j + \Psi_j\right)\right]
  \]
- 因为方程完全解耦，各玩家可独立采样，**无需空间网格或反向传播计算**。

### 5. 反馈策略与最优分布恢复
- 最优控制由对数导数给出：
  \[
  u_i^* = \bar{u}_i + g(x)^\top \nabla_x \log Z_i
  \]
- 最优概率分布 \(P_i^*\) 是参考测度的指数加权形式（Theorem 4），与路径成本函数指数相关。

### 1. 实验场景

  - 状态空间：\(\mathbb{R}\)，时间窗口 \( [0, T] \)。  
  - 玩家目标：分别趋向于动态的 "吸引井"（quadratic wells）中心 \(m_1(t)\) 与 \(m_2(t)\)。  
  - 两者初始于相同位置，井中心随时间向两侧移动。

### 2. 模型交互设定
- 交互矩阵：
  \[
  \alpha(\gamma) = 
  \begin{bmatrix} 1 & \gamma \\ \gamma & 1 \end{bmatrix},
  \quad |\gamma| < 1
  \]
- 参数 γ 调节交互性质：
  - γ = 0：无交互，各自最优控制；
  - γ > 0：拥塞避免（互斥）；
  - γ < 0：协同聚合。

### 3. 计算流程
- 利用 Feynman–Kac 路径积分对参考随机轨迹采样。  
- 比较两种等价的结果：

  2. **反馈控制轨迹**（通过最优反馈律直接生成演化）。

### 4. 对比与验证
- 未使用外部 benchmark，仅相互比较三种交互状态（γ=−0.6, 0, +0.6）和一个非对称设置。
