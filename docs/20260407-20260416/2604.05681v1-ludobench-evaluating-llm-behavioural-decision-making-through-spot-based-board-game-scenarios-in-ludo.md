## 摘要
我们提出了 LudoBench，这是一项用于评估大型语言模型（LLM）在 Ludo 游戏中战略推理能力的基准测试。Ludo 是一种带有随机性的多智能体棋盘游戏，其骰子机制、棋子捕获、安全格导航以及归家路径推进等元素共同带来了有意义的规划复杂性。LudoBench 包含 480 个人工设计的局部场景，分属 12 个具有行为区分的决策类别，每一类都针对特定的战略选择进行了隔离。我们同时提供一个完整可运行的四人 Ludo 模拟器，支持随机（Random）、启发式（Heuristic）、博弈论（Game-Theory）以及 LLM 代理。博弈论代理使用深度受限的 Expectiminimax 搜索，通过有限前瞻提供了超越贪心启发式的原则性战略上限。在对跨越四个模型家族的六个模型的评估中，我们发现所有模型仅在 40–46% 的情况下与博弈论基线达成一致。模型呈现出截然不同的行为原型：一类是“完成人”，他们专注于推进棋子但忽视发展；另一类是“建设者”，他们努力发展但从未完成。每种原型仅捕捉到博弈论策略的一半。模型在相同棋盘状态下的历史依赖性怨恨框架（grudge framing）中还显示出可测量的行为变化，揭示了提示敏感性是关键漏洞。LudoBench 为在不确定性条件下基准测试 LLM 战略推理提供了一个轻量且可解释的框架。所有代码、局部场景数据集（480 条记录）及模型输出均可在 https://anonymous.4open.science/r/LudoBench-5CBF/ 获取。

## Abstract
We introduce LudoBench, a benchmark for evaluating LLM strategic reasoning in Ludo, a stochastic multi-agent board game whose dice mechanics, piece capture, safe-square navigation, and home-path progression introduce meaningful planning complexity. LudoBench comprises 480 handcrafted spot scenarios across 12 behaviorally distinct decision categories, each isolating a specific strategic choice. We additionally contribute a fully functional 4-player Ludo simulator supporting Random, Heuristic, Game-Theory, and LLM agents. The game-theory agent uses Expectiminimax search with depth-limited lookahead to provide a principled strategic ceiling beyond greedy heuristics. Evaluating six models spanning four model families, we find that all models agree with the game-theory baseline only 40-46% of the time. Models split into distinct behavioral archetypes: finishers that complete pieces but neglect development, and builders that develop but never finish. Each archetype captures only half of the game theory strategy. Models also display measurable behavioral shifts under history-conditioned grudge framing on identical board states, revealing prompt-sensitivity as a key vulnerability. LudoBench provides a lightweight and interpretable framework for benchmarking LLM strategic reasoning under uncertainty. All code, the spot dataset (480 entries) and model outputs are available at https://anonymous.4open.science/r/LudoBench-5CBF/

- **研究动机**：  
  当前大型语言模型（LLM）已在语言理解、数学推理和常识问答等基准中表现出接近人类的能力，但其在**战略规划与不确定决策**中是否真正具备推理能力仍存在争议。

- **目标问题**：

## 二、方法论：LudoBench框架设计与核心思想

- **核心思想**：  
  构建一个可解释、轻量的实验平台，用于评估LLM在随机博弈环境中的战略推理能力。通过“局部场景（Spot-based）”评估方式，隔离单步决策，便于定量分析不同模型的行为倾向。

- **主要组件与技术要点**：

  1. **Ludo模拟器**  
     - 支持 4 名玩家，遵循标准 Ludo 规则（掷骰、棋子捕获、进入安全格、归家路径等）。
     - 实现三类基线代理：随机(RandomAgent)、启发式(HeuristicAgent)、博弈论(GameTheoryAgent)。

  2. **Game-Theory Agent（博弈论基线）**  
     - 实现 **Expectiminimax** 搜索（考虑随机性与对手策略响应），并在多玩家场景下扩展为 **Expectimax-MaxN** 模型。  
     - 使用深度受限（h=2）的搜索及线性评分函数：
       \[
       V_{eval} = 0.0025·Δprog + 0.20·Δfin − 0.05·Δbase + 0.01·Δsafe
       \]
       衡量棋子进度、完成数、底座数量与安全格分布，代表理性策略上限。

  3. **Spot-based 评估机制**  
     - 设计**480个人工场景**（每类40个），共12个决策类别，覆盖：
       - 攻击与防御权衡（如 capture vs. safe）
       - 推进与完成（home entry, finish）
       - 合规性测试（blocked, overshoot）
       - 历史情境测试（grudge paired）
     - 每个场景仅含一个关键决策，排除多步规划干扰。

  4. **Prompt与输出规范**  
     模型输入：棋盘状态、骰子值、规则说明，可附加 *persona* 或历史描述文本。  
     输出格式严格限制为 `<piece index> | <reason>`，以便自动化验证决策合法性与理由解析。

## 三、实验设计与评估配置

- **Benchmark组成**  
  - 480 个局部场景 × 12 类别 × 5 种人格设定（none, aggressive, greedy, safe, unforgiving）
  - 每个LLM共进行**2,400次评估**（480×5），六个模型总计**14,400次决策测试**

- **模型与对照组**  
  - 六个LLM：Qwen-Plus (QP)、Qwen-2.5-7B (Q7B)、DeepSeek-Chat (DS)、Claude-3.5-Haiku (Haiku)、Llama-4-Scout (Scout)、Gemma-3-12B (Gemma)
  - 三个基线：RandomAgent、HeuristicAgent、Game-TheoryAgent  

- **评估指标**  
  - 合规性（invalid rate）  
  - 战略一致性（GT alignment）  
  - 行为偏向（capture rate、safe rate、home finish rate 等）  
  - 历史敏感性（grudge change rate）  
  - 人格效应（persona alignment）

- **实验配置**  
  - 全部为**零样本（zero-shot）**模式，温度设定为0（确定性输出）。  
  - 无系统提示，仅使用标准场景模板。

## 六、主要结论与发现

1. **规则合规性 ≠ 战略能力**  
   - DeepSeek 与 Haiku 几乎无非法动作（invalid <1%），但战略风格极端相反。
2. **模型一致率低**  
   - 与博弈论基线一致率仅 **40–46%**，超过半数决策为非理性选择。
3. **行为原型分化明显**  
   - “完成人”（Finisher）：注重棋子完成但忽略发展（Haiku、Qwen类）。  
   - “建设者”（Builder）：注重棋子发展但忽略完成（DeepSeek、Gemma类）。  
   - 无模型能平衡两者，均只掌握博弈策略的一半。
4. **情境敏感性显著**  
   - 在相同棋盘状态下，仅因“怨恨叙述”改变，部分模型（如Qwen-Plus）决策变化率达33%。  
   - 显示LLM在叙事语境下决策易受提示诱导。
5. **人格指令效果弱**  
   - 多数人格设定（如safe/aggressive）对行为影响微弱或方向错误，提示人格控制的不可预测性。
