## 摘要
多机器人系统在家庭和医院等社会环境中具有巨大的应用潜力，然而现有的多机器人研究通常将机器人视为功能上相同的个体，忽视了机器人个体身份如何影响用户感知，以及在存在个体差异的情况下，协调机制如何塑造多机器人的整体行为。为了解决这一问题，我们提出了 M2HRI —— 一个基于大型语言模型（LLM）的多模态多智能体框架。该框架为每个机器人赋予独特的个性与长期记忆机制，并基于这些差异设计了相应的协调机制。在一个多智能体人机交互（HRI）场景下的控制用户研究（n = 105）中，我们发现 LLM 驱动的个性特征具有显著可区分性，并能提升交互质量；长期记忆有助于增强个性化与偏好认知；而集中式协调显著减少了任务重叠并整体提升了交互质量。综合来看，这些结果表明，智能体个体性与结构化协调对于实现连贯且社会适宜的多智能体人机交互至关重要。项目网站与代码可在 https://project-m2hri.github.io/ 获取。

## Abstract
Multi-robot systems hold significant promise for social environments such as homes and hospitals, yet existing multi-robot works treat robots as functionally identical, overlooking how robots individual identity shape user perception and how coordination shapes multi-robot behavior when such individuality is present. To address this, we introduce M2HRI, a multimodal multi-agent framework built on large language models that equips each robot with distinct personality and long-term memory, alongside a coordination mechanism conditioned on these differences. In a controlled user study (n = 105) in a multi-agent human-robot interaction (HRI) scenario, we find that LLM-driven personality traits are significantly distinguishable and enhance interaction quality, long-term memory improves personalization and preference awareness, and centralized coordination significantly reduces overlap while improving overall interaction quality. Together, these results demonstrate that both agent individuality and structured coordination are essential for coherent and socially appropriate multi-agent HRI. Project website and code are available at https://project-m2hri.github.io/.

  提出一个基于 LLM 的多模态、多智能体人机交互（HRI）框架 **M2HRI**，系统性地研究：

  2. 长期记忆（Memory）是否能提升个性化；

## 2. 方法论与技术框架

### 核心思想
M2HRI 框架通过 **LLM + VLM 驱动** 的多模态多智能体结构，使机器人能够：
1. 感知用户语音、视觉与环境信息；
2. 结合个性（Big Five人格模型）与长期记忆生成响应；
3. 通过中央协调器（Coordinator）分配发言、控制交互节奏与避免冲突。

### 架构组成
每个机器人代理 \( a_i \) 由两个层次组成：
- **身份层 (Identity Profile)** ：定义个体特征  
  \[
  \phi_i = \langle M_i, B_i, \Gamma_i\rangle
  \]
  - \( M_i \)：记忆（semantic、episodic、working）  
  - \( B_i \)：人格特征向量（O, C, E, A, N）  
  - \( \Gamma_i \)：可执行动作集（语音、手势、移动等）

- **认知循环 (Cognitive Loop)** ：定义行为过程  
  \[
  \Lambda_i = \langle P_i, R_i, E_i\rangle
  \]
  - \( P_i \)：多模态感知模块（基于VLM）  
  - \( R_i \)：规划与推理模块（基于LLM）  
  - \( E_i \)：执行模块，负责将策略转化为具体动作

### 关键技术实现
- **个性建模**  
  基于 Big Five (OCEAN) 模型，五维人格值 1-5 量化，并映射为语言描述词嵌入至 LLM prompt，使人格特征直接影响语言生成。
  
- **记忆结构**  
  由工作记忆（Working Memory）、语义记忆（Semantic Memory）、情节记忆（Episodic Memory）三层构成，实现历史信息的存取。  
  使用 **ChromaDB** 存储语义与情节记忆，并通过 LLM memory controller 执行动态查询与更新。

- **中央协调机制**  
  由 LLM 实现的协调器根据每个代理的观察 \( o_t^{(i)} \)，计算响应概率：
  \[
  r_{i,t} = f_{coord}(Z_t) \in [0,1]
  \]
  其中 \( Z_t \) 聚合了多代理的观测、上下文、人格特征。  
  超过阈值 τ 的代理被选作发言者，以避免重叠。

- **规划与执行**  
  LLM 生成结构化策略 \(\pi_t = \{(a_j, \theta_j)\}\)，再通过 **NAO 机器人** 的动作库执行“语音+手势+表情”等多模态输出。

## 6. 主要结论与发现

1. **人格效果显著（RQ1）**
   - 用户能可靠区分不同人格机器人（四个特质显著，尽责性差异较弱）。
   - 人格表达稳定且一致，尤其神经质与开放性特征最突出。
   - 个性提升了体验参与度，其中表现性强的特质对参与度贡献更大。

2. **长期记忆提升个性化（RQ2）**
   - 有记忆条件下，回忆准确率、偏好觉察、个性化自然度均显著提升。  
   - 记忆的主要作用是“连续性与偏好意识”，而对整体流畅性的提升较弱。

3. **集中协调优化交互质量（RQ3）**
   - 协调机制显著改善对话流畅度与发言适配性。  
   - 交叠与打断次数减少最明显（效应最大）。  
   - 社会规范性（turn-taking）得到强化。

总体结论：**个体化 + 协作结构** 是实现自然、连贯、多机器人社会化交互的关键。
