## 摘要
多模态大型语言模型（Multimodal Large Language Models，MLLMs）在地球观测领域展现出巨大潜力。然而，在处理超高分辨率（Ultra-High-Resolution，UHR）影像时，生成的大量视觉token引入了极高的计算开销，严重限制了推理效率。现有的视觉token压缩方法主要采用静态且均匀的压缩策略，忽视了遥感解译任务中固有的“语义-几何双重性”。具体而言，对象语义任务关注于物体的抽象语义，可通过激进的背景修剪受益；而场景几何任务则关键依赖于空间拓扑的完整性。为应对这一挑战，我们提出DualComp，一种任务自适应的双流token压缩框架。DualComp由一个轻量级预训练路由器动态引导，将特征处理解耦为两个专用路径。在对象语义流中，空间连续语义聚合器（Spatially-Contiguous Semantic Aggregator，SCSA）利用尺寸自适应聚类聚合冗余背景，同时保护小目标；在场景几何流中，基于指令引导的结构恢复器（Instruction-Guided Structure Recoverer，IGSR）通过贪心路径追踪拓扑补全机制重建空间骨架。在超高分辨率遥感基准数据集XLRS-Bench上的实验表明，DualComp以极低的计算成本实现了高保真度的遥感解译，在效率与精度方面均取得了同步提升。

## Abstract
Multimodal Large Language Models (MLLMs) have demonstrated immense potential in Earth observation. However, the massive visual tokens generated when processing Ultra-High-Resolution (UHR) imagery introduce prohibitive computational overhead, severely bottlenecking their inference efficiency. Existing visual token compression methods predominantly adopt static and uniform compression strategies, neglecting the inherent "Semantic-Geometric Duality" in remote sensing interpretation tasks. Specifically, object semantic tasks focus on the abstract semantics of objects and benefit from aggressive background pruning, whereas scene geometric tasks critically rely on the integrity of spatial topology. To address this challenge, we propose DualComp, a task-adaptive dual-stream token compression framework. Dynamically guided by a lightweight pre-trained router, DualComp decouples feature processing into two dedicated pathways. In the object semantic stream, the Spatially-Contiguous Semantic Aggregator (SCSA) utilizes size-adaptive clustering to aggregates redundant background while protecting small object. In the scene geometric stream, the Instruction-Guided Structure Recoverer (IGSR) introduces a greedy path-tracing topology completion mechanism to reconstruct spatial skeletons. Experiments on the UHR remote sensing benchmark XLRS-Bench demonstrate that DualComp accomplishes high-fidelity remote sensing interpretation at an exceptionally low computational cost, achieving simultaneous improvements in both efficiency and accuracy.

## 一、研究背景与核心问题

- **研究动机**：  
  多模态大语言模型（MLLMs）已在地球观测和遥感场景中展现出强大潜力，但在处理**超高分辨率（UHR）**图像时会生成海量视觉 token（通常十几万级别）。

  - 模型推理时延显著，难以在实际遥感任务中部署。

- **核心问题**：  
  以往视觉 token 压缩方法大多采用**静态、统一**的压缩比例，忽略了遥感任务的特性差异。  
  作者通过实验发现，遥感任务天然存在 **“语义–几何双重性（Semantic–Geometric Duality）”**：
  - **语义类任务（semantic-dominant）**：如目标检测、计数等，关注“物体是什么”，可容忍背景压缩；
  - **几何类任务（geometric-dominant）**：如路径规划、地类划分等，依赖“物体位置与拓扑”，对空间连续性极为敏感；
  - **混合任务（balanced）**：需兼顾两者平衡。

因此，论文目标是：  
> **提出一种任务自适应、免训练的视觉 token 压缩方案，在不修改主 MLLM 结构的情况下动态平衡语义与几何两类证据。**

## 二、方法论：DualComp 框架

### 1. 整体思想
- **DualComp**（Semantic–Geometric Dual Compression）是一个**训练无关、可即插即用**的双流视觉压缩框架。
- 主体由三个部分组成：
  1. **轻量级路由器（Duality-Aware Router）**
  2. **语义流模块（SCSA）**
  3. **几何流模块（IGSR）**
- 它在输入阶段根据任务指令动态划分 token 预算，在“语义保持”与“几何保持”之间自适应分配。

### 2. 动态决策模块：Duality-Aware Router
- 输入：来自 MLLM 的**文本指令特征**。
- 输出两个关键参数：
  - **λ**：语义/几何偏好系数（λ 越小偏向语义任务，λ 越大偏向几何任务）；
  - **ρ**：总体压缩比（控制压缩强度）。
- 分配公式：
  - 总保留 token 数：`n_keep = N_max * ρ`
  - 分流分配：`n_sem = n_keep * (1 - λ)`, `n_geo = n_keep * λ`
- 路由器仅约 1M 参数，通过离线问答打分与规则融合方式生成 λ 标签进行预训练，推理时冻结。

### 3. 语义流：SCSA（Spatially-Contiguous Semantic Aggregator）
- 用于**萃取语义显著的目标区域**、压缩冗余背景。
- 核心流程：
  1. **τ(λ)-自适应局部聚类**：利用 frozen CLIP 特征及余弦相似度，将相邻且相似的 token 聚成空间连续簇，阈值 τ 由 λ 控制。
  2. **[CLS] 注意力打分**：利用预训练 ViT 的 [CLS] 对 patch 注意力评估语义重要度。
  3. **尺寸感知表示**：小簇保留代表 token，大簇采用注意力加权平均表示。
- 输出语义压缩后的 token 集 `T_sem`。

### 4. 几何流：IGSR（Instruction-Guided Structure Recoverer）
- 用于**恢复空间结构和拓扑连续性**。
- 包含三步：
  1. **局部差异锚点提取**：通过局部特征差分识别边界/结构区域；
  2. **文本指令调制（TASM）**：利用 CLIP 文本–视觉相似度，调节结构得分，突出与任务相关结构；
  3. **贪心路径追踪（Greedy Topology Completion）**：并行连接高得分锚点，重建空间连通骨架。
- 输出几何序列 `T_geo`，保证结构连续性。

### 5. 融合阶段
- **λ 加权融合**：输出融合 `T_fused = [(1−λ)T_sem, λT_geo]`。
- **拓扑展开**：对 `T_geo` 根据空间连通性排序，将二维空间拓扑映射为一维 token 序列，确保 Transformer 能捕获相邻结构关系。
