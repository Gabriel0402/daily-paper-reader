## 摘要
按设计正确的综合（Correct-by-design synthesis）为随机多智能体系统（MAS）的形式化安全性保证提供了一个有原则的框架。然而，基于有限抽象的传统方法在智能体数量和时序逻辑规范复杂度增加时，往往会带来极高的计算成本。在本研究中，我们针对计数线性时序逻辑（counting linear temporal logic, cLTL）规范下的同质随机多智能体系统进行了研究，并展示了其相应的满足概率可以通过确定性有限自动机（deterministic finite automata, DFA）获得结构化的张量分解。基于这一结构，我们提出了一个基于双树（dual-tree）的值迭代框架，从而在动态规划过程中减少冗余计算。数值结果表明，所提出的方法在处理复杂规范和大规模多智能体系统时具有高效性与可扩展性。

## Abstract
Correct-by-design synthesis provides a principled framework for establishing formal safety guarantees for stochastic multi-agent systems (MAS). However, conventional approaches based on finite abstractions often incur prohibitive computational costs as the number of agents and the complexity of temporal logic specifications increase. In this work, we study homogeneous stochastic MAS under counting linear temporal logic (cLTL) specifications, and show that the corresponding satisfaction probability admits a structured tensor decomposition via leveraging deterministic finite automata (DFA). Building on this structure, we develop a dual-tree-based value iteration framework that reduces redundant computation in the process of dynamic programming. Numerical results demonstrate the proposed approach's effectiveness and scalability for complex specifications and large-scale MAS.

## 一、核心问题与研究背景

- **研究动机：**  
  随着随机多智能体系统（Stochastic Multi-Agent Systems, MAS）在交通管理、无人机编队、搜索救援等安全关键应用中的广泛部署，对系统行为进行形式化安全保证变得至关重要。  
- **现有挑战：**  
  基于线性时序逻辑（Linear Temporal Logic, LTL）进行的正确设计综合虽可提供理论安全性，但在智能体数量增加与规范复杂化时会出现严重的**维度诅咒（curse of dimensionality）**，使计算成本和存储需求呈指数级增长。  
- **研究缺口：**  
  当前针对随机系统的正确设计综合多集中在系统抽象构建阶段的优化，而缺乏提升综合算法本身计算效率的研究。此外，传统 LTL 对多智能体群体行为表达能力有限，难以捕捉集体任务特征。  
- **解决方向：**  
  本文引入计数线性时序逻辑（Counting LTL, cLTL），结合确定性有限自动机（DFA）与低秩张量分解技术，提出在保证形式化正确性的前提下实现动态规划的高效化。

## 二、方法论与核心技术

### （1）总体思想
- 将同质随机多智能体系统建模为若干**动态解耦的马尔可夫决策过程（MDP）**的并行组合；
- 利用 **cLTL 规范的可满足性概率** 的结构化特征，推导出其可以表示为若干**秩为 1 的张量（rank-1 tensors）**的求和；
- 在此基础上，提出**双树（dual-tree）值迭代框架**，通过分离多智能体与单智能体层次，显著降低动态规划中的冗余计算。

### （2）关键技术要点
- **Rank-1 张量分解：**  
  每个满足前缀（witness）概率可分解为智能体层面的外积形式，满足概率表示为  
  \[
  P_{M×C_\pi}(∃t≤T:l[0,t]∈W) = Σ_{l[0,t]∈W} ⊗_{i∈I} P^{C_i}_π(l_i[0,t])
  \]
  从而从指数级内存需求降至线性级。
- **DFA 限制策略记忆：**  
  用 DFA 状态代替完整历史前缀，实现策略映射 \( C_i(x_i, q) \) ，从而控制策略复杂度。
- **双树结构：**
  - **多智能体树（Gₘ）**：保存分析结构（见论文 Fig. 1），用于表示 witness 路径。
  - **单智能体树（Gₛ）**：存储唯一的智能体层值向量，并通过关系映射 R 将其与 Gₘ 关联。
  - 值函数的更新：  
    \[
    W_{new}(κ’) = T_{π q,l_i}(W(κ))
    \]
- **三类算法模块：**
  - **Algorithm 1**：双树的同步扩展；
  - **Algorithm 2**：基于单智能体树的自底向上的值迭代；
  - **Algorithm 3**：基于阈值的树节点剪枝（Pruning），去除低概率叶节点以控制内存增长。
- **辅助技术：**  
  利用**二叉决策图（BDD）**压缩满足计数命题的字母集合，减少树宽度。

  状态区间：\( X_c = [-10, 10] \)，动作区间：\( A_c = [-2, 2] \)。

- **主要规范（cLTL 公式）：**

- **Benchmark 指标：**

## 六、主要结论与发现

- **理论层面：**
  - 证明了 cLTL 满足概率可表示为 rank-1 张量的结构化分解；
  - 建立了基于双树结构的值函数迭代的严格下界保证。
- **算法层面：**
  - 双树框架显著减少了冗余计算及内存储存量；
  - 利用 BDD 与剪枝策略进一步提高了大规模系统下的可扩展性。
- **实验结果：**
  - 在 μ₁ 规范下，内存占用比前作减小近一数量级；
  - 对于 N ≥ 8 的场景，先前方法无法运行，而双树方法仍可稳定完成；
  - 运行时间与智能体数量呈近线性增长趋势。
