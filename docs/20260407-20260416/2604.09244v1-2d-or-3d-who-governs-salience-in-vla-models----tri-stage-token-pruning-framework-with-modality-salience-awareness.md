## 摘要
视觉-语言-动作（Vision-Language-Action，VLA）模型已成为具身智能的主流。近期的 VLA 模型将输入模态从仅限二维扩展到二维与三维结合的范式，形成了多视觉模态 VLA（MVLA）模型。尽管空间感知能力有所提升，MVLA 由于模态扩展导致输入 token 数量增加，从而面临更高的加速需求。Token 剪枝是针对 MVLA 模型的一种有效优化方法。然而，现有的 token 剪枝方案主要为仅含二维模态的 VLA 模型设计，忽略了二维与三维模态显著性差异。本文遵循 MVLA 模型在多模态数据处理中的应用过程，提出了一个三阶段分析方法，以捕捉二维/三维模态显著性的差异与动态变化。基于此，我们设计了一个针对 MVLA 模型的三阶段 token 剪枝框架，实现最优的二维/三维 token 选择与高效剪枝。实验结果表明，该框架在仅带来 5.8% 额外开销的情况下，可实现最高达 2.55 倍的推理速度提升且精度损失极小。我们的代码即将发布。

## Abstract
Vision-Language-Action (VLA) models have emerged as the mainstream of embodied intelligence. Recent VLA models have expanded their input modalities from 2D-only to 2D+3D paradigms, forming multi-visual-modal VLA (MVLA) models. Despite achieving improved spatial perception, MVLA faces a greater acceleration demand due to the increased number of input tokens caused by modal expansion. Token pruning is an effective optimization methods tailored to MVLA models. However, existing token pruning schemes are designed for 2D-only VLA models, ignoring 2D/3D modality salience differences. In this paper, we follow the application process of multi-modal data in MVLA models and develop a tri-stage analysis to capture the discrepancy and dynamics of 2D/3D modality salience. Based on these, we propose a corresponding tri-stage token pruning framework for MVLA models to achieve optimal 2D/3D token selection and efficient pruning. Experiments show that our framework achieves up to a 2.55x inference speedup with minimal accuracy loss, while only costing 5.8% overhead. Our Code is coming soon.

## 一、研究背景与核心问题

- **研究背景**：  
  随着具身智能的发展，视觉-语言-动作（Vision-Language-Action, VLA）模型成为机器人感知和交互的主流框架。这些模型通过整合视觉编码器、语言模型（LLM）和动作解码器，将自然语言指令直接映射为机器人控制信号。  
  传统 VLA 模型主要基于 2D 图像输入，但在复杂场景下二维视觉无法充分捕获空间关系与深度信息。因此，研究者开始将 3D 数据（如点云）引入，形成多视觉模态 VLA（MVLA）模型。

- **问题与瓶颈**：  
  - 模态扩展导致输入 token 数量激增（2D+3D 双通道各产生大量视觉 token），引发显著的推理延迟。  
  - 现有的 token 剪枝方法大多针对纯 2D 模型设计，未考虑 2D/3D 模态间显著性差异。  
  - 缺乏明确的机制来分析多模态显著性的阶段性动态：从数据预处理到语义推理再到动作执行的各阶段，二维和三维特征的重要性在不断变化。  

- **研究目标**：  
  作者希望**回答“2D 或 3D 谁主导显著性？”**的问题，并提出一种可动态感知模态显著性的**三阶段 token 剪枝框架**，以兼顾加速与性能保持。

## 二、方法论概述

### 1. 核心思想
- 提出一个**“三阶段显著性分析 + 剪枝框架”**，用以捕捉 2D/3D 模态显著性在三个推理阶段中的差异与动态：
  1. **数据预处理阶段**：关注 2D/3D 特征本身的贡献。
  2. **语义合成阶段**：分析不同语义区域（背景、机器人、目标对象）中的 2D/3D 显著性结构。
  3. **动作迭代阶段**：建模显著性的时间动态变化，实现基于历史信息的自适应剪枝。

### 2. 关键技术与指标
- **模态显著性量化指标（MS）**：  
  - 在第 1 阶段，作者通过模型输出特征的 L1 范数计算 2D 与 3D 模态显著性：  
    \( MS1_{modal} = avg_p(‖f_{modal,p}‖_1 / (‖f_{2D,p}‖_1 + ‖f_{3D,p}‖_1)) \)
  - 在语义合成阶段，利用注意力权重分解（平行部分 / 正交部分）度量语义区域内的模态独立信息量。
  - 在动作阶段，引入时间步 \(t\) 上的滑动窗口与动量因子 \(β\)，通过指数滑动平均（EMA）预测显著性随时序的变化：
    \( \hat{M}_{3D,p}^{t} = β \hat{M}_{3D,p}^{t-1} + (1-β)M_{3D,p}^{t} \)

### 3. 算法流程
- 该框架整体包括三个剪枝候选确定阶段和一个候选融合阶段：
  1. **基于显著性阈值的模态筛选（S1）**：以双阈值机制将每个 patch 分类为 2D 保留、3D 保留、或二者皆保留。
  2. **基于语义集的自适应筛选（S2）**：使用 K-Means 聚类将注意力矩阵划分为目标、机器人和背景区，通过全局基线约束不同区域的保留率。
  3. **基于动作动态的EMA平滑（S3）**：通过时间窗口预测防止帧间抖动。
  4. **候选融合规则**：以交集为主导，若存在冲突则保留语义候选结果，最终生成 token mask。

## 六、主要结论与发现

- 通过三阶段模态显著性分析，明确了：
  - 在**数据预处理阶段**，2D 模态显著性更高；
  - 在**语义合成阶段**，3D 模态在目标与机器人语义区域中显著性上升；
  - 在**动作迭代阶段**，2D/3D 显著性随时序动态变化，需引入预测机制。
