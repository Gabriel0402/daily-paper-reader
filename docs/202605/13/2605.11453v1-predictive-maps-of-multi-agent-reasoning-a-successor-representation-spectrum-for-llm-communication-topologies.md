## 摘要
部署多智能体大语言模型（LLM）系统的从业者目前必须在链式、星形、网格及更复杂的通信拓扑变体之间做出选择，但在推理前缺乏诊断手段来判断哪种拓扑会放大漂移、收敛至共识或在扰动下保持鲁棒。现有的评估仅在事后针对特定任务回答这些问题。我们基于行随机通信算子的继任者表示（Successor Representation）$M = (I - γP)^{-1}$，为多智能体 LLM 通信图引入了一种结构诊断方法，并将三个谱量——谱半径 $ρ(M)$、谱间隙 $Δ(M)$ 和条件数 $κ(M)$——与三种不同的故障模式联系起来。我们推导了行随机归一化下链式、星形和网格拓扑的闭式解谱，并在 Qwen2.5-7B-Instruct 上通过 100 次独立试验的 12 步结构化状态追踪任务验证了预测结果。结果表明，条件数是经验扰动鲁棒性的完美秩相关预测指标（$r_s = 1.0$）；谱间隙能部分预测共识动态（$r_s = 0.5$）；而谱半径与累积误差呈完美的负相关（$r_s = -1.0$）。我们将这种反转归因于线性谱对非收缩偏差漂移不敏感的情况，并提出了一种仿射噪声扩展预测图，从而恢复了经验排序。我们认为这是迈向多智能体 LLM 系统表征性、漂移感知结构诊断的第一步，与经典的谱理论和共识理论并列。

## Abstract
Practitioners deploying multi-agent large language model (LLM) systems must currently choose between communication topologies such as chain, star, mesh, and richer variants without any pre-inference diagnostic for which topology will amplify drift, converge to consensus, or remain robust under perturbation. Existing evaluation answers these questions only post hoc and only for the task measured. We introduce a structural diagnostic for multi-agent LLM communication graphs based on the successor representation $M = (I - γP)^{-1}$ of the row-stochastic communication operator, and we connect three of its spectral quantities, the spectral radius $ρ(M)$, the spectral gap $Δ(M)$, and the condition number $κ(M)$, to three distinct failure modes. We derive closed-form spectra for the chain, star, and mesh under row-stochastic normalization, and validate the predictions on a 12-step structured state-tracking task with Qwen2.5-7B-Instruct over 100 independent trials. The condition number is a perfect rank-order predictor of empirical perturbation robustness ($r_s = 1.0$); the spectral gap partially predicts consensus dynamics ($r_s = 0.5$); and the spectral radius is perfectly \emph{inverted} with respect to cumulative error ($r_s = -1.0$). We trace this inversion to a regime in which linear spectra are blind to non-contracting bias drift, and we propose an affine-noise extension of the predictive map that recovers the empirical ordering. We read this as a first step toward representational, drift-aware structural diagnostics for multi-agent LLM systems, sitting alongside classical spectral and consensus theory.

在构建多智能体 LLM 系统时，开发者通常需要在不同的通信拓扑（如链式、星形、网格等）中做出选择。然而，目前缺乏一种**推理前（Pre-inference）**的诊断工具来预测特定拓扑是否会导致误差累积（漂移）、是否能达成共识以及在面对扰动时是否稳健。现有的评估方法大多是事后的（Post hoc），且高度依赖于具体任务。本研究旨在通过分析通信图的结构属性，建立一套理论框架，在不运行模型的情况下预测系统的失效模式。

### 2. 方法论：核心思想与技术细节
论文将多智能体系统建模为有向图，并引入了强化学习和神经科学中的**继任者表示（Successor Representation, SR）**作为核心诊断工具。

*   **核心公式**：定义行随机转移矩阵 $P$（由通信图的邻接矩阵归一化得到），其继任者表示矩阵为 $M = (I - \gamma P)^{-1}$，其中 $\gamma \in [0, 1)$ 是折扣因子（实验中取 0.9）。$M$ 压缩了所有多步影响路径。
*   **三个谱指标（Spectral Quantities）**：
    1.  **谱半径 $\rho(M)$**：衡量线性系统的几何增长率，用于预测误差放大倾向。
    2.  **谱间隙 $\Delta(M)$**：衡量马尔可夫链的混合速度，用于预测智能体间达成共识的动态。
    3.  **条件数 $\kappa(M)$**：衡量系统对输入扰动的敏感度，用于预测鲁棒性。
*   **稳定性悖论与修正**：研究发现原始谱半径与 LLM 的实际漂移误差呈负相关（稳定性悖论）。为此，作者提出了**仿射噪声模型（Affine-noise Model）**，推导出漂移修正增益 $\tilde{\rho}$，考虑了聚合步骤（如投票、法官总结）带来的方差缩减效应。

### 3. 实验设计
*   **任务场景**：一个 12 步的结构化状态追踪任务。智能体需要根据复杂的 JSON 规则更新包含数值（Value）、奇偶性（Parity）和等级（Level）的状态。

    *   **网格（Mesh）**：4 个智能体进行同伴评议和多数投票。
*   **实验模型**：本地部署的 Qwen2.5-7B-Instruct。

### 5. 实验数量与充分性
*   **实验规模**：每种拓扑配置均进行了 100 次独立重复试验，以获取均值和标准差。
*   **充分性评价**：
    *   **优点**：100 次试验提供了较好的统计显著性，能够观察到明显的拓扑效应。
    *   **局限**：仅对比了 3 种基础拓扑和 1 个模型系列。虽然对于验证理论框架是足够的，但对于建立“普适定律”而言，样本点（拓扑种类）较少。

### 6. 主要结论与发现
*   **条件数 $\kappa(M)$ 是完美的预测指标**：它与经验扰动鲁棒性的秩相关系数达到了 $r_s = 1.0$，证明了谱分析在预测系统脆弱性方面的极高准确性。
*   **谱间隙 $\Delta(M)$ 部分有效**：它能预测共识趋势，但由于星形拓扑中的“法官”角色形成了物理瓶颈，实际共识速度超出了纯图论的预测。
*   **稳定性悖论**：链式拓扑在数学上最稳定（$\rho=1$），但在 LLM 推理中误差最大。这是因为线性谱忽略了 LLM 逐步产生的随机偏差。
*   **漂移修正的有效性**：引入考虑聚合次数 $k$ 的修正增益 $\tilde{\rho} \propto 1/\sqrt{k}$ 后，理论预测与实际累积误差的排序完全一致。

### 7. 优点：亮点与创新
*   **理论前瞻性**：首次将 SR 谱理论引入多智能体 LLM 拓扑设计，将“试错法”转变为“解析法”。
*   **跨学科融合**：结合了控制理论、图论和认知科学的视角，解释了 LLM 推理中的结构性失效。
*   **实用性**：计算谱指标仅需毫秒级时间，且无需运行昂贵的 LLM 推理，具有极高的工程诊断价值。

### 8. 不足与局限
*   **拓扑覆盖有限**：未测试更复杂的动态图或大规模智能体集群（如 100+ 智能体）。
*   **任务单一性**：实验集中在结构化状态追踪，对于开放式创意写作或多轮对话等非结构化任务的适用性有待验证。
*   **噪声假设**：仿射噪声模型假设智能体误差是独立同分布（i.i.d.）的，但在实际应用中，若多个智能体共享同一基座模型，可能存在系统性偏差（相关噪声），这会削弱聚合的效果。
