## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记或全局潜变量。当指令涉及特定对象时，动作解码器很难处理这些表示，尤其是在对象身份与上下文纠缠的场景偏移下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并在块因果序列中与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码 16 步连续动作块。通过仅地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，从而强制实现可寻址性，在不添加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO（97.8%）和 SimplerEnv（79.3%）上达到了强 VLA 和 WAM 基线的水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试产生的交换绑定余弦值为 0.87，而整体基线最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

### 1. 核心问题与整体含义（研究动机和背景）
*   **研究动机**：现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）在标准基准测试中表现优异，但在面对场景扰动（如相机视角切换、物体布局改变、背景变化）时极易崩溃。
*   **核心问题**：作者认为这种不鲁棒性源于**“对象可寻址性（Object Addressability）”的缺失**。目前的模型通常将场景处理为整体图像或全局标记（Tokens），导致目标物体的身份（Identity）与周围上下文（Context）发生纠缠。当场景发生几何偏移时，模型无法稳定地定位指令所指的对象。
*   **整体含义**：论文旨在构建一种能够将“操作哪个物体”与“物体当前状态”在张量层面解耦的模型，使机器人能够像人类一样，通过稳定的对象接口进行推理和操作。

### 2. 方法论：核心思想与关键技术
*   **核心思想**：提出 **OA-WAM**，通过将每一帧分解为 $N+1$ 个可寻址的对象槽位（Slots），并在 Transformer 架构中强制执行对象路由约束。
*   **关键技术细节**：
    *   **对象槽位表示**：每个槽位由两部分组成：**地址向量（$addr_k$）**（由语言标签和初始特征计算，每集固定，代表“是谁”）和**内容向量（$cnt_t^k$）**（随时间变化，代表“在哪/什么状态”）。
    *   **对象可寻址注意力（OA Attention）**：在 7B 参数的 Chameleon 骨干网络中，跨槽位注意力的 **Key 投影仅读取地址子向量**。
    *   **残差流重置（Residual Reset）**：在每一层 Transformer 处理后，强制将残差流中的地址部分重置为初始值，防止身份信息在深层网络中发生漂移或纠缠。
    *   **双头预测**：
        *   **世界头（World Head）**：回归预测下一帧各槽位的内容和位姿。
        *   **动作头（Action Head）**：基于流匹配（Flow-matching）在单次前向传播中解码 16 步连续动作块。
*   **算法流程**：感知层（SAM3 + DINOv3）提取槽位 -> 槽位适配器投影 -> 块因果序列组装 -> 槽位感知骨干网络处理 -> 动作与未来状态并行输出。

### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准操作基准，包含 Spatial、Object、Goal、Long 四个子集。
    *   **SimplerEnv**：评估模拟到真实的视觉匹配能力（WidowX 机器人）。
    *   **LIBERO-Plus**：核心鲁棒性测试，包含相机、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动轴。
*   **对比方法**：
    *   **VLA 基线**：OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1 等。
    *   **WAM 基线**：WorldVLA, VLA-JEPA, Cosmos-Policy, GE-Act 等。
*   **特殊测试**：**因果槽位干预测试**。通过交换两个物体的地址向量，观察机器人是否会改变操作目标，以此验证模型是否真正实现了对象绑定。

### 4. 资源与算力
*   **预训练（Stage 0）**：使用 **384 张 A100-80GB GPU**，耗时 **18 天**（约 16.6 万 A100 小时），处理了 2.5T Tokens。
*   **微调（Stage I & II）**：使用 **8 张 A100-80GB GPU**，每个阶段耗时 **3-4 天**。
*   **推理延迟**：在单张 A100 上，感知耗时约 138ms，骨干网络约 80ms，动作头约 10ms，总计约 **233ms/动作块**（约 4.3 Hz 控制频率）。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了 3 个主流基准测试，进行了 3 个随机种子的平均实验。
*   **消融实验**：非常充分。作者设计了 V0-V3 四个变体，分别验证了“地址掩码”和“残差重置”的独立贡献，并对比了有无世界预测头的影响。
*   **客观性**：使用了官方的扰动生成器，并在 LIBERO-Plus 上进行了零样本（Zero-shot）评估（即训练集不包含扰动因素），保证了评估的公平性和客观性。

### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO（97.8%）和 SimplerEnv（79.3%）上达到或超过了 SOTA 水平。
*   **几何鲁棒性极强**：在相机偏移、机器人初始位姿变化等几何扰动下，OA-WAM 的表现显著优于所有整体表示模型（几何平均分 84.3%，比 $\pi_{0.5}$ 高 4.8%）。
*   **机制验证**：在地址交换干预测试中，OA-WAM 的绑定余弦值达到 **0.87**，而所有整体表示模型均低于 **0.09**，证明其确实通过地址空间实现了目标选择。

### 7. 优点
*   **架构创新**：通过简单的张量掩码和重置操作，在不增加额外 Token 的情况下实现了对象级寻址，设计优雅且高效。
*   **可验证性**：不同于黑盒 VLA 模型，OA-WAM 的决策逻辑（即关注哪个槽位）是可以通过干预实验直接验证的。
*   **分布外（OOD）泛化**：通过引入对象中心化的归纳偏置，显著提升了模型在未见过的几何配置下的生存能力。

### 8. 不足与局限
*   **感知瓶颈**：模型高度依赖上游的感知系统（SAM3/DINOv3）。如果感知层未能提取出物体（如在强传感器噪声下），策略层无法纠正。
*   **实时性挑战**：感知部分耗时占总延迟的 60% 以上，对于高频闭环控制仍有优化空间。
*   **模拟器依赖**：目前所有验证均在模拟器中完成，尚未展示在真实物理环境中的部署效果。
*   **弱耦合假设**：模型假设目标与干扰物之间是弱耦合的，在处理复杂的物体间物理遮挡或强交互时可能存在局限。
