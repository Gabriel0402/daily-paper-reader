## 摘要
Segment Anything Model 2 (SAM2) 已成为通用分割的基础模型。由于其具有泛化能力的视觉表示，SAM2 已成功应用于各种下游任务。然而，将 SAM2 扩展到 RGB-D 视频显著性目标检测 (RGB-D VSOD) 任务面临三个挑战，包括线性 LoRA 的空间建模能力有限、对 SAM 多尺度特征利用不足，以及初始化依赖于显式提示。为了解决这些问题，我们提出了 M$^4$-SAM（Multi-Modal Mixture-of-Experts with Memory-Augmented SAM），它为 SAM2 配备了模态相关的参数高效微调 (PEFT)、分层特征融合和无提示记忆初始化。首先，我们在 SAM2 的编码器中注入了模态感知混合专家 LoRA (Modality-Aware MoE-LORA)，该模块利用卷积专家编码局部空间先验，并引入模态调度器以实现高效的多模态微调。其次，我们部署了门控多级特征融合 (Gated Multi-Level Feature Fusion)，通过自适应门控机制分层聚合多尺度编码器特征，以平衡空间细节和语义上下文。最后，为了在没有人工提示的情况下进行零样本 VSOD，我们利用了伪引导初始化 (Pseudo-Guided Initialization)，将粗略掩码视为伪先验并用于引导记忆库。广泛的实验表明，M$^4$-SAM 在三个公开的 RGB-D VSOD 数据集上的所有评估指标均达到了最先进的性能。

## Abstract
The Segment Anything Model 2 (SAM2) has emerged as a foundation model for universal segmentation. Owing to its generalizable visual representations, SAM2 has been successfully applied to various downstream tasks. However, extending SAM2 to the RGB-D video salient object detection (RGB-D VSOD) task encounters three challenges including limited spatial modeling of linear LoRA, insufficient employment of SAM's multi-scale features, and dependence of initialization on explicit prompts. To address the issues, we present Multi-Modal Mixture-of-Experts with Memory-Augmented SAM (M$^4$-SAM), which equips SAM2 with modality-related PEFT, hierarchical feature fusion, and prompt-free memory initialization. Firstly, we inject Modality-Aware MoE-LORA, which employs convolutional experts to encode local spatial priors and introduces a modality dispatcher for efficient multi-modal fine-tuning, into SAM2's encoder. Secondly, we deploy Gated Multi-Level Feature Fusion, which hierarchically aggregates multi-scale encoder features with an adaptive gating mechanism, to balance spatial details and semantic context. Finally, to conduct zero-shot VSOD without manual prompts, we utilize a Pseudo-Guided Initialization, where a coarse mask is regarded as a pseudo prior and used to bootstrap the memory bank. Extensive experiments demonstrate that M$^4$-SAM achieves the state-of-the-art performance across all evaluation metrics on three public RGB-D VSOD datasets.

显著性目标检测（SOD）旨在识别场景中最吸引人类注意力的物体。传统的基于 RGB 的方法在复杂背景、弱光或运动模糊下表现不佳。因此，引入深度信息（RGB-D）和时间信息（Video）的 **RGB-D VSOD** 任务应运而生。
尽管 Segment Anything Model 2 (SAM2) 在通用分割上表现强劲，但将其直接应用于 RGB-D VSOD 面临三大挑战：
*   **空间建模受限**：传统的 LoRA 微调仅使用线性投影，缺乏捕捉局部空间结构的能力。
*   **特征利用不足**：SAM2 默认主要利用深层特征，忽略了对显著性检测至关重要的多尺度空间细节。

### 2. 方法论：核心思想与关键技术
论文提出了 **M$^4$-SAM** 框架，通过以下三个核心模块对 SAM2 进行改造：
*   **模态感知混合专家 LoRA (Modality-Aware MoE-LoRA)**：
    *   在 LoRA 分支中引入了多个卷积专家（3x3, 5x5 卷积等），以引入空间局部先验。

*   **门控多级特征融合 (Gated Multi-Level Feature Fusion, Gated-MLF)**：
    *   分层聚合来自编码器的多尺度特征。
    *   利用自适应门控机制平衡浅层细节和深层语义，并结合双重注意力（空间和通道）增强特征表示。
*   **伪引导初始化 (Pseudo-Guided Initialization)**：
    *   为了摆脱人工提示，模型利用解码器生成的首帧粗略掩码（Coarse Mask）作为“伪先验”。
    *   通过线性投影将该伪掩码与特征结合，初始化 SAM2 的记忆库（Memory Bank），从而实现自动化的时序建模。

*   M$^4$-SAM 在所有三个数据集的所有指标上均刷新了 SOTA 记录。

*   **伪掩码质量依赖**：虽然引入了门控机制抑制噪声，但如果首帧的粗略预测（Coarse Mask）出现严重偏差，可能会通过记忆库产生误差累积。
*   **应用范围**：目前主要针对显著性目标检测，对于需要区分特定语义类别的多目标视频分割任务，其泛化能力尚待验证。
*   **长序列挑战**：虽然使用了记忆库，但在极长视频或目标长时间消失又出现的场景下，模型的一致性表现未做专门的长程压力测试。
