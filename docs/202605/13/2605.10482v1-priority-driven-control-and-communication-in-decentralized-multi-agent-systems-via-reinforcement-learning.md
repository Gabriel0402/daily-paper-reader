## 摘要
事件触发控制提供了一种避免网络化多智能体系统中受限通信带宽过度使用的机制。然而，大多数现有方法依赖于精确的系统模型，而这在实践中往往难以获取。在这项工作中，我们提出了一种无模型的、优先级驱动的强化学习算法，该算法能够在去中心化多智能体系统中从数据中联合学习通信优先级和控制策略。通过学习通信优先级，我们规避了在具有二元通信决策的事件触发控制中常见的混合动作空间问题。我们在基准任务上对算法进行了评估，结果表明其性能优于基准方法。

## Abstract
Event-triggered control provides a mechanism for avoiding excessive use of constrained communication bandwidth in networked multi-agent systems. However, most existing methods rely on accurate system models, which may be unavailable in practice. In this work, we propose a model-free, priority-driven reinforcement learning algorithm that learns communication priorities and control policies jointly from data in decentralized multi-agent systems. By learning communication priorities, we circumvent the hybrid action space typical in event-triggered control with binary communication decisions. We evaluate our algorithm on benchmark tasks and demonstrate that it outperforms the baseline method.

### 1. 论文的核心问题与整体含义（研究动机和背景）
在网络化多智能体系统（NCS）中，通信带宽通常是受限的共享资源。传统的**事件触发控制（ETC）**通过仅在必要时传输数据来节省带宽，但大多依赖精确的数学模型，难以处理高维非线性系统。
现有的强化学习（RL）方法在解决此问题时面临三大挑战：
*   **混合动作空间**：智能体需要同时决定“是否通信”（离散）和“控制输入”（连续），导致优化困难。
*   **中心化依赖**：许多方法在训练阶段假设可以无限制获取全局信息，这在实际去中心化场景中难以实现。
*   **网络过载风险**：简单的二值化决策可能导致大量智能体同时触发通信，造成网络拥塞。

本研究旨在提出一种**无模型、去中心化**的框架，通过学习“通信优先级”来联合优化控制与通信策略。

### 2. 论文提出的方法论
核心思想是将离散的通信决策转化为**连续的优先级（Priority）**学习，从而将混合动作空间简化为纯连续动作空间。

*   **核心思想**：每个智能体输出一个连续值 $a_{i,p} \in (0, 1)$ 作为通信优先级。系统根据优先级高低分配 $L$ 个通信槽位。
*   **关键技术细节**：
    *   **动作构成**：智能体的动作 $A_i$ 包含控制动作 $a_{i,c}$ 和优先级动作 $a_{i,p}$。
    *   **奖励函数设计**：在原始控制奖励 $r_{i,c}$ 基础上增加通信惩罚项 $r_{i,p} = -\xi a_{i,p}$，以抑制智能体盲目争夺带宽。
    *   **通信机制**：采用“优先级交换”机制。由于优先级数值极小（如3比特整数），其传输开销远小于观测数据（如16字节浮点向量）。仅优先级最高的 $L$ 个智能体获准广播其观测信息。
*   **算法流程（Control-Priority IPPO）**：
    1.  基于独立近端策略优化（IPPO）框架。
    2.  智能体根据局部观测和收到的历史信息生成控制量和优先级。
    3.  环境根据优先级排序，选出 Top-$L$ 智能体进行广播。
    4.  所有智能体根据反馈的联合奖励和通信惩罚更新 Actor 和 Critic 网络。
