## 摘要
虽然大语言模型 (LLMs) 和视觉语言模型 (VLMs) 在高层推理和语义理解方面表现出卓越的能力，但由于缺乏明确的物理基础且无法执行自适应控制，将它们直接应用于丰富接触 (contact-rich) 的操作任务仍然是一个挑战。为了弥补这一差距，我们提出了 CoRAL (Contact-Rich Adaptive LLM-based control)，这是一个模块化框架，通过将高层推理与底层控制解耦来实现零样本规划。与黑盒策略不同，CoRAL 不将 LLM 用作直接控制器，而是作为代价设计器 (cost designers)，为基于采样的运动规划器 (MPPI) 合成上下文感知的目标函数。为了解决视觉数据中物理参数的模糊性，我们引入了一个神经符号自适应循环：VLM 提供环境动力学的语义先验（如质量和摩擦力估计），随后通过在线系统辨识进行实时显式细化，同时 LLM 根据交互反馈迭代调节代价函数结构以纠正策略错误。此外，基于检索的存储单元允许系统在循环任务中重用成功的策略。这种分层架构通过将高层语义推理与反应式执行解耦，确保了实时控制的稳定性，有效地弥合了慢速 LLM 推理与动态接触需求之间的差距。我们在仿真和真实硬件上验证了 CoRAL，涵盖了具有挑战性的新颖任务，例如利用外部接触将物体靠墙翻转。实验表明，CoRAL 在未见的丰富接触场景中平均成功率提升了 50% 以上，优于最先进的 VLA 和基于基础模型的规划器基准，并通过其自适应物理理解有效地处理了从仿真到现实 (sim-to-real) 的差距。

## Abstract
While Large Language Models (LLMs) and Vision-Language Models (VLMs) demonstrate remarkable capabilities in high-level reasoning and semantic understanding, applying them directly to contact-rich manipulation remains a challenge due to their lack of explicit physical grounding and inability to perform adaptive control. To bridge this gap, we propose CoRAL (Contact-Rich Adaptive LLM-based control), a modular framework that enables zero-shot planning by decoupling high-level reasoning from low-level control. Unlike black-box policies, CoRAL uses LLMs not as direct controllers, but as cost designers that synthesize context-aware objective functions for a sampling-based motion planner (MPPI). To address the ambiguity of physical parameters in visual data, we introduce a neuro-symbolic adaptation loop: a VLM provides semantic priors for environmental dynamics, such as mass and friction estimates, which are then explicitly refined in real time via online system identification, while the LLM iteratively modulates the cost-function structure to correct strategic errors based on interaction feedback. Furthermore, a retrieval-based memory unit allows the system to reuse successful strategies across recurrent tasks. This hierarchical architecture ensures real-time control stability by decoupling high-level semantic reasoning from reactive execution, effectively bridging the gap between slow LLM inference and dynamic contact requirements. We validate CoRAL on both simulation and real-world hardware across challenging and novel tasks, such as flipping objects against walls by leveraging extrinsic contacts. Experiments demonstrate that CoRAL outperforms state-of-the-art VLA and foundation-model-based planner baselines by boosting success rates over 50% on average in unseen contact-rich scenarios, effectively handling sim-to-real gaps through its adaptive physical understanding.

机器人操作中存在大量“接触密集型”（Contact-Rich）任务（如靠墙翻转物体、受力控制的推行），这类任务对物理参数（质量、摩擦力）和力控精度要求极高。现有的视觉-语言-动作（VLA）模型虽然具备语义理解能力，但往往是黑盒策略，缺乏显式的物理接地（Physical Grounding），且在面对未见过的物理环境时缺乏自适应能力。本文旨在解决如何将大语言模型（LLM）的高层推理能力与底层的高频动力学控制相结合，实现零样本（Zero-shot）且具备物理自适应性的复杂操作。

#### 2. 论文提出的方法论：核心思想与关键技术
CoRAL 采用了一种**分层解耦的神经符号架构**，核心思想是将 LLM 从直接控制器转变为“代价函数设计者”。
*   **感知层（VLM + Tracker）：** 使用 FoundationPose 进行 6-DoF 物体跟踪，并利用 VLM（如 GPT-4o）根据视觉上下文推断物体的物理先验（如：这看起来像个沉重的金属块，质量设为 2kg）。
*   **推理层（LLM Task Formulation）：** LLM 不输出具体动作，而是生成 Python 格式的**代价函数（Cost Function）**和**接触策略（Contact Strategy）**。它定义了任务的目标（如：最小化物体与目标的距离）和约束（如：保持特定的接触力）。
*   **执行层（MPPI 控制器）：** 采用模型预测路径积分（MPPI）算法，在 LLM 定义的代价函数指导下，通过高频采样（10Hz）生成最优控制序列。
*   **双重自适应循环：**
    *   **内环（反应式控制）：** MPPI 根据实时传感器反馈（力/力矩）调整动作。
    *   **外环（LLM 细化）：** 当任务持续失败时，LLM 作为“诊断员”分析交互历史，修正物理参数估计（如：物体推不动，调大质量参数）或重写代价函数逻辑。
*   **存储单元：** 引入基于检索增强生成（RAG）的记忆模块，存储成功的经验以供相似任务复用。

#### 3. 实验设计
*   **实验场景：** 包含 6 类挑战性任务：推并抓取剪切板（多阶段）、抓取方块、杂乱环境取物、恒力推送、翻转方块、靠墙翻转物体（极度依赖外部接触）。
*   **Benchmark 与对比方法：**

*   **硬件配置：** 实验在一台配备 **Intel Core i9-13900K CPU**、**64GB RAM** 和单块 **NVIDIA RTX 4060 Ti GPU** 的台式 PC 上运行。

*   **训练时长：** 由于是零样本框架，**无需针对特定任务进行策略训练**，仅需调用预训练模型。

#### 5. 实验数量与充分性
*   **实验规模：** 每个任务进行 10 次随机初始化的试验。涵盖了仿真环境（Robosuite/MuJoCo）和真实世界（Franka Emika Panda 机器人）的验证。
*   **充分性评价：** 实验设计较为充分。不仅对比了最先进的端到端模型，还通过消融实验深入探讨了模块化设计的价值。特别是针对“物理参数估计错误”的鲁棒性测试，证明了系统在初始参数大幅偏差时仍能通过自适应循环恢复，具有较强的客观性和说服力。

#### 6. 论文的主要结论与发现
*   **性能领先：** CoRAL 在复杂接触任务中的成功率比现有 VLA 模型平均高出 50% 以上。
*   **解耦的必要性：** 将感知（VLM）与策略推理（LLM）分离至关重要，统一的 VLM 往往无法同时处理精确的物理估计和复杂的逻辑规划。
*   **自适应能力：** 在线细化循环能有效弥合“仿真到现实”的差距，LLM 能够根据物理反馈识别并纠正错误的物理假设。
*   **符号化的优势：** 生成 Python 代码形式的代价函数比直接预测动作更具可解释性，且允许利用成熟的控制理论工具。

#### 7. 优点
*   **零样本泛化：** 无需任何演示数据即可处理新颖的、力控敏感的任务。
*   **高可解释性：** LLM 生成的代价函数和诊断日志清晰地展示了机器人的意图和失败原因。
*   **物理接地：** 通过将 LLM 嵌入到 MPPI 循环中，解决了大模型“幻觉”导致的动作不连贯问题。

#### 8. 不足与局限
*   **延迟问题：** 依赖云端 LLM API 导致外环自适应存在秒级延迟，无法处理极高动态的突发物理变化。
*   **感知依赖：** 系统高度依赖 6-DoF 姿态跟踪的准确性，在物体遮挡严重或形状极其复杂的场景下可能失效。
*   **VLM 局限：** VLM 对物理参数（如摩擦系数）的初始估计仍带有猜测性，虽然有后续修正，但初始失败可能导致硬件损耗。
*   **应用限制：** 目前主要针对刚性物体操作，尚未验证在柔性物体或流体操作中的有效性。
