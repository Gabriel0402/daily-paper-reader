## 摘要
将多模态基础模型集成到企业生态系统中提出了一个根本性的软件架构挑战。架构师必须平衡相互竞争的质量属性：视觉语言动作（VLA）模型的高延迟和非确定性，与企业控制回路所需的严格确定性和实时性能。在本研究中，我们提出了一种用于视觉智能体的架构模式语言，该语言将快速、确定性的反射与慢速、概率性的监督相分离。它由四种架构设计模式组成：(1) 混合示能集成，(2) 自适应视觉锚定，(3) 视觉层级合成，以及 (4) 语义场景图。

## Abstract
Integrating multimodal foundation models into enterprise ecosystems presents a fundamental software architecture challenge. Architects must balance competing quality attributes: the high latency and non-determinism of vision language action (VLA) models versus the strict determinism and real-time performance required by enterprise control loops. In this study, we propose an architectural pattern language for visual agents that separates fast, deterministic reflexes from slow, probabilistic supervision. It consists of four architectural design patterns: (1) Hybrid Affordance Integration, (2) Adaptive Visual Anchoring, (3) Visual Hierarchy Synthesis, and (4) Semantic Scene Graph.

### 2. 论文提出的方法论
核心思想是**“摊销推理”（Amortized Inference）**：将快速、确定性的“反射层”（系统1）与慢速、概率性的“监督层”（系统2）分离。

#### 四大核心架构模式：
1.  **混合示能集成（Hybrid Affordance Integration）**：
    *   **细节**：建立多模态感知融合层。并行运行轻量级局部模型（如UI对象检测器和OCR），通过几何融合和冲突解决机制，过滤感知噪声，避免“示能幻觉”。
2.  **自适应视觉锚定（Adaptive Visual Anchoring）**：
    *   **细节**：建立“回退路由”连接器。正常情况下使用本地缓存的视觉锚点（<50ms）进行操作；当匹配置信度低于阈值（如 $\tau < 0.9$）时，触发VLA监督者进行语义定位并更新缓存。
3.  **视觉层级合成（Visual Hierarchy Synthesis）**：
    *   **细节**：利用格式塔原理（邻近、对齐等）将扁平的检测结果转化为层级化的UI树。这解决了“相对寻址”问题，使智能体能理解“A表格中的提交按钮”而非仅靠绝对坐标。
4.  **语义场景图（Semantic Scene Graph）**：
    *   **细节**：构建显式的、可查询的世界模型。在执行高风险操作前，通过遍历场景图验证前提条件，增强了决策的可解释性和安全性。
