## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变化以及人类纠错机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略出发，利用在机器人车队中收集的自主展开和人类干预，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，用于流式 VLA 动作生成器中的策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和 3-5 分钟的长程任务。随着车队经验的积累，单一通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

#### 1. 核心问题与整体含义（研究动机和背景）
论文针对通用机器人策略（如 VLA 模型）在现实部署中面临的**分布偏移（Distribution Shift）**、**长尾失败**和**任务变体**等核心挑战。尽管大规模离线预训练能提供基础能力，但固定的数据集无法涵盖所有实际使用场景。
*   **研究动机**：将“部署”从训练的终点转变为持续改进的起点。
*   **核心理念**：构建一个“数据飞轮”，通过分布式机器人车队（Fleet）在真实环境中自主运行并收集经验（包括成功、失败和人类干预），利用强化学习（RL）实现策略的持续进化。

#### 2. 方法论：核心思想与关键技术
LWD 提出了一个车队规模的“离线到在线”强化学习框架，包含两个关键算法组件：

*   **分布隐式价值学习（DIVL, Distributional Implicit Value Learning）**：
    *   **思想**：改进了隐式 Q 学习（IQL）。由于车队数据具有异构性和多模态特征（不同策略版本、不同环境），传统的标量价值函数会模糊稀疏的高回报路径。
    *   **技术细节**：DIVL 学习动作价值的概率分布而非单一标量，并提取特定分位数（Quantile）作为 TD 目标。引入了**自适应 $\tau$ 策略**，根据分布的熵（不确定性）动态调整引导目标的乐观程度。
*   **伴随匹配 Q 学习（QAM, Q-learning with Adjoint Matching）**：
    *   **思想**：解决流式（Flow-based）VLA 策略的优化难题。直接对多步去噪过程进行梯度回传既昂贵又不稳定。
    *   **技术细节**：QAM 将 Critic 提供的动作梯度转化为流模型（Flow Model）的步进式监督信号，通过伴随状态（Adjoint State）实现高效且稳定的策略提取。
*   **训练流程**：分为离线预训练（利用 652 小时混合数据）和在线持续后训练（利用机群实时回传的混合缓冲区数据）。

#### 3. 实验设计
*   **实验场景**：涵盖 8 项真实世界任务，分为两类：
    1.  **语义杂货补货**：平架补货、错位纠正、冰柜补货、开放式冷柜补货（侧重语义泛化）。
    2.  **长程操纵任务（3-5 分钟）**：功夫茶冲泡、果汁制作、鸡尾酒调制、鞋盒打包（侧重长程依赖和精度）。
*   **硬件平台**：16 台 **Agibot G1** 双臂协作机器人。
*   **对比基准（Baselines）**：
    *   **SFT**：仅使用演示数据的监督微调。
    *   **RECAP**：一种基于优势加权的离线后训练方法。
    *   **HG-DAgger**：一种依赖人类干预的交互式模仿学习方法。
    *   **LWD (Offline)**：仅使用离线 RL 阶段的模型。

*   **模型架构**：
    *   **Critic/Value 网络**：基于 Gemma 3-270M-IT 语言模型和 SigLIP-So400M 视觉编码器。
    *   **Actor 网络**：基于 PaliGemma (Gemma-2B + SigLIP) 骨干网络，配合 300M 参数的动作专家模块。

#### 5. 实验数量与充分性
*   **实验规模**：在 16 台机器人上进行了大规模部署，这在真实机器人 RL 研究中属于较高规格。
*   **消融实验**：
    *   对比了 DIVL 与传统的 Expectile 回归（IQL 模式），证明了分布学习在长程任务上的优越性。
    *   验证了自适应 $\tau$ 策略对价值估计校准的有效性。
*   **充分性评价**：实验设计较为充分，涵盖了从短程语义任务到极长程操纵任务的梯度。通过对比 SFT、模仿学习变体和离线 RL 变体，客观地展示了在线机群学习的增量价值。

#### 6. 主要结论与发现
*   **性能提升**：LWD (Online) 在所有任务上达到 **95% 的平均成功率**，显著优于所有基线。
*   **长程任务优势**：在功夫茶、调酒等复杂任务中，LWD 的提升最为巨大（相比 SFT 提升了约 20-30%），证明了 RL 在处理复合误差和长程奖励传播方面的能力。
*   **效率优化**：相比 SFT，LWD 训练出的策略在完成任务时更加果断，平均循环时间（Cycle Time）缩短了约 23.75 秒。
*   **价值信号**：可视化证明了学习到的价值函数能够准确追踪任务进度，即使在只有终端稀疏奖励的情况下。

#### 7. 优点（亮点）
*   **闭环系统工程**：成功实现了从分布式部署到云端集中训练再到重新部署的完整自动化流水线。
*   **算法创新**：DIVL 和 QAM 的结合解决了 VLA 模型在 RL 训练中的稳定性和效率问题。
*   **真实感强**：实验完全在真实物理环境中进行，任务复杂度高（如 5 分钟的长程任务），具有极强的实用参考价值。

#### 8. 不足与局限
*   **安全建模缺失**：目前的框架未显式考虑在线探索过程中的物理安全约束。
*   **指令理解深度**：长程任务仍依赖单一的简短指令，缺乏对复杂视觉-语言推理或多轮对话纠错的深入探讨。
*   **算力成本**：虽然在线训练时间短，但维持大规模 VLM 级别的 Critic 训练和机群同步需要极高的基础设施支持。
*   **人工依赖**：虽然支持自主学习，但在处理极端失败时仍需人工干预（Intervention），完全脱离人类的自主进化尚有距离。
