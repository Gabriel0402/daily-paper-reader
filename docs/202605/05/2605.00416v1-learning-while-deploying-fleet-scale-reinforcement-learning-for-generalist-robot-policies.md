## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变化以及人类纠错机会，这些是固定演示数据集无法完全捕捉的。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略出发，通过利用在机器人车队中收集的自主运行和人类干预数据，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，以便在基于流的 VLA 动作生成器中进行策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界操纵任务，包括语义杂货补货和 3-5 分钟的长程任务。随着车队经验的积累，单一通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

### 1. 核心问题与整体含义
论文针对通用机器人策略（如 VLA 模型）在现实部署中面临的**分布偏移（Distribution Shift）**和**长尾失败（Long-tail Failures）**问题。尽管大规模离线预训练赋予了机器人基础能力，但固定的数据集无法涵盖所有现实场景。
**核心动机**是将“部署”从训练的终点转变为持续学习的起点。通过构建一个“数据飞轮”，让部署在全球各地的机器人集群（Fleet）在执行任务时自动收集成功、失败及人类干预的数据，并利用强化学习（RL）持续优化单一的通用策略。

### 2. 方法论：LWD 框架
LWD 采用了一种**离线到在线（Offline-to-Online）的强化学习**架构，包含两个关键技术组件：

*   **分布隐式价值学习（DIVL, Distributional Implicit Value Learning）：**
    *   **核心思想：** 改进了传统的 IQL 算法。由于集群数据高度异构（包含不同策略版本、环境和人类干预），单一的标量价值估计会模糊掉稀疏但关键的成功信号。
    *   **技术细节：** DIVL 学习动作价值的完整分布（Categorical Distribution），并提取特定的**分位数（Quantile）**作为 TD 目标。
    *   **自适应策略：** 引入基于分布熵的自适应分位数参数 $\tau$，在不确定性高时保持保守，在确定性高时追求乐观。
*   **伴随匹配 Q 学习（QAM, Q-learning with Adjoint Matching）：**
    *   **核心思想：** 解决在大型生成式 VLA 模型（基于 Flow Matching）中提取策略的稳定性问题。
    *   **技术细节：** 避免了直接对复杂的 ODE 求解过程进行反向传播，而是将 Critic 提供的动作梯度转化为流模型每一步的局部回归目标，从而稳定地将价值提升转化为动作改进。
*   **训练流程：** 分为离线预训练（在 652 小时混合数据上）和在线持续微调（在集群实时回传的数据上）两个阶段，两者共享统一的 RL 目标。

### 3. 实验设计
*   **实验场景：** 使用了 **16 台 Agibot G1 双臂机器人**，涵盖 8 项真实任务：
    *   **长程任务（3-5 分钟）：** 功夫茶、制作鸡尾酒、制作果汁、鞋盒打包。
    *   **语义泛化任务：** 杂货店补货（冷柜、货架、纠错等）。
*   **Benchmark 与对比方法：**
    *   **SFT（监督微调）：** 仅使用人类演示数据的基准。
    *   **RECAP：** 一种基于优势加权的离线 RL 改进方法。
    *   **HG-DAgger：** 交互式模仿学习基准，利用人类干预数据。
    *   **LWD (Offline)：** 仅完成离线 RL 阶段的模型。

### 5. 实验数量与充分性
*   **实验规模：** 实验涵盖了从短程语义任务到极具挑战性的长程操纵任务，总交互时长和机器人数量在同类研究中属于顶尖水平。

### 6. 主要结论与发现
*   **性能飞跃：** LWD 在所有任务上的平均成功率达到 **95%**，显著优于 SFT (76%) 和其他基准。
*   **长程任务优势：** 在功夫茶等复杂任务中，RL 的价值传播机制有效解决了模仿学习中常见的复合误差问题，成功率提升最为显著。
*   **效率提升：** 相比 SFT，LWD 优化后的策略动作更果断，平均任务完成时间缩短了约 23.75 秒。
*   **分布式的价值：** 证明了分布价值学习（DIVL）在处理异构集群数据时比传统标量 RL 更稳健。

### 7. 优点与亮点
*   **闭环系统：** 真正实现了“边部署边学习”的工业级闭环，将机器人集群视为一个整体的学习系统。
*   **算法创新：** 成功将分布 RL（Distributional RL）与流匹配（Flow Matching）策略结合，解决了大型 VLA 模型难以进行 RL 微调的痛点。
*   **实战导向：** 实验全部在真实物理机器人上完成，而非仿真环境，任务复杂度（如功夫茶）极高。

### 8. 不足与局限
*   **安全建模缺失：** 框架目前未显式加入安全约束，在真实部署中可能存在碰撞或损坏风险。
