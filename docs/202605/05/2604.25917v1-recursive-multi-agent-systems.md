## 摘要
递归或循环语言模型最近作为一种新的扩展轴出现，通过在潜状态上迭代优化相同的模型计算来深化推理。我们将这种扩展原则从单一模型推广到多智能体系统，并提出疑问：智能体协作本身是否可以通过递归进行扩展？为此，我们引入了 RecursiveMAS，这是一个递归多智能体框架，将整个系统建模为统一的潜空间递归计算。RecursiveMAS 通过轻量级的 RecursiveLink 模块将异构智能体连接为协作循环，实现了分布内潜思维生成和跨智能体潜状态传输。为了优化该框架，我们开发了一种内外循环学习算法，通过跨递归轮次的共享基于梯度的信用分配，实现迭代式的全系统协同优化。对运行时间复杂度和学习动力学的理论分析表明，RecursiveMAS 比标准的基于文本的多智能体系统更高效，并在递归训练期间保持稳定的梯度。在实验方面，我们在 4 种代表性的智能体协作模式下实例化了 RecursiveMAS，并在涵盖数学、科学、医学、搜索和代码生成的 9 个基准测试中进行了评估。与先进的单/多智能体和递归计算基准相比，RecursiveMAS 始终提供 8.3% 的平均准确率提升，同时实现 1.2 倍至 2.4 倍的端到端推理加速，以及 34.6% 至 75.6% 的 Token 使用量减少。代码和数据已在 https://recursivemas.github.io 提供。

## Abstract
Recursive or looped language models have recently emerged as a new scaling axis by iteratively refining the same model computation over latent states to deepen reasoning. We extend such scaling principle from a single model to multi-agent systems, and ask: Can agent collaboration itself be scaled through recursion? To this end, we introduce RecursiveMAS, a recursive multi-agent framework that casts the entire system as a unified latent-space recursive computation. RecursiveMAS connects heterogeneous agents as a collaboration loop through the lightweight RecursiveLink module, enabling in-distribution latent thoughts generation and cross-agent latent state transfer. To optimize our framework, we develop an inner-outer loop learning algorithm for iterative whole-system co-optimization through shared gradient-based credit assignment across recursion rounds. Theoretical analyses of runtime complexity and learning dynamics establish that RecursiveMAS is more efficient than standard text-based MAS and maintains stable gradients during recursive training. Empirically, we instantiate RecursiveMAS under 4 representative agent collaboration patterns and evaluate across 9 benchmarks spanning mathematics, science, medicine, search, and code generation. In comparison with advanced single/multi-agent and recursive computation baselines, RecursiveMAS consistently delivers an average accuracy improvement of 8.3%, together with 1.2$\times$-2.4$\times$ end-to-end inference speedup, and 34.6%-75.6% token usage reduction. Code and Data are provided in https://recursivemas.github.io.

## 1. 核心问题与整体含义
*   **研究背景**：单体语言模型在处理复杂推理任务时常面临容量瓶颈。虽然多智能体系统（MAS）通过协作提升了性能，但传统的基于文本的交互方式存在三大痛点：**高推理延迟**（需反复解码/编码）、**Token 成本高昂**、以及**难以进行端到端优化**（文本交互阻断了梯度流）。
*   **核心问题**：能否将“递归计算”这一扩展轴从单模型引入多智能体系统，让异构智能体在**潜空间（Latent Space）**中进行循环协作与协同进化？
*   **整体含义**：论文提出了 **RecursiveMAS** 框架，旨在通过潜空间递归实现 MAS 的性能扩展（Scaling），同时大幅提升推理效率。

## 2. 方法论
*   **核心思想**：将整个多智能体系统视为一个统一的递归计算图，智能体之间不再传递文本，而是传递经过轻量级模块转换的潜状态（Hidden States）。
*   **关键技术细节**：
    *   **RecursiveLink 模块**：一个轻量级的两层残差投影网络。
        *   **Inner Link**：用于智能体内部，将最后一层的隐藏状态映射回输入空间，支持“潜思维（Latent Thoughts）”的自回归生成。
        *   **Outer Link**：用于跨智能体，将源智能体的潜状态转换为目标智能体（可能具有不同维度和架构）可理解的输入嵌入。
    *   **内外环学习算法（Inner-Outer Loop）**：
        *   **内环（模型级预热）**：并行训练每个智能体的 Inner Link，使其生成的潜思维在语义分布上与原始文本嵌入对齐。
        *   **外环（系统级优化）**：在全系统递归路径上进行端到端训练，通过跨轮次的梯度回传（BPTT 思想）优化 Outer Link，实现全系统的协同进化。
*   **理论支撑**：论文通过数学证明，RecursiveMAS 的运行复杂度远低于文本 MAS（用 $d_h^2$ 复杂度替代了 $|V|d_h$ 复杂度），且在递归训练中能保持稳定的梯度流，避免了文本交互导致的梯度消失。

## 3. 实验设计
*   **数据集与场景**：涵盖 9 个主流基准测试，分为四大类：
    *   **数学推理**：MATH500, AIME2025, AIME2026。
    *   **科学与医学**：GPQA-Diamond, MedQA。
    *   **代码生成**：LiveCodeBench-v6, MBPP Plus。
    *   **搜索问答**：HotpotQA, Bamboogle。
*   **协作模式（Benchmark 结构）**：验证了顺序型（Sequential）、混合型（Mixture）、专家-学习者蒸馏型（Distillation）和工具调用审议型（Deliberation）四种拓扑结构。
*   **对比方法**：
    *   **单智能体**：经过 LoRA 或全量微调（Full-SFT）的单体模型。
    *   **MAS 框架**：Mixture-of-Agents (MoA), TextGrad。
    *   **递归基准**：LoopLM（单模型递归）、Recursive-TextMAS（基于文本的递归协作）。

## 4. 资源与算力
*   **硬件环境**：实验使用了 **NVIDIA H100 和 A100 GPU**。
*   **训练配置**：采用 AdamW 优化器，学习率 5e-4，Batch size 为 4。
*   **成本分析**：论文明确对比了训练成本。RecursiveMAS 的可训练参数量极低（仅占总量的 0.31%），单次训练估算成本约为 **$4.27**，远低于全量微调（$9.67）。其 GPU 显存占用（15.29GB）也优于 LoRA 和 Full-SFT。

## 5. 实验数量与充分性
*   **实验规模**：论文在 9 个数据集上进行了多轮递归（r=1, 2, 3）的深度对比，并针对 4 种不同的 MAS 架构进行了泛化性测试。
*   **消融实验**：详细探讨了 RecursiveLink 的残差设计、潜思维长度 $m$ 的影响、以及训练轮次与推理轮次之间的 Scaling Law。
*   **客观性**：使用了多种异构模型（Qwen3/3.5, Llama-3, Gemma3, Mistral）作为骨干网络，确保结论不依赖于特定模型。通过 PCA 可视化展示了潜空间语义随递归轮次向 Ground Truth 靠拢的过程，增强了实验的说服力。

## 6. 主要结论与发现
*   **性能飞跃**：RecursiveMAS 在所有基准上平均准确率提升 **8.3%**，且性能随递归深度增加而持续增长。
*   **效率革命**：相比文本 MAS，实现了 **1.2x 至 2.4x** 的端到端推理加速，并减少了 **34.6% 至 75.6%** 的 Token 消耗。
*   **训练优势**：RecursiveLink 能够维持稳定的梯度，使得系统级优化比传统的文本反馈优化（如 TextGrad）更有效。
*   **Scaling Law**：发现增加训练递归轮次可以提升性能天花板，而增加推理递归轮次则能进一步压榨模型潜力。

## 7. 优点
*   **架构优雅**：通过轻量级模块解决了异构模型间“鸡同讲讲”的对齐问题。
*   **极高效率**：潜空间协作规避了文本生成的巨大开销，是目前 MAS 走向实用化的重要方向。
*   **理论与实践结合**：不仅有显著的实验提升，还提供了关于梯度稳定性和复杂度的严谨理论证明。

## 8. 不足与局限
*   **可解释性缺失**：由于智能体之间传递的是潜向量，人类无法直接观察中间协作过程，这在需要高度透明度的场景（如医疗诊断决策过程）中可能存在风险。
*   **对齐敏感性**：系统性能高度依赖于 RecursiveLink 的对齐质量，若基础模型分布差异过大，可能需要更多的训练数据来收敛。
*   **静态拓扑限制**：虽然支持多种模式，但实验中的 MAS 拓扑在推理时是预设好的，尚未实现动态自适应的递归路径选择。
