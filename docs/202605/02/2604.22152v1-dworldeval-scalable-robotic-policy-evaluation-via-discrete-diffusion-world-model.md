## 摘要
在数千个环境和数千个任务中评估机器人策略，对于现有方法而言是不可行的。这促使我们需要一种新的方法来进行可扩展的机器人策略评估。在本文中，我们提出了 dWorldEval，它使用离散扩散世界模型作为机器人策略的可扩展评估代理。具体而言，dWorldEval 将包括视觉、语言和机器人动作在内的所有模态映射到一个统一的 token 空间中，并通过单个基于 Transformer 的去噪网络对其进行建模。基于该架构，我们采用了稀疏关键帧记忆来维持时空一致性。我们还引入了一个进度 token，用于指示任务完成度。在推理时，该模型联合预测未来观测结果和进度 token，从而在进度达到 1 时自动判定任务成功。大量实验表明，dWorldEval 在 LIBERO、RoboTwin 以及多个真实机器人任务上显著优于以往的方法（如 WorldEval、Ctrl-World 和 WorldGym）。它为构建用于大规模机器人评估的世界模拟器开辟了新的架构范式。

## Abstract
Evaluating robotics policies across thousands of environments and thousands of tasks is infeasible with existing approaches. This motivates the need for a new methodology for scalable robotics policy evaluation. In this paper, we propose dWorldEval, which uses a discrete diffusion world model as a scalable evaluation proxy for robotics policies. Specifically, dWorldEval maps all modalities - including vision, language, and robotic actions - into a unified token space, modeling them via a single transformer-based denoising network. In this paper, we propose dWorldEval, using a discrete diffusion world model as a scalable evaluation proxy for robotics policy. Specifically, it maps all modalities, including vision, language, and robotics action into a unified token space, then denoises them with a single transformer network. Building on this architecture, we employ a sparse keyframe memory to maintain spatiotemporal consistency. We also introduce a progress token that indicates the degree of task completion. At inference, the model jointly predicts future observations and progress token, allowing automatically determine success when the progress reaches 1. Extensive experiments demonstrate that dWorldEval significantly outperforms previous approaches, i.e., WorldEval, Ctrl-World, and WorldGym, on LIBERO, RoboTwin, and multiple real-robot tasks. It paves the way for a new architectural paradigm in building world simulators for robotics evaluation at scale.

### 1. 核心问题与整体含义（研究动机和背景）
随着通用机器人操纵策略的快速发展，如何在成千上万的任务和环境中高效评估这些策略成为了重大挑战。传统的实机测试成本极高，而基于物理引擎的模拟器则面临资产建模繁重的问题。
*   **核心痛点**：现有的生成式世界模型（如基于视频扩散模型的方法）在评估机器人策略时不够可靠。主要原因在于：
    1.  **动作控制力弱**：动作通常作为辅助条件注入，容易被强大的视觉先验覆盖，导致模型在面对错误动作时“幻觉”出成功的结局。
    2.  **时空不一致性**：长程预测中物体容易变形或消失。
*   **研究目标**：构建一个可扩展、高保真、且能自动判定任务成功的机器人策略评估代理模型。

### 2. 方法论：核心思想与关键技术
dWorldEval 采用**掩码离散扩散（Masked Discrete Diffusion, MDD）**架构，将世界模型建模为一个统一的 Token 序列预测问题。
*   **统一 Token 空间**：将视觉（MAGVIT-v2）、语言（LLaDA）和连续动作块（FAST）全部映射为离散 Token。通过 Self-Attention，视觉 Token 可以直接对等地位地关注动作 Token，从而增强动作的可控性。
*   **稀疏关键帧记忆（Sparse Keyframe Memory）**：为了解决长程漂移，模型维护了一个滑动窗口的低分辨率历史关键帧序列，作为全局时空一致性的锚点。
*   **离散进度 Token（Discrete Progress Token）**：在 Token 序列末尾引入一个表示任务完成度（0.0 到 1.0）的 Token。该 Token 由 VLM（SEED-1.5VL）在训练集上自动标注，模型在推理时联合预测视觉结果和进度分值。
*   **自动成功检测**：当预测的进度 Token 达到 1.0 时，自动判定该策略执行成功，无需外部奖励函数或人工干预。

### 3. 实验设计
*   **数据集与场景**：
    1.  **LIBERO**：包含 Object, Spatial, Goal, 100 四个子集，涵盖大量桌面操纵任务。
    2.  **RoboTwin**：双臂机器人基准，涉及 10 个接触密集型任务（如堆叠、放置）。
    3.  **真实世界**：使用 AgileX 双臂平台，采集了 5.2k 条轨迹（含 1k 条失败样本），涵盖 5 类复杂任务。
*   **对比基准（Baselines）**：对比了当前主流的视频扩散世界模型：**WorldEval**、**WorldGym** 和 **Ctrl-World**。
*   **评估指标**：除了标准的 LPIPS，提出了 **$\Delta$-LPIPS**（测量状态转换的感知保真度），以及与真实执行成功率的 **Pearson 相关系数 (r)** 和 **秩违规率 (MMRV)**。

*   **硬件环境**：使用了 **8 台 NVIDIA H800 GPU**。
*   **训练细节**：在 8 台 H800 上训练了 15 个 epoch，采用 AdamW 优化器，全局 Batch Size 为 128。
*   **推理效率**：在单台 H800 上，评估一条完整轨迹大约需要 30-90 秒（约 1.5s/帧）。

### 5. 实验数量与充分性
*   **实验规模**：实验涵盖了仿真与现实、单臂与双臂、专家数据与失败数据。
*   **消融实验**：针对“稀疏记忆模块”、“动作控制力（动作洗牌实验）”、“进度 Token 的准确性”进行了详尽的消融研究。
*   **充分性评价**：实验设计非常充分且客观。通过引入失败轨迹训练和动作洗牌（Action Shuffling）测试，有力证明了模型不是在靠视觉先验“脑补”，而是真实地根据动作输入产生预测。

### 6. 主要结论与发现
*   **强相关性**：dWorldEval 估计的成功率与真实物理执行的成功率高度线性相关（Pearson r ≈ 0.9），能够准确对不同架构、不同训练阶段的策略进行排名。
*   **动作敏感性**：相比基准模型，dWorldEval 在处理分布外（OOD）或错误动作时表现更稳健，不会错误地将失败动作渲染为成功结果。
*   **时空稳定性**：稀疏记忆机制显著降低了长程预测中的视觉漂移。

### 7. 优点（亮点）
*   **架构创新**：摒弃了将动作作为辅助条件的传统做法，采用统一 Token 流，从底层架构上解决了动作控制力弱的问题。
*   **闭环评估**：实现了“想象”中的闭环交互，且自带进度评分，形成了一个完整的自动化评估闭环。
*   **诊断工具**：提出的 $\Delta$-LPIPS 为世界模型的可控性提供了一个有效的量化诊断指标。

### 8. 不足与局限
*   **计算开销**：尽管比实机测试快，但 1.5s/帧 的推理速度在面对超大规模策略池时仍有优化空间。
*   **VLM 依赖**：进度 Token 的训练依赖于预训练 VLM 的标注质量，若 VLM 对某些极端精细动作判断失误，会影响世界模型的评分准确性。
*   **物理精确度**：虽然视觉上一致，但作为基于学习的模型，其在处理极端复杂的流体或多体动力学时的物理精确性仍待进一步验证。
