## 摘要
递归或循环语言模型最近作为一种新的扩展轴出现，通过在潜状态上迭代优化相同的模型计算来深化推理。我们将这种扩展原则从单一模型扩展到多智能体系统，并探讨：智能体协作本身是否可以通过递归进行扩展？为此，我们引入了 RecursiveMAS，这是一个递归多智能体框架，将整个系统建模为统一的潜空间递归计算。RecursiveMAS 通过轻量级的 RecursiveLink 模块将异构智能体连接为协作循环，实现了分布内潜思维生成和跨智能体潜状态传输。为了优化我们的框架，我们开发了一种内外循环学习算法，通过跨递归轮次的共享基于梯度的信用分配，实现迭代式的全系统协同优化。对运行时间复杂度和学习动态的理论分析表明，RecursiveMAS 比标准的基于文本的多智能体系统（MAS）更高效，并在递归训练期间保持稳定的梯度。在实验方面，我们在 4 种具有代表性的智能体协作模式下实例化了 RecursiveMAS，并在涵盖数学、科学、医学、搜索和代码生成的 9 个基准测试中进行了评估。与先进的单/多智能体和递归计算基准相比，RecursiveMAS 始终提供 8.3% 的平均准确率提升，同时实现 1.2 倍至 2.4 倍的端到端推理加速，以及 34.6% 至 75.6% 的 Token 使用量减少。代码和数据已在 https://recursivemas.github.io 提供。

## Abstract
Recursive or looped language models have recently emerged as a new scaling axis by iteratively refining the same model computation over latent states to deepen reasoning. We extend such scaling principle from a single model to multi-agent systems, and ask: Can agent collaboration itself be scaled through recursion? To this end, we introduce RecursiveMAS, a recursive multi-agent framework that casts the entire system as a unified latent-space recursive computation. RecursiveMAS connects heterogeneous agents as a collaboration loop through the lightweight RecursiveLink module, enabling in-distribution latent thoughts generation and cross-agent latent state transfer. To optimize our framework, we develop an inner-outer loop learning algorithm for iterative whole-system co-optimization through shared gradient-based credit assignment across recursion rounds. Theoretical analyses of runtime complexity and learning dynamics establish that RecursiveMAS is more efficient than standard text-based MAS and maintains stable gradients during recursive training. Empirically, we instantiate RecursiveMAS under 4 representative agent collaboration patterns and evaluate across 9 benchmarks spanning mathematics, science, medicine, search, and code generation. In comparison with advanced single/multi-agent and recursive computation baselines, RecursiveMAS consistently delivers an average accuracy improvement of 8.3%, together with 1.2$\times$-2.4$\times$ end-to-end inference speedup, and 34.6%-75.6% token usage reduction. Code and Data are provided in https://recursivemas.github.io.

*   **背景**：单模型推理能力有限，多智能体系统虽能通过分工提升性能，但传统的基于文本（Text-based）的协作存在效率低下（需反复解码/编码）、梯度无法跨智能体有效传播、以及推理深度受限等问题。

### 2. 方法论：核心思想与关键技术
论文提出了 **RecursiveMAS** 框架，其核心思想是将异构智能体连接成一个闭环，在潜空间内进行信息的迭代优化。
*   **RecursiveLink 模块**：这是连接智能体的关键轻量级组件（两层残差投影模块）。
    *   **Inner Link（内环链接）**：在单个智能体内部，将最后一层的隐藏状态映射回输入空间，支持“潜思维（Latent Thoughts）”的自回归生成。
    *   **Outer Link（外环链接）**：在不同架构/尺寸的智能体之间进行分布对齐和维度转换，实现跨智能体的潜状态传输。
*   **内外环学习算法（Inner-Outer Loop Training）**：
    *   **内环训练**：对每个智能体进行预热，使其 Inner Link 能够生成符合语义分布的潜思维。
    *   **外环训练**：将整个 MAS 展开为递归路径，通过最终输出的交叉熵损失，利用梯度回传对所有 Outer Link 进行协同优化。
*   **理论支撑**：论文证明了 RecursiveMAS 在时间复杂度上优于文本 MAS（避免了 $|V|$ 维度的词表投影），且在学习动态上能保持梯度稳定，避免了文本交互导致的梯度消失。

### 6. 主要结论与发现
*   **性能提升**：RecursiveMAS 在所有基准上平均准确率提升 **8.3%**。
*   **效率优势**：相比文本递归 MAS，实现了 **1.2x - 2.4x** 的推理加速，并减少了 **34.6% - 75.6%** 的 Token 使用量。
*   **递归缩放律**：发现增加训练递归轮次和推理递归轮次均能提升性能，且两者具有协同效应。
*   **语义对齐**：可视化证明了随着递归轮次增加，生成的潜思维在语义上越来越接近真实答案的分布。

### 7. 优点与亮点
*   **架构创新**：首次将递归缩放原则从单模型扩展到异构多智能体系统。
*   **潜空间协作**：通过潜空间交互绕过了文本生成的效率瓶颈，并解决了端到端训练中的梯度断裂问题。
*   **通用性强**：不限制智能体类型和协作拓扑，能够无缝适配顺序、混合、蒸馏等多种 MAS 模式。
*   **高性价比**：仅需训练极少量的投影参数即可实现全系统协同进化。

### 8. 不足与局限
*   **黑盒性质**：潜空间协作虽然高效，但相比文本协作失去了可解释性，人类无法直接观察智能体中间轮次的“思考”内容。
*   **训练依赖**：虽然 RecursiveLink 很轻量，但仍需要针对特定任务的高质量数据进行内外环训练，无法像 Prompt-based MAS 那样完全即插即用。
*   **任务覆盖**：虽然涵盖了推理类任务，但对于需要极长文本生成的创造性写作等任务，潜空间递归的效果尚未验证。
