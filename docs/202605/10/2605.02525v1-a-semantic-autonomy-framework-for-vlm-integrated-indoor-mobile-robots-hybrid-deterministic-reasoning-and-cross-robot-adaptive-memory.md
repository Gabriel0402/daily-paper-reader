## 摘要
室内自主移动机器人可以使用 ROS 2 Navigation 2 等成熟框架可靠地导航到度量坐标，但它们缺乏解释表达意图而非位置的自然语言指令的能力。视觉语言模型（VLM）提供了弥合这一差距所需的语义推理，但其推理延迟（在消费级硬件上每次决策需 2-9 秒）和会话间的遗忘限制了实际部署。本文提出了语义自主栈（Semantic Autonomy Stack），这是一个用于语义自主室内导航的六层参考框架，并在配备现成边缘硬件的物理机器人上验证了一个包含混合确定性-VLM 推理和跨机器人自适应记忆的完整实例。一个七步参数解析器在不调用语言模型、摄像头或 GPU 的情况下，在 0.1 毫秒内处理了 88% 的指令；只有真正模糊的指令才会升级到 VLM 推理。一个具有明确范围分类（全局环境知识、特定操作员偏好、特定机器人能力）的五类语义记忆框架实现了跨会话学习和跨机器人知识迁移：通过一台机器人上的 VLM 交互学习到的偏好被提升为确定性解析，并通过共享的编译摘要迁移到第二台机器人，实现了实测 103,000 倍的延迟降低。在两台定制的差分驱动机器人上进行的 82 次场景级决策和三个会话的实验验证表明，语义迁移准确率达到 100%（33/33, 95% CI [0.894, 1.000]），语义解析准确率达到 100%，且具备并发多机器人运行的可行性——所有这些均在无板载 GPU 的 Raspberry Pi 5 平台上实现，且无需任何训练数据。

## Abstract
Autonomous indoor mobile robots can navigate reliably to metric coordinates using established frameworks such as ROS 2 Navigation 2, yet they lack the ability to interpret natural language instructions that express intent rather than positions. Vision-Language Models offer the semantic reasoning required to bridge this gap, but their inference latency (2-9 seconds per decision on consumer hardware) and session-by-session amnesia limit practical deployment. This paper presents the Semantic Autonomy Stack, a six-layer reference framework for semantically autonomous indoor navigation, and validates a complete instance featuring hybrid deterministic-VLM reasoning and cross-robot adaptive memory on physical robots with off-the-shelf edge hardware. A seven-step parametric resolver handles 88% of instructions in under 0.1 milliseconds without invoking a language model, camera, or GPU; only genuinely ambiguous instructions escalate to VLM reasoning. A five-category semantic memory framework with explicit scope taxonomy (global environment knowledge, per-operator preferences, per-robot capabilities) enables cross-session learning and cross-robot knowledge transfer: preferences learned through VLM interactions on one robot are promoted to deterministic resolution and transferred to a second robot via a shared compiled digest, achieving a measured latency reduction of 103,000-fold. Experimental validation on two custom-built differential-drive robots across 82 scenario-level decisions and three sessions demonstrates 100% semantic transfer accuracy (33/33, 95% CI [0.894, 1.000]), 100% semantic resolution accuracy, and concurrent multi-robot operation feasibility - all on Raspberry Pi 5 platforms with no onboard GPU, requiring zero training data.

#### 1. 核心问题与整体含义（研究动机和背景）
传统的室内移动机器人（如基于 ROS 2 Nav2 的系统）在度量空间导航（坐标到坐标）方面已非常成熟，但缺乏对人类自然语言意图（如“带我去可以休息的地方”）的理解能力。虽然视觉语言模型（VLM）能提供语义推理，但在实际部署中面临三大挑战：
*   **高推理延迟**：在消费级硬件上每次决策需 2-9 秒，难以满足实时性需求。
*   **会话遗忘（Amnesia）**：机器人无法记住之前的交互偏好，每次重启都需重新推理。
*   **跨机器人协作难**：缺乏一种机制将一台机器人学到的语义知识高效迁移到其他机器人。

#### 2. 方法论：核心思想与关键技术
论文提出了 **语义自主栈（Semantic Autonomy Stack, SAS）**，这是一个六层架构（L0-L5），核心思想是**双过程推理（Dual-process reasoning）**：

*   **混合推理架构（L3）**：
    *   **L3a（确定性快速解析）**：采用一个**七步参数化级联解析器**。它首先检查记忆中的偏好（Step 0），然后依次匹配节点 ID、名称、对象 ID、属性得分、类别及邻近度。该过程不调用 VLM，延迟极低（<0.1ms）。
    *   **L3b（VLM 语义推理）**：仅当 L3a 无法解析时，才激活 VLM（如 Qwen 3.5:4b）。系统会获取图像、构建包含环境拓扑和语义对象的 Prompt 进行深度推理。
*   **语义记忆框架（L5）**：
    *   将记忆分为五类（M1-M5）：环境统计、时间模式、**操作员偏好（M3）**、平台能力和任务历史。
    *   **知识提升机制**：当某个 VLM 推理结果在频率和一致性上达到阈值时，会被“提升”为确定性规则，存入共享的编译摘要（Digest）中。
*   **执行合约 $\langle A, O, V, L \rangle$**：通过能力清单、观测归一化、动作验证和审计日志，确保 VLM 的输出符合安全策略且可追溯。

#### 3. 实验设计与对比
*   **实验场景**：在大学教学楼走廊环境进行，包含 24 个节点、60 条边和 18 个静态语义对象（实验室、洗手间、植物等）。
*   **实验流程**：分为三个阶段：
    1.  **学习周期（Session A）**：机器人 Xplorer-C 通过 VLM 学习模糊指令（如“找个能坐下放松的地方”）。
    2.  **跨机器人迁移（Session B）**：将学到的知识通过 Digest 迁移到从未接触过该指令的 Xplorer-B。
    3.  **并发运行（Session C）**：验证多台机器人在共享记忆下的独立运行能力。
*   **对比基准**：主要对比了 **L3b（纯 VLM 推理）** 与 **L3a（基于记忆的确定性解析）** 在延迟和准确率上的差异。

#### 4. 资源与算力
*   **机器人端（边缘）**：使用 **Raspberry Pi 5 (16GB)**，无板载 GPU。负责导航（Nav2）、感知（YOLO26n CPU 推理）和局部语义地图构建。

*   **通信**：机器人与工作站通过 HTTP（获取上下文）和 ROS 2 DDS（发送动作指令）连接。

*   **充分性**：实验涵盖了从 VLM 推理到记忆提升、再到跨机器人迁移的完整闭环。通过统计学检验（如 Mann-Whitney U 检验、Fisher 精确检验）验证了结果的显著性。

#### 6. 主要结论与发现
*   **效率提升**：确定性解析器（L3a）处理了 **88%** 的指令，无需调用 VLM。
*   **延迟骤降**：通过记忆迁移，指令解析延迟从 VLM 的 6733ms 降至 0.065ms，**缩减了 103,000 倍**。
*   **高准确性**：跨机器人语义迁移准确率达到 **100% (33/33)**。

#### 7. 优点与亮点
*   **架构优雅**：SAS 六层模型解耦了感知、推理与执行，具有很强的平台移植性。
*   **实用性强**：通过“双过程推理”巧妙解决了 VLM 算力消耗大的痛点，适合工业和室内服务场景。
*   **零训练数据**：无需针对特定环境重新训练模型，依靠预训练 VLM 的零样本能力和在线记忆学习。

#### 8. 不足与局限
*   **环境单一**：实验仅在单一走廊环境进行，尚未在复杂多房间或动态室外环境验证。

*   **VLM 串行化**：在当前架构下，若多台机器人同时请求 VLM 推理，服务器端会存在排队延迟。
