## 摘要
利用大语言模型（LLM）丰富的世界知识来增强强化学习（RL）智能体，是通往通用智能的一条充满前景的路径。然而，一种根本性的“先验-动力学”不匹配阻碍了现有方法：静态的 LLM 知识无法直接适应长时程任务中复杂的转移动力学。将 LLM 先验作为固定策略使用会限制探索的多样性，因为先验对特定环境的动力学是盲目的；而端到端微调则面临优化不稳定和信用分配问题。为了弥补这一差距，我们提出了 PriorZero，这是一个统一的框架，通过解耦的“展开-训练”设计，将源自 LLM 的概念先验集成到基于世界模型的规划中。在展开阶段，一种新颖的根节点先验注入机制仅在蒙特卡洛树搜索（MCTS）的根节点引入 LLM 先验，使搜索集中在语义上有前景的动作上，同时保留世界模型的深度前瞻能力。在训练阶段，PriorZero 将世界模型学习与 LLM 适配解耦：世界模型在交互数据上不断优化，以共同提高其动力学、策略和价值预测；随后利用其价值估计，通过交替优化为稳定的 LLM 微调提供细粒度的信用分配信号。在包括 Jericho 中的文本冒险游戏和 BabyAI 中的指令遵循网格世界任务在内的多个基准测试上的实验表明，PriorZero 一致地提高了探索效率和渐近性能，为 LLM 赋能的决策制定建立了一个极具前景的框架。我们的代码可在 https://github.com/opendilab/LightZero 获取。

## Abstract
Leveraging the rich world knowledge of Large Language Models (LLMs) to enhance Reinforcement Learning (RL) agents offers a promising path toward general intelligence. However, a fundamental prior-dynamics mismatch hinders existing approaches: static LLM knowledge cannot directly adapt to the complex transition dynamics of long-horizon tasks. Using LLM priors as fixed policies limits exploration diversity, as the prior is blind to environment-specific dynamics; while end-to-end fine-tuning suffers from optimization instability and credit assignment issues. To bridge this gap, we propose PriorZero, a unified framework that integrates LLM-derived conceptual priors into world-model-based planning through a decoupled rollout-training design. During rollout, a novel root-prior injection mechanism incorporates LLM priors exclusively at the root node of Monte Carlo Tree Search (MCTS), focusing search on semantically promising actions while preserving the world model's deep lookahead capability. During training, PriorZero decouples world-model learning from LLM adaptation: the world model is continuously refined on interaction data to jointly improve its dynamics, policy, and value predictions, its value estimates are then leveraged to provide fine-grained credit assignment signals for stable LLM fine-tuning via alternating optimization. Experiments across diverse benchmarks, including text-based adventure games in Jericho and instruction-following gridworld tasks in BabyAI, demonstrate that PriorZero consistently improves both exploration efficiency and asymptotic performance, establishing a promising framework for LLM-empowered decision-making. Our code is available at https://github.com/opendilab/LightZero.

#### 1. 核心问题与研究动机
论文的核心问题在于解决**大语言模型（LLM）的静态语义先验与强化学习（RL）环境动态特性之间的不匹配（Prior-Dynamics Mismatch）**。
*   **背景**：LLM 拥有丰富的常识，但在长时程、奖励稀疏的任务中，LLM 往往表现短视（Knowing-Doing Gap），且无法直接理解特定环境的转移动力学。
*   **现有挑战**：直接将 LLM 作为策略（P1）缺乏环境适配；端到端微调 LLM（P2）面临高方差的信用分配问题；将 LLM 作为动力学模型（P3）存在 Token 级别与状态-动作级别之间的粒度差异。

#### 2. 方法论：PriorZero 框架
PriorZero 提出了一个解耦的“展开-训练”架构，将 LLM 的语义引导与世界模型的规划能力相结合：
*   **根节点先验注入（Root-Prior Injection）**：
    *   在蒙特卡洛树搜索（MCTS）过程中，仅在**根节点**融合 LLM 的语义先验 $\pi_{LLM}$ 和世界模型的策略 $\pi_{WM}$。
    *   **核心思想**：根节点对应真实的文本观测，LLM 能够理解；而搜索树的深层节点是世界模型生成的潜在状态（Latent States），没有直接的文本对应，因此深层节点完全依赖世界模型的动力学预测，避免了 LLM 引入系统性偏差。
*   **交替强化微调（Alternating Reinforcement Fine-Tuning）**：
    *   **解耦优化**：世界模型（WM）和 LLM 采用交替更新策略（比例通常为 10:1）。
    *   **低方差信用分配**：利用训练好的世界模型提供价值估计（Value Estimates），构建 $n$ 步自助法（Bootstrapped）优势信号（Advantage），通过 PPO 算法稳定地微调 LLM。
*   **结构化先验提取**：利用思维链（CoT）推理，让 LLM 在输出动作前先进行“中立情境分析”，减少决策偏见。

    2.  **BabyAI**：网格世界指令遵循任务（包含 18 个不同难度的关卡），侧重多任务泛化和空间推理。
*   **对比方法（Baseline）**：
    *   **UniZero**：不带 LLM 先验的基础世界模型。
    *   **P1 (Naive Policy)**：直接 Prompt LLM 输出动作。
    *   **P2 (Naive RLFT)**：端到端强化学习微调 LLM。
    *   **P3 (Dynamics Model)**：用 LLM 作为动力学骨干网络。
    *   **P4 (Text Encoder)**：将 LLM 作为冻结的特征提取器。

*   **消融实验**：非常充分，分别验证了交替训练策略、CoT 损失权重、模型规模（3B vs 7B）以及 MCTS 规划组件的必要性。
*   **客观性**：通过对比多种 LLM 知识迁移范式（P1-P4），客观地展示了 PriorZero 在收敛速度和最终性能上的优势。

#### 6. 主要结论与发现
*   **协同效应**：LLM 先验能显著缩小搜索空间，而世界模型的规划能纠正 LLM 的短视错误。
*   **微调的必要性**：冻结的 LLM 先验在后期会成为瓶颈，通过世界模型提供的优势信号进行微调，能使 LLM 先验与环境动态实现对齐。
*   **根节点注入的有效性**：仅在根节点注入先验既保留了语义引导，又维持了世界模型在潜在空间中模拟的忠实度。

#### 7. 优点
*   **架构优雅**：巧妙地解决了 LLM 无法直接处理潜在空间动力学的问题。

*   **可扩展性**：实验证明随着 LLM 规模增大（3B 到 7B），性能有进一步提升的潜力。

*   **动作空间限制**：目前主要针对离散动作空间（文本或网格动作），在连续控制任务中的表现尚未验证。

*   **环境依赖**：高度依赖于能够提供有效文本观测的环境，对于纯视觉且难以文本化的任务可能受限。
