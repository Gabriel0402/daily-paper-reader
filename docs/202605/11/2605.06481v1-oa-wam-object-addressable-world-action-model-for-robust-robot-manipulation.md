## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记或全局潜变量。当指令涉及特定对象时，动作解码器很难处理这些表示，特别是在场景偏移导致对象身份与上下文纠缠的情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并在块因果序列中与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码 16 步连续动作块。通过仅地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，从而强制实现可寻址性，在不添加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强大的 VLA 和 WAM 基准水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试产生的交换绑定余弦值为 0.87，而整体基准模型最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）通常将场景视为一个整体（如全局图像标记或潜变量）。这种“整体性”表示导致模型在面对场景偏移（如相机视角变化、物体布局改变）时，难以将指令中的特定物体与视觉背景解耦，从而导致动作预测失效。
*   **研究动机**：机器人操作本质上是对象级的任务。作者认为，鲁棒的操作需要一种“对象可寻址性”（Object Addressability），即模型应能稳定地定位并查询任务相关的物体，而不受周围环境或非相关属性变化的干扰。

### 2. 方法论：核心思想与关键技术
*   **核心思想**：将每一帧分解为 $N+1$ 个独立的对象槽位（Slots），并在架构层面强制实现“身份地址”与“时变内容”的分离。
*   **关键技术细节**：
    *   **槽位表示**：每个槽位向量 $s_k^t$ 由两部分组成：**地址向量（addr）**（由初始帧特征和语言标签计算，每集固定，代表“它是哪个物体”）和**内容向量（cnt）**（每帧更新，代表“它当前的状态”）。
    *   **对象可寻址注意力（OA Attention）**：在 Transformer 的每一层中，跨槽位注意力的键（Key）投影被限制为仅读取 `addr` 子向量。
    *   **残差重置（Residual Reset）**：在每个 Transformer 块之后，通过前向钩子将残差流中的地址部分重置为原始地址，防止身份信息在深层网络中与环境内容发生纠缠。
    *   **统一序列建模**：基于 7B 参数的 Chameleon 风格骨干网，融合了文本、图像 VQ、本体感受、过去动作和对象槽位五种流。
    *   **双头输出**：**世界头（World Head）**回归预测下一帧的槽位状态（内容和位姿）；**动作头（Action Head）**利用流匹配（Flow-matching）在单次前向传播中生成 16 步连续动作块。

### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准操作基准，包含 Spatial、Object、Goal、Long 四个子集。
    *   **SimplerEnv (WidowX)**：评估模拟与真实环境的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性测试，包含相机视角、机器人初始状态、布局、光照、背景、语言描述、传感器噪声 7 类扰动。
*   **对比方法**：
    *   **VLA 基准**：OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1, X-VLA 等。
    *   **WAM 基准**：WorldVLA, VLA-JEPA, GE-Act, Cosmos-Policy 等。
*   **特殊测试**：**因果槽位干预测试**。通过在推理时交换两个物体的地址向量，观察机器人末端执行器的轨迹是否随之偏移，以验证模型是否真正实现了对象级绑定。

*   **推理延迟**：在单张 A100 上，感知阶段（SAM3+DINOv3）约 138ms，骨干网+动作头约 95ms，总闭环控制频率约为 4.3 Hz。

### 5. 实验数量与充分性
*   **实验规模**：论文在三个主流基准上进行了详尽测试，每个实验均采用 3 个随机种子取平均值。
*   **消融实验**：设计了 A1-A4 四组消融实验，分别验证了 OA 约束的必要性、因果绑定的有效性、世界头的作用以及扰动一致性损失的影响。
*   **客观性与公平性**：对比了超过 15 种最新的 SOTA 模型，且在鲁棒性测试中严格遵守“仅在标准数据上训练，零样本迁移至扰动场景”的原则，实验设计非常严谨且具有说服力。

### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超越了现有最强模型。
*   **几何鲁棒性极强**：在 LIBERO-Plus 的几何轴（相机、机器人位姿、布局）上，OA-WAM 显著领先，Geo-Avg 达到 84.3%，比 $\pi_{0.5}$ 高出 4.8%。
*   **机制验证**：因果干预测试显示 OA-WAM 的交换绑定余弦值高达 **0.87**，而所有整体式基准模型均低于 0.09，证明了其确实通过地址空间实现了精确的物体选择。

### 7. 优点（亮点）
*   **架构创新**：通过简单的张量级掩码和重置操作（无需额外参数），在 Transformer 内部实现了硬性的对象寻址约束。
*   **解耦彻底**：成功将“操作哪个物体”与“物体当前状态”在计算图上分离，解决了 VLA 模型长期存在的场景纠缠问题。
*   **验证严谨**：不仅关注成功率，还通过因果干预实验深入探究了模型内部的决策机制。

### 8. 不足与局限
*   **感知依赖**：模型的表现高度依赖于上游感知系统（SAM3 和 DINOv3）。在传感器噪声轴上表现较差（-17.1%），主要是因为图像模糊导致槽位提取失败，而非策略本身问题。
*   **计算开销**：虽然骨干网推理较快，但前端感知（对象分割与特征提取）占据了总耗时的 60% 以上，限制了实时性。
*   **假设限制**：扰动一致性损失假设目标与干扰项之间是弱耦合的，在极度拥挤或物体间有复杂物理交互的场景下可能失效。
*   **环境限制**：目前仅在模拟器中验证，尚未展示在真实物理机器人上的部署效果。
