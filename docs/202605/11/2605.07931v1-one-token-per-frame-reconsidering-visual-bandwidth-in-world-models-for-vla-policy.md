## 摘要
视觉-语言-动作（VLA）模型日益依赖辅助世界模块进行长时程规划，然而如何在预训练 VLA 之上对这些模块进行参数化仍是一个开放的设计问题。现有的增强了世界模型的 VLA 通常以高视觉带宽将逐帧视觉流输入世界模块，并将其推演（rollout）视为动作预测的副产品；在冻结骨干网络且适应预算受限的情况下，这使得逐帧表示和潜在动作耦合都未得到充分研究。我们提出了 OneWM-VLA，它通过自适应注意力池化（Adaptive Attention Pooling）将每个视角压缩为每帧一个语义 Token，并在单一的流匹配（flow-matching）目标下生成结果潜在流和动作轨迹，而不是通过单独的解码器将它们连接。实验表明，在我们的设置下，逐帧视觉带宽可以减少到单个 Token 而不损害长时程性能。在 π₀ (2B) 骨干网络上使用 14.71M LoRA 参数进行训练，OneWM-VLA 在 MetaWorld MT50 上的平均成功率从 47.9% 提高到 61.3%，在 LIBERO-Long 上达到 95.6%（π₀ 为 85.2%），并在真实 Piper 机械臂的长时程可变形任务“折叠衣服”中达到 60.0%（π₀ 为 20.0%）。

## Abstract
Vision-language-action (VLA) models increasingly rely on auxiliary world modules to plan over long horizons, yet how such modules should be parameterized on top of a pretrained VLA remains an open design question. Existing world-model-augmented VLAs typically pass the per-frame visual stream into the world module at high visual bandwidth and treat its rollout as a side product of action prediction; under a constrained adaptation budget on a frozen backbone, this leaves both the per-frame representation and the latent action coupling under-examined. We introduce OneWM-VLA, which compresses each view into a single semantic token per frame through an Adaptive Attention Pooling, and produces the resulting latent stream and the action trajectory under a single flow-matching objective rather than connecting them through a separate decoder. Empirically, we find that per-frame visual bandwidth can be reduced to a single token without compromising long-horizon performance under our setup. Trained with 14.71M LoRA parameters on a $π_0$ (2B) backbone, OneWM-VLA improves the average success rate from 47.9% to 61.3% on MetaWorld~MT50, reaches 95.6% on LIBERO-Long (vs.85.2% for $π_0$), and reaches 60.0% on the long-horizon deformable task Fold Cloth on a real Piper arm (vs.20.0% for $π_0$).

*   **核心问题**：现有的“世界模型+VLA”方案通常采用高带宽视觉输入（每帧多个 Token），导致计算开销巨大，且在冻结骨干网络进行微调的场景下，视觉表征与动作生成的耦合不够紧密。

#### 2. 方法论
OneWM-VLA 提出了“瓶颈-推演耦合”（Bottleneck-Rollout Coupling）的核心思想：
*   **自适应注意力池化（Adaptive Attention Pooling）**：
    *   **多策略池化**：利用三种打分函数（MAX：峰值响应；SUM：总响应；LEARN：可学习 MLP）对 PaliGemma 提取的 256 个视觉 Token 进行筛选。
    *   **自适应融合**：通过可学习的权重将上述三种策略生成的 Token 融合成每帧仅有的**一个**语义 Token。这极大地降低了视觉带宽，使计算量不随规划时程线性爆炸。
*   **联合流匹配（Joint Flow Matching）**：
    *   将未来的潜在状态流（Latent Stream）与动作轨迹（Action Trajectory）置于统一的流匹配框架下。
    *   **统一目标函数**：模型同时预测动作和潜在状态的向量场，使潜在状态成为动作生成的结构化先验，而非独立的辅助输出。
    *   **推理过程**：在推理时，动作和潜在 Token 从高斯噪声开始同步去噪，确保环境动力学与机器人控制的高度对齐。

#### 6. 主要结论与发现
*   **单 Token 足够且更优**：在受限微调预算下，每帧 1 个 Token 的效果优于更多 Token。小尺寸潜在空间起到了隐式正则化作用，提升了泛化性。
*   **性能大幅提升**：OneWM-VLA 在 MetaWorld MT50 上的平均成功率提升了 13.3% 以上；在真实“折叠衣服”任务中，成功率从 $\pi_0$ 的 20% 提升至 60%。

*   **极高的效率**：通过极度压缩视觉带宽，解决了长程规划中的内存溢出（OOM）问题，显著提升了推理 FPS。
*   **优雅的设计**：将世界模型与策略模型通过联合流匹配深度融合，避免了传统方案中“预测归预测，控制归控制”的脱节现象。
*   **强悍的长程能力**：在处理复杂的柔性物体（如折叠衣服）时表现出极强的鲁棒性。
