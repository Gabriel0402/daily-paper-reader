## 摘要
基于大型多模态模型的多智能体系统（MAS）通过专业化智能体实现协作式复杂问题解决。然而，MAS 容易受到传染性越狱的影响，即攻击单个智能体可能会扩散到其他智能体，导致大规模沦陷。现有的防御措施通过训练一种更具传染性的治愈因子来应对，使智能体倾向于检索该因子而非病毒对抗样本（VirAEs）。然而，这会导致智能体响应的同质化，仅提供表层抑制而非真正的恢复。我们重新审视了这些防御措施，它们通过共享的治愈因子进行全局操作，而传染性越狱则源于局部交互行为。这种不匹配限制了它们的有效性。为此，我们提出了一种无需训练的前瞻引导局部净化（FLP）框架，其中每个智能体通过对未来交互进行推理来追踪行为演变并消除感染。具体而言，每个智能体在随后的对话轮次中模拟未来的行为轨迹。为了反映 MAS 中的多样性，我们引入了一种多角色模拟策略，以便在不同交互语境下进行鲁棒预测。然后，我们利用响应多样性作为诊断信号，通过分析基于角色的预测在检索结果和语义层面的不一致性来检测感染。对于受感染的智能体，我们应用局部净化：通过立即回滚相册来缓解近期感染，而长期感染则使用递归二分诊断（RBD）处理，该方法递归地划分图像相册并应用相同的诊断策略来定位和消除 VirAEs。实验表明，FLP 将最大累积感染率从 95% 以上降低到 5.47% 以下。此外，检索和语义指标与良性基准高度吻合，表明其有效地保留了交互多样性。

## Abstract
Large multimodal model-based Multi-Agent Systems (MASs) enable collaborative complex problem solving through specialized agents. However, MASs are vulnerable to infectious jailbreak, where compromising a single agent can spread to others, leading to widespread compromise. Existing defenses counter this by training a more contagious cure factor, biasing agents to retrieve it over virus adversarial examples (VirAEs). However, this homogenizes agent responses, providing only superficial suppression rather than true recovery. We revisit these defenses, which operate globally via a shared cure factor, while infectious jailbreak arise from localized interaction behaviors. This mismatch limits their effectiveness. To address this, we propose a training-free Foresight-Guided Local Purification (FLP) framework, where each agent reasons over future interactions to track behavioral evolution and eliminate infections. Specifically, each agent simulates future behavioral trajectories over subsequent chat rounds. To reflect diversity in MASs, we introduce a multi-persona simulation strategy for robust prediction across interaction contexts. We then use response diversity as a diagnostic signal to detect infection by analyzing inconsistencies across persona-based predictions at both retrieval-result and semantic levels. For infected agents, we apply localized purification: recent infections are mitigated via immediate album rollback, while long-term infections are handled using Recursive Binary Diagnosis (RBD), which recursively partitions the image album and applies the same diagnosis strategy to localize and eliminate VirAEs. Experiments show that FLP reduces the maximum cumulative infection rate from over 95% to below 5.47%. Moreover, retrieval and semantic metrics closely match benign baselines, indicating effective preservation of interaction diversity.

*   **研究动机**：
    *   **现有防御的局限性**：目前的防御手段（如 Cowpox）通常采用全局共享的“治愈因子”，强制智能体优先检索治愈因子而非病毒。这虽然能抑制恶意输出，但会导致智能体响应的**同质化**，丧失了 MAS 原有的交互多样性，且属于“表面抑制”而非“彻底清除”。
    *   **局部与全局的错位**：传染性越狱源于局部的交互行为，而现有防御多从全局介入，缺乏对局部感染动态的精准追踪。

### 2. 论文提出的方法论：FLP 框架
论文提出了一种**无需训练**的**前瞻引导局部净化（Foresight-Guided Local Purification, FLP）**框架，核心思想是让智能体在正式交互前进行“自我预演”，从而识别并清除感染源。

*   **核心技术细节**：
    1.  **多角色模拟（Multi-Persona Simulation）**：智能体从预定义的角色池中实例化多个不同的角色（Persona），在内部模拟多轮“计划-检索-提问-回答”的交互。其逻辑是：良性智能体在不同角色下应表现出多样化的响应，而受感染智能体会因 VirAE 的诱导而表现出行为坍缩（即无论什么角色都输出相似的恶意内容）。
    2.  **感染诊断（Infection Diagnosis）**：
        *   **检索熵（Retrieval Entropy）**：量化检索结果的集中程度。如果智能体反复检索同一个图像（VirAE），熵值会显著降低。
        *   **语义多样性（Semantic Diversity）**：计算不同角色生成文本的余弦相似度。恶意行为通常会导致语义空间的高度聚集。
        *   当两个指标均低于设定阈值时，判定智能体已被感染。
    3.  **局部净化（Localized Purification）**：
        *   **近期感染**：通过计算**语义漂移（Semantic Drift）**识别。若近期行为发生剧烈变化，则执行“相册回滚”，删除最近的记录。
        *   **长期感染**：采用**递归二分诊断（Recursive Binary Diagnosis, RBD）**。将相册递归地划分为子集并进行模拟诊断，像二分查找一样精准定位并剔除隐藏在相册深处的 VirAE。

### 6. 论文的主要结论与发现
*   **高效防御**：FLP 能将 MAS 的最大累积感染率从 95% 以上降低到 **5.47% 以下**，在多数情况下甚至能实现 0% 的感染。

*   **前瞻性优势**：通过内部模拟，智能体能够在恶意信息传播给其他智能体**之前**就将其拦截（Pre-propagation Intervention）。

### 7. 优点（亮点）
*   **范式创新**：从“全局治愈”转向“局部前瞻净化”，解决了防御导致的响应同质化问题。
*   **无需训练**：作为一种推理侧防御，它不需要昂贵的重新训练或微调，易于部署到现有的 MAS 框架中。
*   **精准定位**：提出的 RBD 算法能够像“手术刀”一样从海量历史数据中精准剔除对抗样本，而非盲目清空存储。

### 8. 不足与局限
*   **推理开销**：虽然无需训练，但内部多角色模拟会显著增加推理时的计算延迟（文中提到约 2-4 倍的时间开销），这在实时性要求极高的场景下可能受限。
*   **模态限制**：实验主要集中在多模态（图像诱导）越狱，对于纯文本越狱或其他复杂任务类型的覆盖相对较少。
*   **角色池依赖**：防御效果可能受到预定义角色池（Persona Pool）质量的影响，若角色多样性不足，诊断的准确率可能会下降。
