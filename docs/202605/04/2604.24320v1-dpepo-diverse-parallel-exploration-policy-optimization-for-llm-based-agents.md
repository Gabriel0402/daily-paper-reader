## 摘要
遵循序列化“推理-行动”（reason-then-act）范式的大语言模型（LLM）智能体在许多复杂任务中取得了卓越的性能。然而，由于这些方法每步仅与单个环境交互，因此面临探索受限和环境理解不完整的问题。在本文中，我们首先引入了一种新颖的范式，使智能体能够同时与多个环境交互并共享跨轨迹经验。基于这一范式，我们进一步提出了 DPEPO，这是一种鼓励智能体进行多样化并行探索的强化学习（RL）算法。DPEPO 包含两个阶段：初始的有监督微调（SFT）赋予智能体基础的并行推理和动作生成能力，随后是采用分层奖励方案的强化学习阶段。我们设计了一个并行轨迹级成功奖励和两个步级奖励：多样化动作奖励（Diverse Action Reward）和多样化状态转移奖励（Diverse State Transition Reward），旨在主动惩罚行为冗余并促进广泛探索。在 ALFWorld 和 ScienceWorld 上的广泛实验表明，DPEPO 达到了最先进的（SOTA）成功率，同时保持了与强序列基线相当的效率。（代码见 https://github.com/LePanda026/Code-for-DPEPO）

## Abstract
Large language model (LLM) agents that follow the sequential "reason-then-act" paradigm have achieved superior performance in many complex tasks.However, these methods suffer from limited exploration and incomplete environmental understanding, as they interact with only a single environment per step. In this paper, we first introduce a novel paradigm that enables an agent to interact with multiple environments simultaneously and share cross-trajectory experiences. Building upon this paradigm, we further propose DPEPO, a reinforcement learning (RL) algorithm that encourages the agent to perform diverse parallel exploration. There are two stages in DPEPO: initial supervised fine-tuning (SFT) imparts basic parallel reasoning and action generation, followed by reinforcement learning stage with a hierarchical reward scheme. We design a parallel trajectory-level success reward and two step-level rewards: Diverse Action Reward and Diverse State Transition Reward, which actively penalize behavioral redundancy and promote broad exploration. Extensive experiments on ALFWorld and ScienceWorld show that DPEPO achieves state-of-the-art (SOTA) success rates, while maintaining comparable efficiency to strong sequential baselines. ### 1. 论文的核心问题与整体含义
*   **研究动机**：现有的 LLM 智能体大多遵循 **ReAct**（推理-行动）范式，这是一种线性的、序列化的交互方式。智能体每一步只能执行一个动作并观察一个反馈，导致其对环境的理解是片面且带有偏见的。
*   **核心问题**：在复杂、长程的任务中，单路径探索容易陷入局部最优或因信息不足而失败。简单的多样本采样（Multi-sampling）又存在动作趋同（缺乏多样性）和效率低下（轨迹间经验不共享）的问题。
*   **整体含义**：论文提出了一种**并行探索范式**，允许智能体同时与多个环境副本交互，并通过强化学习算法 **DPEPO** 鼓励探索的多样性，从而构建更全面的环境认知。

### 2. 方法论：核心思想与关键技术
DPEPO 的核心思想是将“并行交互”与“多样性激励”相结合，分为两个阶段：
*   **阶段一：冷启动（SFT）**
    *   通过少量人工标注和模型合成的并行探索轨迹，对 LLM 进行有监督微调，使其掌握基础的并行推理格式（使用特定的 `<parallel>` 和 `<env_i>` 标签）。
*   **阶段二：多样化并行策略优化（RL）**
    *   **并行轨迹级成功奖励**：只要 $K$ 个并行环境中有任何一个达到目标状态，即判定为成功并给予奖励。
    *   **并行步级多样性奖励（核心创新）**：
        1.  **多样化动作奖励 (DAR)**：同时惩罚“深度冗余”（在同一个环境中重复无效动作）和“广度冗余”（在不同并行环境中执行相同的动作）。
        2.  **多样化状态转移奖励 (DTR)**：惩罚重复的“状态-动作”转换对，强制智能体尝试未曾探索过的路径。
    *   **策略优化算法**：改进自 **GRPO**（组相对策略优化）。它取消了 Critic 模型，通过组内多个轨迹的相对得分来估计优势函数（Advantage），并结合轨迹级和步级的层次化信号进行更新。

### 5. 实验数量与充分性
*   **实验规模**：在两个主流基准测试的所有任务切片（Seen/Unseen, L0/L1/L2）上进行了全面测试。

### 6. 主要结论与发现
*   **性能领先**：DPEPO 在 ALFWorld 达到 **98.2%** 的成功率，在 ScienceWorld 达到 **61.4%**，显著超过了 DeepSeek-R1 和其他 RL 智能体方法。
*   **效率优势**：尽管是并行探索，但由于智能体能更快找到正确路径，其**平均轨迹步数更短**。在推理时间上，由于并行动作是同步执行的，DPEPO 的实际耗时比序列化的 DeepSeek-R1 更短。
*   **涌现行为**：观察到智能体学会了“结构化搜索”（例如在不同环境分别检查柜子、抽屉和桌面）和“假设验证”等高级探索策略。

### 7. 优点（亮点）
*   **范式创新**：打破了传统的单线 ReAct 模式，提出了更符合人类“多路尝试”逻辑的并行探索范式。
*   **奖励设计精巧**：通过 DAR 和 DTR 有效解决了强化学习中常见的“行为塌缩”和“探索冗余”问题。
*   **高性价比**：相比于大规模采样，DPEPO 在提升性能的同时，通过精简的 Prompt 设计和并行机制，控制了 Token 成本和训练时间。

### 8. 不足与局限
*   **环境依赖性**：该方法依赖于能够被“并行实例化”的环境。在现实世界的具身智能体（如实体机器人）场景中，很难同时开启多个完全一致的物理副本。
*   **Token 峰值压力**：虽然平均效率高，但在单步交互中，由于需要同时处理多个环境的反馈，瞬时的上下文长度（Context Length）会显著增加，对模型的长文本处理能力有一定要求。
*   **应用限制**：目前主要在文本交互环境中验证，在多模态（视觉）环境下的泛化能力尚待进一步研究。
