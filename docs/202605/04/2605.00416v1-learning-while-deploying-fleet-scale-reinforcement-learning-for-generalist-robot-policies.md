## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变化以及人类纠错机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略开始，通过利用在机器人车队中收集的自主运行和人类干预，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，用于在基于流的 VLA 动作生成器中进行策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和 3-5 分钟的长程任务。随着车队经验的积累，单一通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

*   **研究动机**：尽管视觉-语言-动作（VLA）模型在离线预训练中表现出色，但在真实部署中仍面临分布偏移（Distribution Shift）、长尾失败和环境多样性等挑战。
*   **核心问题**：如何利用机器人机群（Fleet）在部署过程中产生的真实交互数据（包括成功、失败、部分进展和人类干预），实现通用策略的持续进化，而非仅仅将其视为静态的测试过程。
*   **整体含义**：提出了 **LWD (Learning While Deploying)** 框架，将部署转化为数据飞轮的起点，通过离线到在线的强化学习（RL）闭环，使通用策略能够随使用规模的扩大而自动增强。

### 2. 方法论
LWD 结合了分布价值学习与高效的策略提取技术，主要包含两个核心算法组件：
*   **分布隐式价值学习 (DIVL)**：
    *   **核心思想**：改进了隐式 Q 学习（IQL）。由于机群数据具有高度异构性（不同策略版本、不同场景），同一状态-动作对的回报可能是多峰或重尾分布的。
    *   **技术细节**：DIVL 不再估计标量期望值，而是学习回报的**分类分布（Categorical Distribution）**。它使用分位数（Quantile）作为自举（Bootstrap）目标，通过调节分位数参数 $\tau$ 来平衡乐观与保守。
    *   **自适应策略**：引入基于分布熵（Entropy）的自适应 $\tau$ 机制。当价值估计不确定（熵高）时，降低 $\tau$ 以减少过估计；当确定时，提高 $\tau$ 以追求更高回报。
*   **基于伴随匹配的 Q 学习 (QAM)**：
    *   **核心思想**：解决在基于流匹配（Flow Matching）的 VLA 模型中进行策略提取的稳定性问题。
    *   **技术细节**：直接对多步去噪过程进行梯度回传既昂贵又不稳定。QAM 将 Critic 提供的动作梯度转化为流模型每一步的局部回归目标，从而在不破坏生成模型表达能力的前提下，利用 Critic 指引策略改进。
*   **训练流程**：分为离线预训练（在包含演示、历史运行和人类引导失败数据的离线缓冲池上训练）和在线持续后训练（在机群实时回传的混合数据上训练）。

### 6. 主要结论与发现

### 7. 优点
*   **系统性闭环**：成功构建了一个从机群部署到云端学习再到重新部署的完整“数据飞轮”系统。
*   **算法创新**：DIVL 和 QAM 的结合解决了 VLA 模型在强化学习微调中的稳定性和异构数据利用问题。
*   **实战价值**：在极高难度的长程任务（如功夫茶）上展示了极高的成功率，证明了该框架处理复杂现实逻辑的能力。

### 8. 不足与局限
*   **安全建模缺失**：目前的框架未显式加入执行安全约束，在更敏感的环境中部署可能存在风险。
*   **语言理解深度**：长程任务仍依赖相对简单的指令，对于需要复杂视觉-语言推理的动态任务分解能力尚待进一步验证。

*   **算力透明度**：未详细披露后端训练服务器的具体配置，难以评估中小规模团队复现该系统的成本。
