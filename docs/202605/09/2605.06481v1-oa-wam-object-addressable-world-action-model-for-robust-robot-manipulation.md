## 摘要
世界动作模型（WAMs）通过联合预测场景演变和机器人动作来增强视觉-语言-动作（VLA）策略，但现有方法通常将预测的世界表示为整体图像、视频标记（tokens）或全局潜变量。当指令涉及特定对象时，动作解码器很难处理这些表示，尤其是在场景偏移导致对象身份与上下文纠缠的情况下。我们提出了 OA-WAM，一种用于鲁棒机器人操作的对象可寻址世界动作模型。OA-WAM 将每一帧分解为 N+1 个槽位（slot）状态，包括一个机器人槽位和 N 个对象槽位。每个槽位包含一个持久的地址向量和一个随时间变化的内容向量，并以块因果序列的形式与文本、图像、本体感受和过去动作标记融合。世界头预测下一帧的槽位状态，而流匹配动作头在同一次前向传播中解码 16 步连续动作块。通过仅通过地址键路由跨槽位注意力，并在每个 Transformer 层重置地址切片，强制实现可寻址性，从而在不增加额外标记的情况下，将“操作哪个对象”与“该对象当前是什么”分离开来。OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到了强 VLA 和 WAM 基线的水平，在最相关的 LIBERO-Plus 几何轴上达到了最先进的性能，并在七轴综合评估中保持竞争力。因果槽位干预测试产生的交换绑定余弦值为 0.87，而整体基线最高仅为 0.09。这些结果表明，可寻址的对象状态为场景扰动下的鲁棒世界动作建模提供了一个有效的接口。

## Abstract
World Action Models (WAMs) enhance Vision-Language-Action policies by jointly predicting scene evolution and robot actions, but existing methods usually represent the predicted world as holistic images, video tokens, or global latents. These representations are difficult for an action decoder to address when an instruction refers to a particular object, especially under scene shifts where object identity is entangled with context. We propose OA-WAM, an Object-Addressable World Action Model for robust robot manipulation. OA-WAM decomposes each frame into N+1 slot states, with one robot slot and N object slots. Each slot contains a persistent address vector and a time-varying content vector, and is fused with text, image, proprioception, and past-action tokens in a block-causal sequence. A world head predicts next-frame slot states, while a flow-matching action head decodes a 16-step continuous action chunk in the same forward pass. Addressability is enforced by routing cross-slot attention through address-only keys and resetting the address slice at every transformer layer, separating which object to act on from what that object currently is without adding extra tokens. OA-WAM matches strong VLA and WAM baselines on LIBERO (97.8%) and SimplerEnv (79.3%), reaches state-of-the-art performance on the most relevant LIBERO-Plus geometric axes, and remains competitive on the seven-axis aggregate. A causal slot-intervention test yields a swap-binding cosine of 0.87, versus at most 0.09 for holistic baselines. These results suggest that addressable object states provide an effective interface for robust world-action modeling under scene perturbations.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的视觉-语言-动作（VLA）模型和世界动作模型（WAM）通常将场景表示为整体图像、视频标记流或全局潜变量。这种“整体性”表示导致模型在处理复杂指令时，难以将特定物体与周围环境解耦。
*   **研究动机**：在面对场景偏移（如相机视角变化、背景更换、物体布局调整）时，虽然目标物体依然可见，但整体表示会将物体身份与上下文纠缠在一起，导致动作解码器无法准确“寻址”目标，从而引发操作失败。
*   **整体含义**：论文提出了**对象可寻址性（Object Addressability）**的概念，旨在通过架构设计强制模型在处理物体时，将“它是哪个物体（身份地址）”与“它当前状态如何（内容/姿态）”在张量层面进行分离，从而提升机器人操作的鲁棒性。

### 2. 方法论：核心思想与关键技术
*   **核心思想**：将每一帧分解为 $N+1$ 个独立的对象槽位（Slots），并引入架构约束，使得模型在推理过程中必须通过固定的“地址”来定位物体。
*   **关键技术细节**：
    *   **槽位表示**：每个槽位标记由两部分拼接而成：`addr`（身份地址向量，由初始帧和指令确定，每集固定）和 `content`（时变内容向量，每帧更新）。
    *   **对象可寻址注意力（OA Attention）**：在 Transformer 的每一层中，跨槽位注意力的 **Key（键）** 投影仅读取 `addr` 部分，而 **Value（值）** 和 **Query（查询）** 保持完整。
    *   **地址重置钩子（Reset Hook）**：在每层残差连接后，强制将地址切片重置为初始值，防止身份信息在深层网络中被上下文污染。
    *   **统一序列建模**：采用类似 Chameleon 的 7B 参数骨干网络，将文本、图像 VQ 码、槽位标记、本体感受和过去动作组装成块因果序列。
    *   **双头预测**：
        *   **世界头（World Head）**：回归预测下一帧各槽位的内容和姿态。
        *   **动作头（Action Head）**：基于流匹配（Flow-matching）在单次前向传播中生成 16 步连续动作块。

### 3. 实验设计
*   **数据集与场景**：
    *   **LIBERO**：标准基准，测试分布内（ID）的成功率。
    *   **SimplerEnv (WidowX)**：测试模拟到真实的视觉匹配能力。
    *   **LIBERO-Plus**：核心鲁棒性基准，包含相机、机器人初始状态、布局、光照、背景、语言、传感器噪声 7 个扰动轴。
*   **对比方法**：
    *   **VLA 基线**：OpenVLA, $\pi_0$, $\pi_{0.5}$, InternVLA-M1, X-VLA 等。
    *   **WAM 基线**：WorldVLA, VLA-JEPA, GE-Act, Cosmos-Policy 等。

### 4. 资源与算力
*   **预训练（Stage 0）**：使用 **384 张 NVIDIA A100-80GB GPU**，训练时长约 **18 天**（总计约 16.6 万 A100 小时），处理了 2.5T tokens。
*   **对齐与微调（Stage I & II）**：在 **8 张 A100** 上进行，每个阶段耗时约 **3-4 天**。
*   **推理延迟**：在单张 A100 上，感知部分（SAM3+DINOv3）约 138ms，骨干网络+动作头约 90ms，总闭环频率约为 4.3 Hz。

### 5. 实验数量与充分性
*   **实验规模**：论文在三个主流基准上进行了详尽测试，每个任务均运行 3 个随机种子并取平均值。
*   **消融实验**：设计了 A1（对象可寻址性隔离）、A3（世界头作用）、A4（干扰项一致性）等多组消融，验证了 OA 约束是提升鲁棒性的关键。
*   **因果干预测试（A2）**：通过在测试时交换两个物体的 `addr` 向量，观察机器人是否会转向错误的物体。这一实验极具说服力，证明了模型确实学会了通过地址进行寻址。
*   **客观性**：对比了几乎所有最新的 SOTA 模型，并使用了官方的扰动生成器，实验设计严谨、公平。

### 6. 主要结论与发现
*   **性能领先**：OA-WAM 在 LIBERO (97.8%) 和 SimplerEnv (79.3%) 上达到或超过了现有 SOTA 水平。
*   **鲁棒性飞跃**：在 LIBERO-Plus 的几何扰动轴（相机、机器人状态、布局）上，OA-WAM 的几何平均成功率比 $\pi_{0.5}$ 高出 **4.8%**。
*   **因果绑定**：在槽位交换干预中，OA-WAM 的绑定余弦值达到 **0.87**，而所有整体式基线均低于 **0.09**，证明了其独特的对象级控制能力。

### 7. 优点
*   **架构创新**：通过简单的张量掩码和重置钩子实现了强力的归纳偏置，无需复杂的端到端语义保证。
*   **可解释性与可干预性**：提供了一个明确的对象级接口，允许通过操作 `addr` 向量来直接干预机器人的目标选择。
*   **鲁棒性强**：有效地解决了 VLA 模型在场景几何变化时容易“迷失目标”的顽疾。

### 8. 不足与局限
*   **感知瓶颈**：高度依赖上游的感知模型（SAM3, DINOv3, Qwen3-VL）。如果感知层未能正确提取物体或姿态，策略层无法纠正。
*   **传感器噪声敏感**：在 LIBERO-Plus 的“传感器噪声”轴上表现较差（比基线低 17.1%），主要是因为图像噪声破坏了槽位提取阶段。
*   **计算开销**：虽然骨干网络推理快，但前端感知（尤其是 SAM3 和多视图处理）带来了显著的延迟，限制了更高频率的实时控制。
*   **环境限制**：目前主要在模拟器中验证，尚未展示大规模真机部署的复杂长程任务结果。
