## 摘要
在多摄像头自动驾驶中，同时感知透视图中的 2D 物体和鸟瞰图（BEV）中的 3D 物体具有挑战性。现有的两阶段流水线仅将 2D 结果作为 3D 检测的一次性线索。我们提出了 SimPB++，它可以同时从多个摄像头中检测透视图中的 2D 物体和 BEV 中的 3D 物体。它通过混合解码器架构将这两项任务统一到一个端到端模型中，并以交互方式耦合多视图 2D 和 3D 解码器。两个新颖的模块实现了深度交互：动态查询分配（Dynamic Query Allocation）自适应地将 2D 查询分配给 3D 候选目标，而自适应查询聚合（Adaptive Query Aggregation）利用多视图 2D 特征优化 3D 表示，形成循环的 3D-2D-3D 优化过程。对于多视图 2D 检测，我们使用查询组注意力（Query-group Attention）进行组内通信。我们还设计了一种用于远距离感知的裁剪与缩放（Crop-and-Scale）策略，以及一种带有辅助 RoI 检测器的传播去噪（Propagating Denoising）策略。SimPB++ 支持仅 2D 标注和全标注数据的混合监督，减少了对昂贵 3D 标签的依赖。实验表明，该方法在 nuScenes 数据集的两项任务上均达到了最先进的性能，并在 Argoverse2 上实现了强大的远距离检测（可达 150 米）。

## Abstract
Simultaneous perception of 2D objects in perspective view and 3D objects in Bird's Eye View (BEV) is challenging for multi-camera autonomous driving. Existing two-stage pipelines use 2D results only as a one-time cue for 3D detection. We propose SimPB++, which simultaneously detects 2D objects in perspective and 3D objects in BEV from multiple cameras. It unifies both tasks into an end-to-end model with a hybrid decoder architecture, coupling multi-view 2D and 3D decoders interactively. Two novel modules enable deep interaction: Dynamic Query Allocation adaptively assigns 2D queries to 3D candidates, and Adaptive Query Aggregation refines 3D representations using multi-view 2D features, forming a cyclic 3D-2D-3D refinement. For multi-view 2D detection, we use Query-group Attention for intra-group communication. We also design a Crop-and-Scale strategy for long-range perception and a Propagating Denoising strategy with an auxiliary RoI detector. SimPB++ supports mixed supervision with 2D-only and fully annotated data, reducing reliance on expensive 3D labels. Experiments show state-of-the-art performance on nuScenes for both tasks and strong long-range detection (up to 150m) on Argoverse2.

### 1. 论文的核心问题与整体含义
在自动驾驶的多摄像头感知系统中，同时获取透视图（Perspective View）中的 2D 检测和鸟瞰图（BEV）中的 3D 检测至关重要。
*   **研究动机**：现有的方法通常将 2D 和 3D 检测视为分离的任务，或者采用两阶段流水线（2D 检测结果仅作为 3D 查询的单次初始化线索）。这种设计导致 2D 和 3D 特征之间缺乏深度交互，且 2D 检测在不同相机间相互独立，无法形成对物体的全局理解。
*   **核心目标**：提出一个统一的查询驱动（Query-based）框架 **SimPB++**，通过单一模型同时输出 2D 和 3D 结果，并实现 3D-2D-3D 的循环特征增强。

### 2. 论文提出的方法论
SimPB++ 采用了一种基于 DETR 风格的端到端架构，其核心技术包括：
*   **混合解码器（Hybrid Decoder）**：每一层解码器都耦合了一个多视图 2D 解码器和一个 3D 解码器。
*   **动态查询分配（Dynamic Query Allocation）**：利用相机内外参将 3D 锚点（Anchors）投影到各个图像平面。如果 3D 锚点在某个视图中可见，则为其分配对应的 2D 查询。这建立了 3D 到 2D 的显式关联。
*   **自适应查询聚合（Adaptive Query Aggregation）**：将经过图像特征细化后的 2D 查询重新聚合回 3D 查询空间。该模块能识别物体是否被截断，并据此调整权重，从而实现 3D 特征的循环细化。
*   **裁剪与缩放策略（Crop-and-Scale）**：针对远距离小目标，模型自适应地对原始高分辨率图像的特定区域进行裁剪并缩放至输入尺寸，生成额外的“远距离视图”，通过动态采样获取更精细的特征。
*   **传播去噪（Propagating Denoising）**：在 2D 和 3D 查询之间同步传播噪声锚点，加速模型收敛并增强特征对齐。
*   **混合监督训练**：支持同时使用“全标注（2D+3D）”和“仅 2D 标注”的数据进行训练，有效利用低成本标签提升 3D 检测性能。

### 6. 主要结论与发现
*   **性能领先**：SimPB++ 在 nuScenes 验证集上显著超越了之前的 SOTA 模型（如在 ResNet101 下达到 0.639 NDS）。
*   **长距离优势**：在 Argoverse2 上，SimPB++ 在 120-150 米范围内的检测能力大幅领先于专门针对长距离设计的 Far3D。
*   **协同效应**：2D 检测任务不仅是副产品，其特征反馈能显著提升 3D 定位的准确性（尤其是方位角误差 mAOE 的降低）。
*   **成本降低**：通过混合监督，仅需少量 3D 标注结合大量 2D 标注即可达到接近全 3D 标注的性能。

### 7. 优点
*   **架构优雅**：将 2D 和 3D 检测任务在查询层面深度耦合，避免了传统两阶段方法的信息丢失。
*   **循环优化**：3D-2D-3D 的设计比单向的 2D-to-3D 关联更具鲁棒性，能更好地处理跨相机视图的物体。
*   **长距离感知**：Crop-and-Scale 策略在不显著增加计算量的前提下，有效解决了远距离像素稀疏的问题。
*   **实用性强**：支持混合标签训练，对工业界降低标注成本具有直接意义。

### 8. 不足与局限
*   **相机参数依赖**：动态查询分配高度依赖于精确的相机内外参，在相机抖动或校准偏移的情况下，性能可能会受影响。
*   **计算开销**：虽然是稀疏框架，但引入额外的 2D 解码层和辅助分支（训练阶段）会增加显存占用和训练时间。
