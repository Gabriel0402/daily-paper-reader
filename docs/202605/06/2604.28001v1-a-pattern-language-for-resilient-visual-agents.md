## 摘要
将多模态基础模型集成到企业生态系统中面临着根本性的软件架构挑战。架构师必须平衡相互竞争的质量属性：视觉语言动作（VLA）模型的高延迟和非确定性，与企业控制回路所要求的严格确定性和实时性能。在本研究中，我们为视觉智能体提出了一种架构模式语言，旨在将快速、确定性的反射与缓慢、概率性的监督相分离。该语言由四种架构设计模式组成：(1) 混合可负担性集成，(2) 自适应视觉锚定，(3) 视觉层级合成，以及 (4) 语义场景图。

## Abstract
Integrating multimodal foundation models into enterprise ecosystems presents a fundamental software architecture challenge. Architects must balance competing quality attributes: the high latency and non-determinism of vision language action (VLA) models versus the strict determinism and real-time performance required by enterprise control loops. In this study, we propose an architectural pattern language for visual agents that separates fast, deterministic reflexes from slow, probabilistic supervision. It consists of four architectural design patterns: (1) Hybrid Affordance Integration, (2) Adaptive Visual Anchoring, (3) Visual Hierarchy Synthesis, and (4) Semantic Scene Graph.

### 2. 论文提出的方法论：核心思想与关键技术
论文借鉴了行为机器人学（Subsumption Architecture）和认知心理学（双系统理论），提出了一套包含四种架构模式的语言：

*   **核心思想**：**摊销推理（Amortized Inference）**。将高成本的概率性推理（System 2）仅用于修复和指导低成本的确定性反射（System 1），从而在保持弹性的同时降低平均延迟和成本。
*   **四种关键架构模式**：
    1.  **混合可负担性集成（Hybrid Affordance Integration）**：多模态感知融合层。并行运行目标检测（定位组件）和 OCR（提取文本），通过几何融合解决感知冲突，消除“感知幻觉”。
    2.  **自适应视觉锚定（Adaptive Visual Anchoring）**：建立 System 1（本地缓存/模板匹配，<50ms）与 System 2（VLA 监督者）的边界。当 System 1 置信度低于阈值时，触发 System 2 进行语义重定位并更新缓存。
    3.  **视觉层级合成（Visual Hierarchy Synthesis）**：将扁平的感知结果（坐标点）转化为具有层级关系的 UI 树（如表单、表格、模态框），解决绝对坐标失效后的相对寻址问题。
    4.  **语义场景图（Semantic Scene Graph）**：构建显式的、可查询的世界模型。在执行高风险操作前进行逻辑验证，提升系统的可解释性和安全性。
