## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变化以及人类干预机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略开始，通过利用在机器人车队中收集的自主运行和人类干预数据，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与用于流式 VLA 动作生成器中策略提取的伴随匹配 Q 学习（QAM）相结合。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和时长 3-5 分钟的长程任务。随着车队经验的积累，单个通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

### 论文总结：边部署边学习（Learning While Deploying, LWD）

#### 1. 核心问题与整体含义
论文针对通用机器人策略（如 VLA 模型）在真实世界部署中的局限性提出方案。虽然大规模离线预训练赋予了机器人基础能力，但在实际应用中，机器人常面临**分布偏移（Distribution Shift）**、**长尾失败场景**以及**任务变体**。传统的“训练后部署”模式无法处理这些动态变化。
**核心动机**：构建一个“数据飞轮”，使机器人能够从部署过程中的自主尝试和人类干预中持续学习，实现车队规模（Fleet-scale）的离线到在线强化学习（RL）闭环。

#### 2. 方法论：LWD 框架
LWD 结合了分布价值学习与高效的策略提取技术，主要包含两个关键组件：

*   **分布隐式价值学习（DIVL, Distributional Implicit Value Learning）**：
    *   **核心思想**：改进自隐式 Q 学习（IQL）。由于机群数据具有异构性（包含成功、失败、人类干预），同一状态-动作对的收益可能是多峰或重尾分布的。
    *   **技术细节**：DIVL 不学习标量价值，而是学习收益的**分类分布（Categorical Distribution）**。它使用分位数（Quantile）作为自举（Bootstrap）目标，既保留了 IQL 不查询分布外动作的稳定性，又能捕捉高收益的模式。
    *   **自适应 $\tau$ 策略**：根据价值分布的**归一化熵**动态调整分位数参数 $\tau$。当分布不确定性高（熵大）时，采取保守策略；当确定性高时，采取激进（高 $\tau$）的改进目标。

*   **伴随匹配 Q 学习（QAM, Q-learning with Adjoint Matching）**：
    *   **核心思想**：解决流匹配（Flow Matching）策略在 RL 中的优化难题。
    *   **技术细节**：直接通过多步去噪过程对 VLA 策略进行梯度回传既不稳定又耗时。QAM 将 Critic 提供的动作梯度转化为流模型每一步的局部回归目标，从而稳定地将价值提升转化为策略改进。

*   **算法流程**：
    1.  **阶段 1（离线）**：在包含演示、历史运行和人类引导失败数据的离线缓冲区上预训练策略、Critic 和价值模型。
    2.  **阶段 2（在线）**：在 16 台机器人上部署策略，收集自主运行和干预数据，存入在线缓冲区。
    3.  **持续更新**：中心化学习器利用离线与在线混合数据（约 1:1 比例）持续更新模型，并定期向机群同步新策略。

#### 3. 实验设计
*   **场景与任务**：共 8 项真实世界任务。
    *   **杂货补货（4项）**：平铺货架补货、错位纠正、冷柜补货、冷藏箱处理。侧重语义泛化。
    *   **长时程任务（4项）**：功夫茶（泡茶）、制作果汁、调制鸡尾酒、鞋盒打包。单次任务时长 3-5 分钟，包含 5-8 个子步骤。
*   **机器人平台**：16 台 Agibot G1 双臂人形/类人操纵平台。
*   **对比基准（Baselines）**：
    *   **SFT**：仅使用演示数据进行监督微调。
    *   **RECAP**：一种基于优势加权的离线 RL 改进方法。
    *   **HG-DAgger**：交互式模仿学习，利用人类干预数据。
    *   **LWD (Offline)**：仅使用离线 RL 阶段的模型。

#### 4. 资源与算力
*   **模型架构**：
    *   **策略网络**：基于 $\pi_{0.5}$ 架构，使用 PaliGemma (Gemma-2B + SigLIP) 作为骨干，Gemma-300M 作为动作专家。
    *   **价值/Critic 网络**：使用 Gemma 3-270M-IT 和 SigLIP-So400M。
*   **算力实现**：使用多主机 SPMD JAX 程序进行分布式训练。
*   **训练时长**：在线训练阶段，每个实验分配 **4 小时墙钟时间**，通过 16 台机器人并行，相当于累积了约 **60 小时**的真实交互数据。

#### 5. 实验数量与充分性
*   **实验规模**：在 16 台机器人上进行了大规模验证，这在机器人 RL 论文中属于较高规格。
*   **消融实验**：
    *   对比了 DIVL 与标量期望回归（Expectile Regression），证明了分布建模在长时程任务上的显著优势（提升 16.7%）。
    *   验证了自适应 $\tau$ 策略对训练稳定性的贡献。

#### 6. 主要结论与发现
*   **性能卓越**：LWD 在所有任务中达到 **95% 的平均成功率**，显著优于 SFT (76%) 和其他基准。
*   **长时程任务优势**：在 3-5 分钟的任务中，LWD 的提升最为明显，因为它能通过 TD 误差将奖励信号传播到早期决策中，而模仿学习易受复合误差影响。
*   **效率提升**：相比 SFT，LWD 训练出的策略执行更果断，平均循环时间（Cycle Time）缩短了约 23.75 秒。
*   **数据飞轮效应**：证明了通过机群部署收集的异构数据（包括失败和干预）可以有效转化为策略的鲁棒性。

#### 7. 优点
*   **系统性创新**：成功闭合了大规模机器人机群从部署到学习的环路，具有极强的工程实践意义。
*   **算法适配性**：DIVL 和 QAM 的结合巧妙解决了 VLA 模型（尤其是 Flow-based 策略）在强化学习中梯度难传、数据异构的痛点。
*   **真实感强**：完全基于真实世界机器人实验，任务复杂度高（如功夫茶），证明了方案的实用性。

#### 8. 不足与局限
*   **安全建模缺失**：目前的框架未显式考虑执行过程中的安全性约束。
*   **指令简单化**：长时程任务仍依赖单一的高级语言指令，缺乏对复杂视觉-语言推理的深度挖掘。
*   **更新策略**：在线更新采用简单的实时调度，对于更大规模（如成千上万台）的机群，可能需要更复杂的异步同步机制。
*   **算力细节模糊**：未披露具体的硬件成本，难以评估该方案的经济性。
