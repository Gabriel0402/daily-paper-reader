## 摘要
视觉-语言-动作（VLA）模型在具身操纵中展现出了卓越的能力和泛化性。然而，它们的决策依赖于一种快速、本能的过程，缺乏深思熟虑。在面对需要更多考量的复杂或模糊场景时，这种策略往往会导致次优或灾难性的动作。在本文中，我们介绍了 VLA-ATTC，这是一个赋予 VLA 模型自适应测试时计算（TTC）能力的框架。VLA-ATTC 采用了一种基于不确定性的“认知离合器”，在必要时动态地从反射式执行过渡到 TTC 审议阶段。在 TTC 阶段，一种新型的相对动作评论家（RAC）模型通过成对比较从生成的候选动作中识别出最佳动作。这种相对机制取代了不稳定的绝对值估计，显著简化了学习目标。此外，我们还引入了一种高效的采样策略来分摊计算成本，以及一个无需人工标注即可整理偏好对的自动化数据流水线。在 LIBERO-LONG 基准测试中，VLA-ATTC 将 SOTA 模型 PI0.5 的失败率降低了 50% 以上。我们将开源所有代码和权重。

## Abstract
Vision-Language-Action (VLA) models have demonstrated remarkable capabilities and generalization in embodied manipulation. However, their decision-making relies on a fast, instinctive process that lacks deliberation. This strategy often leads to suboptimal or catastrophic actions when facing complex or ambiguous scenarios that require greater consideration. In this paper, we introduce \textbf{VLA-ATTC}, a framework that endows VLA models with adaptive test-time compute (TTC). VLA-ATTC employs an uncertainty-based ``cognitive clutch'' to dynamically transition from reflexive execution to a TTC deliberation phase when necessary. During TTC phase, a novel \textbf{Relative Action Critic} (RAC) model identifies the optimal action from generated candidates via pairwise comparisons. This relative mechanism replaces unstable absolute value estimation, significantly simplifying the learning objective. Furthermore, we introduce an efficient sampling strategy to amortize computational costs and an automated data pipeline that curates preference pairs without manual annotation. On the LIBERO-LONG benchmark, VLA-ATTC reduces the failure rate of the SOTA model PI0.5 by over 50\%. We will open-source all the code and weights.

目前的 VLA 模型（如 PI0, OpenVLA）主要依赖“直觉式”的快速推理（类似人类的系统 1），在面对简单任务时表现良好，但在复杂、模糊或关键决策点时，由于缺乏“深思熟虑”（系统 2 推理），容易产生次优甚至灾难性的动作。
*   **核心挑战**：如何在不显著增加推理延迟的前提下，为 VLA 模型引入类似“思考”的审议过程？
*   **研究目标**：开发一种自适应机制，仅在必要时触发额外的计算资源进行动作优化。

### 2. 方法论：核心思想与关键技术
VLA-ATTC 框架由三个核心组件构成：
*   **认知离合器（Cognitive Clutch）**：
    *   **原理**：利用不确定性作为触发器。在每个时间步，共享 VLM 的上下文特征，但使用不同随机种子生成两个动作块。
    *   **度量**：使用动态时间规整（DTW）计算两个动作序列之间的距离。如果距离超过阈值 $\tau$，则认为模型不确定，触发 TTC 审议阶段；否则直接执行。
*   **相对动作评论家（Relative Action Critic, RAC）**：
    *   **核心思想**：放弃难以训练的“绝对评分”，转而采用“成对比较”（A 是否优于 B）。
    *   **架构**：一个轻量级的 Transformer 结构，通过“多分支注意力”同时融合 VLM 的原始特征、蒸馏后的查询特征（Query Features）以及待比较的动作对。
    *   **锦标赛机制**：在 TTC 阶段并行生成 $N$ 个候选动作，通过 RAC 进行多轮两两对决，最终选出最优动作。
*   **自动化数据流水线**：
    *   利用 Flow-matching 模型的特性，通过调整 ODE 积分步数（步数多则质量高，步数少则质量低）自动生成“优/劣”动作对，无需人工标注即可训练 RAC。

*   **性能飞跃**：在 LIBERO-LONG 上，VLA-ATTC 将 PI0.5 的失败率降低了 50% 以上；在真实世界任务中，成功率平均提升了 17.3%。

### 7. 优点
*   **即插即用**：无需对原始 VLA 基座模型进行微调，保持了基座模型的泛化能力。
*   **自适应性**：通过“认知离合器”实现了计算资源的按需分配，解决了测试时计算缩放的效率瓶颈。
*   **鲁棒的评价机制**：RAC 的相对比较机制比绝对价值函数更容易学习且更稳定。
*   **全自动闭环**：从数据生成到模型训练再到推理优化，实现了高度自动化。

### 8. 不足与局限
*   **阈值敏感性**：虽然文中给出了 K 百分位设置方法，但在全新环境下可能仍需少量离线数据来确定最优的触发阈值 $\tau$。

*   **任务覆盖**：虽然在操纵任务上表现优异，但在需要极高实时反应（如高速避障）或极长逻辑推理（如多步规划）的任务中，该框架的增益尚未得到验证。
