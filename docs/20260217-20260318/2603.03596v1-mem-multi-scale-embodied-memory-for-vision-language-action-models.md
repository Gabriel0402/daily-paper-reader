## 摘要
传统上，端到端机器人学习中的记忆涉及将一系列过去的观测输入到学习到的策略中。然而，在复杂的现实世界多阶段任务中，机器人的记忆必须在多个粒度级别上表示过去的事件：从捕捉抽象语义概念的长期记忆（例如，正在做饭的机器人应该记住食谱的哪些阶段已经完成），到捕捉近期事件并补偿遮挡的短期记忆（例如，当机器人的手臂遮挡住物体时，它仍能记住想要拾取的物体）。在这项工作中，我们的核心见解是，用于长时程机器人控制的有效记忆架构应该结合多种模态，以捕捉这些不同层次的抽象。我们引入了多尺度具身记忆（MEM），这是一种用于机器人策略中混合模态长时程记忆的方法。MEM 将通过视频编码器压缩的基于视频的短时程记忆与基于文本的长期记忆相结合。它们共同使机器人策略能够执行长达 15 分钟的任务，例如清理厨房或制作烤奶酪三明治。此外，我们发现记忆使 MEM 策略能够智能地在上下文中调整操作策略。

## Abstract
Conventionally, memory in end-to-end robotic learning involves inputting a sequence of past observations into the learned policy. However, in complex multi-stage real-world tasks, the robot's memory must represent past events at multiple levels of granularity: from long-term memory that captures abstracted semantic concepts (e.g., a robot cooking dinner should remember which stages of the recipe are already done) to short-term memory that captures recent events and compensates for occlusions (e.g., a robot remembering the object it wants to pick up once its arm occludes it). In this work, our main insight is that an effective memory architecture for long-horizon robotic control should combine multiple modalities to capture these different levels of abstraction. We introduce Multi-Scale Embodied Memory (MEM), an approach for mixed-modal long-horizon memory in robot policies. MEM combines video-based short-horizon memory, compressed via a video encoder, with text-based long-horizon memory. Together, they enable robot policies to perform tasks that span up to fifteen minutes, like cleaning up a kitchen, or preparing a grilled cheese sandwich. Additionally, we find that memory enables MEM policies to intelligently adapt manipulation strategies in-context.

### 2. 方法论
核心思想是将记忆分解为两种模态和尺度，并集成到 $\pi_{0.6}$ VLA 模型中：
*   **短期视频记忆（Video Memory）**：

    *   **关键技术**：在预训练的 ViT 每一层中交替使用“空间注意力”和“因果时间注意力”（Space-Time Separable Attention）。
    *   **压缩机制**：在编码器的高层丢弃过去帧的 Token，仅保留当前帧的表示，从而在不增加 VLA 主干网络负担的情况下注入时间上下文。
*   **长期语言记忆（Language Memory）**：
    *   将过去发生的语义事件总结为自然语言。
    *   **更新机制**：高层策略（$\pi_{HL}$）根据当前观察、任务目标和旧记忆 $m_t$ 生成新的子任务指令 $l_{t+1}$ 和更新后的记忆 $m_{t+1}$。
    *   **数据生成**：利用大语言模型（LLM）对训练数据中的子任务序列进行压缩和摘要，去除冗余信息，减少训练与推理间的分布偏移。

### 6. 主要结论与发现
*   **多尺度必要性**：单一模态（仅视频或仅语言）均无法完美解决长时程任务。视频记忆对动作纠错至关重要，语言记忆对跨分钟的任务逻辑至关重要。
*   **上下文自适应**：MEM 赋予了机器人“从失败中学习”的能力。例如，如果第一次抓取筷子失败，记忆能让机器人意识到高度偏差并调整策略，而无记忆模型会陷入重复失败。
*   **预训练的重要性**：在多样化视频数据上预训练记忆模块，比仅在目标任务上微调记忆效果显著更好。
*   **压缩的价值**：通过 LLM 压缩语言记忆能有效缓解因推理时多次尝试失败导致的“分布偏移”问题。

### 7. 优点
*   **架构优雅**：视频编码器设计巧妙，不增加额外参数即可复用 ViT 权重，且推理效率极高。
*   **突破时程限制**：成功演示了长达 15 分钟的复杂机器人任务，这在端到端 VLA 领域属于领先水平。
*   **多模态融合**：将视觉的“精细度”与语言的“抽象性”结合，符合人类处理复杂任务的认知逻辑。

### 8. 不足与局限
*   **标注依赖**：长期语言记忆的训练依赖于高质量的子任务语言标注，这在缺乏标注的大规模数据集中可能难以扩展。
*   **因果混淆风险**：虽然论文称多样化数据缓解了此问题，但记忆模型在模仿学习中仍存在“机械重复过去动作”的潜在风险。
*   **实时性挑战**：尽管优化了编码器，但随着输入摄像头数量增加（如文中用到4个），处理极长视频上下文的计算开销依然是实时控制的瓶颈。
