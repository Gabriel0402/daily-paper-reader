## Abstract
Vision-language-action (VLA) models for closed-loop robot control are typically cast under the Markov assumption, making them prone to errors on tasks requiring historical context. To incorporate memory, existing VLAs either retrieve from a memory bank, which can be misled by distractors, or extend the frame window, whose fixed horizon still limits long-term retention. In this paper, we introduce ReMem-VLA, a Recurrent Memory VLA model equipped with two sets of learnable queries: frame-level recurrent memory queries for propagating information across consecutive frames to support short-term memory, and chunk-level recurrent memory queries for carrying context across temporal chunks for long-term memory. These queries are trained end-to-end to aggregate and maintain relevant context over time, implicitly guiding the model's decisions without additional training or inference cost. Furthermore, to enhance visual memory, we introduce Past Observation Prediction as an auxiliary training objective. Through extensive memory-centric simulation and real-world robot experiments, we demonstrate that ReMem-VLA exhibits strong memory capabilities across multiple dimensions, including spatial, sequential, episodic, temporal, and visual memory. ReMem-VLA significantly outperforms memory-free VLA baselines $π$0.5 and OpenVLA-OFT and surpasses MemoryVLA on memory-dependent tasks by a large margin.

### 论文总结：ReMem-VLA：通过双层循环查询为视觉-语言-动作模型赋能记忆能力

#### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言-动作（VLA）模型大多遵循**马尔可夫假设**，即仅根据当前或极短时间的观察来预测动作。这导致机器人在处理需要历史上下文的任务（如“将杯子放回原处”或执行多步长程任务）时经常失败。现有的记忆增强方案（如记忆库检索或扩展观察窗口）存在易受干扰项影响、固定窗口限制长时记忆等缺陷。本文旨在为 VLA 模型引入一种能够同时处理短期和长期历史信息的统一记忆机制。

#### 2. 方法论：核心思想与关键技术
ReMem-VLA 的核心是引入了**双层循环查询（Dual-Level Recurrent Queries）**机制：
*   **双层循环查询结构**：
    *   **帧级循环查询（Frame-level）**：每一帧更新一次，通过指数移动平均（EMA）传播信息，捕捉短期上下文（如维持姿态、近期物体状态）。
    *   **块级循环查询（Chunk-level）**：每隔 $k$ 帧（一个动作块）更新一次，更新频率较低，用于保留长期任务进度和初始环境配置。
*   **无梯度循环路径**：为了克服截断反向传播（TBPTT）在长序列训练中的局限性，模型采用固定的、无梯度的 EMA 更新路径。这意味着模型学习的是“往记忆里写什么”，而记忆的传播方式是确定的，从而保证了长程信息的稳定性。
*   **过去观测预测（POP）**：引入辅助训练目标，要求模型根据当前状态重建过去的 RGB 图像，以此增强视觉记忆能力。
*   **槽位式流式训练（Slot-based Streaming Training）**：提出一种新的批处理范式，通过维护固定数量的槽位来并行处理不同长度的片段，确保循环状态在整个片段内连续积累，避免了跨片段的状态泄露。

#### 3. 实验设计
*   **仿真实验**：使用 **MemoryBench** 基准测试（包含放回方块、重排方块、重开抽屉），并额外设计了一个超过 600 帧的**长程任务（Long Horizon Task）**。
*   **真实世界实验**：设计了四类任务，分别测试：
    1.  **时间记忆**：浇花（需持续 6 秒）。
    2.  **情节记忆**：舀两勺米（需计数并停止）。
    3.  **序列记忆**：按顺序按压不同颜色的按钮。
    4.  **视觉记忆**：将水果放回初始位置。
*   **对比基准**：对比了 OpenVLA-OFT、$\pi 0.5$（无记忆基准）以及 MemoryVLA（基于检索的记忆模型）。

#### 4. 资源与算力
*   **训练硬件**：使用了 **8 台 NVIDIA A100 GPU**。
*   **训练细节**：总 Batch Size 为 64，训练步数为 150k 步。
*   **推理硬件**：在单张 **RTX 4090 GPU** 上部署，推理频率约为 10Hz（通过重用动作块可达 20Hz）。
*   **模型规模**：基于 Qwen2-VL-2B 骨干网络。

#### 5. 实验数量与充分性
*   **实验规模**：仿真实验中每个任务进行 100 次测试，真实世界实验中每个任务进行 50 次测试。
*   **消融实验**：非常充分。作者分别验证了：
    *   双层查询的必要性（对比仅帧级、仅块级、无记忆）。
    *   POP 辅助任务对视觉记忆的提升。
    *   无梯度路径与可学习路径（GRU/MLP）的优劣对比。
    *   EMA 超参数、查询数量、更新间隔的影响。
*   **客观性**：通过在不同维度（空间、时间、序列等）的记忆任务上进行测试，证明了方法的全面性。

#### 6. 主要结论与发现
*   **性能领先**：ReMem-VLA 在 MemoryBench 上的平均成功率达到 94.5%，远超 $\pi 0.5$（8.25%）和 MemoryVLA（1.5%）。
*   **长程优势**：在长程任务中，双层循环机制表现出极强的稳定性，而传统 VLA 几乎无法完成。
*   **无梯度路径的关键性**：实验发现，在 TBPTT 训练下，使用可学习的循环参数（如 GRU）反而会破坏长时记忆，而固定的 EMA 路径效果最好。
*   **视觉记忆增强**：POP 任务显著提升了机器人找回初始位置的能力（成功率从 34% 提升至 82%）。

#### 7. 优点
*   **架构优雅**：通过简单的查询机制实现了复杂的长短期记忆融合，无需复杂的外部记忆库。
*   **训练高效**：槽位式流式训练解决了循环神经网络在变长序列上的批处理难题。
*   **端到端**：记忆的提取和利用是端到端学习的，无需人工定义记忆特征。
*   **实用性强**：在真实机器人上验证了多种记忆维度，证明了其在复杂现实场景下的潜力。

#### 8. 不足与局限
*   **泛化性风险**：目前主要在特定任务数据集上训练，尚未在 Open-X-Embodiment 等大规模机器人数据集上进行预训练，其通用泛化能力有待验证。
*   **推理开销**：虽然在 4090 上能跑，但相比于纯马尔可夫模型，维护循环状态和多层 Transformer Connector 增加了一定的计算负担。
*   **生命周期限制**：目前仅针对单个任务片段（Episode）内的记忆，尚不支持跨天、跨 session 的超长时（Lifelong）记忆。
