## 摘要
视觉-语言-动作（VLA）模型通过利用大视觉语言模型（VLM）共同解释指令和视觉输入，取得了重大突破。然而，视觉 Token 的大幅增加，特别是来自多视图输入的 Token，给实时机器人操作带来了严峻挑战。现有的 VLM 加速技术（如 Token 剪枝）在直接应用于 VLA 模型时，往往会导致性能下降，因为它们忽略了不同视图之间的关系，且未能考虑到机器人操作的动态性和任务特定特征。为此，我们提出了 BFA++，这是一个专门为 VLA 模型设计的动态 Token 剪枝框架。BFA++ 引入了一种由两级重要性预测器引导的分层剪枝策略：视图内预测器突出显示每张图像中与任务相关的区域以抑制空间噪声，而视图间预测器则在不同的操作阶段识别关键的摄像机视图以减少跨视图冗余。这种设计在保留基本视觉线索的同时实现了高效的 Token 选择，从而提高了计算效率和操作成功率。在 RoboTwin 基准测试和真实世界机器人任务上的评估表明，BFA++ 一致优于现有方法。BFA++ 在 π0 和 RDT 模型上的成功率均提高了约 10%，分别实现了 1.8 倍和 1.5 倍的加速。我们的结果强调，上下文敏感且任务感知的 Token 剪枝比全视觉处理更有效，能够在真实世界的机器人系统中实现更快的推理和更高的操作精度。

## Abstract
Vision-Language-Action (VLA) models have achieved significant breakthroughs by leveraging Large Vision Language Models (VLMs) to jointly interpret instructions and visual inputs. However, the substantial increase in visual tokens, particularly from multi-view inputs, poses serious challenges to real-time robotic manipulation. Existing acceleration techniques for VLMs, such as token pruning, often result in degraded performance when directly applied to VLA models, as they overlook the relationships between different views and fail to account for the dynamic and task-specific characteristics of robotic operation. To address this, we propose BFA++, a dynamic token pruning framework designed specifically for VLA models. BFA++ introduces a hierarchical pruning strategy guided by two-level importance predictors: an intra-view predictor highlights task-relevant regions within each image to suppress spatial noise, while an inter-view predictor identifies critical camera views throughout different manipulation phases to reduce cross-view redundancy. This design enables efficient token selection while preserving essential visual cues, resulting in improved computational efficiency and higher manipulation success rates. Evaluations on the RoboTwin benchmark and real-world robotic tasks demonstrate that BFA++ consistently outperforms existing methods. BFA++ improves the success rate by about 10% on both the π0 and RDT models, achieving speedup of 1.8X and 1.5X, respectively. Our results highlight that context-sensitive and task-aware token pruning serves as a more effective strategy than full visual processing, enabling faster inference and improved manipulation accuracy in real-world robotic systems.

### 1. 核心问题与研究动机
*   **核心问题**：现代 VLA 模型（如 $\pi_0$、RDT）倾向于使用多视图输入以增强感知，但这导致视觉 Token 数量激增，严重影响推理速度（实时性）。
*   **研究背景**：现有的视觉大模型（VLM）剪枝技术（如 DART、FastV）直接用于机器人领域效果不佳。原因在于：
    1.  忽略了不同摄像头视图之间的动态关系（例如：接近阶段侧重全局视图，操作阶段侧重手腕视图）。
    2.  缺乏任务感知的监督信号，容易误删关键的细微操作线索（如夹爪边缘或小物体）。
    3.  机器人数据集规模有限，模型难以自发学习到鲁棒的注意力分配。

### 2. 方法论：BFA++ 框架
BFA++ 提出了一种**分层（Hierarchical）且任务感知**的剪枝策略，核心组件包括：
*   **两级重要性预测器**：
    *   **视图内预测器 (Intra-IP)**：识别单张图像中与任务相关的区域（如夹爪、目标物体），抑制背景噪声。
    *   **视图间预测器 (Inter-IP)**：根据操作阶段（接近、交互、撤回）动态评估不同相机视图的重要性。
*   **分层剪枝流程**：
    1.  **局部剪枝 (Local Prune)**：在每个视图内部，根据 Intra-IP 分数移除固定比例的次要 Token。引入**空间自适应加权**（Spatial Adaptive Weighting），通过考虑邻近 Token 距离来平滑分数，防止孤立的重要区域被误删。
    2.  **全局剪枝 (Global Prune)**：将剩余 Token 的 Intra 分数与 Inter 权重相乘，进行全局排序，剔除跨视图冗余。
*   **监督训练**：利用离线标注系统（结合 VLM 逻辑推理、边界框检测和人工标注）生成重要性标签，通过二元交叉熵损失（BCE Loss）与 VLA 主模型共同微调。

### 6. 主要结论与发现
*   **性能提升**：在 $\pi_0$ 和 RDT 上，成功率均提升了约 **10%**。
*   **推理加速**：$\pi_0$ 实现 **1.8倍** 加速（从 6.5Hz 提升至 10.3Hz），RDT 实现 **1.5倍** 加速。
*   **关键发现**：适当的剪枝不仅能加速，还能起到“去噪”作用。移除与任务无关的背景 Token 有助于模型在有限的数据下更好地提取关键交互特征。

### 7. 优点与亮点
*   **任务感知性**：不同于无监督剪枝，BFA++ 引入了机器人操作的先验知识（如手腕相机在交互瞬间的重要性）。
*   **即插即用**：作为一种后训练（Post-training）方法，可以适配不同架构的 VLA 模型（如 Flow-matching 或 Diffusion 架构）。
*   **分层策略**：先局部后全局的逻辑有效平衡了“保留细节”与“减少冗余”之间的矛盾。

### 8. 不足与局限
*   **泛化风险**：重要性预测器是在特定任务分布上训练的，对于从未见过的物体或全新的相机布局，预测器的准确性可能会下降。
*   **标注依赖**：虽然提供了自动标注方案，但高性能的预测器仍依赖于高质量的离线标注系统。
*   **硬件限制**：在 $\pi_0$ 等模型中，为了不破坏 KV Cache 的连续性，剪枝只能在进入 LLM Backbone 之前进行，限制了更深层动态剪枝的潜力。
