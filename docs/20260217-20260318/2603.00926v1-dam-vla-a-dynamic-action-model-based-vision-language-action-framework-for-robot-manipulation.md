## 摘要
在仓库、医院和家庭等动态环境中，机器人必须在粗略运动和精确操纵之间无缝切换，以完成复杂任务。然而，目前的视觉-语言-动作（VLA）框架大多改编自预训练的视觉-语言模型（VLM），往往难以在通用任务适应性与复杂操纵所需的专门精度之间取得平衡。为了应对这一挑战，我们提出了 DAM-VLA，一种基于动态动作模型的 VLA 框架。DAM-VLA 将 VLM 推理与专门用于机械臂和夹持器控制的基于扩散的动作模型相结合。具体而言，它引入了：(i) 一种动作路由机制，利用特定任务的视觉和语言线索来选择合适的动作模型（如机械臂移动或夹持器操纵）；(ii) 一种动态动作模型，融合了高层 VLM 认知与底层视觉特征以预测动作；以及 (iii) 一种双尺度动作加权机制，实现了机械臂移动模型与夹持器操纵模型之间的动态协调。在广泛的评估中，DAM-VLA 在模拟环境（SIMPLER、FurnitureBench）和真实场景中均取得了优于现有最先进 VLA 方法的成功率，展示了从标准取放任务到具有挑战性的长时程及接触密集型任务的稳健泛化能力。

## Abstract
In dynamic environments such as warehouses, hospitals, and homes, robots must seamlessly transition between gross motion and precise manipulations to complete complex tasks. However, current Vision-Language-Action (VLA) frameworks, largely adapted from pre-trained Vision-Language Models (VLMs), often struggle to reconcile general task adaptability with the specialized precision required for intricate manipulation. To address this challenge, we propose DAM-VLA, a dynamic action model-based VLA framework. DAM-VLA integrates VLM reasoning with diffusion-based action models specialized for arm and gripper control. Specifically, it introduces (i) an action routing mechanism, using task-specific visual and linguistic cues to select appropriate action models (e.g., arm movement or gripper manipulation), (ii) a dynamic action model that fuses high-level VLM cognition with low-level visual features to predict actions, and (iii) a dual-scale action weighting mechanism that enables dynamic coordination between the arm-movement and gripper-manipulation models. Across extensive evaluations, DAM-VLA achieves superior success rates compared to state-of-the-art VLA methods in simulated (SIMPLER, FurnitureBench) and real-world settings, showing robust generalization from standard pick-and-place to demanding long-horizon and contact-rich tasks.

*   **现有局限**：目前的 VLA 模型（如 RT-2, OpenVLA）多由预训练 VLM 改编，通常将连续动作离散化或使用单一扩散头。这种“一刀切”的方法难以平衡通用的任务推理能力与复杂操纵所需的极高精度。
*   **核心思想**：论文指出机械臂移动（路径约束少、需全局视觉）与夹持器操纵（约束严苛、需局部视觉）存在本质差异。因此，DAM-VLA 旨在通过 VLM 的推理能力动态路由不同的专门化动作模型，实现高精度与强泛化性的结合。

### 2. 方法论
DAM-VLA 框架由三个关键组件构成：
*   **动作路由机制 (Action Routing)**：利用 VLM（LLaMA-2）提取的推理潜变量 ($f_{rea}$)，通过一个全连接层预测权重 $w$。该权重决定当前状态属于“机械臂移动”还是“夹持器操纵”，从而选择对应的动作模型。
*   **动态动作模型 (Dynamic Action Model)**：采用双头扩散转换器（DiT）结构。
    *   **机械臂模型**：输入全局视觉特征（DINOv2 的 class token），负责大范围路径规划。
    *   **夹持器模型**：输入局部视觉特征（DINOv2 的 register token），负责精细对准和抓取。
    *   两者均结合了 VLM 输出的高层认知潜变量 ($f_{cog}$)。
*   **双尺度动作加权机制 (Dual-Scale Action Weighting)**：
    *   **轨迹级权重**：在夹持器状态改变（开/关）附近应用非对称高斯分布，强化关键操纵阶段的监督。
    *   **动作块级权重**：对预测的动作序列应用指数衰减，优先保证近期动作的准确性。

### 6. 主要结论与发现
*   **性能领先**：DAM-VLA 在 SIMPLER 模拟中的平均成功率显著高于现有模型（如在 Google Robot VM 模式下达到 83%，优于 CogACT 的 72%）。
*   **复杂任务优势**：在需要精细操作的“打开抽屉并放置苹果”任务中，DAM-VLA 表现出大幅领先。
*   **强鲁棒性**：在真实世界的 OOD 场景下，DAM-VLA 保持了 82.2% 的成功率，证明了其良好的泛化能力。
*   **协同效应**：将 VLM 的高层推理与专门的底层扩散模型结合，比单一模型更能有效处理动态环境下的操纵挑战。

### 7. 优点
*   **架构创新**：通过“路由”思想解决了 VLA 模型在宏观移动与微观操纵之间的矛盾。
*   **视觉特征精细化**：巧妙地为不同动作模型分配了 class token（全局）和 register token（局部），符合机器人学的直觉。
*   **加权策略科学**：双尺度加权机制考虑了动作的时间不确定性和任务关键节点，提升了训练效率。

### 8. 不足与局限
*   **任务多样性尚可提升**：虽然涵盖了组装任务，但主要还是集中在取放类动作，需进一步验证在更多样化（如折叠、擦拭等）任务中的表现。
*   **特定场景增益有限**：在“堆叠绿块”等极少数任务中，相比基准模型的提升幅度较小，说明在极精细的协调稳定性上仍有优化空间。
*   **模型扩展性限制**：目前仅在两个动作模型（臂 vs 夹）之间路由，未来若要处理更多自由度或多机器人协作，路由机制的复杂度和计算开销会增加。
