## 摘要
通用价值模型在大规模混合质量数据中扩展机器人策略学习方面发挥着关键作用。从数学上讲，准确的价值估计需要深度的时序理解，这要求模型既能利用历史上下文来锚定当前信念，又能对未来结果进行规划。然而，现有的大多数机器人价值模型都基于视觉-语言模型（VLM）主干网络，这些网络主要在静态或时序稀疏的视觉观测上进行预训练，缺乏价值估计所需的时序建模能力。与VLM不同，世界模型天然擅长时序建模和未来规划，使其成为学习可泛化价值函数的理想基础。受此启发，我们将世界模型与价值估计相结合，构建了一种新的通用机器人价值模型——世界价值模型（WVM），该模型提供准确的任务进展以评估数据质量。在标准基准上，WVM实现了最先进的（SOTA）价值顺序相关性（VOC）结果。作为仅包含专家数据的标准评估套件的补充，我们进一步引入了Suboptimal-Value-Bench，这是一个多具身基准，包含800条次优轨迹，并带有高保真度、人工标注的帧级注释。我们的评估表明，WVM在Suboptimal-Value-Bench上保持了其SOTA性能，确立了其在处理专家数据和次优数据时的鲁棒性。当用于策略学习时，WVM在模拟和实际部署中提升了各种策略提取方法的操作性能，为从混合质量数据中学习提供了稳健的指导。

## Abstract
Generalist value models play a pivotal role in scaling robotic policy learning from large-scale, mixed-quality data. Mathematically, accurate value estimation demands deep temporal understanding, requiring models to both ground the current belief using historical context and plan over future outcomes. However, most existing robotic value models are built on Vision-Language Model (VLM) backbones that are pretrained primarily on static or temporally sparse visual observations, lacking the requisite temporal modeling capabilities for value estimation. Unlike VLMs, world models naturally excel at temporal modeling and future planning, making them ideal foundations for learning generalizable value functions. Driven by this insight, we marry world models with value estimation to construct a new generalist robotic value model, World Value Model (WVM), that offers accurate task progressions to assess data quality. On standard benchmarks, WVM delivers state-of-the-art (SOTA) Value-Order Correlation (VOC) results. Complementing standard evaluation suites that contains only expert data, we further introduce Suboptimal-Value-Bench, a multi-embodiment benchmark consisting of 800 suboptimal trajectories with high-fidelity, human-labeled frame annotations. Our evaluations show that WVM maintains its SOTA performance on Suboptimal-Value-Bench, establishing its robustness in handling both expert and suboptimal data. When deployed for policy learning, WVM improves manipulation performance across various policy extraction approaches in both simulated and real-world deployment, providing robust guidance for learning from mixed-quality data.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：现有机器人通用价值模型（generalist value models）大多基于视觉-语言模型（VLM）主干，如GVL、VLAC、Robometer等，其在静态或时序稀疏的视觉数据上预训练，缺乏深度时序建模和未来规划能力，难以准确估计混合质量数据（包含次优轨迹）中的任务进展。
- **整体含义**：准确的价值估计需要同时具备历史上下文理解和前瞻性未来规划。世界模型（world models）天然擅长时序动态建模和未来预测，因此将世界模型的时空先验引入价值估计，可以克服VLM主干的瓶颈。本文提出World Value Model (WVM)，利用预训练视频世界模型的时空能力，实现高保真、可泛化的价值函数，进而有效评估数据质量并改进下游策略学习。

## 2. 方法论

### 核心思想
- 将价值估计重新定义为**分布性价值块预测**（distributional value chunk）：对长度为h的帧序列预测一个连续值块，而非单个标量。
- 利用**预训练视频世界模型**（Wan2.2-TI2V-5B）作为丰富特征提取器，通过**Mixture-of-Transformers (MoT)** 架构将视频流与轻量价值DiT耦合，使价值流能够选择性关注视频潜在特征，同时通过不对称注意力掩码避免干扰视频生成流。

### 关键技术细节
- **视频流**：基于Wan2.2的Video VAE和Video DiT。输入一个长度为(2h+1)的视频片段：1帧前缀 + h帧当前观测 + h帧未来目标帧。VAE编码后保留当前潜在作为上下文，损坏未来潜在用于视频去噪。
- **价值流**：轻量级DiT，与视频DiT共享深度但隐藏维度更小。价值令牌通过MoT多模态自注意力与视频DiT的中间特征交互，且仅价值流关注视频流，视频流不关注价值流。
- **训练目标**：采用**flow matching**对价值块和未来视频潜在进行监督。总损失 L = L_value + λ L_video，其中λ=1.0。L_value预测价值块的velocity场，L_video预测未来视频潜在。
- **前缀随机化（Prefix Randomization）**：以概率p将前缀价值替换为[0,1]的随机标量，防止价值流从前缀中获取捷径，维持跨块连续性。默认p=0.5。
- **视频倒带（Video Rewinding）**：对观测窗口应用上升、平台、下降三种时序模式，对应平稳推进、犹豫、重试，使价值流暴露于局部进展剖面。

### 推理
- 使用单步Euler求解器从流匹配速度场去噪得到价值块，再利用重叠窗口平均得到帧级预测。

## 3. 实验设计

### 数据集/场景
- **标准Expert VOC基准**：包含六个数据集：OXE（从RoboReward过滤得到10个子集）、RoboCOIN、EgoDex，以及自收集的三个具身任务（RoboSuite仿真、AgileX单臂/双臂实物、ARX双臂实物）。共约1,605条专家轨迹。
- **Suboptimal-Value-Bench**（新提出的基准）：包含800条人工标注的次优轨迹，涵盖三个具身（AgileX、ARX、RoboSuite）共15个任务。每条轨迹标注两种次优行为模式：**犹豫（Hesitation）**（任务进展平台期）和**重试（Retry）**（进展下降段）。提供了帧级真值价值曲线（分段线性构造）。
- **下游策略学习**：模拟任务：RoboSuite的3个任务（Stack cubes, Put cereal in tray, Put milk in tray）。真实任务：AgileX双机械臂的3个任务（Put mouse in box, Put shrimp in pot, Put marker in holder）。仅使用次优数据（每任务10条模拟、50条真实）。

### 对比方法
- **价值质量评估**：对比6个基线：GVL（GPT-5.4）、VLAC（InternVL）、Robometer（Qwen3-VL-4B）、TopReward（Qwen3-VL-8B）、RoboReward（Qwen3-VL-4B）、Robo-Dopamine（3B GRM）。
- **下游策略**：基础策略为π0.5-base，对比方法：vanilla BC、AWR、Filtered BC（Adv>0）、Filtered BC（Top 70%）。

### 评价指标
- Expert-VOC：价值顺序相关性，衡量专家轨迹上的单调性。
- Hesitation-RMSE：犹豫段预测值与恒值真值的均方根误差。
- Retry-VOC：重试段价值下降趋势的VOC（仅计算单调下降窗口）。
- 下游策略成功率。

## 6. 主要结论与发现

- WVM在Expert-VOC上平均得分0.95，远超最强基线RoboReward的0.88，在四个数据集上排名第一。
- 在Suboptimal-Value-Bench上，WVM的Hesitation-RMSE平均0.05（最接近基线0.14），Retry-VOC平均0.78（基线0.62），显著优于所有基线。
- 在次优数据上训练下游策略：所有三种WVM引导的变体（AWR、Filtered BC两类）均优于vanilla BC，成功率提升显著（模拟任务平均从~10%升至~30-45%，真实任务从~40%升至~70-90%）。
- 消融证实：视频协同训练不可或缺（w/o L_video性能下降）；前缀随机化p=0.5平衡了捷径抑制和跨块一致性；Flow Matching头优于HL-Gaussian分类头。

## 7. 优点

- **方法创新**：首个将预训练视频世界模型作为价值函数的主干，充分利用其时空建模和未来预测能力，克服VLM的先天不足。
- **价值形式创新**：采用分布性价值块（flow matching）替代标量回归，提供更密集的训练信号和更丰富的表达力，适合捕捉局部进展剖面（如平台、下降）。
- **训练增强**：前缀随机化和视频倒带使模型能处理次优轨迹，提升鲁棒性。
- **次优价值基准**：Suboptimal-Value-Bench填补了仅评估专家轨迹的空白，提供人工标注的高保真次优数据，助力更全面的价值模型评估。
- **实验全面**：覆盖多个公共数据集、多个具身、仿真和真实场景，消融清晰，对比基线全面且复现公平（使用统一视频采样管道和官方推荐配置）。

## 8. 不足与局限

- **训练数据规模受限**：受限于算力，WVM训练混合数据量相对有限，因此在完全未见过的任务和场景中零样本泛化能力仍显不足。
- **基准覆盖范围**：Suboptimal-Value-Bench主要集中于pick-and-place任务，缺乏对更灵巧操作（如手内操作）和长时序任务（如多步骤组装）的覆盖。未来需要扩展评估多样性。
- **计算成本高**：需要32张A100训练40小时，且总参数量达5.7B，部署和微调成本较高。
- **推理虽快（单步）**，但依赖于大规模预训练世界模型，模型本身较重。
- **仅验证了次优数据下的策略学习**，未探索WVM在完全在线或场景真实RL中的表现。
- **价值标签构造**依赖人工标注和分段线性简化（重试段假设均匀后退），与实际次优轨迹的复杂动态可能存在偏差。
