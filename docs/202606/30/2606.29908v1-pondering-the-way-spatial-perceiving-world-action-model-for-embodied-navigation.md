## 摘要
现有的基于世界模型的视觉导航规划器通常遵循以验证为中心的范式，将目标意图与轨迹合成解耦。这种方法存在候选依赖、计算开销大以及采样动作与预测视觉不一致等问题。为解决这些问题，我们提出了SWAM（空间感知世界动作模型），一种以任务为中心的联合观察-动作生成框架。给定起始和目标RGB观察，SWAM通过单次推理同时生成中间RGB-D序列和相应的动作轨迹，促进目标一致的轨迹生成并改善空间可行性。虽然SWAM在训练期间利用深度伪标签来内化空间先验，但在推理时仅需单目RGB输入。我们进一步引入了视觉引导的动作细化模块和轨迹尺度正则化损失，以强制运动和视觉线索之间的细粒度对齐，同时稳定不同距离下的预测。大量实验表明，SWAM在成功率、轨迹准确性和推理效率方面显著优于最先进的两阶段规划器，并在未见环境中展现出稳健的零样本泛化能力。

## Abstract
Existing world model-based planners for visual navigation typically follow a verification-centric paradigm, decoupling goal intent from trajectory synthesis. This approach suffers from candidate dependence, heavy computational overhead, and inconsistencies between sampled actions and predicted visuals. To address these issues, we propose SWAM (Spatial-perceiving World Action Model), a task-centric joint observation-action generation framework. Given start and goal RGB observations, SWAM performs single-pass inference to simultaneously generate intermediate RGB-D sequences and corresponding action trajectories, promoting goal-consistent trajectory generation and improved spatial feasibility. While SWAM leverages depth pseudo-labels during training to internalize spatial priors, it requires only monocular RGB input at inference time. We further introduce a visual-guided action refinement module and a trajectory-scale regularization loss to enforce fine-grained alignment between motion and visual cues while stabilizing predictions across varying distances. Extensive experiments show that SWAM significantly outperforms state-of-the-art two-stage planners in success rate, trajectory accuracy, and inference efficiency, while demonstrating robust zero-shot generalization to unseen environments.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：现有视觉导航的世界模型规划器通常采用“验证中心”的两阶段范式：先由外部策略采样候选动作轨迹，再通过动作条件视频预测模型生成未来观测，最后基于目标相似度选择最佳轨迹。这种范式存在三个核心缺陷：
  - **候选依赖**：决策质量受限于候选轨迹的覆盖范围；
  - **计算效率低**：需要大量候选采样与 rollout，推理成本极高（尤其在长距离或复杂场景）；
  - **几何不一致**：缺乏显式空间约束，导致预测中出现突兀视角变化或不可行轨迹。
- **核心动机**：作者认为这些问题的根源在于**路径提议与环境预测的解耦**。世界模型被降级为被动评估器，目标意图与轨迹生成分离，空间推理被隐式地交由视觉模式处理，忽略了导航的核心需求（即生成空间可行、目标一致的动作序列）。
- **研究含义**：提出一种**任务中心的联合观测-动作生成框架**，将“如何行动”与“将看到什么”在同一过程中协同生成，从而克服上述局限。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- **联合生成**：给定起始 RGB 观察 \(o_0\) 和目标 RGB 观察 \(o_G\)，单次推理同时生成中间 RGB-D 观察序列 \(\{o_n\}_{n=1}^{N}\) 和对应的动作轨迹 \(\{a_n\}_{n=1}^{N}\)（动作定义为局部坐标系下的平面位移 \((\Delta x_n, \Delta y_n)\)）。
- **空间感知**：训练时利用预训练单目深度估计器（DepthAnything v3）为每帧生成深度伪标签，为生成过程提供几何先验，推理时仅需单目 RGB 输入。
- **视觉引导动作细化**：在动作解码前引入交叉注意力机制，将生成的 RGB-D 特征注入动作 token，增强动作与视觉的细粒度对齐。
- **轨迹尺度正则化**：在扩散损失之外，额外监督预测终点与真实终点的累积位移误差，稳定长距离预测。

### 关键技术细节

1. **整体架构**（基于预训练视频扩散模型 CogVideoX 的 Diffusion Transformer, DiT）：
   - **潜在表示与分词**：使用冻结的 3D VAE 将 RGB 和深度图分别编码为潜在特征 \(z_i, z_d\)；将动作序列通过 MLP 投影为 token 矩阵 \(x_a \in \mathbb{R}^{N \times D}\)。
   - **多模态序列构建**：将中间 RGB 潜在 \( \{z_i^n\}_{n=1}^{N} \)、中间深度潜在 \( \{z_d^n\}_{n=1}^{N} \) 和动作 token \(x_a\) 拼接为统一序列 \(X_t\)，起始/目标潜在作为条件 \(c = (z_i^0, z_d^0, z_i^G, z_d^G)\)。
   - **去噪过程**：采用 DDPM 风格的前向加噪与迭代去噪，扩散 Transformer \(F_\theta\) 预测噪声残差：
     \[
     X_{t-1} = \frac{1}{\sqrt{\alpha_t}}\left(X_t - \frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(X_t, c, t)\right) + \sigma_t z
     \]
     去噪完成后，VAE 解码器还原 RGB 与深度图，动作 token 通过 VGAR 模块和轻量网络解码为动作。

2. **视觉引导动作细化（VGAR）**：
   - 对扩散 Transformer 最后一层的视觉 token \(X_v\) 和动作 token \(X_a\)，执行门控残差交叉注意力：
     \[
     C = \text{CrossAttn}(\text{LN}(X_a), \text{LN}(X_v), \text{LN}(X_v))
     \]
     \[
     \Delta X_a = G \odot W(C), \quad G = \sigma(\text{MLP}([\text{LN}(X_a); C]))
     \]
     \[
     X'_a = X_a + \Delta X_a
     \]
   - 门控机制 \(G\) 控制视觉证据对动作表示的注入程度。

3. **轨迹尺度正则化（TSR）**：
   - 计算预测终点位置 \(\hat{p}_N = \sum_{n=1}^N \hat{a}_n\) 与真实终点 \(p_G\) 的 L2 距离，归一化到序列长度：
     \[
     L_{\text{TSR}} = \frac{\|\hat{p}_N - p_G\|_2}{N}
     \]
   - 联合训练目标：\(L = L_{\text{DDPM}} + \lambda_{\text{TSR}} L_{\text{TSR}}\)。

4. **模型初始化**：新增模块（深度嵌入、VGAR 等）全部初始化为零权重，确保初始阶段与预训练行为一致，逐步学习耦合关系。
