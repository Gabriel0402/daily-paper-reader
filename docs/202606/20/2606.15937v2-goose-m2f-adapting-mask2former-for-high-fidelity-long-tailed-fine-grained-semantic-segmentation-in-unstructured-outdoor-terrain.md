## 摘要
我们提出了GOOSE-M2F，这是Mask2Former针对ICRA 2026 GOOSE 2D细粒度语义分割（FGSS）挑战赛的任务特定适配。GOOSE基准测试涵盖跨非结构化户外地形的64个细粒度类别，具有严重的长尾分布，其中稀有类别每张图像中占不到50个像素。我们通过三个针对性贡献扩展了Swin-Large Mask2Former基线：（1）200个对象查询以消除表示饱和；（2）结合ASPP-lite和CBAM双注意力的特征精化模块（FRM）；（3）辅助监督头，为稀有类别提供直接的逐像素梯度。多阶段训练策略结合了分布平衡损失、稀有类别复制粘贴增强、动态IoU感知重新加权和指数移动平均（EMA）。在推理时，采用密集滑动窗口引擎，配合2D高斯核混合和4尺度TTA，提升了+10.57%。GOOSE-M2F在官方复合mIoU上达到70.08%（细粒度63.55%，粗粒度76.61%），在GOOSE 2D FGSS排行榜上位列第三。代码和训练模型已在GitHub（https://github.com/Aditya-Lingam-9000/GOOSE-M2F）和Hugging Face（https://huggingface.co/XYZ9843/GOOSE-M2F）上公开。

## Abstract
We present GOOSE-M2F, a task-specific adaptation of Mask2Former for the GOOSE 2D Fine-Grained Semantic Segmentation (FGSS) Challenge at ICRA 2026. The GOOSE benchmark spans 64 fine-grained classes across unstructured outdoor terrain with a severely long-tailed distribution, where rare classes occupy fewer than 50 pixels per image. We extend the Swin-Large Mask2Former baseline with three targeted contributions: (1) 200 object queries to eliminate representational saturation; (2) a Feature Refinement Module (FRM) combining ASPP-lite and CBAM dual-attention; and (3) an Auxiliary Supervision Head that delivers direct per-pixel gradients for rare classes. A multi-stage training strategy pairs Distribution-Balanced loss, Rare-Class Copy-Paste augmentation, dynamic IoU-aware re-weighting, and EMA. At inference, a dense sliding-window engine with 2D Gaussian kernel blending and 4-scale TTA adds +10.57%. GOOSE-M2F achieves 70.08% Official Composite mIoU (63.55% fine, 76.61% coarse), placing 3rd on the GOOSE 2D FGSS leaderboard. Code and trained models are publicly available at GitHub: https://github.com/Aditya-Lingam-9000/GOOSE-M2F and Hugging Face: https://huggingface.co/XYZ9843/GOOSE-M2F.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：非结构化户外地形（如森林、碎石、雪地、军事地形）的语义分割对机器人（如足式机器人）在自然环境中运行至关重要，但现有城市基准（Cityscapes、ADE20K）无法处理细粒度类别、严重类别不平衡以及缺乏大规模标注数据的问题。
- **核心问题**：GOOSE 2D细粒度语义分割挑战赛（ICRA 2026）提出64个语义类，面临三个结构困难：
  - **长尾分布**：主导类别占据>50%像素，而稀有类别出现在<1%图像中，每张图像仅<50像素。
  - **模糊边界**：植被子类连续融合，需要宽上下文特征。
  - **微小结构**：如电线、杆、交通锥等，在标准交叉熵下提供可忽略的梯度信号。
- **整体含义**：现有Mask2Former基线（100个查询）在处理64类、每场景15-20个语义区域时出现表征饱和和匹配失败，亟需针对性改进。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
在保留Swin-Large预训练特征的前提下，通过三个轻量级组件直接对抗长尾、模糊边界和小目标问题，并结合多阶段训练策略和强化推理引擎。

### 关键技术细节

- **架构概览**：基于HuggingFace的`mask2former-swin-large-cityscapes-semantic`，冻结后微调。前向流程：
  ```
  ŷ = HungarianMatch(Dec_200(FRM(PixDec(fSwin(x)))), y)
  ```
  其中fSwin为Swin-Large骨干，PixDec为6层MSDeformAttn像素解码器（输出B×256×H/4×W/4），FRM为特征精化模块，Dec_200为含200个查询的Transformer解码器。

- **扩展对象查询（200 Queries）**：
  - 默认100查询针对Cityscapes 19类设计。GOOSE有64类且场景常含15-20个语义区域，查询饱和导致分割冲突。
  - 扩展至200个查询：保留原始100个嵌入，新增100个查询初始化为高斯扰动（ϵ ~ N(0, 0.02²I)），保持预训练流形同时增加容量。
  - 贡献+2~3%复合mIoU。

- **特征精化模块（FRM）**（图1）：
  - 解决像素解码器单尺度（H/4）限制。
  - **ASPP-lite**：以扩张率d∈{1,3,6,12}计算并行空洞卷积（各64中间通道）加全局池化，经1×1卷积融合。d=1用于微小结构，d=3/6用于中型物体，d=12用于大区域（森林），全局池化用于场景上下文（天空、雪）。
  - **CBAM**：依次应用通道注意力（MLP基于平均/最大池化）和空间注意力（7×7卷积），残差连接保持梯度流。
  - 贡献+3~4%在植被和地形类别。

- **辅助监督头（Auxiliary Supervision Head）**：
  - 稀有类别（<50像素/裁剪）在Mask2Former主损失中得不到梯度，因为二分图匹配总会将微小段分配给成本更低的较大竞争者。
  - 辅助头在H/4分辨率直接应用逐像素交叉熵，独立于查询匹配，采用全卷积密集预测范式。
  - 损失：`L_total = L_M2F + 0.4 * L_aux`，辅助头为两个Conv2d层（256→256→64），推理时去除，无VRAM开销。
  - 贡献+5~8%稀有类别。

### 训练策略

- **分布平衡损失（DB Loss）**：`w_c = (1-β)/(1-β^n_c)`，β=0.9999，稀有类权重最高放大50倍，裁剪至[0.1,10]。
- **动态IoU感知重新加权**：每验证轮次后，根据EMA模型的当前IoU更新类别损失权重，`w_c^(dyn) = f_tier(IoU_c)`，将[0%,80%+]映射到乘数[5×,1×]。
- **稀有类别复制粘贴（RCCP）**：预提取稀有类裁剪，以概率0.85 alpha融合到训练图像（软5×5高斯掩码）。微小物体额外缩放约U(1.2,2.5)以对抗匈牙利匹配失败。
- **类别感知重复采样（CAS）**：根据图像中最稀有类别过采样，`r_c = min(p*t / f_c, 3.0)`，t=0.06，稀有图像最多放大3倍。
- **EMA**：影子权重`θ_EMA^(t)=0.9995θ_EMA^(t-1)+0.0005θ^(t)`，用于验证和提交，比原始权重高+1~1.5%。
- **多阶段调度**：AdamW（权重衰减0.03），多项式LR衰减，1000步线性warmup，梯度裁剪5.0，骨干学习率比解码器低5倍。
  - **warmup阶段**（stage 1-3）：骨干1e-6，解码器5e-6，平台期约55.6%。
  - **加速阶段**（stage 4）：有意将LR跳升10倍（骨干1e-5，解码器5e-5），突破平台期至56.4%；stage 5-7巩固至59.5%。
  - **精炼阶段**（stage 8）：退火至骨干5e-6，解码器2.5e-5。
  - 其他设置：裁剪576×1152，有效batch 16（1样本×16梯度累积），标签平滑0.10，FP16混合精度。

### 推理引擎（最大单项增益+10.57%）

- **密集滑动窗口**：图像划分为重叠的896×896裁剪，步长384（57%重叠），确保每像素被多裁剪覆盖，+4~5%。
- **2D高斯核混合**：每个裁剪的logit图乘以2D高斯权重（σ=0.5c），中心置信度高，边缘低，消除拼接伪影，对细长结构（电线、围栏）关键。
- **4尺度×2翻转TTA**：每图像在{0.5×,0.75×,1.0×,1.5×}尺度下加水平翻转，共8视图，logit平均池化后argmax。1.5×上采样对小类尤为重要。
- 总TTA提升：59.51%→70.08%（+10.57%）。
