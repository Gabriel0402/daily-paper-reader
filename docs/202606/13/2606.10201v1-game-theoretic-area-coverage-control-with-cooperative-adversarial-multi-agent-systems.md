## 摘要
我们将多智能体区域覆盖控制问题建模为两个目标冲突的智能体群体之间的两人零和博弈。传统的覆盖控制根据环境风险密度场分配资源。相比之下，我们通过允许第二组对抗智能体生成空间风险场来推广这一指标。耦合的智能体动力学通过区域覆盖度量（作为博弈奖励）相互关联。该框架为群体导出了耦合的梯度下降-上升控制器。对一个低维情况的分析揭示了由群体控制增益比决定的霍普夫分岔。在对抗智能体主导的机制中，系统被驱动进入周期性追逐-逃避循环。在常规智能体主导的机制中，系统收敛到一个固定配置。数值模拟验证了这些理论见解。最后，我们刻画了纳什均衡条件。在该均衡下，常规智能体收敛到广义质心Voronoi剖分，而对抗智能体则停留在其相应的均衡质心。

## Abstract
We formulate a multi-agent area coverage control problem as a two-player zero-sum game between two agent groups with conflicting goals. Conventional coverage control allocates resources based on an environmental risk density field. In contrast, we generalize this metric by allowing a second group of adversarial agents to generate the spatial risk field. Coupled agent dynamics are linked through the area coverage metric, which functions as the game reward. This framework induces coupled gradient-descent-ascent controllers for the groups. Analysis of a low-dimensional case reveals a Hopf bifurcation dictated by the ratio of the groups' control gains. In the regime dominated by adversarial agents, the system is driven into a periodic chase-evasion cycle. In the regime dominated by ordinary agents, the system converges to a fixed configuration. Numerical simulations validate these theoretical insights. Finally, we characterize the Nash equilibrium conditions. Under this equilibrium, ordinary agents converge to a generalized centroidal Voronoi tessellation, whereas adversarial agents settle at their corresponding equilibrium centroids.

传统多智能体区域覆盖控制问题通常假设风险密度场（risk density field）是预先定义且与智能体行为无关的静态环境场。然而，实际场景中（如监视、侦察、防御）通常存在主动对抗的智能体，它们能够观察并调整自身策略以规避被覆盖或检测，从而形成“追逐-逃避”的动态交互。现有方法无法刻画这种对抗性耦合。本文的动机是：将区域覆盖控制重新解释为两人零和博弈，其中一组“普通智能体”（ordinary agents）负责覆盖，另一组“对抗智能体”（adversarial agents）主动生成风险场以破坏覆盖，从而建立统一的博弈论框架来研究耦合动力学行为及其均衡性质。

## 2. 方法论

### 核心思想
- 将覆盖性能指标 \( H(P,S,\mathcal{V}) = \sum_{i=1}^M \int_{\mathcal{V}_i} f(r_i) \bar{\phi}(S) d\Omega \) 作为零和博弈的奖励函数。

- 采用梯度下降-上升（Gradient Descent-Ascent, GDA）控制器。

### 关键技术细节
- 智能体动力学：单积分器 \(\dot{p}_i = u_{p_i}\)，\(\dot{s}_j = u_{s_j}\)。
- 普通智能体控制器（Lloyd算法）：\(u_{p_i} = k_p (c_{p_i} - p_i)\)，其中 \(c_{p_i}\) 是Voronoi单元质心。
- 对抗智能体控制器：\(u_{s_j} = k_s (c_{s_j} - s_j)\)，其中 \(c_{s_j}\) 是通过加权风险密度定义的质心。
- 定义控制增益比 \(\gamma = k_p / k_s\)。
- 低维（1个普通、1个对抗）情况下，推导出固定点为工作空间中心 \((a+b)/2\)，并证明其为鞍点。

- 纳什均衡条件：普通端Hessian块正定（\(H_{pp} \succ 0\)），对抗端负定（\(H_{ss} \prec 0\)），要求风险密度充分平滑（\(\sigma\)足够大）。

### 算法流程（文字说明）
1. 初始化普通智能体和对抗智能体位置。
2. 循环：

## 6. 主要结论与发现

1. 控制增益比 \(\gamma = k_p/k_s\) 是系统行为的关键调控参数。
2. 当 \(\gamma > \gamma_c\) 时，系统收敛到固定配置：普通智能体形成质心Voronoi剖分，对抗智能体到达对应均衡质心。
3. 当 \(\gamma < \gamma_c\) 时（对抗主导），系统出现Hopf分岔，产生周期性追逐-逃避极限环。
4. 纳什均衡存在条件：风险密度足够平滑（\(\sigma\)大，\(\epsilon < \epsilon_c\)），此时Hessian满足 \(H_{pp} \succ 0\) 且 \(H_{ss} \prec 0\)。
5. 低维系统的局部Nash均衡在对称条件下是全局唯一的。
