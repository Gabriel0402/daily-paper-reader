## 摘要
遥感语义分割需要昂贵的像素级标注，而且几乎每个问题都需要新的数据集，因为模型很少能在传感器、平台或地理区域之间迁移。现有的人机协同框架通过辅助机制（伪标签、传播、条件随机场、基础模型提示、辅助头）将稀疏点击扩展为密集监督，所有这些都基于模型的预测分布。在该分布中，一个自信错误的像素与一个自信正确的像素在构造上无法区分，因此任何读取该分布的规则都无法区分二者；区分信号位于模型之外。本文假设，针对模型自信错误的专家点击（而非任意像素）足以匹配密集监督，无需扩展机制。iSAGE（专家引导的迭代稀疏标注）在一个集成的开源平台上实现了这一假设，其中错误加权损失放大每次点击的梯度，标注记录本身即是数据集，可扩展、可修正、可审计。实验采用最小工作量方案：每帧每类最多一个标记像素。在BsB Aerial数据集上，iSAGE恢复了97.2%的密集监督性能（在0.040%的像素上达到74.79% mIoU），具有对比鲜明的类别动态：无定形类别（如渗透区域）从初始种子即饱和，而小类别（如汽车）需要后期迭代努力。在ISPRS Vaihingen（外部基准）上，iSAGE在0.011%的像素上达到76.78% mIoU，与密集基线（76.65%）持平，并超过了所有已发表的方法。在相同流程下，四种输出读取机制（预算1-100倍的天真熵、阈值0.90-0.99的伪标签、基于条件随机场的传播、均匀随机）的瓶颈比iSAGE低7.4到14.5个百分点。在31种被调查的方法中，iSAGE是唯一无需辅助机制即可运行的迭代人机协同框架。

## Abstract
Semantic segmentation in remote sensing requires costly pixel-level annotations, and nearly every problem demands a new dataset since models rarely transfer across sensors, platforms, or geographies. Existing human-in-the-loop frameworks expand sparse clicks into dense supervision via auxiliary machinery (pseudo-labels, propagation, CRFs, foundation-model prompts, auxiliary heads), all operating on the model's predictive distribution. A confidently wrong pixel is indistinguishable from a confidently correct one in that distribution by construction, so no rule reading it can separate the two; the distinguishing signal is external to the model. This paper hypothesizes that expert clicks targeting confident model errors, not arbitrary pixels, suffice to match dense supervision, with no expansion machinery. iSAGE (Iterative Sparse Annotation Guided by Expert) realizes this hypothesis on an integrated open-source platform, where an error-weighted loss amplifies the gradient at each click and the annotation record itself is the dataset, extensible, correctable, and auditable. Experiments use a minimum-effort regime: at most one labeled pixel per class per frame. On BsB Aerial, iSAGE recovers 97.2% of dense supervision (74.79% mIoU on 0.040% of pixels) with contrasting class dynamics: amorphous classes (permeable areas) saturate from the seed, while small classes (cars) require late-iteration effort. On ISPRS Vaihingen (external benchmark), iSAGE reaches 76.78% mIoU with 0.011% of pixels, matching the dense baseline (76.65%) and exceeding all published methods. Under the same pipeline, four output-reading mechanisms (oracle entropy across budgets 1--100x, pseudo-labels across thresholds 0.90--0.99, CRF-based propagation, uniform random) plateau 7.4 to 14.5 pp below iSAGE. Across 31 surveyed methods, iSAGE is the only iterative human-in-the-loop framework operating without auxiliary machinery.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：遥感语义分割需要昂贵的像素级密集标注，但模型在不同传感器、平台或地理区域间几乎无法迁移，导致每个新问题都需要从头标注数据集，成本极高。
- **现有方法的局限性**：现有的人机协同（HITL）框架通过辅助机制（伪标签、传播、条件随机场CRF、基础模型提示、辅助头等）将稀疏点击扩展为密集监督，但所有这些机制都基于模型自身的预测分布（output-reading）。在该分布中，一个“自信错误”的像素与一个“自信正确”的像素在构造上无法区分——因为分布本身不包含外部真值信息。因此，任何依赖于该分布的规则都无法分离这两类像素，而最有价值的训练信号恰恰来自“自信错误”的像素。
- **核心假设**：专家仅点击模型自信错误的像素（而非任意像素），无需任何扩展机制，即可匹配密集监督的性能。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：提出 iSAGE（Iterative Sparse Annotation Guided by Expert）框架，在迭代训练循环中，专家观察模型预测叠加图，点击模型自信错误的像素（即预测与视觉常识明显不符的区域）。这些点击直接作为训练监督，不经过任何伪标签、CRF、基础模型等扩增步骤。
- **关键技术细节**：
  - **稀疏点标注**：每个标注精确对应一个像素坐标和类别标签，记录为JSON（可审计、可修正、可扩展）。无区域扩增，无边界绘制。
  - **迭代错误驱动精炼**：每轮迭代，先用当前模型生成预测，专家从误分类区域（视觉叠加图中可见的错误）选取新标注点，加入训练集后重新训练模型。实验采用最严格的预算：每帧每类每轮最多标注1个像素。
  - **误差加权Dice损失（EWDL）**：对预测错误的像素赋予更高权重（λ > 1），放大其梯度贡献，迫使模型聚焦于错误区域。公式如下：
    - 对于每个标注像素，定义 correctness 指标。权重 w(x) = λ if incorrect else 1。
    - 加权Dice损失：$L^{(c)}_{EWDL} = 1 - \frac{2 \sum_x w(x) \cdot y^{(c)}_{gt}(x) \cdot y^{(c)}_{pr}(x) + \epsilon}{\sum_x w(x) \cdot (y^{(c)}_{gt}(x) + y^{(c)}_{pr}(x)) + \epsilon}$
    - 多类时取各类平均。λ=5为默认值。
  - **软件平台**：集成了预测叠加、点击标注、记录持久化、重新训练为一体的开源平台，确保实验可复现。平台领域无关，只需更换预处理和编码器。
- **算法流程**（文字描述）：
  1. 初始稀疏标注训练初始模型 M0。
  2. 对于迭代 i=1..N：
     - 用 Mi-1 生成预测。
     - 人工从误分类区域选择新标注点 S'i。
     - 扩展训练集 Si = Si-1 ∪ S'i。
     - 使用 EWDL 训练更新模型 Mi。
  3. 输出最终模型 MN。

## 3. 实验设计：数据集、基准、对比方法

- **数据集**：
  - **BsB Aerial**（控制实验）：巴西首都巴西利亚的0.24m分辨率RGB正射影像，1250个256×256图块（1000训练，250测试）。四类：汽车、道路、建筑、渗透区域。用于比较密集监督、随机选择、替代损失函数等。
  - **ISPRS Vaihingen**（外部基准）：德国Vaihingen的0.09m分辨率IRRG影像，33张航空图块。六个语义类（不含杂项类），采用EasySeg的17图块测试分割，训练集1000个512×512图块（比EasySeg少44%）。
- **基准方法**：
  - 密集监督基线：使用相同架构的全监督模型。
  - 随机选择基线：迭代随机选择像素而非错误驱动选择。
  - 替代损失函数：BCE、CE、Dice、Focal Loss。
  - 输出读取基线（四种）：
    1. 天真熵（oracle entropy）：选择每类每帧最高熵像素，给定真值标签，预算从1×到100×。
    2. 伪标签（pseudo-labeling）：阈值0.90/0.95/0.99，扩增至~90%像素。
    3. CRF传播：DenseCRF后接阈值。
    4. 均匀随机：每类每帧随机1个像素。
- **对比方法**（外部基准）：包括 EasySeg、D2ADA、ILM-ASSL、RIPU 等弱监督/领域自适应方法。
- **架构**：默认U-Net + EfficientNet-B7，额外测试ResNet-101、DeepLabV3+、SegFormer。

## 6. 主要结论与发现

- iSAGE在极稀疏标注下匹配甚至超越密集监督性能：
  - **BsB Aerial**：多分类任务74.79% mIoU（0.040%像素），恢复密集标注97.2%性能（76.93% mIoU）。
  - **ISPRS Vaihingen**：76.78% mIoU（0.011%像素），超过密集基线76.65%，领先所有已发表方法（如EasySeg 72.83%）。
- **错误驱动 vs 随机**：错误驱动选择显著优于随机（差距5.19 mIoU），尤其在汽车类（差距9.0 IoU）。
- **输出读取极限验证**：四种基线（天真熵、伪标签、CRF、均匀随机）均无法达到iSAGE性能，即使预算增加100倍或阈值调优也差距7.4~14.5百分点。证实自信错误的信息无法从模型预测分布中提取。
- **类别动态**：无定形类别（渗透区域）早期饱和，小物体（汽车）需要更多后期迭代精力。
- EWDL损失与错误驱动点击一致，适当放大错误像素梯度（λ=5）有益，但Focal损失也可达相近性能。

## 7. 优点

- **概念创新**：明确指出并验证了输出读取机制的固有信息极限，提出了“去除所有辅助机制”的极简主义方案，挑战了现有框架的复杂性。
- **方法简洁性**：仅需单像素点击 + 简单加权损失，无需任何扩增、伪标签、条件随机场等，降低工程复杂度和超参数调优。
- **可审计性和可扩展性**：标注记录为JSON，可版本化、回滚、修正，且可直接作为数据集；新增类别只需点击新类实例，无需重新标注旧数据。
- **开源集成平台**：将所有环节打包，便于复现和推广应用。
- **公平比较**：采用最严苛的预算（每帧每类每轮1个像素），且与多种基线在同一框架下比较，结论可信。
- **跨架构验证**：在四种不同架构下性能稳定保持密集监督的93%以上。

## 8. 不足与局限

- **实验覆盖有限**：仅在两个遥感数据集（RGB/IRRG、0.09~0.24m）上验证，未涵盖SAR、高光谱、多光谱卫星图像，也未扩展到医学、工业等其他领域。作者自承仅覆盖“遥感航空影像家族”。
- **缺乏多标注者研究**：BsB Aerial仅由单一专家标注，未验证不同专家之间的变异性对迭代过程的影响。作者指出这是最关键的后续工作。
- **认知与效率考量未量化**：虽提及点点击的认知优势（避免边界歧义、可延迟决策），但未进行用户时间实验或疲劳度测量。仅给出密集矢量化的参考时间（每汽车7秒等），缺乏iSAGE实际时间测量。
- **外部基线复现问题**：Vaihingen对比方法的结果引用自EasySeg原文，而非在相同条件下复现，存在实现细节差异的可能。
- **单像素点击作为唯一原始点**：未探索多模式混合（如某些区域使用scribbles、排斥点击等），可能限制某些场景下的效率。
- **EWDL的局限性**：设计用于稀疏监督，在密集标注且存在错误时可能过度惩罚，但该场景不在论文范围内。
- **推理时间成本**：每轮迭代需重训练模型（100 epochs），对于大规模数据集可能耗时；但论文强调迭代次数很少（实验中仅5轮）即可收敛。
