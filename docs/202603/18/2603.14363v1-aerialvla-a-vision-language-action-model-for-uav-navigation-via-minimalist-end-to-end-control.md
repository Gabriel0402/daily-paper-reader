## 摘要
无人机（UAV）的视觉语言导航（VLN）要求在动态 3D 环境中进行复杂的视觉解释和连续控制。现有的分层方法依赖于密集的先验指导或辅助目标检测器，从而产生了语义鸿沟并限制了真正的自主性。我们提出了 AerialVLA，这是一个极简的端到端视觉-语言-动作框架，可将原始视觉观测和模糊语言指令直接映射为连续的物理控制信号。首先，我们引入了一种精简的双视图感知策略，在减少视觉冗余的同时保留了前向导航和精确落地的关键线索，这进一步促进了未来的仿真到现实迁移。为了恢复真正的自主性，我们部署了一种仅源自机载传感器的模糊方向提示机制，完全消除了对密集先验指导的依赖。最后，我们构建了一个统一的控制空间，将连续的 3 自由度（3-DoF）运动学指令与内在降落信号相结合，使智能体无需外部目标检测器即可实现精准降落。在 TravelUAV 基准测试上的大量实验表明，AerialVLA 在已知环境中达到了最先进的性能。此外，它在未知场景中表现出卓越的泛化能力，成功率接近领先基线模型的三倍，验证了以自主为中心的极简范式比复杂的模块化系统能捕获更稳健的视觉-运动表征。

## Abstract
Vision-Language Navigation (VLN) for Unmanned Aerial Vehicles (UAVs) demands complex visual interpretation and continuous control in dynamic 3D environments. Existing hierarchical approaches rely on dense oracle guidance or auxiliary object detectors, creating semantic gaps and limiting genuine autonomy. We propose AerialVLA, a minimalist end-to-end Vision-Language-Action framework mapping raw visual observations and fuzzy linguistic instructions directly to continuous physical control signals. First, we introduce a streamlined dual-view perception strategy that reduces visual redundancy while preserving essential cues for forward navigation and precise grounding, which additionally facilitates future simulation-to-reality transfer. To reclaim genuine autonomy, we deploy a fuzzy directional prompting mechanism derived solely from onboard sensors, completely eliminating the dependency on dense oracle guidance. Ultimately, we formulate a unified control space that integrates continuous 3-Degree-of-Freedom (3-DoF) kinematic commands with an intrinsic landing signal, freeing the agent from external object detectors for precision landing. Extensive experiments on the TravelUAV benchmark demonstrate that AerialVLA achieves state-of-the-art performance in seen environments. Furthermore, it exhibits superior generalization in unseen scenarios by achieving nearly three times the success rate of leading baselines, validating that a minimalist, autonomy-centric paradigm captures more robust visual-motor representations than complex modular systems.

### 1. 核心问题与研究背景
*   **核心问题**：现有的无人机视觉语言导航（UAV-VLN）方法普遍依赖“双重拐杖”：一是**密集预言机引导**（依赖每一步的精确指令，如“向右转”），使智能体沦为被动跟随者；二是**外部目标检测器**（如 Grounding DINO），用于触发降落，导致感知与控制链路断裂。
*   **研究背景**：在动态 3D 环境中，无人机需要处理 6 自由度的复杂运动。现有模块化系统存在严重的语义鸿沟、推理延迟高且在未知场景中泛化能力差。

### 2. 方法论：极简端到端 VLA 框架
AerialVLA 建立在 OpenVLA-7B 骨干网络之上，核心思想是**将原始视觉和模糊指令直接映射为物理控制信号**。
*   **极简双视图感知**：放弃冗余的多摄像头阵列，仅融合**前视**（避障与识别）和**下视**（对准与降落）视图，垂直拼接为 224x224 图像，通过 SigLIP 和 DINOv2 编码器提取特征。
*   **模糊定向提示（Fuzzy Directional Prompting）**：利用机载 IMU/GPS 获取目标的大致方位，将其离散化为粗粒度的语义标签（如“在你右后方”），迫使模型进行主动视觉寻址而非被动指令跟随。
*   **数值动作标记化（Numerical Tokenization）**：将连续的 3 自由度动作空间（前向位移 $\Delta x$、垂直位移 $\Delta z$、偏航角变化 $\Delta \psi$）映射到 LLM 词表已有的数字 Token（0-98）。这利用了预训练模型对数值大小的先验理解，加速收敛。
*   **内在降落机制**：将降落任务统一到行为克隆目标中，通过预测 `LAND` 标记或零位移信号实现端到端停机，无需外部检测器。
*   **几何一致性监督**：利用侧向深度图过滤掉专家数据中因反应延迟导致的歧义样本，确保模型学习到正确的避障逻辑。

### 3. 实验设计
*   **数据集与基准**：使用 **TravelUAV** 基准（UAV-Need-Help 任务），包含约 1.2 万条人类驾驶轨迹。
*   **测试场景**：分为三个子集：**Seen**（已知环境）、**Unseen Object**（未知目标类别）、**Unseen Map**（全新地图/几何环境）。
*   **对比方法**：

*   **评价指标**：导航误差（NE）、成功率（SR）、预言机成功率（OSR）、路径长度加权成功率（SPL）。

### 5. 实验数量与充分性
*   **实验规模**：在三个不同维度的测试集上进行了全面评估，涵盖了从简单到困难的多个等级。
*   **消融实验**：针对数据清洗（几何过滤）、感知视图数量（2 视图 vs 5 视图）、动作标记化方式（数值 Token vs 自定义 Token）进行了对比。

### 6. 主要结论与发现
*   **泛化能力惊人**：在完全陌生的地图中，AerialVLA 的成功率（37.58%）是之前最强基线 LongFly（11.27%）的 **3.3 倍**。
*   **极简胜过复杂**：减少视觉冗余（双视图优于五视图）能有效防止模型过拟合到背景杂波中。
*   **端到端优势**：统一的控制空间和内在降落机制显著提高了从“接近目标”到“成功降落”的转化效率。

### 7. 优点与亮点
*   **高自主性**：摆脱了对密集外部指令和黑盒检测器的依赖，更接近真实世界的部署需求。
*   **推理高效**：通过数值 Token 化和极简感知设计，降低了系统复杂度和推理延迟。
*   **零样本迁移**：展现了强大的视觉伺服能力，能够处理训练集中未见过的目标几何形状。

### 8. 不足与局限
*   **缺乏全局记忆**：由于采用纯反应式控制（Reactive Policy）以降低延迟，在具有高度重复结构的城市地图中，智能体偶尔会陷入局部循环，缺乏全局回溯能力。
*   **行为保守**：在极端分布外（OOD）场景（如目标被茂密树冠完全遮挡）下，模型倾向于安全悬停而非进行多阶段的主动探索。
*   **仿真依赖**：目前主要在 AirSim 仿真环境中验证，尚未展示在真实物理环境中的迁移表现。
