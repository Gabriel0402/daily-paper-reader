## 摘要
视觉-语言-动作（VLA）模型旨在根据视觉观测和自然语言指令控制机器人进行操作。然而，现有的分层和自回归范式通常会引入架构开销，面临时间不一致性和长时程误差累积的问题，并且缺乏在不增加额外模块的情况下捕获环境动力学的机制。为此，我们提出了 MMaDA-VLA，这是一个完全原生的预训练扩散 VLA 大模型，它在单一框架中统一了多模态理解与生成。我们的核心思想是一种原生的离散扩散公式，它将语言、图像和连续机器人控制嵌入到一个离散的 token 空间中，并使用掩码 token 去噪训练单个骨干网络，以并行地联合生成未来的目标观测和动作块（action chunk）。迭代去噪实现了全局、无序的细化，在不使用辅助世界模型的情况下，将动作锚定在预测的未来视觉结果中，从而提高了长时程一致性。在仿真基准测试和真实世界任务中的实验表明，该模型达到了最先进的性能，在 LIBERO 上实现了 98.0% 的平均成功率，在 CALVIN 上实现了 4.78 的平均长度。

## Abstract
Vision-Language-Action (VLA) models aim to control robots for manipulation from visual observations and natural-language instructions. However, existing hierarchical and autoregressive paradigms often introduce architectural overhead, suffer from temporal inconsistency and long-horizon error accumulation, and lack a mechanism to capture environment dynamics without extra modules. To this end, we present MMaDA-VLA, a fully native pre-trained large diffusion VLA model that unifies multi-modal understanding and generation in a single framework. Our key idea is a native discrete diffusion formulation that embeds language, images, and continuous robot controls into one discrete token space and trains a single backbone with masked token denoising to jointly generate a future goal observation and an action chunk in parallel. Iterative denoising enables global, order-free refinement, improving long-horizon consistency while grounding actions in predicted future visual outcomes without auxiliary world models. Experiments across simulation benchmarks and real-world tasks show state-of-the-art performance, achieving 98.0% average success on LIBERO and 4.78 average length on CALVIN.

*   **核心目标**：提出一种原生的、预训练的大型离散扩散VLA模型，旨在单一框架内统一多模态理解与生成，通过并行预测未来视觉目标和动作序列来增强机器人的操作精度和长程一致性。

### 2. 方法论
*   **核心思想**：采用**原生离散扩散（Native Discrete Diffusion）**公式，将语言指令、图像观测和连续机器人动作全部映射到统一的离散 Token 空间中，利用掩码 Token 去噪（Masked Token Denoising）任务进行训练。
*   **关键技术细节**：
    *   **统一 Token 化**：文本使用 LLaDA 分词器，图像使用 MAGVIT-v2 量化器，动作则将每个维度离散化为 256 个 bin。
    *   **并行生成架构**：模型在推理时同时生成“未来目标观测图像”和“动作块（Action Chunk）”。这种设计允许动作生成参考预测的视觉演变，起到隐式世界模型的作用。
    *   **混合注意力机制（Hybrid Attention）**：在模态内部采用双向全注意力（实现全局信息交换），在模态之间采用因果注意力（确保信息流向正确），从而解耦生成目标并稳定特征。
    *   **迭代去噪推理**：通过多步迭代细化 Token，避免了自回归模型中动作维度之间人为设定的顺序偏见。
    *   **KV 缓存优化**：引入训练无关的缓存框架，针对生成部分的稀疏演变进行选择性刷新，以降低扩散模型的推理延迟。

### 6. 主要结论与发现
*   **性能领先**：MMaDA-VLA 在 LIBERO 上达到 **98.0%** 的平均成功率，在 CALVIN 上达到 **4.78** 的平均完成长度，均刷新了纪录。
*   **长程一致性**：通过并行生成目标图像和动作块，模型在长程任务（如 CALVIN 的连续 5 个子任务）中的表现显著优于纯动作预测模型。
*   **世界模型效应**：消融实验证明，预测未来观测图像能显著提升动作生成的质量（在 CALVIN 上平均长度提升了 0.48）。
*   **预训练价值**：大规模跨具身预训练显著增强了模型的泛化能力，使其在未见过的环境和任务中表现更稳健。

### 7. 优点
*   **架构优雅**：将理解与生成、视觉与动作统一在离散扩散框架下，无需额外的策略头或复杂的世界模型模块。
*   **无序细化**：利用扩散模型的全局细化能力，解决了自回归模型在处理无序动作维度时的局限性。
*   **高效推理**：通过 KV 缓存和并行解码技术，缓解了扩散模型通常面临的计算开销问题。

### 8. 不足与局限
*   **视觉保真度有限**：生成的未来目标图像在细节（如夹爪几何形状、微小物体）上存在模糊，虽然足以引导动作，但尚未达到高质量视频生成的水平。
*   **推理延迟挑战**：尽管有缓存优化，但多步迭代去噪的实时性仍可能低于单步前向传播的 VLA 模型，在极高频控制场景下可能受限。
*   **数据依赖**：模型性能高度依赖于大规模高质量的机器人演示数据，对于数据稀缺的新型机器人形态可能需要较长的微调过程。
