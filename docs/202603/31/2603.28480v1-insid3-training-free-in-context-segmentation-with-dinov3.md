## 摘要
上下文分割 (In-context segmentation, ICS) 旨在给定一个带标注的视觉示例时，分割任意概念（如物体、部件或个性化实例）。现有工作通常依赖于：(i) 微调视觉基础模型 (VFM)，这虽能提升域内性能但会损害泛化能力；或 (ii) 组合多个冻结的 VFM，这虽保留了泛化性但增加了架构复杂性且分割粒度固定。我们从极简主义视角重新审视 ICS 并提出：单个自监督骨干网络能否在无需任何监督或辅助模型的情况下，同时支持语义匹配与分割？研究表明，DINOv3 的大规模密集自监督特征展现出极强的空间结构和语义对应性。据此，我们提出了 INSID3，这是一种免训练的方法，仅需冻结的 DINOv3 特征即可根据上下文示例在不同粒度下分割概念。INSID3 在单样本语义分割、部件分割及个性化分割任务中均达到了最先进水平，在参数量减少 3 倍且无需掩码或类别级监督的情况下，mIoU 提升了 7.5%。代码已开源至 https://github.com/visinf/INSID3 。

## Abstract
In-context segmentation (ICS) aims to segment arbitrary concepts, e.g., objects, parts, or personalized instances, given one annotated visual examples. Existing work relies on (i) fine-tuning vision foundation models (VFMs), which improves in-domain results but harms generalization, or (ii) combines multiple frozen VFMs, which preserves generalization but yields architectural complexity and fixed segmentation granularities. We revisit ICS from a minimalist perspective and ask: Can a single self-supervised backbone support both semantic matching and segmentation, without any supervision or auxiliary models? We show that scaled-up dense self-supervised features from DINOv3 exhibit strong spatial structure and semantic correspondence. We introduce INSID3, a training-free approach that segments concepts at varying granularities only from frozen DINOv3 features, given an in-context example. INSID3 achieves state-of-the-art results across one-shot semantic, part, and personalized segmentation, outperforming previous work by +7.5 % mIoU, while using 3x fewer parameters and without any mask or category-level supervision. Code is available at https://github.com/visinf/INSID3 .

### 论文总结：INSID3: Training-Free In-Context Segmentation with DINOv3

#### 1. 论文的核心问题与整体含义
上下文分割（In-Context Segmentation, ICS）的任务是：给定一个或多个带有掩码（Mask）标注的参考图像，在目标图像中分割出相同的视觉概念（如特定物体、部件或实例）。
*   **研究动机**：现有的 ICS 方法主要分为两类：一是**微调类**（如 SegGPT），虽然在特定领域表现好，但泛化能力差且需要大量标注数据；二是**多模型组合类**（如 DINOv2 + SAM），虽然泛化性强，但架构复杂、计算开销大，且受限于 SAM 固定的分割粒度。
*   **核心问题**：能否仅使用一个单一的、冻结的自监督骨干网络，在不进行任何微调或模型组合的情况下，实现高质量、多粒度的上下文分割？

#### 2. 方法论
INSID3 提出了一种极简的免训练流程，完全依赖 DINOv3 的自监督特征。其核心步骤包括：
*   **位置去偏（Positional Debiasing）**：作者发现 DINOv3 特征存在“位置偏差”，即不同图像中相同坐标的特征倾向于产生虚假匹配。INSID3 通过输入噪声图像估计位置子空间，并将其从特征中投影去除，从而显著提升跨图像的语义匹配精度。
*   **细粒度聚类（Fine-grained Clustering）**：利用 DINOv3 强大的局部一致性，对目标图像特征进行**凝聚层次聚类**（Agglomerative Clustering），将图像分解为具有语义意义的细粒度区域（簇）。
*   **种子簇选择（Seed-cluster Selection）**：
    *   **候选定位**：通过反向对应（Backward Correspondence）过滤掉无关区域。
    *   **语义匹配**：在去偏特征空间中，计算参考原型与目标簇之间的相似度，选出最匹配的“种子簇”。
*   **簇聚合（Cluster Aggregation）**：为了恢复物体的完整范围，INSID3 结合了**跨图像语义相似度**和**目标图内自相似度**，将与种子簇高度相关的其他候选簇进行合并，生成最终掩码。

*   **实验规模**：论文在 9 个不同领域的基准数据集上进行了广泛测试，涵盖了从日常物体到医疗、遥感、水下等极端域外场景。

#### 6. 论文的主要结论与发现
*   **DINOv3 的潜力**：大规模自监督预训练的 DINOv3 已经具备了极强的密集特征表达能力，足以在没有掩码监督的情况下直接“涌现”出分割能力。
*   **性能领先**：INSID3 在平均 mIoU 上比之前的免训练 SOTA 方法（如 GF-SAM）高出 **7.5%**，且参数量仅为其 1/3。
*   **位置偏差的发现**：揭示了 DINOv3 中存在的系统性位置偏差，并证明简单的线性投影去偏不仅能提升 ICS 性能，还能显著增强语义对应（Semantic Correspondence）任务的精度（PCK 提升高达 6.6%）。

#### 7. 优点
*   **极简高效**：无需微调、无需解码器、无需 SAM 等辅助模型，仅靠单一骨干网络完成任务。
*   **极强的泛化性**：在医疗、遥感等与预训练数据差异巨大的领域表现优异，远超微调类方法。
*   **多粒度支持**：能够灵活处理从整体物体到精细部件（如“狗的耳朵”）的分割，不受限于预定义的掩码提议。

#### 8. 不足与局限
*   **单目标限制**：目前主要针对单类概念进行分割，若目标图中存在多个不同类别的参考概念，需要多次推理。
*   **提示形式**：目前仅支持掩码提示（Mask Prompt），尚未直接支持点（Point）或框（Box）等更轻量级的交互方式。
*   **聚类复杂度**：凝聚层次聚类虽然效果好，但其计算复杂度随图像分辨率增加呈二次方增长，在极高分辨率下可能存在性能瓶颈。
