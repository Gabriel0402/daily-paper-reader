## 摘要
视觉-语言模型（VLM）在不同视角下对同一物体的描述往往不一致，这阻碍了具身智能体随时间构建一致语义表示的能力。以往的方法通过离线多视图聚合或将探索、数据关联和描述学习解耦的多阶段流水线来解决不一致性，但对先前观察到的物体的推理能力有限。在本文中，我们提出了一种统一的记忆增强视觉-语言智能体，它在单个自回归框架内同时处理数据关联、物体描述和探索策略。该模型处理当前的 RGB 观测、自顶向下的探索地图以及序列化为物体级 token 的物体级情节记忆，从而确保在长序列中保持持久的物体身份和语义一致性。为了以自监督方式训练模型，我们利用基于分歧的策略和伪描述模型在逼真的 3D 环境中收集了一个数据集，该伪描述模型强制要求多视图描述历史之间的一致性。在手动标注的物体级测试集上进行的广泛评估表明，与基准模型相比，标准描述分数提高了高达 11.86%，描述自相似性提高了 7.39%，同时通过紧凑的场景表示实现了可扩展的性能。代码、模型权重和数据可在 https://hsp-iit.github.io/epos-vlm/ 获取。

## Abstract
Vision-Language Models (VLMs) often yield inconsistent descriptions of the same object across viewpoints, hindering the ability of embodied agents to construct consistent semantic representations over time. Previous methods resolved inconsistencies using offline multi-view aggregation or multi-stage pipelines that decouple exploration, data association, and caption learning, with limited capacity to reason over previously observed objects. In this paper, we introduce a unified, memory-augmented Vision-Language agent that simultaneously handles data association, object captioning, and exploration policy within a single autoregressive framework. The model processes the current RGB observation, a top-down explored map, and an object-level episodic memory serialized into object-level tokens, ensuring persistent object identity and semantic consistency across extended sequences. To train the model in a self-supervised manner, we collect a dataset in photorealistic 3D environments using a disagreement-based policy and a pseudo-captioning model that enforces consistency across multi-view caption histories. Extensive evaluation on a manually annotated object-level test set, demonstrate improvements of up to +11.86% in standard captioning scores and +7.39% in caption self-similarity over baseline models, while enabling scalable performance through a compact scene representation. Code, model weights, and data are available at https://hsp-iit.github.io/epos-vlm/.

#### 1. 核心问题与整体含义
在具身 AI 领域，视觉-语言模型（VLM）在不同视角、距离或遮挡情况下，对同一个物理对象的描述往往存在不一致性（语义漂移）。这种不一致性导致智能体难以构建持久的语义地图，影响后续的推理和决策。现有的解决方法通常将探索、数据关联和描述生成解耦为多个独立阶段，或者依赖离线的后期聚合，缺乏实时处理能力和主动消除歧义的策略。本文提出的 **EPOS-VLM** 旨在通过一个统一的、记忆增强的自回归框架，让智能体在导航过程中实现持久且语义一致的物体描述。

#### 2. 方法论
EPOS-VLM 的核心思想是将具身任务转化为一个统一的序列建模问题，通过引入**物体级情节记忆（Episodic Object Memory）**来增强 VLM 的长程推理能力。

*   **统一框架**：在单个自回归 Transformer 中同时预测：
    1.  **数据关联**：通过 `[MATCH]` token 将当前帧检测到的物体与记忆中的持久 ID 关联，或标记为新物体（`[NEW_ID]`）。
    2.  **物体描述**：为当前观察到的物体生成视角无关的描述。
    3.  **导航动作**：预测下一步动作（如前进、左转等）。
*   **记忆 token 化**：将物体的持久 ID、离散化的 3D 坐标、以及该物体过去的描述历史（及其出现频率）序列化为文本 token，直接作为 VLM 的上下文输入。
*   **自监督训练流程**：
    *   **主动探索**：使用基于“描述分歧”的启发式策略收集数据，优先访问那些描述不稳定的物体。
    *   **3D-CPS 伪标签生成**：利用 3D 几何信息选择最佳视角，并结合多视图描述频率，通过大型 VLM 生成高质量、视角无关的物体伪描述作为训练目标。
*   **端到端微调**：使用 LoRA 技术对预训练 VLM（如 Qwen3-VL-2B）进行微调，使其学会利用记忆来纠正当前的感知偏差。

#### 6. 主要结论与发现
*   **性能提升**：EPOS-VLM 在标准描述指标（SPICE）上比最强基准提升了 **11.86%**，在语义一致性（Mean CS）上提升了 **7.39%**。
*   **记忆的重要性**：情节记忆不仅帮助模型识别“老熟人”，还能通过历史描述的统计信息消除当前视角的视觉歧义。
*   **主动探索的价值**：相比于单纯的几何覆盖，针对语义不确定性进行探索能显著提高智能体对环境的理解深度。
*   **可扩展性**：基于文本 token 的稀疏记忆表示在处理长序列任务时，比传统的 3D 点云方法更具计算效率。

#### 7. 优点
*   **架构优雅**：将感知、记忆、关联和行动统一在单个自回归模型中，避免了多阶段系统的误差累积。
*   **紧凑的表示**：通过 token 化记忆解决了具身智能体长期运行时的内存爆炸问题。
*   **闭环优化**：模型能够根据语义不确定性主动选择观测视角，体现了“感知-行动”的紧密耦合。

#### 8. 不足与局限
*   **依赖外部模块**：模型仍依赖预训练的实例分割模型（Mask2Former），若分割失败，后续的数据关联和描述都会受影响。
*   **环境局限**：实验主要在静态、光照稳定的模拟环境中进行，未探讨动态物体或极端传感器噪声对记忆维护的挑战。
*   **实时性挑战**：虽然推理时间稳定，但 0.7s/步 的速度在需要极高反应速度的实时机器人任务中可能仍显不足。
