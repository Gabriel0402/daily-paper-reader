## 摘要
我们研究了语言条件视觉导航（LCVN），其中具身智能体仅需根据初始的第一视角观测来遵循自然语言指令。在无法获取目标图像的情况下，智能体必须依靠语言来引导其感知和连续控制，这使得语言对齐（grounding）问题变得尤为具有挑战性。我们将此问题表述为以语言指令为条件的开环轨迹预测，并引入了 LCVN 数据集。该数据集是一个包含 39,016 条轨迹和 117,048 条经人工验证指令的基准，支持在多种环境和指令风格下进行可重复研究。基于该数据集，我们开发了 LCVN 框架，通过两个互补的模型系列将语言对齐、未来状态预测和动作生成联系起来。第一个系列结合了基于扩散的世界模型 LCVN-WM 和在世界模型潜空间中训练的 Actor-Critic 智能体 LCVN-AC。第二个系列 LCVN-Uni 采用自回归多模态架构，同时预测动作和未来观测。实验表明，这两个系列各有优势：前者能提供更具时间连贯性的预测展开（rollouts），而后者在未知环境中的泛化表现更好。总而言之，这些观察结果表明了在统一任务设置下共同研究语言对齐、想象和策略学习的价值，LCVN 为进一步探索语言条件世界模型提供了具体基础。代码开源于 https://github.com/F1y1113/LCVN。

## Abstract
We study language-conditioned visual navigation (LCVN), in which an embodied agent is asked to follow a natural language instruction based only on an initial egocentric observation. Without access to goal images, the agent must rely on language to shape its perception and continuous control, making the grounding problem particularly challenging. We formulate this problem as open-loop trajectory prediction conditioned on linguistic instructions and introduce the LCVN Dataset, a benchmark of 39,016 trajectories and 117,048 human-verified instructions that supports reproducible research across a range of environments and instruction styles. Using this dataset, we develop LCVN frameworks that link language grounding, future-state prediction, and action generation through two complementary model families. The first family combines LCVN-WM, a diffusion-based world model, with LCVN-AC, an actor-critic agent trained in the latent space of the world model. The second family, LCVN-Uni, adopts an autoregressive multimodal architecture that predicts both actions and future observations. Experiments show that these families offer different advantages: the former provides more temporally coherent rollouts, whereas the latter generalizes better to unseen environments. Taken together, these observations point to the value of jointly studying language grounding, imagination, and policy learning in a unified task setting, and LCVN provides a concrete basis for further investigation of language-conditioned world models. The code is available at https://github.com/F1y1113/LCVN.

*   **整体含义**：论文旨在统一语言对齐（Grounding）、世界建模（想象未来）与策略学习，探索如何让具身智能体在缺乏实时环境反馈的情况下，通过“脑补”完成复杂的导航任务。

### 2. 方法论：核心思想与关键技术
论文提出了两个互补的模型家族来解决 LCVN 任务：

*   **LCVN-WM + LCVN-AC（基于扩散的世界模型家族）**：
    *   **LCVN-WM**：基于扩散变换器（DiT）构建。利用 **Diffusion Forcing** 技术，对上下文窗口内的潜变量施加不同强度的噪声，增强长程时间建模能力。它以动作和语言指令为条件，预测下一帧的潜空间状态。
    *   **LCVN-AC**：在世界模型的潜空间内训练的 Actor-Critic 智能体。它通过“内在奖励”（预测状态与专家状态的一致性）进行学习，避免了高维图像处理的开销。
*   **LCVN-Uni（自回归多模态家族）**：
    *   **核心思想**：将动作预测和视觉想象统一为一个序列建模任务。
    *   **技术细节**：基于 Chameleon/Anole-7B 架构，使用 VQ 编码器处理图像、BPE 处理文本、Bin 编码器处理连续动作。模型在单次前向传播中同时输出下一跳的动作标记和视觉标记。
*   **LCVN 数据集**：构建了一个包含 3.9 万条轨迹的大规模数据集，并利用 Qwen-VL-2.5 生成并经人工校验了三种风格（简洁、复杂、基于地标）的共 11.7 万条指令。

### 3. 实验设计
*   **数据集/场景**：整合了 Go Stanford、ReCon、SCAND、HuRoN 四个数据集用于训练和域内评估；使用 TartanDrive 作为完全未见的测试场景（Zero-shot 评估）。
*   **Benchmark 评价指标**：
    *   **导航质量**：成功率（SR）、绝对轨迹误差（ATE）、相对位姿误差（RPE）。
    *   **想象质量**：SSIM、PSNR、LPIPS 以及更符合人类感知的 DreamSim。
*   **对比方法**：对比了 Diamond（基于 UNet 的扩散模型）、NWM（导航世界模型）及其语言增强版。

### 4. 资源与算力
*   **硬件环境**：所有模型均在 **4 张 NVIDIA A100 (80GB)** GPU 上训练。
*   **训练细节**：
    *   LCVN-WM 使用 AdamW 优化器，学习率 1e-4。
    *   LCVN-Uni 采用了 **LoRA 微调**（秩为 16），仅更新 Transformer 的 QKV 投影层，训练 20 个 epoch。
    *   图像被压缩为 32x32 的潜变量（WM）或 448x448 分辨率下的视觉标记（Uni）。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了大规模的对比实验，涵盖了 5 个原始数据集的迁移。

    2.  **上下文窗口大小**（k=1, 2, 4）对性能的权衡。
    3.  **指令风格**：验证地标性指令是否比简洁指令更有效。

### 6. 主要结论与发现
*   **互补性**：基于扩散的模型（LCVN-WM）生成的视觉序列在时间上更连贯，适合短程精确控制；而自回归模型（LCVN-Uni）在未见过的环境中泛化能力更强。
*   **语言的重要性**：移除语言指令会导致导航成功率大幅下降，证明语言不仅是目标描述，更是约束世界模型演化的关键上下文。
*   **指令风格影响**：基于地标（Landmark-based）的指令对导航的引导效果最好，而过于复杂的描述（Intricate）有时会引入噪声。

### 7. 优点（亮点）
*   **任务创新**：首次系统性地定义了“开环语言条件视觉导航”任务，强调了“想象力”在具身智能中的作用。
*   **高质量数据集**：LCVN 数据集提供了多风格的人工校验指令，是该领域宝贵的开源资源。
*   **双路径探索**：同时探索了扩散模型和自回归模型两条主流技术路线，并给出了详尽的对比分析。

### 8. 不足与局限
*   **开环限制**：虽然开环评估能体现预测能力，但在极长距离导航中，缺乏环境反馈会导致误差累积（Covariate Shift），实际应用中可能仍需结合闭环修正。
*   **推理开销**：LCVN-Uni 基于 7B 参数的大模型，尽管使用了 LoRA，其推理延迟（每步约 20.5 秒）远高于扩散模型（每步约 6.4 秒），难以满足实时机器人控制的需求。
*   **空间分辨率权衡**：在自回归模型中，为了增加时间上下文（更多帧），不得不降低单帧的视觉标记数量，这在一定程度上牺牲了空间感知精度。
