## 摘要
多模态大语言模型（MLLMs）正越来越多地应用于像素级视觉任务，但其内在的空间理解能力仍未得到充分理解。我们通过对整个 MLLM 流水线（视觉编码器、适配器和 LLM）进行逐层线性探测评估，研究了其分割能力。我们进一步进行了基于干预的注意力剔除（attention knockout）分析，以测试跨 token 注意力是否逐步优化了视觉表示，并评估了图像 token 间的双向注意力对空间一致性的影响。我们的分析表明，适配器引入了分割表示的性能下降，但 LLM 层通过注意力介导的优化逐步实现了恢复，其中正确分类的 token 会引导被错误分类的邻近 token 转向正确的标签。在早期的图像 token 位置，这种恢复受到因果注意力的限制，而图像 token 间的双向注意力可以缓解这一问题。这些发现为 MLLM 如何处理视觉信息以进行分割提供了机理性解释，为未来具备分割能力的模型设计提供了参考。

## Abstract
Multimodal Large Language Models (MLLMs) are increasingly applied to pixel-level vision tasks, yet their intrinsic capacity for spatial understanding remains poorly understood. We investigate segmentation capacity through a layerwise linear probing evaluation across the entire MLLM pipeline: vision encoder, adapter, and LLM. We further conduct an intervention based attention knockout analysis to test whether cross-token attention progressively refines visual representations, and an evaluation of bidirectional attention among image tokens on spatial consistency. Our analysis reveals that the adapter introduces a segmentation representation drop-off, but LLM layers progressively recover through attention-mediated refinement, where correctly classified tokens steer misclassified neighbors toward the correct label. At early image token positions, this recovery is bounded by causal attention, which bidirectional attention among image tokens alleviates. These findings provide a mechanistic account of how MLLMs process visual information for segmentation, informing the design of future segmentation-capable models.

### 2. 方法论
论文提出了一个系统性的干预驱动框架，包含三个核心技术手段：
*   **逐层线性探测（Layerwise Linear Probing）**：在视觉编码器输出、适配器输出以及 LLM 的每一层之后，训练独立的线性分类器（Linear Probe），评估该层特征对像素级类别的区分能力。
*   **注意力剔除干预（Attention Knockout）**：为了验证 LLM 层的性能提升是否源于注意力机制，作者在推理时选择性地屏蔽特定类别的注意力信号。

    *   **屏蔽正确类别**：观察是否会阻碍纠错，从而验证“语义锚点”假说。
*   **图像 Token 双向注意力（Bidirectional Attention）**：针对标准 LLM 的因果注意力（Causal Attention）导致早期图像 Token 缺乏全局上下文的问题，修改注意力掩码，允许图像 Token 之间进行双向交互，同时保持文本生成的因果性。

### 6. 主要结论与发现
*   **适配器性能下降（Drop-off）**：适配器在将视觉特征投影到 LLM 空间时，为了对齐模态，会牺牲细粒度的空间保真度，导致分割性能显著下降。
*   **LLM 逐步恢复（Recovery）**：随着层数加深，LLM 层通过跨 Token 注意力逐步修复受损的表征，性能稳步回升。
*   **语义锚点机制**：正确分类的 Token 在注意力机制中充当“锚点”，引导周围分类错误的 Token 转向正确标签。
*   **上下文饥饿（Context Starvation）**：在因果注意力下，图像序列开头的 Token 因为看不到后面的图像内容，恢复效果最差；引入双向注意力可显著提升早期 Token 的准确率（如第一个 Token 准确率提升 23.2%）。
*   **编码器兼容性**：与文本对齐的编码器（CLIP, SigLIP）在 LLM 层中的恢复效果优于纯视觉编码器（DINOv2）。

### 7. 优点
*   **机理性深度**：不仅展示了“是什么”，还通过干预实验解释了“为什么”，揭示了 LLM 内部的自我修正机制。
*   **诊断性强**：指出了适配器是空间信息的瓶颈，为未来 MLLM 架构优化（如改进适配器设计）提供了明确方向。
*   **实用性建议**：证明了在不损害语言能力的前提下，对图像 Token 使用双向注意力是提升空间理解的有效手段。

### 8. 不足与局限
*   **架构局限**：主要针对适配器风格（Adapter-style）的 MLLM，对于交织型（Interleaved）或原生多模态架构的适用性有待验证。
*   **探测器限制**：线性探测可能低估了特征中隐藏的非线性空间能力。
