## 摘要
近期利用多模态大语言模型（MLLM）的分割方法在物体级分割和增强空间感知方面表现出了可靠的性能。然而，几乎所有先前的方法都主要依赖专门的掩码解码器（mask decoders）来从生成的分割相关嵌入和视觉特征中解析掩码，或者引入多个额外的 Token 进行辅助。本文旨在研究是否以及如何通过单个分割嵌入（SELF1E）从 MLLM 自身解锁分割能力，并在无需外部解码器的情况下取得具有竞争力的结果。为此，我们的方法针对 MLLM 中像素重组（pixel-shuffled）图像特征导致的分辨率降低这一根本局限。首先，我们保留原始未压缩分辨率的图像特征，并使用从 MLLM 处理后的压缩特征中提取的残差特征对其进行填充，从而提高特征精度。随后，我们分别对经过和未经 LLM 处理的图像特征集成像素逆重组（pixel-unshuffle）操作，以释放压缩特征的细节，并在未压缩分辨率下放大残差特征，从而进一步增强填充特征的分辨率。此外，我们重新设计了具有双重感知路径（即图像到图像和图像到分割）的注意力掩码，实现了像素与分割 Token 之间丰富的特征交互。在多个分割任务上的全面实验验证了 SELF1E 达到了与基于专门掩码解码器的方法相当的性能，证明了在 MLLM 中实现无解码器分割的可行性。项目主页：https://github.com/ANDYZAQ/SELF1E。

## Abstract
Recent segmentation methods leveraging Multi-modal Large Language Models (MLLMs) have shown reliable object-level segmentation and enhanced spatial perception. However, almost all previous methods predominantly rely on specialist mask decoders to interpret masks from generated segmentation-related embeddings and visual features, or incorporate multiple additional tokens to assist. This paper aims to investigate whether and how we can unlock segmentation from MLLM itSELF with 1 segmentation Embedding (SELF1E) while achieving competitive results, which eliminates the need for external decoders. To this end, our approach targets the fundamental limitation of resolution reduction in pixel-shuffled image features from MLLMs. First, we retain image features at their original uncompressed resolution, and refill them with residual features extracted from MLLM-processed compressed features, thereby improving feature precision. Subsequently, we integrate pixel-unshuffle operations on image features with and without LLM processing, respectively, to unleash the details of compressed features and amplify the residual features under uncompressed resolution, which further enhances the resolution of refilled features. Moreover, we redesign the attention mask with dual perception pathways, i.e., image-to-image and image-to-segmentation, enabling rich feature interaction between pixels and the segmentation token. Comprehensive experiments across multiple segmentation tasks validate that SELF1E achieves performance competitive with specialist mask decoder-based methods, demonstrating the feasibility of decoder-free segmentation in MLLMs. Project page: https://github.com/ANDYZAQ/SELF1E.

#### 1. 核心问题与整体含义（研究动机和背景）
目前的多模态大语言模型（MLLM）在分割任务中通常依赖**外部专门的掩码解码器**（如 SAM 或 Mask2Former）或**多个额外的分割 Token**。这种架构增加了系统的复杂性，且过度依赖外部基础模型。
*   **核心挑战**：MLLM 为了提高 LLM 的处理效率，通常采用像素重组（Pixel-shuffle）和 MLP 来压缩视觉特征，这导致了空间分辨率的严重损失，难以进行精确的像素级分割。
*   **研究动机**：探索是否能仅利用 MLLM 自身的架构，通过**单个分割 Token**（Single [SEG] Token）且**无需外部解码器**（Decoder-free）来实现高性能的图像分割。

#### 2. 方法论
作者提出了 **SELF1E**（unlocking segmentation from MLLM it**SELF** with **1 E**mbedding），核心思想是修复被压缩的视觉特征并增强其分辨率。

*   **残差特征填充 (Residual Features Refilling, RFR)**：
    *   保留视觉编码器输出的原始高分辨率特征（未压缩特征）。
    *   计算 LLM 处理前后的特征差异，提取**残差特征**（$F_R = F_{IMG} - F_{V1}$）。
    *   将残差特征上采样并融合回高分辨率特征中，使特征既具备原始的空间精度，又包含 LLM 提炼的语义信息。
*   **残差特征放大器 (Residual Features Amplifier, RFA)**：
    *   引入带有像素反重组（Pixel-unshuffle）操作的 MLP，对经过和未经 LLM 处理的特征分别进行处理。
    *   通过这种方式在更高分辨率下无缝融合残差，进一步释放压缩特征中的细节。
*   **双感知路径注意力掩码 (Attention Mask with Dual Perception Pathways)**：
    *   重新设计 LLM 内部的注意力机制。除了传统的因果掩码，增加了**图像到图像**和**图像到分割 Token**的双向交互路径。
    *   这使得 [SEG] Token 能够更好地感知全局图像上下文，同时图像 Token 也能根据分割需求进行特征对齐。
*   **掩码生成**：直接通过图像特征 $F'_{IMG}$ 与分割 Token 嵌入 $F'_{SEG}$ 的点积（Dot-product）生成预测掩码 $\hat{y}$。

*   **分辨率是关键**：证明了 MLLM 分割的瓶颈不在于 Token 数量，而在于特征压缩过程中的信息损失。

#### 7. 优点
*   **架构简洁**：实现了真正的“端到端”MLLM 分割，去除了对 SAM 等外部模型的重度依赖。
*   **高效性**：单 Token 预测大幅降低了推理延迟。
*   **特征融合巧妙**：通过残差填充和放大技术，在不改变 LLM 核心计算逻辑的前提下解决了分辨率问题。
*   **通用性强**：在多种分割任务和不同规模的基座模型上均表现稳健。

#### 8. 不足与局限
*   **VQA 能力权衡**：引入分割监督后，模型在 OCR 相关任务和知识密集型任务（如 TextVQA, OKVQA）上的表现有所下降。
*   **推理模式冲突**：重新设计的双向注意力掩码虽然提升了分割精度，但与 LLM 原生的自回归推理模式存在一定冲突，可能影响多轮对话的流畅性。
*   **模板依赖**：分割任务仍需依赖特定的文本模板来触发。
