## 摘要
视觉语言建模的最新进展催生了可提示的检测与分割系统，这些系统在推理时能够接受任意自然语言查询。在这些系统中，SAM3 通过结合 ViT-H/14 骨干网络、跨模态 Transformer 解码以及学习到的对象查询，实现了最先进的准确率。然而，SAM3 在每次前向传播中仅处理单个文本提示。检测 N 个类别需要 N 次独立执行，而每次执行都由拥有 4.39 亿参数的骨干网络占据主要开销。我们提出了“实时检测万物”（DART），这是一个无需训练的框架，通过利用一种结构不变性将 SAM3 转换为实时多类别检测器：视觉骨干网络是类别无关的，其生成的图像特征独立于文本提示。这使得骨干网络的计算可以在所有类别之间共享，将其成本从 O(N) 降低到 O(1)。结合批量多类别解码、仅检测推理以及 TensorRT FP16 部署，这些优化在不修改任何模型权重的情况下，在 3 个类别时实现了 5.6 倍的累积加速，在 80 个类别时加速比可扩展至 25 倍。在 COCO val2017 数据集（5,000 张图像，80 个类别）上，DART 在单张 RTX 4080 上实现了 55.8 AP 和 15.8 FPS（4 个类别，1008x1008），超越了在数百万个框标注上训练的专用开放词汇检测器。针对极低延迟目标，采用冻结编码器-解码器的适配器蒸馏在骨干网络耗时 13.9 毫秒的情况下实现了 38.7 AP。代码和模型可在 https://github.com/mkturkcan/DART 获取。

## Abstract
Recent advances in vision-language modeling have produced promptable detection and segmentation systems that accept arbitrary natural language queries at inference time. Among these, SAM3 achieves state-of-the-art accuracy by combining a ViT-H/14 backbone with cross-modal transformer decoding and learned object queries. However, SAM3 processes a single text prompt per forward pass. Detecting N categories requires N independent executions, each dominated by the 439M-parameter backbone. We present Detect Anything in Real Time (DART), a training-free framework that converts SAM3 into a real-time multi-class detector by exploiting a structural invariant: the visual backbone is class-agnostic, producing image features independent of the text prompt. This allows the backbone computation to be shared between all classes, reducing its cost from O(N) to O(1). Combined with batched multi-class decoding, detection-only inference, and TensorRT FP16 deployment, these optimizations yield 5.6x cumulative speedup at 3 classes, scaling to 25x at 80 classes, without modifying any model weight. On COCO val2017 (5,000 images, 80 classes), DART achieves 55.8 AP at 15.8 FPS (4 classes, 1008x1008) on a single RTX 4080, surpassing purpose-built open-vocabulary detectors trained on millions of box annotations. For extreme latency targets, adapter distillation with a frozen encoder-decoder achieves 38.7 AP with a 13.9 ms backbone. Code and models are available at https://github.com/mkturkcan/DART.

### 1. 核心问题与研究背景
*   **核心问题**：最先进的视觉语言模型（如 SAM3）虽然在开放词汇分割和检测上表现卓越，但其推理机制是“单提示词（Single-Prompt）”模式。这意味着检测 $N$ 个类别需要重复运行 $N$ 次完整的模型。
*   **研究背景**：SAM3 的 ViT-H 骨干网络拥有 4.39 亿参数，占据了约 78% 的计算开销。在多类别检测场景下，这种线性增长的计算成本（$O(N)$）使得实时推理变得不可能。
*   **研究动机**：作者观察到 SAM3 的骨干网络是“类别无关”的（Class-agnostic），即图像特征的提取不依赖于文本提示。利用这一结构不变性，可以在不重新训练模型的情况下，将其转化为高效的实时多类别检测器。

### 2. 方法论：DART 框架
DART（Detect Anything in Real Time）通过一系列无需训练的结构优化实现加速：
*   **骨干网络共享（Backbone Sharing）**：每张图像只运行一次 ViT-H 骨干网络，提取的特征供所有类别共享，将骨干网络成本从 $O(N)$ 降至 $O(1)$。
*   **批量多类别解码（Batched Multi-class Decoding）**：利用 Transformer 解码器跨模态注意力在批次维度上的独立性，将 $N$ 个类别的文本提示堆叠，在单次前向传播中完成处理。
*   **仅检测推理（Detection-only Inference）**：移除耗时的掩码（Mask）预测头，仅保留边界框回归和存在预测头。
*   **TensorRT FP16 部署与图重构**：
    *   针对 ViT-H 在 FP16 下的精度崩溃问题，将复杂的 RoPE（旋转位置嵌入）重构为实值运算，并将注意力机制重构为显式的 QK 乘法，以触发 TensorRT 的优化内核。
*   **帧间流水线（Inter-frame Pipelining）**：在视频处理中，利用 CUDA 流使当前帧的解码器运算与下一帧的骨干网络运算重叠。
*   **适配器蒸馏（Adapter Distillation）**：针对极低延迟需求，冻结 SAM3 的编码器-解码器，仅训练一个轻量级 FPN 适配器（约 5M 参数），将小型骨干网络（如 RepViT）的特征映射到 SAM3 的特征空间。
