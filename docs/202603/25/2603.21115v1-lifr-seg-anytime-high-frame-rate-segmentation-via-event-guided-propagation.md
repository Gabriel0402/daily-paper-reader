## 摘要
动态环境中的密集语义分割从根本上受到标准相机低帧率（LFR）特性的限制，这在帧与帧之间产生了关键的感知间隙。为了解决这一问题，我们引入了“任意时刻帧间语义分割”（Anytime Interframe Semantic Segmentation）：这是一项新任务，旨在仅使用单个过去的 RGB 帧和异步事件数据流来预测任意时刻的分割结果。该任务面临一个核心挑战：如何利用从稀疏且通常含有噪声的事件数据中导出的运动场，稳健地传播密集语义特征，同时减轻高动态场景中的特征退化。我们提出了 LiFR-Seg，这是一个通过随时间传播深度语义特征来直接应对这些挑战的新型框架。我们方法的核心是一个不确定性感知（uncertainty-aware）的扭曲过程，由事件驱动的运动场及其学习到的显式置信度所引导。时间记忆注意力模块进一步确保了动态场景中的连贯性。我们在 DSEC 数据集和我们贡献的一个新的高频合成基准（SHF-DSEC）上验证了我们的方法。值得注意的是，我们的 LFR 系统实现的性能（在 DSEC 上为 73.82% mIoU）与能够完全访问目标帧的高帧率（HFR）上限在统计上几乎没有区别（差距在 0.09% 以内）。这项工作为利用低帧率硬件实现稳健的高帧率感知提供了一种高效的新范式。项目页面：https://candy-crusher.github.io/LiFR_Seg_Proj/#；代码：https://github.com/Candy-Crusher/LiFR-Seg.git。

## Abstract
Dense semantic segmentation in dynamic environments is fundamentally limited by the low-frame-rate (LFR) nature of standard cameras, which creates critical perceptual gaps between frames. To solve this, we introduce Anytime Interframe Semantic Segmentation: a new task for predicting segmentation at any arbitrary time using only a single past RGB frame and a stream of asynchronous event data. This task presents a core challenge: how to robustly propagate dense semantic features using a motion field derived from sparse and often noisy event data, all while mitigating feature degradation in highly dynamic scenes. We propose LiFR-Seg, a novel framework that directly addresses these challenges by propagating deep semantic features through time. The core of our method is an uncertainty-aware warping process, guided by an event-driven motion field and its learned, explicit confidence. A temporal memory attention module further ensures coherence in dynamic scenarios. We validate our method on the DSEC dataset and a new high-frequency synthetic benchmark (SHF-DSEC) we contribute. Remarkably, our LFR system achieves performance (73.82% mIoU on DSEC) that is statistically indistinguishable from an HFR upper-bound (within 0.09%) that has full access to the target frame. This work presents a new, efficient paradigm for achieving robust, high-frame-rate perception with low-frame-rate hardware. #### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：标准 RGB 相机受限于低帧率（LFR，如 20Hz），在处理高速运动物体或突发状况时存在“感知盲区”（Perceptual Gap）。
*   **研究动机**：虽然高帧率（HFR）相机可以解决此问题，但其成本高、功耗大且带宽需求极高。事件相机（Event Camera）具有微秒级响应和高动态范围，但数据空间稀疏，缺乏丰富的语义纹理。
*   **任务定义**：论文提出了**任意时刻帧间语义分割（Anytime Interframe Semantic Segmentation）**任务。即：仅给定一帧过去的 RGB 图像 $I_t$ 和一段连续的异步事件流 $E$，预测未来任意时间点 $t+\delta t$ 的密集语义地图。该任务要求满足**因果性**（不依赖未来帧）和**任意时刻预测能力**。

#### 2. 方法论：核心思想与技术细节
LiFR-Seg 框架通过将 RGB 的深度语义特征沿事件引导的运动场进行传播来填补感知间隙，主要包含三个阶段：
*   **事件驱动的不确定性感知运动场估计**：
    *   利用 RAFT 架构从事件体素（Event Voxel）中估计密集光流 $\hat{M}$。
    *   引入 **ScoreNet** 学习一个置信度图（或称对数精度图）$S$，用于衡量光流在稀疏或噪声事件区域的可靠性。
*   **不确定性引导的特征传播**：
    *   采用 **Softmax Splatting** 技术将 $t$ 时刻的深度特征图 $F_t$ 扭曲（Warp）到 $t+\delta t$ 时刻。
    *   **关键创新**：将学习到的置信度 $S$ 作为权重引入 Splatting 公式，使得不可靠的光流向量在特征融合时贡献较小，从而减少伪影和错误传播。
*   **基于时间记忆注意力的长时一致性**：
    *   为了处理遮挡和长跨度预测，引入 **Memory Bank** 存储历史特征。
    *   通过**交叉注意力机制（Cross-Attention）**，让当前传播的特征查询历史记忆，增强语义连贯性。

#### 3. 实验设计
*   **数据集与场景**：
    *   **DSEC**：真实的自动驾驶场景（20Hz RGB + 事件）。
    *   **SHF-DSEC**：论文自研的高频合成数据集（100Hz），用于严格验证“任意时刻”预测能力。
    *   **M3ED**：包含无人机和四足机器人平台的高动态场景。
    *   **DSEC-Night**：用于评估极端低光环境下的零样本（Zero-shot）泛化能力。
*   **基准（Benchmark）与对比方法**：
    *   **HFR Upper Bound**：理想上限（假设能获取目标时刻的 RGB 帧）。
    *   **LFR Baseline**：仅使用上一帧 RGB 的朴素方法。
    *   **插值法（Interpolation）**：如 TimeLens-XL（非因果，需未来帧）。
    *   **融合法（Fusion）**：如 CMNeXt、EISNet（直接融合 RGB 和事件，非传播机制）。

#### 4. 资源与算力
*   **硬件环境**：训练使用了 **2 张 NVIDIA RTX 4090 GPU**。
*   **训练细节**：采用 AdamW 优化器，训练 **200 个 epoch** 直至收敛，总 Batch Size 为 4。
*   **推理效率**：在 **NVIDIA RTX 3090** 上进行了效率分析。LiFR-Seg-Lite 版本在 440x640 分辨率下可达到约 **65.6 FPS**，且摊销计算量（40.43 GFLOPs）低于直接运行 HFR 模型。

#### 5. 实验数量与充分性
*   **实验规模**：论文在 4 个不同性质的数据集上进行了测试，涵盖了车载、飞行、步行机器人等多种运动模式。
*   **消融实验**：非常充分。分别验证了：
    1.  **不确定性图（ScoreNet）** 的有效性（提升约 1% mIoU）。
    2.  **扭曲域（Warping Domain）** 的选择：证明在特征层扭曲优于在图像层或预测层扭曲。
    3.  **记忆模块** 对长跨度（最高 800ms）预测的支撑作用。
    4.  **光流估计器** 的鲁棒性：更换不同的光流后端（如 RAFT, E-RAFT）均能保持高性能。
*   **客观性**：对比实验均在相同骨干网络（Segformer-B2）下重新训练，确保了公平性。

#### 6. 主要结论与发现
*   **性能极佳**：在 DSEC 数据集上，LiFR-Seg 达到 73.82% mIoU，与拥有未来帧信息的 HFR 上限（73.91%）差距仅为 **0.09%**。
*   **极端环境优势**：在夜间场景（DSEC-Night）中，LiFR-Seg 的表现（41.86%）甚至**超越了基于 RGB 的 HFR 上限**（41.83%），证明了事件相机在低光下的感知优势。
*   **稳定性**：随着预测时间间隔 $\delta t$ 增加，传统方法性能大幅下降，而 LiFR-Seg 凭借传播机制和记忆模块保持了极高的稳定性。

#### 7. 优点
*   **范式创新**：首次定义并解决了“任意时刻帧间语义分割”任务，实现了感知频率与传感器硬件频率的解耦。
*   **鲁棒性设计**：通过显式建模光流不确定性，解决了事件数据稀疏和噪声导致的特征退化问题。
*   **实用性强**：系统具有因果性，适合实时部署，且计算成本随预测帧数增加而摊销，非常高效。

#### 8. 不足与局限
*   **复杂动力学限制**：目前主要针对自我运动（Ego-motion）引起的场景变化，对于极高速的局部非线性变形（如运动员剧烈肢体动作）或源帧本身存在严重运动模糊的情况，处理能力尚待验证。
*   **硬件依赖**：虽然计算量低，但光流估计中的相关性查找（Correlation Lookup）属于内存带宽密集型操作，在某些边缘设备上可能存在瓶颈。
*   **数据集偏差**：合成数据集 SHF-DSEC 虽然频率高，但与真实世界的复杂光影仍有差距。
