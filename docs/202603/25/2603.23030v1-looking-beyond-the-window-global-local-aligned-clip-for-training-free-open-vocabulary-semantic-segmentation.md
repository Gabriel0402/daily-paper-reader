## 摘要
在最近的无需训练的开放词汇语义分割方法中，通常采用滑动窗口推理策略，以克服 CLIP 在处理高分辨率图像时的局限性。然而，这种方法引入了新的挑战：每个窗口都是独立处理的，导致窗口之间存在语义不一致。为了解决这一问题，我们提出了全局-局部对齐 CLIP (GLA-CLIP)，这是一个促进跨窗口全面信息交换的框架。GLA-CLIP 不再将注意力局限于单个窗口内的 token，而是扩展了键值 (key-value) token，以纳入来自所有窗口的上下文线索。尽管如此，我们观察到一种窗口偏差：窗口外的 token 被关注的可能性较低，因为查询 (query) 特征是通过窗口内 patch 的交互产生的，从而缺乏其局部上下文之外的语义基础。为了缓解这一问题，我们引入了代理锚点 (proxy anchor)，它通过聚合来自所有窗口且与给定查询高度相似的 token 构建而成，为衡量窗口内和窗口外 patch 的相似性提供了统一的语义参考。此外，我们提出了一种动态归一化方案，通过动态缩放和阈值化注意力图，根据物体尺度调整注意力强度，以应对小物体场景。此外，GLA-CLIP 可以集成到现有方法中并扩大其感受野。大量的实验验证了 GLA-CLIP 在提升无需训练的开放词汇语义分割性能方面的有效性。代码可在 https://github.com/2btlFe/GLA-CLIP 获取。

## Abstract
A sliding-window inference strategy is commonly adopted in recent training-free open-vocabulary semantic segmentation methods to overcome limitation of the CLIP in processing high-resolution images. However, this approach introduces a new challenge: each window is processed independently, leading to semantic discrepancy across windows. To address this issue, we propose Global-Local Aligned CLIP~(GLA-CLIP), a framework that facilitates comprehensive information exchange across windows. Rather than limiting attention to tokens within individual windows, GLA-CLIP extends key-value tokens to incorporate contextual cues from all windows. Nevertheless, we observe a window bias: outer-window tokens are less likely to be attended, since query features are produced through interactions within the inner window patches, thereby lacking semantic grounding beyond their local context. To mitigate this, we introduce a proxy anchor, constructed by aggregating tokens highly similar to the given query from all windows, which provides a unified semantic reference for measuring similarity across both inner- and outer-window patches. Furthermore, we propose a dynamic normalization scheme that adjusts attention strength according to object scale by dynamically scaling and thresholding the attention map to cope with small-object scenarios. Moreover, GLA-CLIP can be equipped on existing methods and broad their receptive field. Extensive experiments validate the effectiveness of GLA-CLIP in enhancing training-free open-vocabulary semantic segmentation performance. Code is available at https://github.com/2btlFe/GLA-CLIP.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的“免训练开放词汇语义分割（Training-free OVSS）”方法为了处理高分辨率图像，普遍采用**滑动窗口（Sliding-window）**推理策略。然而，由于每个窗口被独立处理，缺乏全局上下文，导致窗口边界出现**语义不一致**（如同一物体在相邻窗口被标为不同类别）和**网格伪影（Grid artifacts）**。
*   **研究背景**：CLIP 等视觉语言模型在低分辨率（如 224x224）预训练，直接应用于高分辨率分割时效果较差。滑动窗口虽然保留了局部细节，但限制了模型的感受野，使其无法“看到窗口之外”的内容。

### 2. 论文提出的方法论：GLA-CLIP
为了解决上述问题，作者提出了 **GLA-CLIP（Global-Local Aligned CLIP）** 框架，包含三个核心组件：
*   **键值扩展机制（Key-Value Extension, KVE）**：
    *   **核心思想**：打破窗口限制，允许当前窗口的查询（Query）访问全图所有窗口生成的键（Key）和值（Value）Token。
    *   **实现**：将所有 $L$ 个窗口的特征聚合，构建全局键矩阵 $K_{global}$ 和全局值矩阵 $V_{global}$，实现跨窗口的注意力计算。
*   **代理锚点（Proxy Anchor）**：
    *   **核心思想**：解决“窗口偏差”。由于 Query 仅由窗口内特征生成，天然倾向于关注窗口内 Token。
    *   **实现**：通过迭代聚合全图中与给定 Query 高度相似的 Token，生成一个代表全局语义的“代理锚点”来替代原始 Query。这为衡量窗口内外 patch 的相似性提供了统一的语义参考。
*   **动态归一化方案（Dynamic Normalization）**：
    *   **核心思想**：根据物体尺度动态调整注意力强度。
    *   **实现**：引入两个自适应变量：$u$（基于窗口数量 $L$ 调整偏移量，抑制全局噪声）和 $w$（基于高置信度 Token 数量估计物体尺度，增强小目标的注意力权重）。这避免了小物体被大量的全局背景 Token 淹没。

*   **性能提升**：GLA-CLIP 在所有 8 个数据集上均显著提升了基准模型（如 ProxyCLIP, ClearCLIP）的性能，平均 mIoU 提升明显。
*   **一致性增强**：通过 BER 指标和定性可视化证明，GLA-CLIP 有效消除了滑动窗口产生的网格伪影，使分割边缘更加平滑自然。
*   **尺度鲁棒性**：动态归一化机制成功解决了全局上下文引入后小物体容易被误判的问题。
*   **即插即用**：该框架具有很强的通用性，可以作为插件提升任何基于滑动窗口的 CLIP 分割方法。

### 7. 优点（亮点）
*   **首创性**：首次识别并系统性地解决了 Training-free OVSS 中滑动窗口导致的局部语义偏差问题。
*   **无需训练**：在不增加任何训练成本的前提下，通过改进推理逻辑大幅提升了 CLIP 的高分辨率处理能力。
*   **优雅的代理机制**：Proxy Anchor 的设计巧妙地平衡了局部查询与全局参考之间的矛盾。
*   **定性效果极佳**：显著改善了分割图的视觉质量，解决了长期存在的网格伪影问题。

### 8. 不足与局限
*   **计算复杂度**：由于引入了跨窗口的键值扩展，注意力计算的复杂度从 $O(LN^2D)$ 增加到 $O(L^2N^2D)$（$L$ 为窗口数）。虽然在实验中内存可控，但在处理极高分辨率或极多窗口时，计算开销可能会显著增加。
*   **超参数依赖**：虽然提出了动态归一化，但代理锚点构建中的阈值 $\rho$ 等参数仍可能需要根据不同领域（如遥感 vs 自然场景）进行微调以达到最优效果。
*   **内存瓶颈**：由于需要同时保留所有窗口的 Key-Value Token，对于显存较小的设备，处理超大图像时可能会面临 OOM（显存溢出）风险。
