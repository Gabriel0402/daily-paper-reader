## 摘要
Segment Anything Model (SAM) 实现了可扩展的医学图像分割，但在作为冻结主干网络部署时，面临推理阶段的不稳定性。在实际应用中，边界框（bounding-box）提示词往往包含定位误差，且固定阈值二值化会引入额外的决策不确定性。这些因素共同导致了较高的预测方差，尤其是在物体边界附近，从而降低了可靠性。我们提出了稳定性感知推理框架（SAIF），这是一种无需训练且即插即用的推理框架，通过显式建模提示词和阈值的不确定性来提高鲁棒性。SAIF 通过结构化边界框扰动和阈值变化构建联合不确定性空间，利用决策稳定性和边界一致性评估每个假设，并引入稳定性-一致性评分来过滤不稳定候选结果，并在概率空间中执行稳定性加权融合。在 Synapse、CVC-ClinicDB、Kvasir-SEG 和 CVC-300 上的实验表明，SAIF 持续提高了分割精度和鲁棒性，在无需重新训练或修改架构的情况下达到了最先进的性能。我们的匿名代码已发布在 https://anonymous.4open.science/r/SAIF。

## Abstract
Segment Anything Model (SAM) enable scalable medical image segmentation but suffer from inference-time instability when deployed as a frozen backbone. In practice, bounding-box prompts often contain localization errors, and fixed threshold binarization introduces additional decision uncertainty. These factors jointly cause high prediction variance, especially near object boundaries, degrading reliability. We propose the Stability-Aware Inference Framework (SAIF), a training-free and plug-and-play inference framework that improves robustness by explicitly modeling prompt and threshold uncertainty. SAIF constructs a joint uncertainty space via structured box perturbations and threshold variations, evaluates each hypothesis using decision stability and boundary consistency, and introduces a stability-consistency score to filter unstable candidates and perform stability-weighted fusion in probability space. Experiments on Synapse, CVC-ClinicDB, Kvasir-SEG, and CVC-300 demonstrate that SAIF consistently improves segmentation accuracy and robustness, achieving state-of-the-art performance without retraining or architectural modification. Our anonymous code is released at https://anonymous.4open.science/r/SAIF.

*   **核心问题**：Segment Anything Model (SAM) 在医学图像分割任务中，当作为“冻结（Frozen）”的主干网络部署时，存在严重的**推理不稳定性**。
*   **研究背景**：
    *   **提示词误差**：实际应用中，由检测器生成或人工标注的边界框（BBox）往往存在定位偏差或尺度误差。
    *   **决策不确定性**：传统的固定阈值（如0.5）二值化方法在面对医学图像的领域偏移时，容易在物体边界产生高方差预测。
    *   **现有方案局限**：现有的医学SAM适配方法多侧重于全量微调或架构修改，忽略了在主干网络不可变的情况下，如何通过优化推理过程来提升可靠性。

### 2. 方法论
SAIF 提出了一种**无需训练、即插即用**的推理框架，核心思想是通过显式建模“提示框-阈值”联合不确定性空间来筛选稳定预测。
*   **不确定性建模**：
    *   **提示框家族**：对原始BBox进行多尺度缩放（Outer candidates）和微小抖动（Inner jitters），生成一组候选提示框。
    *   **自适应阈值集**：基于原始预测概率图的百分位数（10th-90th），构建一个图像特异性的候选阈值搜索空间。
*   **稳定性-一致性评分 (SC Score)**：
    *   利用 **Soft IoU** 衡量候选框在微小抖动下预测结果的一致性。
    *   引入**方差惩罚项**和**面积正则化**，过滤掉产生空掩码或极端不稳定掩码的候选者。
*   **稳定性感知融合**：
    *   **阈值选择**：在候选阈值集中选择能使整体稳定性评分最大化的单一共享阈值。
    *   **加权融合**：选取评分最高的 Top-n 个候选预测，在概率空间进行加权聚合，最后应用选定的阈值生成最终二值掩码。

*   **性能提升**：SAIF 在所有测试的 SAM 变体上均实现了显著提升，mDice 平均提高 4%–8%，mIoU 提高 5%–9%。

### 7. 优点
*   **无需训练**：不需要任何额外的标注数据或重新训练过程，极易集成到现有管线中。
*   **鲁棒性强**：通过显式建模不确定性，有效解决了 SAM 对提示框噪声敏感的问题。
*   **通用性**：作为一种推理策略，它可以提升任何基于框提示的分割模型的性能。

*   **推理延迟**：由于需要多次前向传播来评估稳定性，其推理速度明显慢于单次推理，在实时性要求极高的场景下受限。

*   **计算资源消耗**：在显存较小的设备上，缓存大量概率图可能会带来内存压力。
