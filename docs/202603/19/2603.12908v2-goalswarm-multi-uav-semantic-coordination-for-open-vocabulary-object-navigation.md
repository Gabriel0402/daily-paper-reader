## 摘要
协作式视觉语义导航是在未知环境中运行的空中机器人团队的一项基础能力。然而，由于在机载设备上部署沉重感知模型的计算限制以及去中心化多智能体协同的复杂性，实现鲁棒的开放词汇目标导航仍然具有挑战性。我们提出了 GoalSwarm，这是一个用于零样本语义目标导航的全去中心化多无人机框架。每架无人机通过投影来自空中视角的深度观测，协作构建一个共享的轻量化 2D 俯视语义占据地图，在保留必要的几何和语义结构的同时，消除了完整 3D 表示的计算负担。GoalSwarm 的核心贡献有三点：(1) 集成了零样本基础模型 SAM3，用于开放词汇检测和像素级分割，无需特定任务训练即可实现开放词汇目标识别；(2) 提出了一种贝叶斯价值图，将多视角检测置信度融合为逐像素的目标相关性分布，从而通过置信上限 (UCB) 探索实现知情的边界评分；(3) 一种去中心化协同策略，结合了语义边界提取、基于测地线路径代价的成本效用竞价以及空间分离惩罚，以最大限度地减少集群中的冗余探索。

## Abstract
Cooperative visual semantic navigation is a foundational capability for aerial robot teams operating in unknown environments. However, achieving robust open-vocabulary object-goal navigation remains challenging due to the computational constraints of deploying heavy perception models onboard and the complexity of decentralized multi-agent coordination. We present GoalSwarm, a fully decentralized multi-UAV framework for zero-shot semantic object-goal navigation. Each UAV collaboratively constructs a shared, lightweight 2D top-down semantic occupancy map by projecting depth observations from aerial vantage points, eliminating the computational burden of full 3D representations while preserving essential geometric and semantic structure. The core contributions of GoalSwarm are threefold: (1) integration of zero-shot foundation model -- SAM3 for open vocabulary detection and pixel-level segmentation, enabling open-vocabulary target identification without task-specific training; (2) a Bayesian Value Map that fuses multi-viewpoint detection confidences into a per-pixel goal-relevance distribution, enabling informed frontier scoring via Upper Confidence Bound (UCB) exploration; and (3) a decentralized coordination strategy combining semantic frontier extraction, cost-utility bidding with geodesic path costs, and spatial separation penalties to minimize redundant exploration across the swarm.

#### 1. 核心问题与整体含义（研究动机和背景）
该论文针对**多无人机（Multi-UAV）在未知室内环境中的开放词汇目标导航（ObjectNav）**问题展开研究。
*   **研究动机**：传统的语义导航方法通常依赖闭集检测器（只能识别预定义的类别）或计算昂贵的 3D 神经重建（如 NeuroSwarm），这在机载资源受限的无人机上难以实时运行。
*   **核心挑战**：如何在不进行特定任务训练的情况下，实现对任意目标（如“红椅子”、“雕像”）的识别，并高效地在多机之间分配搜索任务，同时减少重复探索。

#### 2. 方法论：核心思想与关键技术
GoalSwarm 提出了一个全去中心化的框架，主要包含以下四个核心模块：
*   **零样本感知（SAM3 集成）**：利用基础模型 **SAM3** 进行开放词汇的目标检测、分割和跟踪。它通过计算检测置信度与掩码面积的加权值来评估目标的可能性，无需针对特定场景重新训练。
*   **协作式轻量化建图**：无人机将深度图投影为 **2D 俯视语义占据地图**。相比 3D 表示，这种 2D 地图极大地降低了存储和通信开销（单次同步约 6.6 MB），支持多机通过异步广播进行地图融合。
*   **贝叶斯价值图（BVM）**：
    *   **融合逻辑**：利用贝叶斯推断将不同视角、不同距离下的检测置信度融合到逐像素的概率分布中。
    *   **置信度衰减**：引入深度相关的置信锥模型，距离相机中心越远或角度偏离越大的观测，其方差（不确定性）越高。
*   **去中心化协同策略**：
    *   **UCB 边界评分**：利用置信上限（UCB）算法对地图边缘的“边界（Frontier）”进行评分，平衡“探索”（去不确定性高的地方）与“利用”（去目标概率高的地方）。
    *   **成本效用竞标**：结合测地线路径代价（FMM 算法计算）、边界大小和**空间分离惩罚**（防止无人机靠得太近）进行综合评分，通过异步广播解决目标冲突。

#### 3. 实验设计
*   **数据集与场景**：在 **GOAT-Bench** 基准测试上评估，使用 **Habitat** 模拟器中的 **HM3D** 真实室内场景（Unseen 验证集）。
*   **物理模拟**：使用 **VisFly** 框架模拟真实的四旋翼动力学、碰撞检测和 PID 控制。
*   **Benchmark 与对比方法**：
    *   **对比基准**：单智能体（Single Agent）、随机探索（Random Exploration）、无共享地图（No Shared Map，即独立探索）。
    *   **评估指标**：成功率（SR）、路径长度加权成功率（SPL）、平均步数（Avg. Steps）。

#### 4. 资源与算力
*   **硬件环境**：实验在一台配备 **NVIDIA RTX 4090 GPU (24 GB)** 的工作站上运行。
*   **运行模式**：SAM3 模型作为零样本检测器运行在外部 GPU 服务器上，通过 ZMQ 每 3 个仿真步查询一次。
*   **训练时长**：由于该方法基于预训练的基础模型（Zero-shot），**无需针对导航任务进行端到端训练**，主要开销在于推理和仿真。

#### 5. 实验数量与充分性
*   **实验规模**：初始评估基于 20 个 subtasks（来自 HM3D 验证集），随后进行了 **100 个 episode 的扩展评估**以验证一致性。
*   **消融实验**：对比了“无共享地图”和“随机边界选择”等变体，证明了协同机制和贝叶斯价值图的必要性。
*   **客观性**：实验在从未见过的（Unseen）场景中进行，且考虑了无人机的物理动力学，具有较好的客观性。但实验主要在仿真环境中完成，缺乏现实世界的物理干扰测试。

#### 6. 主要结论与发现
*   **协同优势**：多机协同显著优于单机。GoalSwarm 的成功率（45.0%）远高于单机（10.0%），且通过地图共享显著减少了冗余探索。
*   **效率提升**：相比于无协同的多机系统，GoalSwarm 的 SPL（0.179）更高，说明其路径规划更高效，避免了因缺乏信息共享导致的过早误报（False Stops）。
*   **扩展性**：在 100 个 episode 的大规模测试中，成功率进一步提升至 53.8%，证明了框架的鲁棒性。

#### 7. 优点（亮点）
*   **零样本能力**：集成 SAM3 实现了真正的开放词汇导航，能够处理训练集中未出现的物体类别。
*   **计算高效**：通过 2D 语义地图和去中心化架构，解决了无人机机载算力不足和通信带宽限制的问题。
*   **不确定性感知**：贝叶斯价值图与 UCB 的结合，使得集群能够智能地分配搜索任务，优先覆盖高价值或高不确定性的区域。

#### 8. 不足与局限
*   **理想化假设**：实验假设了完美的里程计（Odometry）和无损通信，实际部署中需面对漂移和丢包问题。
*   **维度缺失**：2D 投影地图会丢失垂直方向的信息（如桌子下方的空间或多层货架），可能导致路径规划冲突。
*   **感知瓶颈**：对于小物体或严重遮挡的物体，SAM3 的检测置信度可能低于阈值，导致搜索失败。
*   **环境限制**：目前主要针对室内环境，室外大场景下的表现尚未验证。
