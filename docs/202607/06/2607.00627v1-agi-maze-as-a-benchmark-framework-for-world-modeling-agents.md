## 摘要
大型语言模型（LLM）是强大的模式补全系统，但它们的默认操作模式——从静态上下文预测下一个词元——并不能可靠地产生对外部世界的持久、可操作的表示。许多在文本中看起来像“推理”的任务，一旦环境是部分可观察的、有状态的，并且需要关于隐藏状态的记忆和结构化假设，就会变得困难得多。AGI Maze是一个轻量级框架，用于构建此类环境，无需高维感官输入。它提供了一系列基于网格的迷宫任务，具有清晰的API和多种难度级别。目标是创建基准测试，其中智能体必须学习并使用世界状态表示，而不仅仅是根据直接提供的观测推断局部规则。我们初步评估了几个普通LLM在简单迷宫上的表现，表明它们在LLM推理时无法内部表示迷宫。我们还引入了一个基线智能体，允许它使用消息历史作为工作记忆，在智能体运行时构建观测描述。虽然这可以提高性能，但对于LLM智能体来说，即使在比人类所需步骤预算更充足的情况下，可靠地解决即使是小型迷宫也是不够的。

## Abstract
Large language models (LLMs) are powerful pattern-completion systems, but their default operating mode - predicting the next token from a static context - does not reliably produce persistent, manipulable representations of an external world. Many tasks that look like "reasoning" in text become substantially harder once the environment is partially observable, stateful, and requires memory and structured hypotheses about hidden state. AGI Maze is a lightweight framework for building such environments without requiring high-dimensional sensory inputs. It provides a family of grid-based maze tasks with a clean API and multiple difficulty regimes. The goal is to create benchmarks where agents must learn and use world state representations, not just infer a local rule over readily provided observations. We provide an initial evaluation of several vanilla LLMs on simple mazes showing that they fail to represent mazes internally at LLM inference time. We also introduce a baseline agent, which is allowed to use its message history as a working memory to construct descriptions of observations at agentic runtime. Although this can improve performance, it is still insufficient for an LLM agent to reliably solve even small mazes within a step budget that is more than enough for humans.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：大型语言模型（LLM）本质上是静态的模式补全系统，其默认的“预测下一个词元”操作模式无法可靠地生成持久、可操控的外部世界表示。在部分可观测、有状态、需要记忆和结构化假设的环境中，许多看似“推理”的任务变得极其困难。
- **研究动机**：当前LLM智能体存在两个相互关联的问题：①以文本形式记忆效率低下，导致脆弱行为和高词元成本；②表示不保证稳定，模型倾向于提取产出下一个输出所需的信息，而非维持对世界状态的稳定可操作表示。现有基准（如AGI-ARC-3）多为全可观测、不要求维持世界状态，无法充分测试“智能体能否构建并使用演化世界状态的表示”。
- **整体含义**：世界建模不仅涉及规则发现，还包含表示潜在状态、在不确定性下维持信念、更新信念、基于表示推理、高效使用记忆等。AGI Maze框架旨在填补这一空白，通过轻量级迷宫环境隔离感知复杂性，聚焦于状态、记忆和表示能力。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：设计一套基于网格的迷宫任务，要求智能体在部分可观测、有状态的动态环境中通过文本观测构建并使用世界状态表示（如地图、位置、隐藏状态）。
- **关键技术细节**：
  - **环境设计**：网格迷宫（典型3×3至8×8），包含墙壁、不可破坏边界（只留一个出口）、宝藏和钥匙等基础元素。智能体每次选择上/下/左/右动作，撞墙则位置不变但消耗步数。
  - **动态不确定性机制**：河流（自动沿流向移动）、陷阱循环（传送至下一个陷阱）等，使观测与多个位置对应，迫使智能体维持信念而非精确位置。
  - **步数预算**：根据迷宫大小和难度设定有限步数（硬阈值），要求智能体有效探索与规划。
  - **扩展性**：支持新物品（船、手榴弹、手电筒）、新格子类型、新动作，以防止过拟合硬编码策略。
  - **API设计**：提供HTTP API，包括游戏描述获取、开始游戏、执行步骤，返回文本观测和结构化字段。
- **无公式/算法流程**：论文未给出正式算法，而是描述环境逻辑和智能体接口。

## 3. 实验设计：数据集/场景、基准、对比方法
- **数据集/场景**：迷宫分为五组：

  - 训练（TRAINING）：小型迷宫（3×3至5×4）+ 宽松步数预算。

## 6. 论文的主要结论与发现
- **普通LLM智能体失败**：即使对于小型训练迷宫，普通LLM也无法可靠解决，某些模型（如GPT-4o Mini）甚至比随机行走还差。随着迷宫增大（5×4及以上），所有普通LLM均不如随机行走。
- **规划智能体提升有限**：允许在提示中写笔记可以改善强模型性能（GPT-5.5从30%到60%，Gemini 3.5 Flash从40%到70%），但轻量模型无改善。即使提升后，仍远低于人类水平。
- **关键洞察**：LLM在标准推理过程中并未构建可更新、可查询的显式环境状态表示。将记忆作为文本依然脆弱且低效。
- **框架价值**：AGI Maze有效暴露了LLM在世界建模方面的核心缺陷，并提供了研究状态跟踪、地图构建、记忆管理、长期规划等能力的标准化平台。
