## 摘要
我们研究基于动作的条件世界建模，作为学习机器人学习可迁移动力学先验的一种可扩展方式。通过预训练一个模型来预测动作如何驱动视觉场景演变，所产生的世界模型捕捉到了超越外观级视频生成的可重用交互动力学。具体来说，我们在带有真实动作标注的大规模机器人操作数据上预训练了一个多视角交互基础扩散世界模型A2World。我们从两个互补的角度验证了所学的动力学先验。首先，我们将A2World适配为任务或场景专用的真实世界模拟器A2World-sim，其长程轨迹生成支持基于模拟器的策略评估和可扩展的假设分析，通过用世界模型轨迹替代真实机器人轨迹。其次，从相同的预训练权重出发，我们将A2World适配为视频-动作联合预测模型A2World-policy，该模型在视觉和指令条件作用下预测动作。在模拟基准测试和真实机器人设置上的实验表明，基于动作条件的世界模型预训练产生了可迁移的动力学先验，有利于以模拟器为中心和以策略为中心的机器人学习。

## Abstract
We study action-conditioned world modeling as a scalable way to learn transferable dynamics priors for robot learning. By pretraining a model to predict how actions drive visual scene evolution, the resulting world model captures reusable interaction dynamics beyond appearance-level video generation. Concretely, we pretrain a multi-view interactive base diffusion world model, A2World, on large-scale robot manipulation data with real action annotations. We validate the learned dynamics priors from two complementary perspectives. First, we adapt A2World into a task- or scene-specialized real-world simulator, A2World-sim, whose long-horizon rollouts support simulator-based policy evaluation and scalable what-if analysis by replacing real-robot rollouts with world model rollouts. Second, starting from the same pretrained weights, we adapt A2World into a video-action joint prediction model, A2World-policy, that predicts actions under visual and instruction conditioning. Experiments across simulation benchmarks and real-robot settings demonstrate that action-conditioned world model pretraining yields transferable dynamics priors that benefit both simulator-centric and policy-centric robot learning.

## 1. 核心问题与整体含义（研究动机和背景）

当前机器人学习方法分为两流派：（1）将视频生成模型适配为视觉-语言-动作（VLA）策略，（2）开发基于动作条件的世界模型用于数据增强和策略评估。但现有工作未充分利用机器人数据预训练作为可迁移动力学先验的来源：多数直接微调通用视频生成检查点，缺少大规模基于动作标注的机器人数据预训练；部分在同一数据集预训练，限制了能吸收的具身、相机配置和运动模式；大规模工作常针对单一下游目标优化。本文核心动机：将动作条件世界建模作为学习可迁移动力学先验的可扩展路径。动作提供自然的因果监督信号——对象、场景、视角变化但交互规则受动作如何引起状态变化支配。预训练一个以动作为条件的视觉场景演化模型，强制其编码可控、基于动作的动力学，超越外观级视频预测，产生可重用的交互先验。

## 2. 提出的方法论：核心思想、关键技术细节

### 2.1 核心思想
- 在大规模真实机器人操作数据上预训练一个多视角交互扩散世界模型 **A2World**，学习从动作到视觉演化的可迁移动力学先验。
- 预训练后，将权重适配为两种变体：
  - **A2World-sim**：历史感知自回归世界模型，支持长程轨迹生成，用作任务/场景专用模拟器。
  - **A2World-policy**：MoE风格的视频-动作联合预测模型，作为指令条件机器人策略。

### 2.2 A2World 基础世界模型设计
- **公式**：给定条件帧 \(o_t\) 和未来动作 \(a_{t+1:t+k}\)，预测未来观测 \(o_{t+1:t+k}\)：  
  \(A2World: p(o_{t+1:t+k} | o_t, a_{t+1:t+k})\)。
- **动作条件机制**：动作 \(a\) 经 MLP 编码为嵌入 \(e\)，与扩散时间步嵌入 \(\tau(\sigma)\) 相加，通过自适应层归一化（AdaLN）注入每个 DiT 块。其他条件输入设为零（\(c=0\)）。
- **多视角生成**：将 \(V\) 个视角按时间维度拼接（\(z_{mv} \in \mathbb{R}^{B \times C \times (V \cdot T) \times H \times W}\)），引入可学习视角嵌入 \(\epsilon_{view}(v)\) 拼接到潜变量通道。插入跨视角注意力模块（Cross-View Attention）确保时空一致性。
- **训练目标**：EDM 去噪评分匹配：  
  \(\mathcal{L}_{A2World}(\sigma) = \mathbb{E}_{z,n}[\|A2World(z+n; \sigma, c=0, a) - z\|_2^2]\)。
- **预训练数据**：汇总 2100k+ 轨迹、20+ 具身（AgiBot, DROID, OPEN-X, InternData-A1 等）。采用数据集一致批处理（每个 mini-batch 只来自一个数据集）。

### 2.3 A2World-sim：长程模拟器适配
- **公式**：\(p(o_{t+1:t+k} | o_t, a_{t+1:t+k}, H_{t-1})\)，\(H_{t-1}\) 为历史帧。
- **姿态引导历史采样（Pose-guided History Sampling）**：基于相对动作计算加权弧长，沿弧长均匀采样帧，保留关键状态（如转向点），在固定历史预算下覆盖运动轨迹。
- **历史注入**：两种方式：（1）历史潜变量序列映射为历史 token，通过交叉注意力被当前目标潜变量注意；（2）将历史 token 映射为 key/value 记忆，与当前自注意力拼接（全局历史记忆注入）。
- **自回归生成**：逐块自回归生成，生成帧追加到历史缓存并作为下一轮条件。采用 **Self-forcing 训练策略**：训练时周期性地基于模型自身生成帧（而非始终用地面真值帧），暴露在自身误差中并学会恢复。

### 2.4 A2World-policy：策略模型适配
- **公式**：\(p(o_{t+1:t+k}, a_{t+1:t+k} | o_t, l)\)，\(l\) 为语言指令。
- **指令编码**：使用预训练 T5 文本编码器得到嵌入 \(h_l\)，作为每个 DiT 块的交叉注意力上下文。
- **联合扩散公式**：视频潜变量 \(z_v\) 和动作潜变量 \(z_a\) 独立加噪，采样共享基础噪声水平 \(\sigma_{base}\) 再按模态缩放（\(\sigma_v = m_v \sigma_{base}, \sigma_a = m_a \sigma_{base}\)），提升视频-动作对齐。
- **MoE 风格视频-动作块**：共享一个自注意力模块（视频 token 与动作 token 交互），视频和动作各自保留模态专用的 AdaLN 和 MLP 分支（动作分支从视频分支复制初始化）。动作 token 通过共享自注意力同时注意视频和动作 token。
- **训练目标**：加权联合去噪目标：  
  \(\mathcal{L}_{A2World-policy} = \mathbb{E}[w(\sigma_v)\|\hat{z}_v - z_v\|_2^2 + \lambda_a w(\sigma_a)\|\hat{z}_a - z_a\|_2^2]\)。
- **推理**：模态级分类器自由引导（CFG），可单独设置视频和动作引导强度 \(s_v, s_a\)。

- **动作条件世界模型预训练产生强可迁移动力学先验**，优于文本条件预训练（C-init 80.2% vs A-pre 88.5% OOD 成功率）或任务特定预训练（P-pre 与 A-pre 性能接近但 A-pre 同时支持模拟器）。

## 7. 优点
- **统一框架**：同一预训练权重可同时适配模拟器和策略两种下游任务，体现“先验”的可迁移性。
- **动作条件优于文本条件**：动作提供更确定性的因果监督，减少歧义，产生更强动力学先验（OOD 场景提升显著）。
- **大规模跨具身预训练**：使用 210 万+轨迹、20+ 具身，涵盖多种相机配置和运动模式，提升泛化。
- **多视角一致性**：通过视角嵌入和跨视角注意力实现多视角时空一致生成。
- **Self-forcing 训练**：模拟器训练中引入自回归误差暴露，提升长程稳定性。
- **MoE 风格设计**：视频和动作共享自注意力，保留专用去噪分支，平衡交互与专用能力。
- **消融全面**：验证了历史采样、预训练目标、视频-动作耦合等关键设计选择。

## 8. 不足与局限
- **数据依赖**：依赖高质量的动作标注数据，缺乏对无动作数据或隐藏动作的利用（与部分使用伪标签方法不同）。
- **任务覆盖面**：真实机器人评估仅 5 个任务，且均为桌面操作，未涉及移动操纵、动态环境等。
- **OOD 泛化仍有挑战**：虽然在 LIBERO-Plus-Spatial 上表现较好，但特定场景（如 Cam. 维度仅 24.5% 初始基线的 X-VLA，A2World-policy 为 75.8%）仍有差距。
- **计算资源需求高**：预训练使用 64 块 H200 GPU，模型参数量 2.5-3.0B，对一般研究组可复制性有限。
- **历史采样参数手动设定**：权重 \(w_t=1.0, w_r=0.3\) 需针对任务调整，缺乏自适应选择。
- **模拟器验证指标局限**：仅使用成功率相关性，缺少对接触力、动力学精度等的细粒度度量。
- **策略推理未讨论实时性**：联合视频-动作生成可能引入延迟，实际部署的推理时间未报告。
- **潜在偏差**：预训练数据主要来自开源数据集（AgiBot、DROID 等），可能存在分布偏差（如物体类型、场景光照、文化背景）。
