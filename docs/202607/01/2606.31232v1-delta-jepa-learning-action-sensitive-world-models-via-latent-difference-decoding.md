## 摘要
学习用于规划的视觉世界模型需要紧凑的潜在动态，这些动态必须对动作保持敏感，然而无重建的联合嵌入目标会导致对动作不敏感的表示。我们提出Delta-JEPA，一个端到端无重建的世界模型，它通过潜在差异动作解码器（LDAD）增强潜在前向预测。与从拼接的端点嵌入推断动作的逆解码器不同，LDAD从连续观测之间的潜在位移重建执行的动作。这种位移级别的监督直接正则化变换几何：相邻嵌入不能坍缩而不丢失动作信息，并且不同动作被鼓励引起可区分的潜在变化，以便基于展开的规划。Delta-JEPA仅使用潜在预测和动作重建，避免像素重建和分布匹配正则化器。在四个视觉连续控制任务中，Delta-JEPA相比于基于JEPA和表示学习的基准世界模型改进了规划。消融实验表明，基于位移的动作解码始终比端点拼接更有效，动作敏感性分析显示出更清晰的动作条件潜在响应。这些结果表明监督潜在差异是一种简单有效的机制，用于抗坍缩和动作敏感的世界模型学习。

## Abstract
Learning visual world models for planning requires compact latent dynamics that remain sensitive to actions, yet reconstruction-free joint-embedding objectives can collapse to action-insensitive representations. We propose Delta-JEPA, an end-to-end reconstruction-free world model that augments latent forward prediction with a Latent Difference Action Decoder (LDAD). Unlike inverse decoders that infer actions from concatenated endpoint embeddings, LDAD reconstructs the executed action from the latent displacement between consecutive observations. This displacement-level supervision directly regularizes transition geometry: adjacent embeddings cannot collapse without losing action information, and different actions are encouraged to induce distinguishable latent changes for rollout-based planning. Delta-JEPA uses only latent prediction and action reconstruction, avoiding pixel reconstruction and distribution-matching regularizers. Across four visual continuous-control tasks, Delta-JEPA improves planning over JEPA-based and representation-learning world model baselines. Ablations show that displacement-based action decoding is consistently more effective than endpoint concatenation, and action-sensitivity analyses show clearer action-conditioned latent responses. These results indicate that supervising latent differences is a simple and effective mechanism for collapse-resistant and action-sensitive world model learning.

### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：学习用于规划的视觉世界模型需要紧凑的潜在动态，且这些动态必须对动作保持敏感。然而，现有的无重建（reconstruction-free）联合嵌入预测架构（如JEPA）在仅使用潜在预测目标进行端到端训练时，容易发生**表示坍缩**，即编码器和预测器退化为产生几乎恒定的表示，虽然预测损失很低，但损失了动作信息，导致规划失效。
- **背景**：早期世界模型依赖像素重建（如Dreamer系列），计算开销大且关注视觉细节而非动态相关信息。JEPA等无重建方法虽有潜力，但需额外的崩溃预防策略。现有方案（如LeWorldModel使用SigReg、PLDM使用VICReg+逆动力学）各有局限：前者未显式约束动作敏感性，后者逆解码器从拼接的端点状态[z_t, z_{t+1}]预测动作，可能利用动作相关的单状态捷径，而非真正建模状态转移。
- **研究动机**：提出一种简单有效的机制，在不使用像素重建、冻结编码器或复杂多目标正则化的情况下，同时实现抗坍缩和动作敏感的潜在世界模型学习。

### 2. 方法论：核心思想、关键技术细节、公式
- **核心思想**：在潜在前向预测基础上，增加**潜在差异动作解码器（LDAD）**，通过从相邻观测的潜在位移Δz_t = z_{t+1} - z_t重建执行的动作，来正则化潜在变换几何，鼓励不同动作产生可区分的潜在位移和下一状态嵌入。
- **关键技术细节**：
  - **编码器**：f_θ将观测o_t映射为潜在表示z_t。
  - **前向预测器**：P_ϕ根据z_t和动作a_t预测下一个潜在状态ẑ_{t+1} = P_ϕ(z_t, a_t)。
  - **预测损失**：L_pred = ||ẑ_{t+1} - z_{t+1}||²_2（目标z_{t+1}由同一编码器得到）。
  - **LDAD**：计算Δz_t = z_{t+1} - z_t，解码器D_Θ从Δz_t预测动作â_t = D_Θ(Δz_t)，损失L_action = ||â_t - a_t||²_2。
  - **多步扩展**：D_Θ使用Transformer和N个可学习动作查询，输入长程位移z_{t+N} - z_t，输出N个重建动作，捕捉更长时序结构。
  - **联合优化**：总损失L = L_pred + λ L_action，仅两个目标，无需冻结编码器、stop-gradient或分布匹配。
- **抗坍缩机制**：若相邻观测编码坍缩，Δz_t将无信息，无法重建动作，从而迫使编码器保留区分性特征。
- **避免单状态捷径**：LDAD只依赖位移Δz_t，无法利用单状态中的动作相关线索，使动作重建必须基于实际变换。
- **增强动作敏感性**：不同动作产生不同的Δz_t，使潜在滚动对候选动作更可区分，提升规划效果。
