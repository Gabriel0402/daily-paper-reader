## 摘要
尽管交互式世界模型（IWM）取得了快速进展，但现有基准仅在轨迹层面评估动作跟随，忽略了记忆和交互物理。我们提出WorldOdysseyBench，一个在四个维度上评估长时稳定性的开放世界基准，每个维度都有针对性的创新：（i）动作：逐帧动作度量，跨模型语义尺度差异，暴露被轨迹隐藏的失败；（ii）视觉：基于片段的漂移度量，捕捉起点-终点比较遗漏的非单调中期序列崩溃；（iii）物理：基于可控性门控的力学、光学和3D一致性评估，在忠实动作执行下评分合理性；（iv）记忆：动作解耦协议，通过过渡定位的3D点云重建评估场景记忆，通过追踪加VLM推理评估主体记忆。该基准包含自然、城市和室内场景中的600多个测试案例，基于第一/第三人称视角进行WASD 10-60秒连续交互。评估10多个开源/闭源模型显示，没有模型能可靠满足所有维度；即使最好的模型也仅达到中等分数。在WorldOdysseyBench上的进展是迈向稳定、物理扎实、记忆忠实且可部署于实际应用的IWM的步骤。

## Abstract
Despite rapid progress in interactive world models (IWMs), existing benchmarks evaluate action following only at trajectory level and ignore memory and interaction physics. We introduce WorldOdysseyBench, an open-world benchmark for long-horizon stability across four dimensions, each with tailored innovations: (i) Action: per-frame action metric bypassing cross-model semantic scale disparity and exposing failures hidden by trajectory; (ii) Vision: segment-based drift metric capturing non-monotonic mid-sequence collapse missed by start-vs-end comparisons; (iii) Physics: controllability-gated evaluation over mechanics, optics, and 3D consistency, scoring plausibility under faithful action execution; (iv) Memory: action-decoupled protocol evaluating scene memory via transition-localized 3D point-cloud reconstruction and subject memory via tracking-plus-VLM reasoning. The benchmark comprises 600+ test cases across Nature, Urban, and Indoor scenes in first/third-person views with WASD 10-60s continuous interaction. Evaluating 10+ open/closed-source models reveals none reliably satisfies all dimensions; even the best achieves only moderate scores. Advances on WorldOdysseyBench are steps toward IWMs that are stable, physically grounded, memory-faithful, and deployable in real-world applications.

- **核心问题**：当前交互式世界模型（IWM）的评估基准仅在“轨迹级”评估动作跟随，忽略了长期交互中的**记忆保持**和**物理真实性**，导致模型虽然能短时间跟随指令，但在长程（10–60秒连续交互）中容易出现动作漂移、视觉崩溃、物理不合理、记忆混淆等问题，无法可靠部署到实际应用。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：抛弃传统的轨迹级指标，引入**逐帧级、分段级、门控级、解耦级**的四维评估框架，分别针对 IWM 的四个关键能力。
  - **动作维度**：提出**逐帧动作指标**（per-frame action metric），使用**动作语义尺度对齐**技术消除不同模型输出空间差异（如连续控制 vs. 离散按键），然后计算每帧动作与真实动作之间的**符号/方向一致性**。这能暴露轨迹级指标掩盖的瞬时错误。
  - **视觉维度**：提出**基于片段的漂移度量**（segment-based drift metric），将连续生成帧划分为多个片段，分别计算每个片段与对应真实帧的**感知距离**（如 LPIPS、SSIM），再记录漂移的累积趋势。避免了起点-终点对比遗漏的非单调中期崩溃。
  - **物理维度**：设计**可控性门控评估**（controllability-gated evaluation），首先验证模型是否忠实地执行了给定的动作序列（通过动作维度指标），只有在动作执行正确的前提下，才进一步评估生成帧的**力学合理性**（像物体碰撞后运动方向/速度）、**光学一致性**（光照阴影变化）、**3D空间一致性**（视差、深度、遮挡关系）。评分时按“动作忠实 → 物理合理”级联，防止因动作错误而误判物理。
  - **记忆维度**：提出**动作解耦协议**（action-decoupled protocol），将记忆评估与动作执行分离。对于**场景记忆**，在特定时间点让模型进行**过渡定位**（transition localization），通过双视图/多视图生成重建 3D 点云，与原图重建点云对比误差；对于**主体记忆**，使用**跟踪+视觉语言模型（VLM）推理**，要求模型在长序列中输出穿越时间的主体位置、颜色、类别等属性，与真实跟踪结果比较。

- **关键技术细节**：所有评估指标均设计为**无参、可复现**，且测试用例采用**固定 WASD 动作序列**（类似第一人称游戏的键盘控制），保证不同模型接受完全相同的人类操作指令。每个测试用例持续 10–60 秒，共 600+ 个用例，覆盖自然场景、城市场景、室内场景，包含第一/第三人称视角。
