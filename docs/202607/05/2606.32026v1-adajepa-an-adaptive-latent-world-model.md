## 摘要
潜在世界模型通过压缩潜在空间预测未来状态，实现从高维观测进行规划。然而，这些模型通常在测试时保持不变：当预测不准确时，规划可能失败，尤其是在测试分布偏移的情况下。为了解决这个问题，我们提出AdaJEPA，一种自适应潜在世界模型，在模型预测控制（MPC）的闭环内进行测试时自适应。训练后，AdaJEPA规划并执行第一个动作块，将观察到的下一个状态转换作为自监督自适应信号，并用更新后的模型重新规划。这种闭环更新不断重新校准世界模型，无需额外的专家演示。在一系列目标到达任务中，AdaJEPA在每次MPC重规划步骤中只需一个梯度步骤即可显著提高规划成功率。

## Abstract
Latent world models enable planning from high-dimensional observations by predicting future states in a compact latent space. However, these models are typically kept frozen at test time: when their predictions become inaccurate, planning can fail, especially under test-time distribution shift. To address this, we propose AdaJEPA, an adaptive latent world model that performs test-time adaptation within the closed loop of model predictive control (MPC). After training, AdaJEPA plans and executes the first action chunk, uses the observed next-state transition as a self-supervised adaptation signal, and replans with the updated model. This closed-loop update continuously recalibrates the world model without additional expert demonstrations. Across a range of goal-reaching tasks, AdaJEPA substantially improves planning success with as few as one gradient step per MPC replanning step.

## 1. 核心问题与整体含义（研究动机和背景）

潜在世界模型（Latent World Models）将高维观测编码到紧凑的潜在空间，并在此空间内预测未来状态，从而支持高效的规划。主流方法如联合嵌入预测架构（JEPA）将编码器和预测器联合训练，并在测试时与模型预测控制（MPC）结合进行目标导向规划。**然而，现有世界模型在测试时完全冻结**：当预测不准确时（例如由于训练数据不足或测试时分布偏移），MPC会优化错误的目标——在潜在滚动中看似有效的动作在真实环境中可能失败。分布偏移来源广泛，包括视觉变化（噪声、光照、背景）、动力学变化（摩擦、质量、接触）以及布局变化。这种限制严重阻碍了世界模型在真实世界的部署。

本文提出 AdaJEPA，其核心思想是**让世界模型在部署期间持续学习，而不应保持冻结**。受生物系统中感觉运动适应机制的启发，AdaJEPA 在 MPC 闭环中进行测试时自适应：利用每步执行后观测到的状态转换作为自监督信号更新模型，从而在下一轮规划中获得更准确的预测。

## 2. 论文提出的方法论

### 核心思想
- **计划-执行-适应-重新规划闭环**：在任何一步 MPC 重规划中，先用当前世界模型规划动作序列，执行第一个动作块，观测新状态，然后以该转换作为自监督目标更新模型，再用更新后的模型重新规划。

### 关键技术细节

- **世界模型结构**：由视觉编码器 ℰₛ_φ、动作编码器 ℰₐ_ψ 和预测器 f_θ 组成：
  ```
  z_t = ℰₛ_φ(o_t),   u_t = ℰₐ_ψ(a_t),   ẑ_{t+1} = f_θ(z_t, u_t)
  ```
  训练时使用 JEPA 风格的目标函数，最小化预测潜在与目标潜在之间的损失（$\ell$ 通常为 MSE），并使用 stop-gradient 或正则化防止坍塌。

- **测试时自适应**：
  - **在线缓冲区 $\mathcal{B}$**：存储最近收集的转换对 $(o_i, a_i, o_{i+1})$，限制大小 $N$（默认保持最近 5 个）。
  - **自适应损失**：
    \[
    \mathcal{L}_{\text{ada}}(\mathcal{B}) = \frac{1}{|\mathcal{B}|} \sum_{(o_i,a_i,o_{i+1})\in\mathcal{B}} \ell\big( f_θ(z_i, ℰₐ_ψ(a_i)), \operatorname{sg}(z_{i+1}) \big)
    \]
    使用 stop-gradient（$\operatorname{sg}$）防止在线更新中表示坍塌。
  - **更新参数**：仅更新编码器和预测器的最后几层（默认更新预测器最后一个 transformer block + 编码器最后阶段），使用与训练相同的学习率（预测器 $5\times10^{-4}$，编码器 $10^{-5}$），**一个梯度步**即可。
  - **算法流程**（Algorithm 1）：每步 MPC：规划 → 执行 → 添加转换到缓冲区 → 更新 Ω$（$U$步，默认为1）→ 重新规划。

- **规划器**：使用 MPC 优化动作序列，最小化潜在目标到达成本（平方欧氏距离）。可使用梯度下降（GD）或交叉熵方法（CEM）作为子规划器。

## 3. 实验设计

### 使用的环境与数据集
- **PushT / PushObj**（基于 MuJoCo）：
  - **形状偏移**：训练形状 {T, L, Z, +}，测试包括这些形状及未见过形状 {I, smallT, square}。数据生成时偏置使轨迹包含接触。
  - **视觉偏移**：对原始 PushT 施加模糊、椒盐噪声、暗光照、以及将移动块/固定锚/代理颜色改为红色。

- **标准 PushT 验证轨迹**（从 DINO-WM 继承）用于比较不同 JEPA 实现。

## 6. 论文的主要结论与发现

1. **AdaJEPA 显著提升规划成功率**：在所有测试的分布内和分布外环境中，仅需一个梯度步的自适应即可大幅提高成功率，尤其在形状偏移和视觉偏移下成功率可翻倍。
2. **自适应几乎不增加延迟**：每次重规划增加的时间仅为 0.01~0.03 秒，代价可忽略。
3. **在低数据场景下尤其有效**：当训练数据有限时，自适应可以弥补数据不足，甚至超越用 64倍数据训练的冻结模型。
4. **对不同设计选择鲁棒**：无论是哪种自适应参数、超参数或基础模型，AdaJEPA 均能带来一致改善。
5. **世界模型不应冻结**：在部署时继续学习可以持续校准预测，提升规划鲁棒性。

## 7. 优点

- **方法简洁高效**：仅需一个梯度步即可见效，无需额外数据、奖励或专家演示。
- **计算开销极低**：只更新少量参数，几乎不增加推理时间，适合实时控制。
- **通用性强**：适用于不同 JEPA 架构、不同规划器、不同偏移类型。
- **充分的消融分析**：系统研究了参数选择、超参数、缓冲区设计、数据规模等影响因素，提供了实用的默认配置。
- **对生物启发的印证**：与大脑感觉运动适应机制一致，具有理论上的吸引力。
- **实验设计全面**：覆盖多种偏移和场景，结果统计稳健。

## 8. 不足与局限

- **自适应范围有限**：仅进行轻量级修正，当测试环境需要训练中完全未出现的特征（如新的颜色配置或完全不同的动力学）时，可能无法完全恢复性能（例如红色锚点偏移提升较小）。
- **需要预训练表示具有良好基础**：自适应的有效性取决于预训练潜在空间的覆盖度；若训练数据中缺少关键特征，自适应可能不足以弥补。
- **未在真实机器人上验证**：所有实验均在仿真环境（MuJoCo）中进行，未考虑真实世界的传感噪声、延迟和硬件限制。
- **缺乏与更复杂在线学习方法的对比**：例如未和在线 model-based RL（如 TD-MPC2）或主动学习方法直接比较。
- **潜在过拟合风险**：持续更新可能使模型过度拟合最近轨迹而遗忘之前学习到的全局结构，文中虽提及使用 stop-gradient 和限制更新层数来缓解，但未对遗忘进行量化分析。
- **实验覆盖的偏移类型仍有限**：例如未包含光照方向变化、部分遮挡、非稳态动力学（如随时间变化）等更复杂的场景。
